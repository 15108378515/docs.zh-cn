2. <span data-ttu-id="ffadf-107">保持 .NET Core SDK 和 .NET Core Runtime 之间的版本号一致。</span><span class="sxs-lookup"><span data-stu-id="ffadf-107">Maintain alignment in version numbers between .NET Core SDK and .NET Core Runtime.</span></span>
3. <span data-ttu-id="ffadf-108">对 .NET Core SDK 和 .NET Core Runtime 使用语义版本控制。</span><span class="sxs-lookup"><span data-stu-id="ffadf-108">Use semantic versioning for both the .NET Core SDK and .NET Core Runtime.</span></span>

<span data-ttu-id="ffadf-109">2.0.0 强制执行版本一致性并顺利发布。</span><span class="sxs-lookup"><span data-stu-id="ffadf-109">2.0.0 forced version alignment and proceeded smoothly for one release.</span></span> <span data-ttu-id="ffadf-110">2017 年 12 月，.NET Core SDK 进行了一次功能发布，而 .NET Core Runtime 没有相应的发布。</span><span class="sxs-lookup"><span data-stu-id="ffadf-110">In December 2017 .NET Core SDK had a feature release, with no corresponding release in the .NET Core Runtime.</span></span> <span data-ttu-id="ffadf-111">团队选择了目标 1 和 3，没有保持 .NET Core Runtime 和 SDK 之间的一致性。</span><span class="sxs-lookup"><span data-stu-id="ffadf-111">The team chose goals 1 and 3, losing alignment between the .NET Core Runtime and SDK.</span></span> <span data-ttu-id="ffadf-112">在 .NET Core Runtime 2.1 之前发布了几个 .NET Core SDK 2.1.x 版本。</span><span class="sxs-lookup"><span data-stu-id="ffadf-112">Several .NET Core SDK 2.1.x versions were released before .NET Core Runtime 2.1.</span></span> <span data-ttu-id="ffadf-113">由于 SDK 不是向前兼容，因此这些 2.1.x SDK 版本无法面向 .NET Core Runtime 2.1。</span><span class="sxs-lookup"><span data-stu-id="ffadf-113">Since the SDK is not forwards compatible, these 2.1.x SDK versions could not target .NET Core Runtime 2.1.</span></span> <span data-ttu-id="ffadf-114">该团队为了应对大量混乱，切换到目标 1 和 2，放弃了 [.NET Core 版本控制](index.md#versioning-details)中所述的语义版本控制。</span><span class="sxs-lookup"><span data-stu-id="ffadf-114">The team responded to the considerable confusion by switching to goals 1 and 2, abandoning semantic versioning as described in [.NET Core versioning](index.md#versioning-details).</span></span>

<span data-ttu-id="ffadf-115">由于决定放弃语义版本控制的时间选择，2.1.10x 和 2.1.20x 版本号范围内存在过渡版本，这些版本也无法面向 .NET Core Runtime 2.1。</span><span class="sxs-lookup"><span data-stu-id="ffadf-115">Because of the timing of the decision to abandon semantic versioning, there were transitional releases in the 2.1.10x and 2.1.20x version number ranges that also can't target .NET Core Runtime 2.1.</span></span>

<span data-ttu-id="ffadf-116">版本号的前两位数字与 .NET Core Runtime 的 2.1.0 版本和 .NET Core SDK 的 2.1.300 版本重新保持一致。</span><span class="sxs-lookup"><span data-stu-id="ffadf-116">The first two digits of the version numbers realign with the 2.1.0 version of the .NET Core Runtime and the 2.1.300 version of the .NET Core SDK.</span></span>
