# Compiler Error CS0006
未能找到元数据文件“\<DLL 名称\>”  
  
 编译了程序并给它显式传递了包含元数据的文件的名称，但没有找到 .dll。  有关更多信息，请参见 [\/reference \(Import Metadata\)](../../../csharp/language-reference/compiler-options/reference-compiler-option.md)。
