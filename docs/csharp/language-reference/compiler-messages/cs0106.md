## <a name="example"></a><span data-ttu-id="cccc7-115">示例</span><span class="sxs-lookup"><span data-stu-id="cccc7-115">Example</span></span>  
 <span data-ttu-id="cccc7-116">下面的示例生成 CS0106。</span><span class="sxs-lookup"><span data-stu-id="cccc7-116">The following sample generates CS0106.</span></span>  
  
```csharp  
// CS0106.cs  
namespace MyNamespace  
{  
   interface I  
   {  
      void m();  
      static public void f();   // CS0106  
   }  
  
   public class MyClass  
   {  
      public void I.m() {}   // CS0106  
      public static void Main() {}  
   }  
}  
```
