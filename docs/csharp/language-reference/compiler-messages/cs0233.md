<span data-ttu-id="7ef32-103">“identifier”没有预定义的大小，因此 sizeof 只能在不安全的上下文中使用（请考虑使用 System.Runtime.InteropServices.Marshal.SizeOf）</span><span class="sxs-lookup"><span data-stu-id="7ef32-103">'identifier' does not have a predefined size, therefore sizeof can only be used in an unsafe context (consider using System.Runtime.InteropServices.Marshal.SizeOf)</span></span>  
  
 <span data-ttu-id="7ef32-104">[Sizeof](../../../csharp/language-reference/keywords/sizeof.md) 运算符仅可用于属于编译时常数的类型。</span><span class="sxs-lookup"><span data-stu-id="7ef32-104">The [sizeof](../../../csharp/language-reference/keywords/sizeof.md) operator can only be used for types that are compile-time constants.</span></span> <span data-ttu-id="7ef32-105">如果遇到此错误，请确保可以在编译时确定标识符的大小。</span><span class="sxs-lookup"><span data-stu-id="7ef32-105">If you are getting this error, make sure that the size of the identifier can be determined at compile time.</span></span> <span data-ttu-id="7ef32-106">如果不能，则使用 <xref:System.Runtime.InteropServices.Marshal.SizeOf%2A>，而不是 `sizeof`。</span><span class="sxs-lookup"><span data-stu-id="7ef32-106">If it cannot, then use <xref:System.Runtime.InteropServices.Marshal.SizeOf%2A> instead of `sizeof`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="7ef32-107">示例</span><span class="sxs-lookup"><span data-stu-id="7ef32-107">Example</span></span>  
 <span data-ttu-id="7ef32-108">下面的示例生成 CS0233：</span><span class="sxs-lookup"><span data-stu-id="7ef32-108">The following example generates CS0233:</span></span>  
  
```csharp  
// CS0233.cs  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Sequential)]  
public struct S  
{  
    public int a;  
}  
  
public class MyClass  
{  
    public static void Main()  
    {  
        S myS = new S();  
        Console.WriteLine(sizeof(S));   // CS0233  
        // Try the following line instead:  
        // Console.WriteLine(Marshal.SizeOf(myS));  
   }  
}  
```
