# <a name="compiler-error-cs0433"></a><span data-ttu-id="d56eb-102">编译器错误 CS0433</span><span class="sxs-lookup"><span data-stu-id="d56eb-102">Compiler Error CS0433</span></span>
<span data-ttu-id="d56eb-103">类型 TypeName1 同时存在于 TypeName2 和 TypeName3 中</span><span class="sxs-lookup"><span data-stu-id="d56eb-103">The type TypeName1 exists in both TypeName2 and TypeName3</span></span>  
  
 <span data-ttu-id="d56eb-104">在应用程序中引用的两个不同程序集包含相同的命名空间和类型，这会造成多义性。</span><span class="sxs-lookup"><span data-stu-id="d56eb-104">Two different assemblies referenced in your application contain the same namespace and type, which produces ambiguity.</span></span>  
  
 <span data-ttu-id="d56eb-105">若要解决此错误，请使用 [/reference（C# 编译器选项）](../../../csharp/language-reference/compiler-options/reference-compiler-option.md)编译器选项的别名功能，或者不引用其中一个程序集。</span><span class="sxs-lookup"><span data-stu-id="d56eb-105">To resolve this error, use the alias feature of the [/reference (C# Compiler Options)](../../../csharp/language-reference/compiler-options/reference-compiler-option.md) compiler option or do not reference one of your assemblies.</span></span>  
  
## <a name="example"></a><span data-ttu-id="d56eb-106">示例</span><span class="sxs-lookup"><span data-stu-id="d56eb-106">Example</span></span>

```csharp  
{  
   public class AggPubImpAggPubImp {}  
}  
```

```csharp  
namespace TypeBindConflicts

   public class AggPubImpAggPubImp {}
```  

## <a name="example"></a><span data-ttu-id="d56eb-110">示例</span><span class="sxs-lookup"><span data-stu-id="d56eb-110">Example</span></span>  
 <span data-ttu-id="d56eb-111">下面的示例生成 CS0433。</span><span class="sxs-lookup"><span data-stu-id="d56eb-111">The following example generates CS0433.</span></span>  
  
```csharp  
// CS0433_3.cs  
// compile with: /reference:cs0433_1.dll /reference:cs0433_2.dll  
using TypeBindConflicts;  
public class Test   
{  
   public static void Main()   
   {  
      AggPubImpAggPubImp n6 = new AggPubImpAggPubImp();   // CS0433  
   }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="d56eb-112">示例</span><span class="sxs-lookup"><span data-stu-id="d56eb-112">Example</span></span>

```csharp  
// CS0433_4.cs  
// compile with: /reference:cs0433_1.dll /reference:TypeBindConflicts=cs0433_2.dll  
using TypeBindConflicts;

   public static void Main()   
   {  
      AggPubImpAggPubImp n6 = new AggPubImpAggPubImp();  
   }  
}  
```
