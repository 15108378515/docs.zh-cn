 <span data-ttu-id="d8d3f-104">如果变量是 [const](../../../csharp/language-reference/keywords/const.md) 变量，则它也是 [static](../../../csharp/language-reference/keywords/static.md) 变量。</span><span class="sxs-lookup"><span data-stu-id="d8d3f-104">If a variable is [const](../../../csharp/language-reference/keywords/const.md), it is also [static](../../../csharp/language-reference/keywords/static.md).</span></span> <span data-ttu-id="d8d3f-105">如果需要一个既是 const 又是 static 的变量，只需将该变量声明为 const 即可；如果只需要 static 变量，只需将其标记为 static。</span><span class="sxs-lookup"><span data-stu-id="d8d3f-105">If you want a **const** and **static** variable, just declare that variable as **const**; if all you want is a **static** variable, just mark it **static**.</span></span>
