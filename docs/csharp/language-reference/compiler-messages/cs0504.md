# <a name="compiler-error-cs0504"></a><span data-ttu-id="89175-102">编译器错误 CS0504</span><span class="sxs-lookup"><span data-stu-id="89175-102">Compiler Error CS0504</span></span>
<span data-ttu-id="89175-103">常数“variable”不能标记为 static</span><span class="sxs-lookup"><span data-stu-id="89175-103">The constant 'variable' cannot be marked static</span></span>  
  
 <span data-ttu-id="89175-104">如果变量是 [const](../../../csharp/language-reference/keywords/const.md) 变量，则它也是 [static](../../../csharp/language-reference/keywords/static.md) 变量。</span><span class="sxs-lookup"><span data-stu-id="89175-104">If a variable is [const](../../../csharp/language-reference/keywords/const.md), it is also [static](../../../csharp/language-reference/keywords/static.md).</span></span> <span data-ttu-id="89175-105">如果需要一个既是 const 又是 static 的变量，只需将该变量声明为 const 即可；如果只需要 static 变量，只需将其标记为 static。</span><span class="sxs-lookup"><span data-stu-id="89175-105">If you want a **const** and **static** variable, just declare that variable as **const**; if all you want is a **static** variable, just mark it **static**.</span></span>  
  
 <span data-ttu-id="89175-106">下面的示例生成 CS0504：</span><span class="sxs-lookup"><span data-stu-id="89175-106">The following sample generates CS0504:</span></span>  
  
```csharp  
// CS0504.cs  
namespace x  
{  
   abstract public class clx  
   {  
      static const int i = 0;   // CS0504, cannot be both static and const  
      abstract public void f();  
   }  
}  
```
