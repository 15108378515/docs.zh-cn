# <a name="compiler-error-cs0507"></a><span data-ttu-id="93bb2-102">编译器错误 CS0507</span><span class="sxs-lookup"><span data-stu-id="93bb2-102">Compiler Error CS0507</span></span>
<span data-ttu-id="93bb2-103">“function1”：重写“access”继承成员“function2”时不能更改访问修饰符</span><span class="sxs-lookup"><span data-stu-id="93bb2-103">'function1' : cannot change access modifiers when overriding 'access' inherited member 'function2'</span></span>  
  
 <span data-ttu-id="93bb2-104">尝试在方法重写中更改访问规范。</span><span class="sxs-lookup"><span data-stu-id="93bb2-104">An attempt was made to change the access specification in a method override.</span></span>  
  
## <a name="example"></a><span data-ttu-id="93bb2-105">示例</span><span class="sxs-lookup"><span data-stu-id="93bb2-105">Example</span></span>  
 <span data-ttu-id="93bb2-106">下面的示例生成 CS0507。</span><span class="sxs-lookup"><span data-stu-id="93bb2-106">The following sample generates CS0507.</span></span>  
  
```csharp  
// CS0507.cs  
abstract public class clx  
{  
   virtual protected void f() {}  
}  
  
public class cly : clx  
{  
   public override void f() {}   // CS0507  
   public static void Main() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="93bb2-107">示例</span><span class="sxs-lookup"><span data-stu-id="93bb2-107">Example</span></span>  
 <span data-ttu-id="93bb2-108">如果一个类尝试重写在引用的元数据中定义的标记为 `protected internal` 的方法时，也会发生 CS0507。</span><span class="sxs-lookup"><span data-stu-id="93bb2-108">CS0507 can also occur if a class attempts to override a method marked as `protected internal` defined in referenced metadata.</span></span> <span data-ttu-id="93bb2-109">在此情况下，重写方法应标记为 `protected`。</span><span class="sxs-lookup"><span data-stu-id="93bb2-109">In this situation, the overriding method should be marked as `protected`.</span></span>  
  
```csharp

abstract public class clx  
{  
   virtual protected internal void f() {}  
}  
```
