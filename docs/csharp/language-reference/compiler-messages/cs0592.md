<span data-ttu-id="4b0fd-103">属性“attribute”对于此声明类型无效。</span><span class="sxs-lookup"><span data-stu-id="4b0fd-103">Attribute 'attribute' is not valid on this declaration type.</span></span> <span data-ttu-id="4b0fd-104">仅对“type”声明有效。</span><span class="sxs-lookup"><span data-stu-id="4b0fd-104">It is valid on 'type' declarations only.</span></span>  
  
 <span data-ttu-id="4b0fd-105">定义属性时，通过指定 `AttributeTargets` 值，可以定义该属性可应用于哪些构造。</span><span class="sxs-lookup"><span data-stu-id="4b0fd-105">When you define an attribute, you define what constructs it can be applied to by specifying an `AttributeTargets` value.</span></span> <span data-ttu-id="4b0fd-106">在下面的示例中，`MyAttribute` 属性仅可应用于接口，因为 `AttributeUsage` 属性指定 `AttributeTargets.Iterface`。</span><span class="sxs-lookup"><span data-stu-id="4b0fd-106">In the following example, the `MyAttribute` attribute can be applied to interfaces only, because the `AttributeUsage` attribute specifies `AttributeTargets.Iterface`.</span></span> <span data-ttu-id="4b0fd-107">产生此错误的原因在于属性应用于类（类 `A`）。</span><span class="sxs-lookup"><span data-stu-id="4b0fd-107">The error is generated because the attribute is applied to a class (class `A`).</span></span>  
  
## <a name="example"></a><span data-ttu-id="4b0fd-108">示例</span><span class="sxs-lookup"><span data-stu-id="4b0fd-108">Example</span></span>  
 <span data-ttu-id="4b0fd-109">下面的示例生成 CS0592：</span><span class="sxs-lookup"><span data-stu-id="4b0fd-109">The following sample generates CS0592:</span></span>  
  
```csharp  
// CS0592.cs  
using System;  
  
[AttributeUsage(AttributeTargets.Interface)]  
public class MyAttribute : Attribute   
{  
}  
  
[MyAttribute]  
// Generates CS0592 because MyAttribute is not valid for a class.   
public class A    
{  
    public static void Main()  
    {  
    }  
}  
```
