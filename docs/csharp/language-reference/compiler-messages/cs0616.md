## <a name="example"></a><span data-ttu-id="59b00-106">示例</span><span class="sxs-lookup"><span data-stu-id="59b00-106">Example</span></span>

## <a name="example"></a><span data-ttu-id="59b00-108">示例</span><span class="sxs-lookup"><span data-stu-id="59b00-108">Example</span></span>  
 <span data-ttu-id="59b00-109">下面的示例演示可以如何定义属性：</span><span class="sxs-lookup"><span data-stu-id="59b00-109">The following sample shows how you might define an attribute:</span></span>  
  
```csharp  
// CreateAttrib.cs  
// compile with: /target:library  
using System;  
  
[AttributeUsage(AttributeTargets.Class|AttributeTargets.Interface)]  
public class MyAttr : Attribute  
{  
   public int Name = 0;  
   public int Count = 0;  
  
   public MyAttr (int iCount, int sName)  
   {  
      Count = iCount;  
      Name = sName;  
   }  
}  
  
[MyAttr(5, 50)]  
class Class1 {}  
  
[MyAttr(6, 60)]  
interface Interface1 {}  
```
