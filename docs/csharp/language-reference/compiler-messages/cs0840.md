 <span data-ttu-id="eacae-105">除非常规属性标记为 `abstract` 或 `extern`，或者是 `partial` 类型的成员，否则它必须提供主体。</span><span class="sxs-lookup"><span data-stu-id="eacae-105">Unless a regular property is marked as `abstract` or `extern`, or is a member of a `partial` type, it must supply a body.</span></span> <span data-ttu-id="eacae-106">自动实现的属性不提供访问器主体，但必须指定这两个访问器。</span><span class="sxs-lookup"><span data-stu-id="eacae-106">Auto-implemented properties do not provide accessor bodies, but they must specify both accessors.</span></span> <span data-ttu-id="eacae-107">若要创建自动实现的只读属性，使 set 访问器为 `private`。</span><span class="sxs-lookup"><span data-stu-id="eacae-107">To create a read-only auto-implemented property, make the set accessor `private`.</span></span>

## <a name="example"></a><span data-ttu-id="eacae-110">示例</span><span class="sxs-lookup"><span data-stu-id="eacae-110">Example</span></span>

{  
    public int myProp { get; } // CS0840
