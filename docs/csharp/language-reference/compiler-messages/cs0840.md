# <a name="compiler-error-cs0840"></a><span data-ttu-id="a5a63-102">编译器错误 CS0840</span><span class="sxs-lookup"><span data-stu-id="a5a63-102">Compiler Error CS0840</span></span>
<span data-ttu-id="a5a63-103">“Property name”必须声明主体，因为它未标记为 abstract 或 extern。</span><span class="sxs-lookup"><span data-stu-id="a5a63-103">'Property name' must declare a body because it is not marked abstract or extern.</span></span> <span data-ttu-id="a5a63-104">自动实现的属性必须定义 get 和 set 访问器。</span><span class="sxs-lookup"><span data-stu-id="a5a63-104">Automatically implemented properties must define both get and set accessors.</span></span>  
  
 <span data-ttu-id="a5a63-105">除非常规属性标记为 `abstract` 或 `extern`，或者是 `partial` 类型的成员，否则它必须提供主体。</span><span class="sxs-lookup"><span data-stu-id="a5a63-105">Unless a regular property is marked as `abstract` or `extern`, or is a member of a `partial` type, it must supply a body.</span></span> <span data-ttu-id="a5a63-106">自动实现的属性不提供访问器主体，但必须指定这两个访问器。</span><span class="sxs-lookup"><span data-stu-id="a5a63-106">Auto-implemented properties do not provide accessor bodies, but they must specify both accessors.</span></span> <span data-ttu-id="a5a63-107">若要创建自动实现的只读属性，使 set 访问器为 `private`。</span><span class="sxs-lookup"><span data-stu-id="a5a63-107">To create a read-only auto-implemented property, make the set accessor `private`.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="a5a63-108">更正此错误</span><span class="sxs-lookup"><span data-stu-id="a5a63-108">To correct this error</span></span>  
  
1.  <span data-ttu-id="a5a63-109">提供缺少的正文或访问器，否则对其和/或其封闭类型使用 [abstract](../../../csharp/language-reference/keywords/abstract.md)、[extern](../../../csharp/language-reference/keywords/extern.md) 或 [（类型）](../../../csharp/language-reference/keywords/partial-type.md)修饰符。</span><span class="sxs-lookup"><span data-stu-id="a5a63-109">Supply the missing body or accessor or else use the [abstract](../../../csharp/language-reference/keywords/abstract.md), [extern](../../../csharp/language-reference/keywords/extern.md), or [partial (Type)](../../../csharp/language-reference/keywords/partial-type.md) modifiers on it and/or its enclosing type.</span></span>  
  
## <a name="example"></a><span data-ttu-id="a5a63-110">示例</span><span class="sxs-lookup"><span data-stu-id="a5a63-110">Example</span></span>  
 <span data-ttu-id="a5a63-111">下面的示例生成 CS0840：</span><span class="sxs-lookup"><span data-stu-id="a5a63-111">The following example generates CS0840:</span></span>  
  
```csharp  
// cs0840.cs

using System;  
class Test  
{  
    public int myProp { get; } // CS0840  
  
    // to create a read-only property

```
