# <a name="compiler-error-cs0845"></a><span data-ttu-id="4a616-102">编译器错误 CS0845</span><span class="sxs-lookup"><span data-stu-id="4a616-102">Compiler Error CS0845</span></span>

## <a name="example"></a><span data-ttu-id="4a616-107">示例</span><span class="sxs-lookup"><span data-stu-id="4a616-107">Example</span></span>  
 <span data-ttu-id="4a616-108">下面的代码生成 CS0845：</span><span class="sxs-lookup"><span data-stu-id="4a616-108">The following code generates CS0845:</span></span>  
  
```csharp  
// cs0845.cs  
using System;  
using System.Linq;  
using System.Linq.Expressions;  
  
namespace ConsoleApplication1  
{  
    class Program  
    {  
        static void Main(string[] args)  
        {  
            Expression<Func<object>> e = () => null ?? null; // CS0845  
            // Try the following line instead.  
            // Expression<Func<object>> e = () => (object)null ?? null;  
        }  
    }  
}  
```
