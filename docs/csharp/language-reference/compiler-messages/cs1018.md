# <a name="compiler-error-cs1018"></a><span data-ttu-id="5f230-102">编译器错误 CS1018</span><span class="sxs-lookup"><span data-stu-id="5f230-102">Compiler Error CS1018</span></span>
<span data-ttu-id="5f230-103">应输入关键字“this”或“base”</span><span class="sxs-lookup"><span data-stu-id="5f230-103">Keyword 'this' or 'base' expected</span></span>  
  
 <span data-ttu-id="5f230-104">编译器遇到了不完整的构造函数声明。</span><span class="sxs-lookup"><span data-stu-id="5f230-104">The compiler encountered an incomplete constructor declaration.</span></span>  
  
## <a name="example"></a><span data-ttu-id="5f230-105">示例</span><span class="sxs-lookup"><span data-stu-id="5f230-105">Example</span></span>  
 <span data-ttu-id="5f230-106">下面的示例生成 CS1018 并建议了几种解决此错误的方法：</span><span class="sxs-lookup"><span data-stu-id="5f230-106">The following example generates CS1018, and suggests several ways to resolve the error:</span></span>  
  
```csharp  
// CS1018.cs  
public class C  
{  
}  
  
public class a : C  
{  
    public a(int i)  
    {  
    }  
  
    public a () :   // CS1018  
    // possible resolutions:  
    // public a () resolves by removing the colon  
    // public a () : base() calls C's default constructor  
    // public a () : this(1) calls the assignment constructor of class a  
    {  
    }  
  
    public static int Main()  
    {  
        return 1;  
    }  
}  
```
