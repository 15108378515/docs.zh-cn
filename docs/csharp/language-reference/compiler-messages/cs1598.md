# Compiler Warning (level 1) CS1598
未能加载 XML 分析器，原因如下：“\<原因\>”。将不生成 XML 文档文件“\<文件\>”。  
  
 指定了 [\/doc](../../../csharp/language-reference/compiler-options/doc-compiler-option.md) 选项，但编译器未能找到并加载 msxml3.dll。  请确保 msxml3.dll 文件已安装并注册。
