# <a name="compiler-warning-level-1-cs1690"></a><span data-ttu-id="102b2-102">编译器警告（等级 1）CS1690</span><span class="sxs-lookup"><span data-stu-id="102b2-102">Compiler Warning (level 1) CS1690</span></span>
<span data-ttu-id="102b2-103">访问“member”上的成员会导致运行时异常，因为它是引用封送类的字段</span><span class="sxs-lookup"><span data-stu-id="102b2-103">Accessing a member on 'member' may cause a runtime exception because it is a field of a marshal-by-reference class</span></span>  
  
 <span data-ttu-id="102b2-104">尝试调用派生自 <xref:System.MarshalByRefObject> 的类的成员上的方法、属性或索引器且成员是值类型时，会出现此警告。</span><span class="sxs-lookup"><span data-stu-id="102b2-104">This warning occurs when you try to call a method, property, or indexer on a member of a class that derives from <xref:System.MarshalByRefObject>, and the member is a value type.</span></span> <span data-ttu-id="102b2-105">在应用程序域中，继承自 `MarshalByRefObject` 的对象通常打算以引用方式封送。</span><span class="sxs-lookup"><span data-stu-id="102b2-105">Objects that inherit from `MarshalByRefObject` are typically intended to be marshaled by reference across an application domain.</span></span> <span data-ttu-id="102b2-106">如果任何代码曾尝试在应用程序域中直接访问此类对象的值类型成员，会发生运行时异常。</span><span class="sxs-lookup"><span data-stu-id="102b2-106">If any code ever attempts to directly access the value-type member of such an object across an application domain, a runtime exception will occur.</span></span> <span data-ttu-id="102b2-107">要解决此警告，首先将该成员复制到本地变量中，然后在该变量上调用方法。</span><span class="sxs-lookup"><span data-stu-id="102b2-107">To resolve the warning, first copy the member into a local variable and call the method on that variable.</span></span>  
  
 <span data-ttu-id="102b2-108">下面的示例生成 CS1690：</span><span class="sxs-lookup"><span data-stu-id="102b2-108">The following sample generates CS1690:</span></span>  
  
```csharp  
// CS1690.cs  
using System;  
  
class WarningCS1690: MarshalByRefObject  
{  
   int i = 5;  
  
   public static void Main()   
   {  
     WarningCS1690 e = new WarningCS1690();  
     e.i.ToString();   // CS1690  
  
     // OK  
     int i = e.i;  
     i.ToString();  
     e.i = i;  
   }  
}  
```
