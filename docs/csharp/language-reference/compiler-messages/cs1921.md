# <a name="compiler-error-cs1921"></a><span data-ttu-id="326c2-102">编译器错误 CS1921</span><span class="sxs-lookup"><span data-stu-id="326c2-102">Compiler Error CS1921</span></span>
<span data-ttu-id="326c2-103">匹配“method”的最佳重载方法具有对于初始化表达式元素而言错误的签名。</span><span class="sxs-lookup"><span data-stu-id="326c2-103">The best overloaded method match for 'method' has wrong signature for the initializer element.</span></span> <span data-ttu-id="326c2-104">可初始化的 Add 必须是可访问的实例方法。</span><span class="sxs-lookup"><span data-stu-id="326c2-104">The initializable Add must be an accessible instance method.</span></span>  
  
 <span data-ttu-id="326c2-105">尝试将集合初始值设定项与不具有公共非静态 `Add` 方法的类配合使用时，会生成此错误。</span><span class="sxs-lookup"><span data-stu-id="326c2-105">This error is generated when you try to use a collection initializer with a class that has no public non-static `Add` method.</span></span> <span data-ttu-id="326c2-106">如果 `Add` 方法因其保护级别（`private`、`protected`、`internal`）而不可访问，则会收到 CS0122，从而此错误可能表示该方法定义为 `static`。</span><span class="sxs-lookup"><span data-stu-id="326c2-106">If the `Add` method is not accessible because of its protection level (`private`, `protected`, `internal`) then you will get CS0122, so that this error probably means that the method is defined as `static`.</span></span>
