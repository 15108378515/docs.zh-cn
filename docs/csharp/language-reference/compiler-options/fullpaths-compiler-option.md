# /fullpaths (C# Compiler Options)
**\/fullpaths** 选项可使编译器在列出编译错误和警告时指定文件的完整路径。  
  
## 语法  
  
```  
/fullpaths  
```  
  
## 备注  
 默认情况下，编译所产生的错误和警告指定包含错误的文件名。  **\/fullpaths** 选项使编译器指定文件的完整路径。  
  
 此编译器选项在 Visual Studio 中不可用，且不能通过编程方式进行更改。
