# /help, /? (C# Compiler Options)
该选项将编译器选项的列表和每个选项的简短说明发送到 stdout。  
  
## 语法  
  
```  
/help  
/?  
```  
  
## 备注  
 如果编译中包含该选项，将不会创建输出文件，也不会进行编译。  
  
 此编译器选项在 Visual Studio 中不可用，且不能通过编程方式进行更改。
