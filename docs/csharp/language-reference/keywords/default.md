# default（C# 参考）
`default` 关键字。  
  
 `default` 关键字可在 `switch` 语句或泛型代码中使用。  
  
-   [switch 语句](../../../csharp/language-reference/keywords/switch.md)：指定默认标签。  
  
-   [泛型代码](../../../csharp/programming-guide/generics/default-keyword-in-generic-code.md)：指定类型参数的默认值。  这对于引用类型为 null，对于值类型为零。
