# <a name="in-c-reference"></a>in（C# 参考）

在四种上下文中使用 `in` 上下文关键字：  
  
-   泛型接口和委托中的[泛型类型参数](in-generic-modifier.md)。
-   作为[参数修饰符](in-parameter-modifier.md)，它允许按引用而不是按值向方法传递参数。
-   [foreach](foreach-in.md) 语句。
-   LINQ 查询表达式中的 [join 子句](join-clause.md)。
