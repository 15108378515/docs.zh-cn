# <a name="iteration-statements-c-reference"></a>迭代语句（C# 参考）

可使用迭代语句创建循环。 根据循环终止条件，迭代语句可导致多次执行嵌入式语句。 除非遇到[跳转语句](jump-statements.md)，否则这些语句按顺序执行。

迭代语句中使用下列关键字：

- [do](do.md)

- [for](for.md)

- [foreach, in](foreach-in.md)

- [while](while.md)
