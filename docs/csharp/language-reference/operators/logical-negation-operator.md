# ! 运算符（C# 参考）
逻辑非运算符 \(`!`\) 是对操作数求反的一元运算符。  为 `bool` 定义了该运算符，当且仅当操作数为 `false` 时才返回 `true`。  
  
## 备注  
 用户定义的类型可重载 `!` 运算符（请参见[运算符](../../../csharp/language-reference/keywords/operator.md)）。
