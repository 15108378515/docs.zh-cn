# <a name="compiler-error-cs0021"></a><span data-ttu-id="e56de-102">编译器错误 CS0021</span><span class="sxs-lookup"><span data-stu-id="e56de-102">Compiler Error CS0021</span></span>
<span data-ttu-id="e56de-103">无法将带 [] 的索引应用于“type”类型的表达式</span><span class="sxs-lookup"><span data-stu-id="e56de-103">Cannot apply indexing with [] to an expression of type 'type'</span></span>  
  
 <span data-ttu-id="e56de-104">尝试针对不支持 [Indexers](../../csharp/programming-guide/indexers/index.md)的数据类型使用索引器访问值。</span><span class="sxs-lookup"><span data-stu-id="e56de-104">An attempt was made to access a value through an indexer on a data type that does not support [Indexers](../../csharp/programming-guide/indexers/index.md).</span></span>  
  
 <span data-ttu-id="e56de-105">在 C++ 程序集中尝试使用索引器时，可能会遇到 CS0021。</span><span class="sxs-lookup"><span data-stu-id="e56de-105">You may get CS0021 when trying to use an indexer in a C++ assembly.</span></span> <span data-ttu-id="e56de-106">在这种情况下，请用 `DefaultMember` 特性修饰 C++ 类，让 C# 编译器能够识别默认索引器。</span><span class="sxs-lookup"><span data-stu-id="e56de-106">In this case, decorate the C++ class with the `DefaultMember` attribute so the C# compiler knows which indexer is the default.</span></span> <span data-ttu-id="e56de-107">下面的示例生成 CS0021。</span><span class="sxs-lookup"><span data-stu-id="e56de-107">The following sample generates CS0021.</span></span>  
  
## <a name="example"></a><span data-ttu-id="e56de-108">示例</span><span class="sxs-lookup"><span data-stu-id="e56de-108">Example</span></span>  
 <span data-ttu-id="e56de-109">此文件将编译为 .dll 文件（注释掉了 `DefaultMember` 特性）以生成该错误。</span><span class="sxs-lookup"><span data-stu-id="e56de-109">This file compiles to a .dll file—with the `DefaultMember` attribute commented out—in order to generate the error.</span></span>  
  
```  
// CPP0021.cpp  
// compile with: /clr /LD  
using namespace System::Reflection;  
// Uncomment the following line to resolve  
//[DefaultMember("myItem")]  
public ref class MyClassMC  
{  
        public:  
        property int myItem[int]  
        {  
            int get(int i){  return 5; }  
            void set(int i, int value) {}  
        }  
};  
```  
  
## <a name="example"></a><span data-ttu-id="e56de-110">示例</span><span class="sxs-lookup"><span data-stu-id="e56de-110">Example</span></span>  
 <span data-ttu-id="e56de-111">下面是调用 .dll 文件的 C# 文件。</span><span class="sxs-lookup"><span data-stu-id="e56de-111">The following is the C# file that calls the .dll file.</span></span> <span data-ttu-id="e56de-112">此文件尝试通过索引器访问类，但由于没有将任何成员声明为要使用的默认索引器，所以产生了此错误。</span><span class="sxs-lookup"><span data-stu-id="e56de-112">This file attempts to access the class via an indexer, but because no member was declared as the default indexer to be used, the error is generated.</span></span>  
  
```  
// CS0021.cs  
// compile with: /reference:CPP0021.dll  
public class MyClass  
{  
    public static void Main()  
    {  
        MyClassMC myMC = new MyClassMC();  
        int j = myMC[1]; // CS0021  
    }  
}  
```
