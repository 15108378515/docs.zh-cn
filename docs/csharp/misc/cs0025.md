# <a name="compiler-error-cs0025"></a>编译器错误 CS0025
找不到标准库文件“file”  
  
 找不到编译器需要的文件。 检查路径是否正确并检查该文件是否存在。  
  
 如果该文件是 Visual Studio 系统文件，你可能需要修复你的 Visual Studio 安装，或完全重新安装它。
