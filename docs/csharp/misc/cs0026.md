## <a name="example"></a><span data-ttu-id="9b45f-107">示例</span><span class="sxs-lookup"><span data-stu-id="9b45f-107">Example</span></span>  
 <span data-ttu-id="9b45f-108">下面的示例生成 CS0026：</span><span class="sxs-lookup"><span data-stu-id="9b45f-108">The following example generates CS0026:</span></span>  
  
```csharp  
// CS0026.cs  
public class A  
{  
   public static int i = 0;  
  
   public static void Main()  
   {

      // Try the following line instead:  
      // i = i + 1;  
   }  
}  
```
