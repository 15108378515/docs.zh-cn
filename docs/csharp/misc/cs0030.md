 <span data-ttu-id="6b1a8-104">必须提供转换例程以支持某些运算符重载。</span><span class="sxs-lookup"><span data-stu-id="6b1a8-104">You must provide conversion routines to support certain operator overloads.</span></span> <span data-ttu-id="6b1a8-105">有关详细信息，请参阅[转换运算符](../../csharp/programming-guide/statements-expressions-operators/conversion-operators.md)。</span><span class="sxs-lookup"><span data-stu-id="6b1a8-105">For more information, see [Conversion Operators](../../csharp/programming-guide/statements-expressions-operators/conversion-operators.md).</span></span>  
  
 <span data-ttu-id="6b1a8-106">下面的示例生成 CS0030：</span><span class="sxs-lookup"><span data-stu-id="6b1a8-106">The following sample generates CS0030:</span></span>  
  
```csharp  
// CS0030.cs  
namespace x  
{  
   public class iii  
   {  
      /*  
      public static implicit operator iii(int aa)  
      {  
         return null;  
      }  
  
      public static implicit operator int(iii aa)  
      {  
         return 0;  
      }  
      */  
  
      public static iii operator ++(iii aa)  
      {  
         return (iii)0;   // CS0030  
         // uncomment the conversion routines to resolve CS0030  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
