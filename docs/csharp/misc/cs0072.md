<span data-ttu-id="b0951-103">“event”：无法重写；“method”不是事件</span><span class="sxs-lookup"><span data-stu-id="b0951-103">'event' : cannot override; 'method' is not an event</span></span>  
  
 <span data-ttu-id="b0951-104">[事件](../../csharp/language-reference/keywords/event.md) 只能重写另一事件。</span><span class="sxs-lookup"><span data-stu-id="b0951-104">An [event](../../csharp/language-reference/keywords/event.md) can only override another event.</span></span> <span data-ttu-id="b0951-105">有关详细信息，请参阅[事件](../../csharp/programming-guide/events/index.md)。</span><span class="sxs-lookup"><span data-stu-id="b0951-105">For more information, see [Events](../../csharp/programming-guide/events/index.md).</span></span>  
  
 <span data-ttu-id="b0951-106">下面的示例生成 CS0072：</span><span class="sxs-lookup"><span data-stu-id="b0951-106">The following sample generates CS0072:</span></span>  
  
```csharp  
// CS0072.cs  
delegate void MyDelegate();  
  
class Test1  
{  
   public virtual event MyDelegate MyEvent;  
   public virtual void VMeth()  
   {  
   }  
  
   public void FireAway()  
   {  
      if (MyEvent != null)  
         MyEvent();  
   }  
}  
  
class Test2 : Test1  
{  
   public override event MyDelegate VMeth   // CS0072  
   // uncomment the following lines to resolve  
   // public override event MyDelegate MyEvent  
   {  
      add  
      {  
         VMeth += value;  
         // MyEvent += value;  
      }  
      remove  
      {  
         VMeth -= value;  
         // MyEvent -= value;  
      }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
