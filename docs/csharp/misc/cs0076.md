# <a name="compiler-error-cs0076"></a>编译器错误 CS0076
枚举器名“value__”是保留字，不能使用  
  
 [枚举](../../csharp/language-reference/keywords/enum.md) 中的项不能具有名为 **value__** 的标识符。
