## <a name="example"></a><span data-ttu-id="1ddd8-106">示例</span><span class="sxs-lookup"><span data-stu-id="1ddd8-106">Example</span></span>  
 <span data-ttu-id="1ddd8-107">下面的示例生成 CS0082：</span><span class="sxs-lookup"><span data-stu-id="1ddd8-107">The following example generates CS0082:</span></span>  
  
```  
//cs0082.cs  
class MyClass  
{  
  
    //property  
    public int MyProp  
    {  
        get //CS0082  
        {  
            return 1;  
        }  
    }  
  
    //conflicting Get  
    public int get_MyProp()  
    {  
        return 2;  
    }  
  
    public static int Main()  
    {  
        return 1;  
    }  
}  
```
