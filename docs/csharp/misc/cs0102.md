## <a name="example"></a><span data-ttu-id="58989-106">示例</span><span class="sxs-lookup"><span data-stu-id="58989-106">Example</span></span>  
 <span data-ttu-id="58989-107">下面的示例生成 CS0102。</span><span class="sxs-lookup"><span data-stu-id="58989-107">The following sample generates CS0102.</span></span>  
  
```csharp

{  
   public class MyClass  
   {  
      string s = "Hello";  
      string s = "Goodbye";   // CS0102  
  
      public void GetString()  
      {  
         string s = "Hello again";   // method scope, no error  
      }  
   }  
}  
```
