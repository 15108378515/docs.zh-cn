# <a name="compiler-error-cs0110"></a><span data-ttu-id="4edd7-102">编译器错误 CS0110</span><span class="sxs-lookup"><span data-stu-id="4edd7-102">Compiler Error CS0110</span></span>
<span data-ttu-id="4edd7-103">“const declaration”的常量值计算涉及循环定义</span><span class="sxs-lookup"><span data-stu-id="4edd7-103">The evaluation of the constant value for 'const declaration' involves a circular definition</span></span>  
  
 <span data-ttu-id="4edd7-104">[const](../../csharp/language-reference/keywords/const.md) 变量 (`a`) 的声明不能引用另一个也引用 (`b`) 的 const 变量 (`a`)。</span><span class="sxs-lookup"><span data-stu-id="4edd7-104">The declaration of a [const](../../csharp/language-reference/keywords/const.md) variable (`a`) cannot reference another const variable (`b`) that also references (`a`).</span></span>  
  
 <span data-ttu-id="4edd7-105">下面的示例生成 CS0110：</span><span class="sxs-lookup"><span data-stu-id="4edd7-105">The following sample generates CS0110:</span></span>  
  
```csharp  
// CS0110.cs  
namespace MyNamespace  
{  
   public class A  
   {  
      public static void Main()  
      {  
      }  
   }  
  
   public class B : A  
   {  
      public const int i = c + 1;   // CS0110, c already references i  
      public const int c = i + 1;  
      // the following line would be OK  
      // public const int c = 10;  
   }  
}  
```
