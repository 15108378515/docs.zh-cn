 <span data-ttu-id="79ad2-104">使用 `override`、 **virtual**或 **abstract** 关键字的任何方法声明也不能使用 **static** 关键字。</span><span class="sxs-lookup"><span data-stu-id="79ad2-104">Any method declaration that uses the `override`, **virtual**, or **abstract** keyword cannot also use the **static** keyword.</span></span>  
  
 <span data-ttu-id="79ad2-105">有关详细信息，请参阅[方法](../../csharp/programming-guide/classes-and-structs/methods.md)。</span><span class="sxs-lookup"><span data-stu-id="79ad2-105">For more information, see [Methods](../../csharp/programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="79ad2-106">下面的示例生成 CS0112:</span><span class="sxs-lookup"><span data-stu-id="79ad2-106">The following sample generates CS0112:</span></span>  
  
```csharp  
// CS0112.cs  
namespace MyNamespace  
{  
   abstract public class MyClass  
   {  
      public abstract void Foo();  
   }  
   public class MyClass2 : MyClass  
   {  
      override public static void Foo()   // CS0112, remove static keyword  
      {  
      }  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
