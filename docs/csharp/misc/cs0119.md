# <a name="compiler-error-cs0119"></a>编译器错误 CS0119
“construct1_name”是“construct1”，在指定上下文中无效。  
  
 编译器检测到了以下意外构造:  
  
-   类构造函数不是条件语句中的有效测试表达式。  
  
-   使用了类名而不是实例名来引用数组元素。  
  
-   将方法标识符视为结构或类使用  
  
## <a name="example"></a>示例  
 下面的示例生成 CS0119。  
  
```csharp  
// CS0119.cs  
using System;  
public class MyClass   
{  
   public static void Test() {}  
  
   public static void Main()  
   {  
      Console.WriteLine(Test.x);   // CS0119  
   }  
}  
```
