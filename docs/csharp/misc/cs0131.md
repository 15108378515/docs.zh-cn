## <a name="example"></a><span data-ttu-id="25f90-108">示例</span><span class="sxs-lookup"><span data-stu-id="25f90-108">Example</span></span>  
 <span data-ttu-id="25f90-109">以下示例生成 CS0131。</span><span class="sxs-lookup"><span data-stu-id="25f90-109">The following sample generates CS0131.</span></span>  
  
```  
// CS0131.cs  
public class MyClass  
{  
    public int i = 0;  
    public void MyMethod()  
    {  
        i++ = 1;   // CS0131  
        // try the following line instead  
        // i = 1;  
    }  
    public static void Main() { }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="25f90-110">示例</span><span class="sxs-lookup"><span data-stu-id="25f90-110">Example</span></span>

{  
    public static int Main()  
    {  
        int a = 1, b = 2, c = 3;
