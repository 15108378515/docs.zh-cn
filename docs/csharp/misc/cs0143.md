# <a name="compiler-error-cs0143"></a>编译器错误 CS0143
类型“class”未定义构造函数  
  
 没有合适的构造函数可用。 这是内置数值类型情况，只需通过向它们赋值即可初始化。  
  
 下面的示例生成 CS0143：  
  
```csharp  
// CS0143.cs  
class MyClass  
{  
   static public void Main ()  
   {  
      double d = new double(4.5);   // CS0143  
      // Try this line instead:  
      // double d = 4.5;  
   }  
}  
```
