 <span data-ttu-id="c6848-104">标签在 [switch](../../csharp/language-reference/keywords/switch.md) 语句中重复。</span><span class="sxs-lookup"><span data-stu-id="c6848-104">A label was repeated in a [switch](../../csharp/language-reference/keywords/switch.md) statement.</span></span> <span data-ttu-id="c6848-105">有关详细信息，请参阅 [switch](../../csharp/language-reference/keywords/switch.md)。</span><span class="sxs-lookup"><span data-stu-id="c6848-105">For more information, see [switch](../../csharp/language-reference/keywords/switch.md).</span></span>  
  
 <span data-ttu-id="c6848-106">以下示例生成 CS0152：</span><span class="sxs-lookup"><span data-stu-id="c6848-106">The following sample generates CS0152:</span></span>  
  
```csharp  
// CS0152.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
  
         switch (i)  
         {  
            case 1:  
               i++;  
               return;  
  
            case 1:   // CS0152, two case 1 statements  
               i++;  
               return;  
         }  
      }  
   }  
}  
```
