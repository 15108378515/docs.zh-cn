# <a name="compiler-error-cs0159"></a><span data-ttu-id="1b3a4-102">编译器错误 CS0159</span><span class="sxs-lookup"><span data-stu-id="1b3a4-102">Compiler Error CS0159</span></span>
<span data-ttu-id="1b3a4-103">goto 语句范围内没有“label”这样的标签</span><span class="sxs-lookup"><span data-stu-id="1b3a4-103">No such label 'label' within the scope of the goto statement</span></span>  
  
 <span data-ttu-id="1b3a4-104">在 [语句范围内找不到由](../../csharp/language-reference/keywords/goto.md) goto `goto` 语句引用的标签。</span><span class="sxs-lookup"><span data-stu-id="1b3a4-104">The label referenced by the [goto](../../csharp/language-reference/keywords/goto.md) statement could not be found within the scope of the `goto` statement.</span></span>  
  
 <span data-ttu-id="1b3a4-105">以下示例生成 CS0159：</span><span class="sxs-lookup"><span data-stu-id="1b3a4-105">The following sample generates CS0159:</span></span>  
  
```  
// CS0159.cs  
public class Class1  
{  
   public static void Main()  
   {  
      int i = 0;  
  
      switch (i)  
      {  
         case 1:  
            goto case 3;   // CS0159, case 3 label does not exist  
         case 2:  
            break;  
      }  
      goto NOWHERE;   // CS0159, NOWHERE label does not exist  
   }  
}  
```
