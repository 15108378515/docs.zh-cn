# <a name="compiler-warning-level-3-cs0168"></a><span data-ttu-id="57854-102">编译器警告（等级 3）CS0168</span><span class="sxs-lookup"><span data-stu-id="57854-102">Compiler Warning (level 3) CS0168</span></span>
<span data-ttu-id="57854-103">声明但从未使用变量 var</span><span class="sxs-lookup"><span data-stu-id="57854-103">The variable 'var' is declared but never used</span></span>

<span data-ttu-id="57854-104">当你声明变量，但不是使用时，编译器会发出 3 级警告。</span><span class="sxs-lookup"><span data-stu-id="57854-104">The compiler issues a level-three warning when you declare a variable, but do not use it.</span></span>
  
<span data-ttu-id="57854-105">下面的示例生成 CS0168 警告：</span><span class="sxs-lookup"><span data-stu-id="57854-105">The following sample generates a CS0168 warning:</span></span>  
  
```csharp
// CS0168.cs  
// compile with: /W:3  
public class clx
{
    public int i;
}

public class clz
{
    public static void Main() {
        clx a; // CS0168, the variable 'a' is declared but never used
        // try the following line instead
        // clx a = new clx();  // this does not generate a warning because the clx constructor must execute.
    }
}
```
