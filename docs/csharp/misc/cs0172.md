# <a name="compiler-error-cs0172"></a><span data-ttu-id="d3429-102">编译器错误 CS0172</span><span class="sxs-lookup"><span data-stu-id="d3429-102">Compiler Error CS0172</span></span>
<span data-ttu-id="d3429-103">无法确定条件表达式的类型，因为“type1”和“type2”可相互隐式转换</span><span class="sxs-lookup"><span data-stu-id="d3429-103">Type of conditional expression cannot be determined because 'type1' and 'type2' implicitly convert to one another</span></span>  
  
 <span data-ttu-id="d3429-104">在条件语句中，必须能够转换 `:` 运算符左右两侧的类型。</span><span class="sxs-lookup"><span data-stu-id="d3429-104">In a conditional statement, you must be able to convert the types on either side of the `:` operator.</span></span> <span data-ttu-id="d3429-105">此外，不能有相互转换例程；只需一次转换。</span><span class="sxs-lookup"><span data-stu-id="d3429-105">Also, there cannot be mutual conversion routines; you only need one conversion.</span></span> <span data-ttu-id="d3429-106">有关详细信息，请参阅[转换运算符](../../csharp/programming-guide/statements-expressions-operators/conversion-operators.md)。</span><span class="sxs-lookup"><span data-stu-id="d3429-106">For more information, see [Conversion Operators](../../csharp/programming-guide/statements-expressions-operators/conversion-operators.md).</span></span>  
  
 <span data-ttu-id="d3429-107">下面的示例生成 CS0172：</span><span class="sxs-lookup"><span data-stu-id="d3429-107">The following sample generates CS0172:</span></span>  
  
```  
// CS0172.cs  
public class Square  
{  
   public class Circle  
   {  
      public static implicit operator Circle(Square aa)  
      {  
         return null;  
      }  
  
      public static implicit operator Square(Circle aa)  
      // using explicit resolves this error  
      // public static explicit operator Square(Circle aa)  
      {  
         return null;  
      }  
   }  
  
   public static void Main()  
   {  
      Circle aa = new Circle();  
      Square ii = new Square();  
      object o = (1 == 1) ? aa : ii;   // CS0172  
      // the following cast would resolve this error  
      // (1 == 1) ? aa : (Circle)ii;  
   }  
}  
```
