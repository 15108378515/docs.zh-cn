# <a name="compiler-error-cs0175"></a><span data-ttu-id="e87b0-102">编译器错误 CS0175</span><span class="sxs-lookup"><span data-stu-id="e87b0-102">Compiler Error CS0175</span></span>
<span data-ttu-id="e87b0-103">在此上下文中使用关键字 "base" 无效</span><span class="sxs-lookup"><span data-stu-id="e87b0-103">Use of keyword 'base' is not valid in this context</span></span>  
  
 <span data-ttu-id="e87b0-104">必须使用 [base](../../csharp/language-reference/keywords/base.md) 关键字指定基类的特定成员。</span><span class="sxs-lookup"><span data-stu-id="e87b0-104">The [base](../../csharp/language-reference/keywords/base.md) keyword must be used to specify a particular member of the base class.</span></span> <span data-ttu-id="e87b0-105">有关详细信息，请参阅[构造函数](../../csharp/programming-guide/classes-and-structs/constructors.md)。</span><span class="sxs-lookup"><span data-stu-id="e87b0-105">For more information, see [Constructors](../../csharp/programming-guide/classes-and-structs/constructors.md).</span></span>  
  
 <span data-ttu-id="e87b0-106">下面的示例生成 CS0175：</span><span class="sxs-lookup"><span data-stu-id="e87b0-106">The following sample generates CS0175:</span></span>  
  
```csharp  
// CS0175.cs  
using System;  
class BaseClass  
{  
    public int TestInt = 0;  
}  
  
class MyClass : BaseClass  
{  
    public static void Main()  
    {  
        MyClass aClass = new MyClass();  
        aClass.BaseTest();  
    }  
  
    public void BaseTest()  
    {  
        Console.WriteLine(base); // CS0175  
        // Try the following line instead:  
        // Console.WriteLine(base.TestInt);  
        base = 9;   // CS0175  
  
        // Try the following line instead:  
        // base.TestInt = 9;  
    }  
}  
```
