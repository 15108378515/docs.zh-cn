# <a name="compiler-error-cs0176"></a><span data-ttu-id="2cc15-102">编译器错误 CS0176</span><span class="sxs-lookup"><span data-stu-id="2cc15-102">Compiler Error CS0176</span></span>

 <span data-ttu-id="2cc15-104">仅可使用类名限定 [静态](../../csharp/language-reference/keywords/static.md) 变量；实例名称不能做为限定符。</span><span class="sxs-lookup"><span data-stu-id="2cc15-104">Only a class name can be used to qualify a [static](../../csharp/language-reference/keywords/static.md) variable; an instance name cannot be a qualifier.</span></span> <span data-ttu-id="2cc15-105">有关详细信息，请参阅[静态类和静态类成员](../../csharp/programming-guide/classes-and-structs/static-classes-and-static-class-members.md)。</span><span class="sxs-lookup"><span data-stu-id="2cc15-105">For more information, see [Static Classes and Static Class Members](../../csharp/programming-guide/classes-and-structs/static-classes-and-static-class-members.md).</span></span>  
  
 <span data-ttu-id="2cc15-106">下面的示例生成 CS0176：</span><span class="sxs-lookup"><span data-stu-id="2cc15-106">The following sample generates CS0176:</span></span>  
  
```  
// CS0176.cs  
public class MyClass2  
{  
    public static int num;  
}  
  
public class Test  
{  
    public static void Main()  
    {  
        MyClass2 mc2 = new MyClass2();  
        int i = mc2.num;   // CS0176  
        // try the following line instead  
        // int i = MyClass2.num;  
    }  
}  
```
