# <a name="compiler-warning-level-1-cs0183"></a><span data-ttu-id="24365-102">编译器警告（等级 1）CS0183</span><span class="sxs-lookup"><span data-stu-id="24365-102">Compiler Warning (level 1) CS0183</span></span>
<span data-ttu-id="24365-103">给定表达式始终为所提供的（“类型”）类型</span><span class="sxs-lookup"><span data-stu-id="24365-103">The given expression is always of the provided ('type') type</span></span>  
  
 <span data-ttu-id="24365-104">如果条件语句的计算结果始终为 **true**，则不需要条件语句。</span><span class="sxs-lookup"><span data-stu-id="24365-104">If a conditional statement always evaluates to **true**, then you do not need a conditional statement.</span></span> <span data-ttu-id="24365-105">当尝试使用 **is** 运算符评估类型时，会出现此警告。</span><span class="sxs-lookup"><span data-stu-id="24365-105">This warning occurs when you try to evaluate a type using the **is** operator.</span></span> <span data-ttu-id="24365-106">如果计算结果是值类型，则不需要进行检查。</span><span class="sxs-lookup"><span data-stu-id="24365-106">If the evaluation is a value type, then the check is unnecessary.</span></span>  
  
 <span data-ttu-id="24365-107">下面的示例生成 CS0183:</span><span class="sxs-lookup"><span data-stu-id="24365-107">The following sample generates CS0183:</span></span>  
  
```csharp  
// CS0183.cs  
// compile with: /W:1  
using System;  
public class Test  
{  
   public static void F(Int32 i32, String str)  
   {  
      if (str is Object)          // OK  
         Console.WriteLine( "str is an object" );  
      else  
         Console.WriteLine( "str is not an object" );  
  
      if (i32 is Object)   // CS0183  
         Console.WriteLine( "i32 is an object" );  
      else  
         Console.WriteLine( "i32 is not an object" ); // never reached  
   }  
  
   public static void Main()  
   {  
  
      F(0, "CS0183");  
      F(120, null);   
   }  
}  
```
