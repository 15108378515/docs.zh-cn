## <a name="example"></a><span data-ttu-id="9435f-107">示例</span><span class="sxs-lookup"><span data-stu-id="9435f-107">Example</span></span>  
 <span data-ttu-id="9435f-108">下面的示例生成 CS0191。</span><span class="sxs-lookup"><span data-stu-id="9435f-108">The following sample generates CS0191.</span></span>  
  
```  
// CS0191.cs  
class MyClass  
{  
    public readonly int TestInt = 6;  // OK to assign to readonly field in declaration  
  
    MyClass()  
    {  
        TestInt = 11; // OK to assign to readonly field in constructor  
    }  
  
    public void TestReadOnly()  
    {  
        TestInt = 19;                  // CS0191  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
