<span data-ttu-id="64fb5-103">无法向静态只读字段“name”的字段传递 ref 或 out（在静态构造函数中除外）</span><span class="sxs-lookup"><span data-stu-id="64fb5-103">Fields of static readonly field 'name' cannot be passed ref or out (except in a static constructor)</span></span>  
  
 <span data-ttu-id="64fb5-104">A [readonly](../../csharp/language-reference/keywords/readonly.md)变量必须具有相同[静态](../../csharp/language-reference/keywords/static.md)作为你要在其中将其作为传递构造函数的使用情况[ref](../../csharp/language-reference/keywords/ref.md)或[出](../../csharp/language-reference/keywords/out-parameter-modifier.md)参数。</span><span class="sxs-lookup"><span data-stu-id="64fb5-104">A [readonly](../../csharp/language-reference/keywords/readonly.md) variable must have the same [static](../../csharp/language-reference/keywords/static.md) usage as the constructor in which you want to pass it as a [ref](../../csharp/language-reference/keywords/ref.md) or [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) parameter.</span></span> <span data-ttu-id="64fb5-105">有关详细信息，请参阅[传递参数](../../csharp/programming-guide/classes-and-structs/passing-parameters.md)。</span><span class="sxs-lookup"><span data-stu-id="64fb5-105">For more information, see [Passing Parameters](../../csharp/programming-guide/classes-and-structs/passing-parameters.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="64fb5-106">示例</span><span class="sxs-lookup"><span data-stu-id="64fb5-106">Example</span></span>  
 <span data-ttu-id="64fb5-107">以下示例生成 CS0199：</span><span class="sxs-lookup"><span data-stu-id="64fb5-107">The following sample generates CS0199:</span></span>  
  
```csharp  
// CS0199.cs  
class MyClass  
{  
    public static readonly int TestInt = 6;  
  
    static void TestMethod(ref int testInt)  
    {  
        testInt = 0;  
    }  
  
    MyClass()  
    {  
        TestMethod(ref TestInt);   // CS0199, TestInt is static  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
