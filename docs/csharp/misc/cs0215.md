 <span data-ttu-id="f4b94-104">用户定义的 [true](../../csharp/language-reference/keywords/true.md) 和 [false](../../csharp/language-reference/keywords/false.md) 运算符必须具有 [bool](../../csharp/language-reference/keywords/bool.md)的返回类型。</span><span class="sxs-lookup"><span data-stu-id="f4b94-104">User-defined [true](../../csharp/language-reference/keywords/true.md) and [false](../../csharp/language-reference/keywords/false.md) operators must have a return type of [bool](../../csharp/language-reference/keywords/bool.md).</span></span> <span data-ttu-id="f4b94-105">有关详细信息，请参阅[可重载运算符](../../csharp/programming-guide/statements-expressions-operators/overloadable-operators.md)。</span><span class="sxs-lookup"><span data-stu-id="f4b94-105">For more information, see [Overloadable Operators](../../csharp/programming-guide/statements-expressions-operators/overloadable-operators.md).</span></span>  
  
 <span data-ttu-id="f4b94-106">以下示例生成 CS0215：</span><span class="sxs-lookup"><span data-stu-id="f4b94-106">The following sample generates CS0215:</span></span>  
  
```csharp  
// CS0215.cs  
class MyClass  
{  
   public static int operator true (MyClass MyInt)   // CS0215  
   // try the following line instead  
   // public static bool operator true (MyClass MyInt)  
   {  
      return true;  
   }  
  
   public static int operator false (MyClass MyInt)   // CS0215  
   // try the following line instead  
   // public static bool operator false (MyClass MyInt)  
   {  
      return true;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
