# <a name="compiler-error-cs0216"></a><span data-ttu-id="7822a-102">编译器错误 CS0216</span><span class="sxs-lookup"><span data-stu-id="7822a-102">Compiler Error CS0216</span></span>
<span data-ttu-id="7822a-103">运算符“operator”要求也要定义匹配的运算符“missing_operator”</span><span class="sxs-lookup"><span data-stu-id="7822a-103">The operator 'operator' requires a matching operator 'missing_operator' to also be defined</span></span>  
  
 <span data-ttu-id="7822a-104">用户定义的 [true](../../csharp/language-reference/keywords/true.md) 运算符要求用户定义 [false](../../csharp/language-reference/keywords/false.md) 运算符，反之亦然。</span><span class="sxs-lookup"><span data-stu-id="7822a-104">A user-defined [true](../../csharp/language-reference/keywords/true.md) operator requires a user-defined [false](../../csharp/language-reference/keywords/false.md) operator, and vice versa.</span></span> <span data-ttu-id="7822a-105">有关详细信息，请参阅[运算符](../../csharp/programming-guide/statements-expressions-operators/operators.md)。</span><span class="sxs-lookup"><span data-stu-id="7822a-105">For more information, see [Operators](../../csharp/programming-guide/statements-expressions-operators/operators.md).</span></span>  
  
 <span data-ttu-id="7822a-106">下面的示例生成 CS0216：</span><span class="sxs-lookup"><span data-stu-id="7822a-106">The following sample generates CS0216:</span></span>  
  
```csharp  
// CS0216.cs  
class MyClass  
{  
   public static bool operator true (MyClass MyInt)   // CS0216  
   {  
      return true;  
   }  
  
   // to resolve, uncomment the following operator definition  
   /*  
   public static bool operator false (MyClass MyInt)  
   {  
      return true;  
   }  
   */  
  
   public static void Main()  
   {  
   }  
}  
```
