## <a name="example"></a><span data-ttu-id="e3433-107">示例</span><span class="sxs-lookup"><span data-stu-id="e3433-107">Example</span></span>  
 <span data-ttu-id="e3433-108">以下示例生成 CS0241。</span><span class="sxs-lookup"><span data-stu-id="e3433-108">The following sample generates CS0241.</span></span> <span data-ttu-id="e3433-109">此外，该示例演示如何使用重载模拟具有默认参数的方法。</span><span class="sxs-lookup"><span data-stu-id="e3433-109">In addition, the sample shows how to simulate, with overloading, a method with default arguments.</span></span>  
  
```csharp  
// CS0241.cs  
public class A  
{  
   public void Test(int i = 9) {}   // CS0241  
}  
  
public class B  
{  
   public void Test() { Test(9); }  
   public void Test(int i)  {}  
}  
  
public class C  
{  
   public static void Main()  
   {   
      B x = new B();  
      x.Test();  
   }  
}  
```
