 <span data-ttu-id="8db4e-106">下面的示例生成 CS0252：</span><span class="sxs-lookup"><span data-stu-id="8db4e-106">The following sample generates CS0252:</span></span>  
  
```  
// CS0252.cs  
// compile with: /W:2  
using System;  
  
class MyClass  
{  
   public static void Main()  
   {  
      string s = "11";  
      object o = s + s;  
  
      bool b = o == s;   // CS0252  
      // try the following line instead  
      // bool b = (string)o == s;  
   }  
}  
```
