 <span data-ttu-id="9826a-104">[stackalloc](../../csharp/language-reference/keywords/stackalloc.md) 关键字不能用在 [catch](../../csharp/language-reference/keywords/try-catch.md) 或 [finally](../../csharp/language-reference/keywords/try-catch-finally.md) 块中。</span><span class="sxs-lookup"><span data-stu-id="9826a-104">The [stackalloc](../../csharp/language-reference/keywords/stackalloc.md) keyword may not be used in a [catch](../../csharp/language-reference/keywords/try-catch.md) or [finally](../../csharp/language-reference/keywords/try-catch-finally.md) block.</span></span> <span data-ttu-id="9826a-105">有关详细信息，请参阅[异常和异常处理](../../csharp/programming-guide/exceptions/index.md)。</span><span class="sxs-lookup"><span data-stu-id="9826a-105">For more information, see [Exceptions and Exception Handling](../../csharp/programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="9826a-106">下面的示例生成 CS0255：</span><span class="sxs-lookup"><span data-stu-id="9826a-106">The following sample generates CS0255:</span></span>  
  
```csharp  
// CS0255.cs  
// compile with: /unsafe  
using System;  
  
public class TestTryFinally  
{  
   public static unsafe void Test()  
   {  
      int i = 123;  
      string s = "Some string";  
      object o = s;  
  
      try  
      {  
         // Conversion is not valid; o contains a string not an int  
         i = (int) o;  
      }  
  
      finally  
      {  
         Console.Write("i = {0}", i);  
         int* fib = stackalloc int[100];   // CS0255  
      }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
