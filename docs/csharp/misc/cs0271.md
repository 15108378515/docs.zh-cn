 <span data-ttu-id="1d048-104">当你尝试访问无法访问的 `get` 访问器时出现此错误。</span><span class="sxs-lookup"><span data-stu-id="1d048-104">This error occurs when you try to access an inaccessible `get` accessor.</span></span> <span data-ttu-id="1d048-105">若要解决此错误，增加访问器的可访问性，或更改调用位置。</span><span class="sxs-lookup"><span data-stu-id="1d048-105">To resolve this error, increase the accessibility of the accessor, or change the calling location.</span></span> <span data-ttu-id="1d048-106">有关详细信息，请参阅[访问器可访问性](../../csharp/programming-guide/classes-and-structs/restricting-accessor-accessibility.md)和[属性](../../csharp/programming-guide/classes-and-structs/properties.md)。</span><span class="sxs-lookup"><span data-stu-id="1d048-106">For more information, see [Accessor Accessibility](../../csharp/programming-guide/classes-and-structs/restricting-accessor-accessibility.md) and [Properties](../../csharp/programming-guide/classes-and-structs/properties.md).</span></span>  
  
 <span data-ttu-id="1d048-107">以下示例生成 CS0271：</span><span class="sxs-lookup"><span data-stu-id="1d048-107">The following example generates CS0271:</span></span>  
  
```csharp  
// CS0271.cs  
public class MyClass  
{  
   public int Property  
   {  
      private get { return 0; }  
      set { }  
   }  
  
   public int Property2  
   {  
      get { return 0; }  
      set { }  
   }  
}  
  
public class Test  
{  
   public static void Main(string[] args)   
   {  
      MyClass c = new MyClass();  
      int a = c.Property;   // CS0271  
      int b = c.Property2;   // OK  
   }  
}  
```
