## <a name="example"></a><span data-ttu-id="9a5bd-109">示例</span><span class="sxs-lookup"><span data-stu-id="9a5bd-109">Example</span></span>  
 <span data-ttu-id="9a5bd-110">下面的示例生成 CS0279：</span><span class="sxs-lookup"><span data-stu-id="9a5bd-110">The following example generates CS0279:</span></span>

```  
// CS0279.cs  
  
using System;  
using System.Collections;  
  
public class myTest : IEnumerable  
{  
    IEnumerator IEnumerable.GetEnumerator()  
    {  
        yield return 0;  
    }

    {  
        yield return 0;  
    }  
  
    public static void Main()  
    {  
        foreach (int i in new myTest()) {}  // CS0279  
    }  
}  
```
