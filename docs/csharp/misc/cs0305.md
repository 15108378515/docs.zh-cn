## <a name="example"></a><span data-ttu-id="621a4-106">示例</span><span class="sxs-lookup"><span data-stu-id="621a4-106">Example</span></span>  
 <span data-ttu-id="621a4-107">下面的示例生成 CS0305。</span><span class="sxs-lookup"><span data-stu-id="621a4-107">The following sample generates CS0305.</span></span>  
  
```csharp  
// CS0305.cs  
public class MyList<T> {}  
public class MyClass<T> {}  
  
class MyClass  
{  
   public static void Main()  
   {  
      MyList<MyClass, MyClass> list1 = new MyList<MyClass>();   // CS0305  
      MyList<MyClass> list2 = new MyList<MyClass>();   // OK  
   }  
}  
```
