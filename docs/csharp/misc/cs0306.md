# <a name="compiler-error-cs0306"></a>编译器错误 CS0306
类型“type”不能用作类型参数  
  
 不允许用作类型参数的类型。 这可能是因为类型是指针类型。  
  
 以下示例生成 CS0306：  
  
```csharp  
// CS0306.cs  
class C<T>  
{  
}  
  
class M  
{  
    // CS0306 – int* not allowed as a type parameter  
     C<int*> f;  
}  
```
