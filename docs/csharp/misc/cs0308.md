# <a name="compiler-error-cs0308"></a><span data-ttu-id="fb98a-102">编译器错误 CS0308</span><span class="sxs-lookup"><span data-stu-id="fb98a-102">Compiler Error CS0308</span></span>
<span data-ttu-id="fb98a-103">非泛型类型或方法“identifier”不能与类型参数一起使用。</span><span class="sxs-lookup"><span data-stu-id="fb98a-103">The non-generic type-or-method 'identifier' cannot be used with type arguments.</span></span>  
  
 <span data-ttu-id="fb98a-104">方法或类型并非泛型，却与类型参数一起使用。</span><span class="sxs-lookup"><span data-stu-id="fb98a-104">The method or type is not generic, but it was used with type arguments.</span></span> <span data-ttu-id="fb98a-105">若要避免此错误，删除尖括号和类型参数，或将该方法或类型重新声明为泛型方法或类型。</span><span class="sxs-lookup"><span data-stu-id="fb98a-105">To avoid this error, remove the angled brackets and type arguments, or redeclare the method or type as a generic method or type.</span></span>  
  
 <span data-ttu-id="fb98a-106">以下示例生成 CS0308：</span><span class="sxs-lookup"><span data-stu-id="fb98a-106">The following example generates CS0308:</span></span>  
  
```  
// CS0308a.cs  
class MyClass  
{  
   public void F() {}  
   public static void Main()  
   {  
      F<int>();  // CS0308 – F is not generic.  
      // Try this instead:  
      // F();  
   }  
}  
```  
  
 <span data-ttu-id="fb98a-107">以下示例还生成 CS0308。</span><span class="sxs-lookup"><span data-stu-id="fb98a-107">The following example also generates CS0308.</span></span> <span data-ttu-id="fb98a-108">要解决此错误，请使用"using System.Collections.Generic"指令。</span><span class="sxs-lookup"><span data-stu-id="fb98a-108">To resolve the error, use the directive "using System.Collections.Generic."</span></span>  
  
```  
// CS0308b.cs  
// compile with: /t:library  
using System.Collections;  
// To resolve, uncomment the following line:  
// using System.Collections.Generic;  
public class MyStack<T>  
{  
    // Store the elements of the stack:  
    private T[] items = new T[100];  
    private int stack_counter = 0;  
  
    // Define the iterator block:  
    public IEnumerator<T> GetEnumerator()   // CS0308  
    {  
        for (int i = stack_counter - 1 ; i >= 0; i--)  
        yield return items[i];  
    }  
}  
```
