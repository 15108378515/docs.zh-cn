## <a name="example"></a><span data-ttu-id="1e4a3-111">示例</span><span class="sxs-lookup"><span data-stu-id="1e4a3-111">Example</span></span>  
 <span data-ttu-id="1e4a3-112">以下代码生成 CS0312：</span><span class="sxs-lookup"><span data-stu-id="1e4a3-112">The following code generates CS0312:</span></span>  
  
```

class Program  
{  
    static void MTyVar<T, U>() where T : U { }  
  
    static int Main()  
    {  
        MTyVar<int?, int>(); // CS0312  
        return 1;  
    }  
}  
```
