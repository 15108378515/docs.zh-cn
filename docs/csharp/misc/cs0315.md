# <a name="compiler-error-cs0315"></a><span data-ttu-id="14399-102">编译器错误 CS0315</span><span class="sxs-lookup"><span data-stu-id="14399-102">Compiler Error CS0315</span></span>
<span data-ttu-id="14399-103">类型 valueType 不能用作类型参数 T 中的泛型类型或方法 TypeorMethod\<T >。</span><span class="sxs-lookup"><span data-stu-id="14399-103">The type 'valueType' cannot be used as type parameter 'T' in the generic type or method 'TypeorMethod\<T>'.</span></span> <span data-ttu-id="14399-104">没有从“valueType”到“referenceType”的装箱转换。</span><span class="sxs-lookup"><span data-stu-id="14399-104">There is no boxing conversion from 'valueType' to 'referenceType'.</span></span>  
  
 <span data-ttu-id="14399-105">将泛型类型约束为特定类，并尝试使用不能隐式装箱到它的值类型构造该类的实例时，会发生此错误。</span><span class="sxs-lookup"><span data-stu-id="14399-105">This error occurs when you constrain a generic type to a particular class, and try to construct an instance of that class by using a value type that cannot be implicitly boxed to it.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="14399-106">更正此错误</span><span class="sxs-lookup"><span data-stu-id="14399-106">To correct this error</span></span>  
  
1.  <span data-ttu-id="14399-107">一种解决方案是将结构重新定义为类。</span><span class="sxs-lookup"><span data-stu-id="14399-107">One solution is to redefine the struct as a class.</span></span>  
  
## <a name="example"></a><span data-ttu-id="14399-108">示例</span><span class="sxs-lookup"><span data-stu-id="14399-108">Example</span></span>  
 <span data-ttu-id="14399-109">下面的示例生成 CS0315：</span><span class="sxs-lookup"><span data-stu-id="14399-109">The following example generates CS0315:</span></span>  
  
```csharp  
// cs0315.cs  
public class ClassConstraint { }  
public struct ViolateClassConstraint { }  
  
public class Gen<T> where T : ClassConstraint  
{         
}  
public class Test  
{  
    public static int Main()  
    {  
        Gen<ViolateClassConstraint> g = new Gen<ViolateClassConstraint>(); //CS0315  
        return 1;  
    }  
}  
```
