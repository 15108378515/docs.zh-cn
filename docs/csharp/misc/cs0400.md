 <span data-ttu-id="41e00-104">在全局命名空间中找不到使用全局范围运算符 (`::`) 确定范围的标识符。</span><span class="sxs-lookup"><span data-stu-id="41e00-104">The identifier scoped with the global scope operator (`::`) was not found in the global namespace.</span></span> <span data-ttu-id="41e00-105">你可能缺少包含该标识符的程序集引用，或者该标识符可能已在类或命名空间，而不是全局命名空间中进行了声明。</span><span class="sxs-lookup"><span data-stu-id="41e00-105">You may be missing an assembly reference that contains the identifier, or the identifier may be declared in a class or namespace other than the global namespace.</span></span> <span data-ttu-id="41e00-106">如果未声明全局范围的标识符或其拼写有误，也可能发生此错误。</span><span class="sxs-lookup"><span data-stu-id="41e00-106">This error could also occur if a globally scoped identifier is not declared or is misspelled.</span></span>  
  
 <span data-ttu-id="41e00-107">要避免此错误，找到该标识符的声明，验证拼写是否正确，如果该声明位于单独的程序集中，请确保你具有适当的程序集引用。</span><span class="sxs-lookup"><span data-stu-id="41e00-107">To avoid this error, locate the declaration of the identifier, verify the correct spelling, and if the declaration is in a separate assembly, make sure that you have the appropriate assembly reference.</span></span> <span data-ttu-id="41e00-108">如果该标识符在另一个类型或命名空间内进行了声明，则在 :: 之后使用完全限定的名称。</span><span class="sxs-lookup"><span data-stu-id="41e00-108">If the identifier is declared inside another type or namespace, use the fully-qualified name after the ::.</span></span> <span data-ttu-id="41e00-109">以下示例生成 CS0400：</span><span class="sxs-lookup"><span data-stu-id="41e00-109">The following sample generates CS0400:</span></span>

```  
{  
    public static void Main()  
    {  
        // CS0400 - D could not be found   
        // in the global namespace.  
        global::D d = new global::D();  
   }  
}  
```
