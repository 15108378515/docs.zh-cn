# <a name="compiler-warning-level-4-cs0402"></a>编译器警告（等级 4）CS0402
“identifier”：入口点不能是泛型或属于泛型类型  
  
 在泛型类型中找到了入口点。 若要删除此警告，实现非泛型的类或结构中的 Main。  
  
```csharp  
// CS0402.cs  
// compile with: /W:4  
class C<T>  
{  
   public static void Main()  // CS0402  
   {  
  
   }  
}  
  
class CMain  
{  
   public static void Main() {}  
}  
```
