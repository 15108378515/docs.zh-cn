## <a name="example"></a><span data-ttu-id="70457-107">示例</span><span class="sxs-lookup"><span data-stu-id="70457-107">Example</span></span>  
 <span data-ttu-id="70457-108">以下示例生成 CS0407：</span><span class="sxs-lookup"><span data-stu-id="70457-108">The following sample generates CS0407:</span></span>  
  
```csharp  
// CS0407.cs  
public delegate int MyDelegate();  
  
class C  
{  
    MyDelegate d;  
  
    public C()  
    {  
        d = new MyDelegate(F);  // OK: F returns int  
        d = new MyDelegate(G);  // CS0407 – G doesn't return int  
    }  
  
    public int F()  
    {  
        return 1;  
    }  
  
    public void G()  
    {  
    }  
  
    public static void Main()  
    {  
        C c1 = new C();  
    }  
}  
```
