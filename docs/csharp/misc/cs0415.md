 <span data-ttu-id="9fea4-104">如果在作为显式接口实现的索引器上使用 IndexerName 特性，将出现此错误。</span><span class="sxs-lookup"><span data-stu-id="9fea4-104">This error occurs if you use an IndexerName attribute on an indexer that was an explicit implementation of an interface.</span></span> <span data-ttu-id="9fea4-105">如有可能，可通过从索引器声明中删除接口名称来避免此错误。</span><span class="sxs-lookup"><span data-stu-id="9fea4-105">This error may be avoided by removing the interface name from the declaration of the indexer, if possible.</span></span> <span data-ttu-id="9fea4-106">有关详细信息，请参阅 [IndexerNameAttribute 类](xref:System.Runtime.CompilerServices.IndexerNameAttribute)。</span><span class="sxs-lookup"><span data-stu-id="9fea4-106">For more information, see the [IndexerNameAttribute Class](xref:System.Runtime.CompilerServices.IndexerNameAttribute).</span></span>  
  
 <span data-ttu-id="9fea4-107">以下示例生成 CS0415：</span><span class="sxs-lookup"><span data-stu-id="9fea4-107">The following sample generates CS0415:</span></span>  
  
```csharp
// CS0415.cs  
using System;  
using System.Runtime.CompilerServices;  
  
public interface IA  
{  
    int this[int index]  
    {  
        get;  
        set;  
    }  
}  
  
public class A : IA  
{  
    [IndexerName("Item")]  // CS0415  
    int IA.this[int index]  
    // Try this line instead:  
    // public int this[int index]  
    {  
        get { return 0; }  
        set { }  
    }  
  
    static void Main()  
    {  
    }  
}  
```
