# <a name="compiler-error-cs0416"></a><span data-ttu-id="05782-102">编译器错误 CS0416</span><span class="sxs-lookup"><span data-stu-id="05782-102">Compiler Error CS0416</span></span>
<span data-ttu-id="05782-103">“类型形参”：特性实参不能使用类型形参</span><span class="sxs-lookup"><span data-stu-id="05782-103">'type parameter': an attribute argument cannot use type parameters</span></span>  
  
 <span data-ttu-id="05782-104">类型形参用作了特性实参，这是不被允许的。</span><span class="sxs-lookup"><span data-stu-id="05782-104">A type parameter was used as an attribute argument, which is not allowed.</span></span> <span data-ttu-id="05782-105">使用非泛型类型</span><span class="sxs-lookup"><span data-stu-id="05782-105">Use a non-generic type.</span></span>  
  
 <span data-ttu-id="05782-106">下面的示例生成 CS0416：</span><span class="sxs-lookup"><span data-stu-id="05782-106">The following sample generates CS0416:</span></span>  
  
```csharp  
// CS0416.cs  
public class MyAttribute : System.Attribute  
{  
   public MyAttribute(System.Type t)  
   {  
   }  
}  
  
class G<T>  
{  
  
   [MyAttribute(typeof(G<T>))]  // CS0416  
   public void F()  
   {  
   }  
  
}  
```
