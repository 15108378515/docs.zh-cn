<span data-ttu-id="7552d-103">方法“method”的类型参数“type parameter”的约束必须与接口方法“method”的类型参数“type parameter”的约束相匹配。</span><span class="sxs-lookup"><span data-stu-id="7552d-103">The constraints for type parameter 'type parameter' of method 'method' must match the constraints for type parameter 'type parameter' of interface method 'method'.</span></span> <span data-ttu-id="7552d-104">请考虑改用显式接口实现。</span><span class="sxs-lookup"><span data-stu-id="7552d-104">Consider using an explicit interface implementation instead.</span></span>

## <a name="example"></a><span data-ttu-id="7552d-107">示例</span><span class="sxs-lookup"><span data-stu-id="7552d-107">Example</span></span>  
 <span data-ttu-id="7552d-108">下面的示例生成 CS0425：</span><span class="sxs-lookup"><span data-stu-id="7552d-108">The following example generates CS0425:</span></span>  
  
```  
// CS0425.cs  
  
class C1  
{  
}  
  
class C2  
{  
}  
  
interface IBase  
{  
    void F<ItemType>(ItemType item) where ItemType : C1;  
}  
  
class Derived : IBase  
{  
    public void F<ItemType>(ItemType item) where ItemType : C2  // CS0425  
    {  
    }  
}  
  
class CMain  
{  
    public static void Main()  
    {  
    }  
}  
```
