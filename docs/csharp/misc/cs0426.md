# <a name="compiler-error-cs0426"></a><span data-ttu-id="87334-102">编译器错误 CS0426</span><span class="sxs-lookup"><span data-stu-id="87334-102">Compiler Error CS0426</span></span>
<span data-ttu-id="87334-103">类型“type”中不存在类型名称“identifier”</span><span class="sxs-lookup"><span data-stu-id="87334-103">The type name 'identifier' does not exist in the type 'type'</span></span>  
  
 <span data-ttu-id="87334-104">引用嵌套在另一类型中的类型而这一嵌套类型不存在时，会出现此错误。</span><span class="sxs-lookup"><span data-stu-id="87334-104">This error occurs when you reference a type nested within another type, but no such nested type exists.</span></span> <span data-ttu-id="87334-105">键错嵌套类型的名称时可能出现此错误。</span><span class="sxs-lookup"><span data-stu-id="87334-105">This can occur if you mistype the name of the nested type.</span></span> <span data-ttu-id="87334-106">检查所用名称的拼写，并验证外层类型具有所需成员。</span><span class="sxs-lookup"><span data-stu-id="87334-106">Check the spelling of the names used, and verify that the enclosing type has the expected member.</span></span>  
  
 <span data-ttu-id="87334-107">下面的示例生成 CS0426，因为类 C 不具有嵌套类型 A：</span><span class="sxs-lookup"><span data-stu-id="87334-107">The following sample generates CS0426 because class C has no nested type A:</span></span>  
  
```csharp  
// CS0426.cs  
  
class C  
{  
    // No nested types are declared.     
}  
  
class D  
{  
   public static void Main()  
   {  
       C c = new C();  
       // Attempt to reference a nested type A:  
       C.A a; // CS0426 because no such type C.A  
   }  
}  
```
