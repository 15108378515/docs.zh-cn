 <span data-ttu-id="5638f-104">当你重写 `++` 或 `--` 运算符时，它们必须返回与包含类型相同的类型，或返回派生自包含类型的类型。</span><span class="sxs-lookup"><span data-stu-id="5638f-104">When you override the `++` or `--` operators, they must return the same type as the containing type, or return a type that is derived from the containing type.</span></span>  
  
## <a name="example"></a><span data-ttu-id="5638f-105">示例</span><span class="sxs-lookup"><span data-stu-id="5638f-105">Example</span></span>  
 <span data-ttu-id="5638f-106">以下示例生成 CS0448。</span><span class="sxs-lookup"><span data-stu-id="5638f-106">The following sample generates CS0448.</span></span>  
  
```  
// CS0448.cs  
class C5  
{  
   public static int operator ++(C5 c) { return null; }   // CS0448  
   public static C5 operator --(C5 c) { return null; }   // OK  
   public static void Main() {}  
}  
```
