## <a name="example"></a><span data-ttu-id="1a837-107">示例</span><span class="sxs-lookup"><span data-stu-id="1a837-107">Example</span></span>  
 <span data-ttu-id="1a837-108">下面的示例生成 CS0456。</span><span class="sxs-lookup"><span data-stu-id="1a837-108">The following sample generates CS0456.</span></span>  
  
```  
// CS0456.cs  
// compile with: /target:library  
public class GenericsErrors  
{  
   public class G5<T> where T : struct  
   {  
      public class N<U> where U : T {}   // CS0456  
      public class N2<U> where U : struct {}   // OK  
   }  
}  
```
