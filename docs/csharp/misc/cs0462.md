# <a name="compiler-error-cs0462"></a><span data-ttu-id="90206-102">编译器错误 CS0462</span><span class="sxs-lookup"><span data-stu-id="90206-102">Compiler Error CS0462</span></span>
<span data-ttu-id="90206-103">继承的成员“member1”和“member2”在类型“type”中具有相同的签名，因此不能重写这些成员</span><span class="sxs-lookup"><span data-stu-id="90206-103">The inherited members 'member1' and 'member2' have the same signature in type 'type', so they cannot be overridden</span></span>  
  
 <span data-ttu-id="90206-104">此错误是由于引入泛型而引起的。</span><span class="sxs-lookup"><span data-stu-id="90206-104">This error arises with the introduction of generics.</span></span> <span data-ttu-id="90206-105">正常情况下，类中的方法不能有两个具有相同签名的版本。</span><span class="sxs-lookup"><span data-stu-id="90206-105">Normally, you cannot have two versions of a method in a class with the same signature.</span></span> <span data-ttu-id="90206-106">但是对于泛型，如果泛型方法使用某个特定的类型实例化，则可以指定一个可能与另一个方法重复的泛型方法。</span><span class="sxs-lookup"><span data-stu-id="90206-106">But with generics, you can specify a generic method that might duplicate another method if it is instantiated with a particular type.</span></span>  
  
## <a name="example"></a><span data-ttu-id="90206-107">示例</span><span class="sxs-lookup"><span data-stu-id="90206-107">Example</span></span>  
 <span data-ttu-id="90206-108">当实例化 `C<int>` 时，将为方法 `F` 创建两个具有相同签名的版本，所以类 `D` 中的重写无法确定对哪个方法应用重写。</span><span class="sxs-lookup"><span data-stu-id="90206-108">When `C<int>` is instantiated, two versions of the method `F` are created with the same signature, so the override in class `D` cannot decide which one to apply the override to.</span></span>  
  
 <span data-ttu-id="90206-109">下面的示例生成 CS0462。</span><span class="sxs-lookup"><span data-stu-id="90206-109">The following sample generates CS0462.</span></span>  
  
```csharp  
// CS0462.cs  
// compile with: /target:library  
class C<T>   
{  
   public virtual void F(T t) {}  
   public virtual void F(int t) {}  
}  
  
class D : C<int>   
{  
   public override void F(int t) {}   // CS0462  
}  
```
