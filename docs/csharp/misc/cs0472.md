## <a name="example"></a><span data-ttu-id="53660-105">示例</span><span class="sxs-lookup"><span data-stu-id="53660-105">Example</span></span>  
 <span data-ttu-id="53660-106">下面的示例生成 CS0472。</span><span class="sxs-lookup"><span data-stu-id="53660-106">The following sample generates CS0472.</span></span>  
  
```  
public class Test  
{  
    public static int Main()  
    {  
        int i = 5;  
        int counter = 0;  
  
        // Comparison:  
        if (i == null)  // CS0472  
        // To resolve, use a valid value for i.  
            counter++;   
        return counter;  
    }  
}  
```
