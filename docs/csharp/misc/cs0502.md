# <a name="compiler-error-cs0502"></a><span data-ttu-id="19c6b-102">编译器错误 CS0502</span><span class="sxs-lookup"><span data-stu-id="19c6b-102">Compiler Error CS0502</span></span>

 <span data-ttu-id="19c6b-104">类成员不能既是 [抽象的](../../csharp/language-reference/keywords/abstract.md) 又是 [密封的](../../csharp/language-reference/keywords/sealed.md)。</span><span class="sxs-lookup"><span data-stu-id="19c6b-104">A class member cannot be both [abstract](../../csharp/language-reference/keywords/abstract.md) and [sealed](../../csharp/language-reference/keywords/sealed.md).</span></span>  
  
 <span data-ttu-id="19c6b-105">下面的示例生成 CS0502：</span><span class="sxs-lookup"><span data-stu-id="19c6b-105">The following sample generates CS0502:</span></span>  
  
```csharp  
// CS0502.cs  
public class B  
{  
   abstract public void F();  
}  
  
public class C : B  
{  
   abstract sealed override public void F()   // CS0502  
   {  
   }  
}  
  
public class CMain  
{  
    public static void Main()  
    { }  
}  
```
