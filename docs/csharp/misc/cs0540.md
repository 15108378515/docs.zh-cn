## <a name="example"></a><span data-ttu-id="46f5c-106">示例</span><span class="sxs-lookup"><span data-stu-id="46f5c-106">Example</span></span>  
 <span data-ttu-id="46f5c-107">下面的示例生成 CS0540。</span><span class="sxs-lookup"><span data-stu-id="46f5c-107">The following sample generates CS0540.</span></span>  
  
```  
// CS0540.cs  
interface I  
{  
   void m();  
}  
  
public class Clx  
{  
   void I.m() {}   // CS0540  
}  
  
// OK  
public class Cly : I  
{  
   void I.m() {}  
   public static void Main() {}  
}  
```
