## <a name="example"></a><span data-ttu-id="8c307-106">示例</span><span class="sxs-lookup"><span data-stu-id="8c307-106">Example</span></span>  
 <span data-ttu-id="8c307-107">下面的示例生成 CS0550。</span><span class="sxs-lookup"><span data-stu-id="8c307-107">The following sample generates CS0550.</span></span>  
  
```  
// CS0550.cs  
namespace x  
{  
   interface ii  
   {  
      int i  
      {  
         get;  
         // add the following accessor to resolve this CS0550  
         // set;  
      }  
   }  
  
   public class a : ii  
   {  
      int ii.i  
      {  
         get  
         {  
            return 0;  
         }  
         set {}   // CS0550  no set in interface  
      }  
  
      public static void Main() {}  
   }  
}  
```
