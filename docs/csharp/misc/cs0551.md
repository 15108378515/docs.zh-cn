## <a name="example"></a><span data-ttu-id="46e4e-106">示例</span><span class="sxs-lookup"><span data-stu-id="46e4e-106">Example</span></span>  
 <span data-ttu-id="46e4e-107">以下示例生成 CS0551。</span><span class="sxs-lookup"><span data-stu-id="46e4e-107">The following sample generates CS0551.</span></span>  
  
```csharp  
// CS0551.cs  
// compile with: /target:library  
interface ii  
{  
   int i  
   {  
      get;  
      set;  
   }  
}  
  
public class a : ii  
{  
   int ii.i { set {} }   // CS0551  
  
   // OK  
   int ii.i      
   {  
      set {}  
      get { return 0; }  
   }  
}  
```
