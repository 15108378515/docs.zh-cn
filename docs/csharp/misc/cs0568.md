# <a name="compiler-error-cs0568"></a><span data-ttu-id="12b8f-102">编译器错误 CS0568</span><span class="sxs-lookup"><span data-stu-id="12b8f-102">Compiler Error CS0568</span></span>
<span data-ttu-id="12b8f-103">结构不能包含显式的无参数构造函数</span><span class="sxs-lookup"><span data-stu-id="12b8f-103">Structs cannot contain explicit parameterless constructors</span></span>  
  
 <span data-ttu-id="12b8f-104">每个 [结构](../../csharp/language-reference/keywords/struct.md) 已具有将对象初始化为零的默认构造函数。</span><span class="sxs-lookup"><span data-stu-id="12b8f-104">Each [struct](../../csharp/language-reference/keywords/struct.md) already has a default constructor that initializes the object to zero.</span></span> <span data-ttu-id="12b8f-105">因此，你可以为结构创建构造函数，该构造函数必须采用一个或多个参数。</span><span class="sxs-lookup"><span data-stu-id="12b8f-105">Therefore, the constructors that you can create for a struct must take one or more parameters.</span></span>  
  
 <span data-ttu-id="12b8f-106">下面的示例生成 CS0568：</span><span class="sxs-lookup"><span data-stu-id="12b8f-106">The following sample generates CS0568:</span></span>  
  
```csharp  
// CS0568.cs  
public struct ClassY  
{  
   public int field1;  
   public ClassY(){}   // CS0568, cannot have no param constructor  
   // Try following instead:  
   // public ClassY(int i)  
   // {  
   //    field1 = i;  
   // }  
}  
  
public class ClassX  
{  
   public static void Main()  
   {  
   }  
}  
```
