# <a name="compiler-error-cs0577"></a><span data-ttu-id="6f547-102">编译器错误 CS0577</span><span class="sxs-lookup"><span data-stu-id="6f547-102">Compiler Error CS0577</span></span>
<span data-ttu-id="6f547-103">Conditional 特性对“function”无效，因为它是构造函数、析构函数、运算符或显式接口实现</span><span class="sxs-lookup"><span data-stu-id="6f547-103">The Conditional attribute is not valid on 'function' because it is a constructor, destructor, operator, or explicit interface implementation</span></span>  
  
 <span data-ttu-id="6f547-104">`Conditional` 无法应用于指定的方法。</span><span class="sxs-lookup"><span data-stu-id="6f547-104">`Conditional` cannot be applied to the specified methods.</span></span>  
  
 <span data-ttu-id="6f547-105">例如，你无法对显式接口定义使用某些特性。</span><span class="sxs-lookup"><span data-stu-id="6f547-105">For example, you cannot use some attributes on an explicit interface definition.</span></span> <span data-ttu-id="6f547-106">下面的示例生成 CS0577：</span><span class="sxs-lookup"><span data-stu-id="6f547-106">The following sample generates CS0577:</span></span>  
  
```csharp  
// CS0577.cs  
// compile with: /target:library  
interface I  
{  
   void m();  
}  
  
public class MyClass : I  
{  
   [System.Diagnostics.Conditional("a")]   // CS0577  
   void I.m() {}  
}  
```
