# 编译器错误 CS0584
内部编译器错误: 步骤“stage”符号“symbol”  
  
 尝试确定编译器是否因为无法分析意外的语法而产生故障。 如果不是，请尝试 [与我们交流](/visual-studio/ide/talk-to-us)。
