# <a name="compiler-error-cs0589"></a>编译器错误 CS0589
内部编译器错误: 步骤“PARSE”  
  
 尝试确定编译器是否由于无法分析意外的语法而产生故障。 如果不是这种情况，请尝试[与我们交流](/visualstudio/ide/talk-to-us)。
