# <a name="compiler-error-cs0625"></a><span data-ttu-id="ebb6c-102">编译器错误 CS0625</span><span class="sxs-lookup"><span data-stu-id="ebb6c-102">Compiler Error CS0625</span></span>
<span data-ttu-id="ebb6c-103">field： 标记为 structlayout （layoutkind.explicit） 的实例字段类型必须具有 FieldOffset 特性。</span><span class="sxs-lookup"><span data-stu-id="ebb6c-103">'field': instance field types marked with StructLayout(LayoutKind.Explicit) must have a FieldOffset attribute.</span></span>
  
<span data-ttu-id="ebb6c-104">使用显式 **StructLayout** 特性标记结构时，该结构中的所有字段都必须具有 [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) 特性。</span><span class="sxs-lookup"><span data-stu-id="ebb6c-104">When a struct is marked with an explicit **StructLayout** attribute, all fields in the struct must have the [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) attribute.</span></span> <span data-ttu-id="ebb6c-105">有关详细信息，请参阅 [StructLayoutAttribute 类](xref:System.Runtime.InteropServices.StructLayoutAttribute)。</span><span class="sxs-lookup"><span data-stu-id="ebb6c-105">For more information, see [StructLayoutAttribute Class](xref:System.Runtime.InteropServices.StructLayoutAttribute).</span></span> 

<span data-ttu-id="ebb6c-106">下面的示例生成 CS0625：</span><span class="sxs-lookup"><span data-stu-id="ebb6c-106">The following sample generates CS0625:</span></span>  
  
```csharp  
// CS0625.cs  
// compile with: /target:library  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Explicit)]  
struct A  
{  
   public int i;   // CS0625 not static; an instance field  
}  
  
// OK  
[StructLayout(LayoutKind.Explicit)]  
struct B  
{  
   [FieldOffset(5)]  
   public int i;  
}  
```
