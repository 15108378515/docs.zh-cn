## <a name="example"></a><span data-ttu-id="b0a79-106">示例</span><span class="sxs-lookup"><span data-stu-id="b0a79-106">Example</span></span>  
 <span data-ttu-id="b0a79-107">此示例使用 <xref:System.Diagnostics.ConditionalAttribute>演示 CS0633。</span><span class="sxs-lookup"><span data-stu-id="b0a79-107">This example illustrates CS0633 using the <xref:System.Diagnostics.ConditionalAttribute>.</span></span> <span data-ttu-id="b0a79-108">以下示例生成 CS0633。</span><span class="sxs-lookup"><span data-stu-id="b0a79-108">The following sample generates CS0633.</span></span>  
  
```csharp

#define DEBUG  
using System.Diagnostics;  
public class Test  
{  
   [Conditional("DEB+UG")]   // CS0633  
   // try the following line instead  
   // [Conditional("DEBUG")]  
   public static void Main() { }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="b0a79-109">示例</span><span class="sxs-lookup"><span data-stu-id="b0a79-109">Example</span></span>  
 <span data-ttu-id="b0a79-110">此示例使用 <xref:System.Runtime.CompilerServices.IndexerNameAttribute>演示 CS0633。</span><span class="sxs-lookup"><span data-stu-id="b0a79-110">This example illustrates CS0633 using the <xref:System.Runtime.CompilerServices.IndexerNameAttribute>.</span></span>  
  
```csharp  
// CS0633b.cs  
// compile with: /target:module  
#define DEBUG  
using System.Runtime.CompilerServices;  
public class Test  
{  
   [IndexerName("Invalid+Identifier")]   // CS0633  
   // try the following line instead  
   // [IndexerName("DEBUG")]  
   public int this[int i]   
   {   
      get { return i; }   
   }  
}  
```
