## <a name="example"></a><span data-ttu-id="8937f-106">示例</span><span class="sxs-lookup"><span data-stu-id="8937f-106">Example</span></span>  
 <span data-ttu-id="8937f-107">此示例使用 <xref:System.Diagnostics.ConditionalAttribute>演示 CS0633。</span><span class="sxs-lookup"><span data-stu-id="8937f-107">This example illustrates CS0633 using the <xref:System.Diagnostics.ConditionalAttribute>.</span></span> <span data-ttu-id="8937f-108">以下示例生成 CS0633。</span><span class="sxs-lookup"><span data-stu-id="8937f-108">The following sample generates CS0633.</span></span>  
  
```

using System.Diagnostics;  
public class Test  
{  
   [Conditional("DEB+UG")]   // CS0633  
   // try the following line instead  
   // [Conditional("DEBUG")]  
   public static void Main() { }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="8937f-109">示例</span><span class="sxs-lookup"><span data-stu-id="8937f-109">Example</span></span>  
 <span data-ttu-id="8937f-110">此示例使用 <xref:System.Runtime.CompilerServices.IndexerNameAttribute>演示 CS0633。</span><span class="sxs-lookup"><span data-stu-id="8937f-110">This example illustrates CS0633 using the <xref:System.Runtime.CompilerServices.IndexerNameAttribute>.</span></span>  
  
```  
// CS0633b.cs  
// compile with: /target:module  
#define DEBUG  
using System.Runtime.CompilerServices;  
public class Test  
{  
   [IndexerName("Invalid+Identifier")]   // CS0633  
   // try the following line instead  
   // [IndexerName("DEBUG")]  
   public int this[int i]   
   {   
      get { return i; }   
   }  
}  
```
