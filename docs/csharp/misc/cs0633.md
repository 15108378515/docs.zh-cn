# <a name="compiler-error-cs0633"></a><span data-ttu-id="ba0de-102">编译器错误 CS0633</span><span class="sxs-lookup"><span data-stu-id="ba0de-102">Compiler Error CS0633</span></span>
<span data-ttu-id="ba0de-103">“attribute”特性的参数必须是有效的标识符</span><span class="sxs-lookup"><span data-stu-id="ba0de-103">The argument to the 'attribute' attribute must be a valid identifier</span></span>  
  
 <span data-ttu-id="ba0de-104">传递到 <xref:System.Diagnostics.ConditionalAttribute> 或 <xref:System.Runtime.CompilerServices.IndexerNameAttribute> 特性的参数必须是有效的标识符。</span><span class="sxs-lookup"><span data-stu-id="ba0de-104">Any argument that you pass to the <xref:System.Diagnostics.ConditionalAttribute> or <xref:System.Runtime.CompilerServices.IndexerNameAttribute> attributes must be a valid identifier.</span></span> <span data-ttu-id="ba0de-105">这意味着它不能包含“+”这样的字符，它们出现在标识符中则为非法。</span><span class="sxs-lookup"><span data-stu-id="ba0de-105">This means that it may not contain characters such as "+" that are illegal when they occur in identifiers.</span></span>  
  
## <a name="example"></a><span data-ttu-id="ba0de-106">示例</span><span class="sxs-lookup"><span data-stu-id="ba0de-106">Example</span></span>  
 <span data-ttu-id="ba0de-107">此示例使用 <xref:System.Diagnostics.ConditionalAttribute>演示 CS0633。</span><span class="sxs-lookup"><span data-stu-id="ba0de-107">This example illustrates CS0633 using the <xref:System.Diagnostics.ConditionalAttribute>.</span></span> <span data-ttu-id="ba0de-108">以下示例生成 CS0633。</span><span class="sxs-lookup"><span data-stu-id="ba0de-108">The following sample generates CS0633.</span></span>  
  
```csharp  
// CS0633a.cs  
#define DEBUG  
using System.Diagnostics;  
public class Test  
{  
   [Conditional("DEB+UG")]   // CS0633  
   // try the following line instead  
   // [Conditional("DEBUG")]  
   public static void Main() { }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="ba0de-109">示例</span><span class="sxs-lookup"><span data-stu-id="ba0de-109">Example</span></span>  
 <span data-ttu-id="ba0de-110">此示例使用 <xref:System.Runtime.CompilerServices.IndexerNameAttribute>演示 CS0633。</span><span class="sxs-lookup"><span data-stu-id="ba0de-110">This example illustrates CS0633 using the <xref:System.Runtime.CompilerServices.IndexerNameAttribute>.</span></span>  
  
```csharp  
// CS0633b.cs  
// compile with: /target:module  
#define DEBUG  
using System.Runtime.CompilerServices;  
public class Test  
{  
   [IndexerName("Invalid+Identifier")]   // CS0633  
   // try the following line instead  
   // [IndexerName("DEBUG")]  
   public int this[int i]   
   {   
      get { return i; }   
   }  
}  
```
