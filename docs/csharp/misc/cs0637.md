# <a name="compiler-error-cs0637"></a><span data-ttu-id="564f9-102">编译器错误 CS0637</span><span class="sxs-lookup"><span data-stu-id="564f9-102">Compiler Error CS0637</span></span>

 <span data-ttu-id="564f9-104">[FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) 特性不能用于标记为 [static](../../csharp/language-reference/keywords/static.md) 或 [const](../../csharp/language-reference/keywords/const.md)的字段。</span><span class="sxs-lookup"><span data-stu-id="564f9-104">The [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) attribute cannot be used on fields marked with [static](../../csharp/language-reference/keywords/static.md) or [const](../../csharp/language-reference/keywords/const.md).</span></span>  
  
 <span data-ttu-id="564f9-105">以下示例生成 CS0637:</span><span class="sxs-lookup"><span data-stu-id="564f9-105">The following sample generates CS0637:</span></span>  
  
```csharp  
// CS0637.cs  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Explicit)]  
public class MainClass  
{  
   [FieldOffset(3)]   // CS0637  
   public static int i;  
   public static void Main ()  
   {  
   }  
}  
```
