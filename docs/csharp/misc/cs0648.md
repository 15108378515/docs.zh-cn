# <a name="compiler-error-cs0648"></a>编译器错误 CS0648
“type”是该语言不支持的类型  
  
 从另一种语言（可能是 C++）生成的元数据包含了未标记为托管的类型。 有关该类型的信息包含在元数据中，但该类型不可用于使用 C# 编写的程序中。
