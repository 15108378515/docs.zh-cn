# <a name="compiler-warning-level-4-cs0649"></a><span data-ttu-id="c8a8d-102">编译器警告（等级 4）CS0649</span><span class="sxs-lookup"><span data-stu-id="c8a8d-102">Compiler Warning (level 4) CS0649</span></span>
<span data-ttu-id="c8a8d-103">从未对字段“field”赋值，字段将一直保持其默认值“value”</span><span class="sxs-lookup"><span data-stu-id="c8a8d-103">Field 'field' is never assigned to, and will always have its default value 'value'</span></span>  
  
 <span data-ttu-id="c8a8d-104">编译器检测到未初始化且从未被赋值的私有字段声明或内部字段声明。</span><span class="sxs-lookup"><span data-stu-id="c8a8d-104">The compiler detected an uninitialized private or internal field declaration that is never assigned a value.</span></span>  
  
 <span data-ttu-id="c8a8d-105">下面的示例生成 CS0649：</span><span class="sxs-lookup"><span data-stu-id="c8a8d-105">The following sample generates CS0649:</span></span>  
  
```  
// CS0649.cs  
// compile with: /W:4  
using System.Collections;  
  
class MyClass  
{  
   Hashtable table;  // CS0649  
   // You may have intended to initialize the variable to null  
   // Hashtable table = null;  
  
   // Or you may have meant to create an object here  
   // Hashtable table = new Hashtable();  
  
   public void Func(object o, string p)  
   {  
      // Or here  
      // table = new Hashtable();  
      table[p] = o;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
