 <span data-ttu-id="62d3c-104">不允许仅在对参数使用 [ref](../../csharp/language-reference/keywords/ref.md) 和 [out](../../csharp/language-reference/keywords/out.md) 上存在差异的方法。</span><span class="sxs-lookup"><span data-stu-id="62d3c-104">Methods that differ only on their use of [ref](../../csharp/language-reference/keywords/ref.md) and [out](../../csharp/language-reference/keywords/out.md) on a parameter are not allowed.</span></span>

```  
// CS0663.cs

   {  
   }

   {  
   }

```
