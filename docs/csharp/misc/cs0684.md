# <a name="compiler-warning-level-1-cs0684"></a><span data-ttu-id="906ef-102">编译器警告（等级 1）CS0684</span><span class="sxs-lookup"><span data-stu-id="906ef-102">Compiler Warning (level 1) CS0684</span></span>
<span data-ttu-id="906ef-103">“interface”接口标记为“CoClassAttribute”而不是“ComImportAttribute”</span><span class="sxs-lookup"><span data-stu-id="906ef-103">'interface' interface marked with 'CoClassAttribute' not marked with 'ComImportAttribute'</span></span>  
  
 <span data-ttu-id="906ef-104">如果在接口上指定 **CoClassAttribute** ，则还必须指定 **ComImportAttribute**。</span><span class="sxs-lookup"><span data-stu-id="906ef-104">If you specify **CoClassAttribute** on an interface, you must also specify **ComImportAttribute**.</span></span>  
  
 <span data-ttu-id="906ef-105">下面的示例生成 CS0684：</span><span class="sxs-lookup"><span data-stu-id="906ef-105">The following sample generates CS0684:</span></span>  
  
```csharp  
// CS0684.cs  
// compile with: /W:1  
using System;  
using System.Runtime.InteropServices;  
  
[CoClass(typeof(C))] // CS0684  
// try the following line instead  
// [CoClass(typeof(C)), ComImport]  
interface I  
{  
}  
  
class C  
{  
   static void Main() {}  
}  
```
