# <a name="compiler-error-cs0695"></a><span data-ttu-id="513d1-102">编译器错误 CS0695</span><span class="sxs-lookup"><span data-stu-id="513d1-102">Compiler Error CS0695</span></span>
<span data-ttu-id="513d1-103">“generic type”不能同时实现“generic interface”和“generic interface”，原因是它们可以统一以进行某些类型参数替换</span><span class="sxs-lookup"><span data-stu-id="513d1-103">'generic type' cannot implement both 'generic interface' and 'generic interface' because they may unify for some type parameter substitutions</span></span>  
  
 <span data-ttu-id="513d1-104">当一个泛型类实现同一泛型接口的多个参数化，并且存在一个会使两个接口完全相同的类型参数替换时，将出现此错误。</span><span class="sxs-lookup"><span data-stu-id="513d1-104">This error occurs when a generic class implements more than one parameterization of the same generic interface, and there exists a type parameter substitution which would make the two interfaces identical.</span></span> <span data-ttu-id="513d1-105">若要避免此错误，请仅实现一个接口，或者更改类型参数以避免冲突。</span><span class="sxs-lookup"><span data-stu-id="513d1-105">To avoid this error, implement only one of the interfaces, or change the type parameters to avoid the conflict.</span></span>  
  
 <span data-ttu-id="513d1-106">下面的示例生成 CS0695：</span><span class="sxs-lookup"><span data-stu-id="513d1-106">The following sample generates CS0695:</span></span>  
  
```  
// CS0695.cs  
// compile with: /target:library

{  
}

{  
}  
```
