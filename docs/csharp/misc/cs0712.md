# <a name="compiler-error-cs0712"></a>编译器错误 CS0712
无法创建静态类“static class”的实例  
  
 无法创建静态类的实例。 静态类被设计为包含静态字段和方法，但不能实例化。  
  
## <a name="example"></a>示例  
 下面的示例生成 CS0712：  
  
```csharp  
// CS0712.cs  
public static class SC  
{  
}  
  
public class CMain  
{  
    public static void Main()  
    {  
        SC sc = new SC();  // CS0712  
    }  
}  
```
