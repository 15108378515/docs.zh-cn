# <a name="compiler-error-cs0714"></a>编译器错误 CS0714
“static type”：静态类不能实现接口  
  
 接口可能会在对象上定义非静态方法，因此将不能由静态类实现。 若要解决此错误，请确保你的类不尝试实现任何接口。  
  
## <a name="example"></a>示例  
 以下示例生成 CS0714：  
  
```csharp  
// CS0714.cs  
interface I  
{  
}  
  
public static class C : I  // CS0714  
{  
    public static void Main()  
    {  
    }  
}  
```
