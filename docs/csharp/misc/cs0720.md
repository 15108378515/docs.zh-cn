# <a name="compiler-error-cs0720"></a>编译器错误 CS0720
“static class”：不能在静态类中声明索引器  
  
 索引器在静态类中无意义，因为它们只能与实例一起使用，而你不可能创建静态类型的实例。  
  
## <a name="example"></a>示例  
 以下示例生成 CS0720：  
  
```csharp  
// CS0720.cs  
  
public static class Test  
{  
    public int this[int index]  // CS0720  
    {  
        get { return 1; }  
        set {}  
    }  
  
    static void Main() {}  
}  
```
