# <a name="compiler-error-cs0723"></a>编译器错误 CS0723
无法声明静态类型“type”的变量  
  
 无法创建静态类型的实例。  
  
 下面的示例生成 CS0723：  
  
```csharp  
// CS0723.cs  
public static class SC  
{  
}  
  
public class CMain  
{  
   public static void Main()  
   {  
      SC sc = null;  // CS0723  
   }  
}  
```
