# <a name="compiler-error-cs0724"></a>编译器错误 CS0724
由于程序集没有 CLSCompliant 特性，因此不需要 CLSCompliant 特性  
  
 下面的示例由于 `throw` 子句块内的 `finally` 语句而生成 CS0724。  
  
## <a name="example"></a>示例  
 下面的示例生成 CS0724。  
  
```csharp  
// CS0724.cs  
using System;  
  
class X  
{  
    static void Test()  
    {  
        try  
        {  
            throw new Exception();  
        }  
        catch  
        {  
            try  
            {  
            }  
            finally  
            {  
                throw; // CS0724  
            }  
        }  
    }  
  
    static void Main()  
    {  
    }  
}  
```
