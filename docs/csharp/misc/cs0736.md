# <a name="compiler-error-cs0736"></a><span data-ttu-id="d5e91-102">编译器错误 CS0736</span><span class="sxs-lookup"><span data-stu-id="d5e91-102">Compiler Error CS0736</span></span>
<span data-ttu-id="d5e91-103">“type name”不实现接口成员“member name”。</span><span class="sxs-lookup"><span data-stu-id="d5e91-103">'type name' does not implement interface member 'member name'.</span></span> <span data-ttu-id="d5e91-104">“method name”无法实现接口成员，因为它是静态的。</span><span class="sxs-lookup"><span data-stu-id="d5e91-104">'method name' cannot implement an interface member because it is static.</span></span>

## <a name="example"></a><span data-ttu-id="d5e91-110">示例</span><span class="sxs-lookup"><span data-stu-id="d5e91-110">Example</span></span>  
 <span data-ttu-id="d5e91-111">下面的代码生成 CS0736，因为 `Program.testMethod` 被声明为静态的：</span><span class="sxs-lookup"><span data-stu-id="d5e91-111">The following code generates CS0736 because `Program.testMethod` is declared as static:</span></span>  
  
```csharp  
// cs0736.cs  
namespace CS0736  
{     
  
    interface ITest  
    {  
        int testMethod(int x);  
    }  
  
    class Program : ITest // CS0736  
    {  
        public static int testMethod(int x) { return 0; }  
        // Try the following line instead.  
        // public int testMethod(int x) { return 0; }  
        public static void Main() { }  
    }      
}  
```
