## <a name="example"></a><span data-ttu-id="0d061-108">示例</span><span class="sxs-lookup"><span data-stu-id="0d061-108">Example</span></span>  
 <span data-ttu-id="0d061-109">下面的代码生成 CS0738，因为类方法将返回 `void` ，同名的接口成员返回 `int`：</span><span class="sxs-lookup"><span data-stu-id="0d061-109">The following code generates CS0738 because the class method returns `void` and the interface member of the same name returns `int`:</span></span>

    int TestMethod();
