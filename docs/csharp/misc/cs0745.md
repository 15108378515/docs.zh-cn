## <a name="example"></a><span data-ttu-id="8e7bc-108">示例</span><span class="sxs-lookup"><span data-stu-id="8e7bc-108">Example</span></span>  
 <span data-ttu-id="8e7bc-109">以下代码生成 CS0745：</span><span class="sxs-lookup"><span data-stu-id="8e7bc-109">The following code generates CS0745:</span></span>  
  
```csharp  
// cs0745.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        string[] names = { "Bob", "Bill", "Jonetta", "Mary" };

                    group name name[0]; // CS0745  
  
        return 1;  
    }  
}
```  
