## <a name="example"></a><span data-ttu-id="46544-108">示例</span><span class="sxs-lookup"><span data-stu-id="46544-108">Example</span></span>  
 <span data-ttu-id="46544-109">下面的代码在使用 `incorrect_1` 和 `incorrect_2`的声明时生成 CS0746。</span><span class="sxs-lookup"><span data-stu-id="46544-109">The following code generates CS0746 in the declaration of `incorrect_1` and `incorrect_2`.</span></span> <span data-ttu-id="46544-110">下面的声明显示了两种用于声明匿名类型的正确方法。</span><span class="sxs-lookup"><span data-stu-id="46544-110">The following declarations show two of the correct ways to declare an anonymous type.</span></span>  
  
```  
// cs0746.cs  
public class C  
{  
    public static int Main()  
    {  
        int i = 100;  
        string s = "Bottles of beer.";  
  
        var incorrect_1 = new { a.b = 1 }; // CS0746   
        var incorrect_2 = new {100, "Bottles of beer."}; // CS0746  
        var correct_1 = new { i, s }; //OK  
        var correct_2 = new {num = i, message = s}; // OK  
  
        return 1;  
    }  
}  
```
