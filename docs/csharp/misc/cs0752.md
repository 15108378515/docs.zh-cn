# <a name="compiler-error-cs0752"></a><span data-ttu-id="f3e78-102">编译器错误 CS0752</span><span class="sxs-lookup"><span data-stu-id="f3e78-102">Compiler Error CS0752</span></span>
<span data-ttu-id="f3e78-103">分部方法不能有 out 参数</span><span class="sxs-lookup"><span data-stu-id="f3e78-103">A partial method cannot have out parameters</span></span>  
  
 <span data-ttu-id="f3e78-104">分部方法不能有 [out](../../csharp/language-reference/keywords/out.md) 参数</span><span class="sxs-lookup"><span data-stu-id="f3e78-104">A partial method cannot have an [out](../../csharp/language-reference/keywords/out.md) parameter.</span></span> <span data-ttu-id="f3e78-105">不允许使用 Out 参数，因为如果编译器删除分部方法，则不能保证将 out 参数赋值。</span><span class="sxs-lookup"><span data-stu-id="f3e78-105">Out parameters are not allowed because if the partial method is removed by the compiler then there is no guarantee that the out parameter is ever assigned.</span></span>
