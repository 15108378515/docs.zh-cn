# <a name="compiler-error-cs0754"></a><span data-ttu-id="2b431-102">编译器错误 CS0754</span><span class="sxs-lookup"><span data-stu-id="2b431-102">Compiler Error CS0754</span></span>
<span data-ttu-id="2b431-103">分部方法不能显式实现接口方法。</span><span class="sxs-lookup"><span data-stu-id="2b431-103">A partial method may not explicitly implement an interface method.</span></span>  
  
 <span data-ttu-id="2b431-104">分部方法不能声明为显式实现接口中定义的方法。</span><span class="sxs-lookup"><span data-stu-id="2b431-104">A partial method cannot be declared as an explicit implementation of a method defined in an interface.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="2b431-105">更正此错误</span><span class="sxs-lookup"><span data-stu-id="2b431-105">To correct this error</span></span>  
  
1.  <span data-ttu-id="2b431-106">从方法声明中删除显式接口限定。</span><span class="sxs-lookup"><span data-stu-id="2b431-106">Remove the explicit interface qualification from the method declaration.</span></span>  
  
## <a name="example"></a><span data-ttu-id="2b431-107">示例</span><span class="sxs-lookup"><span data-stu-id="2b431-107">Example</span></span>  
 <span data-ttu-id="2b431-108">下面的代码生成 CS0754:</span><span class="sxs-lookup"><span data-stu-id="2b431-108">The following code generates CS0754:</span></span>  
  
```csharp  
// cs0754.cs  
using System;  
  
    public interface IF  
    {  
        void Part();  
    }  
  
    public partial class C : IF  
    {  
        partial void IF.Part(); //CS0754  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```
