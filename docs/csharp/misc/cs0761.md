 <span data-ttu-id="8d380-104">如果分部方法具有实现，则泛型类型约束必须与方法签名上定义的约束相同。</span><span class="sxs-lookup"><span data-stu-id="8d380-104">If a partial method has an implementation, the generic type constraint must be identical to the constraint defined on the method signature.</span></span>
