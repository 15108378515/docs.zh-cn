<span data-ttu-id="2a735-103">不能在表达式树中使用只有定义声明的分部方法或已移除的条件方法</span><span class="sxs-lookup"><span data-stu-id="2a735-103">Partial methods with only a defining declaration or removed conditional methods cannot be used in expression trees</span></span>  
  
 <span data-ttu-id="2a735-104">虽然对已删除的分部方法的调用是一个表达式，但它在表达式树中不可接受。</span><span class="sxs-lookup"><span data-stu-id="2a735-104">Although a call to a removed partial method is an expression, it is not an acceptable expression in an expression tree.</span></span>

1.  <span data-ttu-id="2a735-106">为分部方法添加实现声明，或删除会导致编译排除条件方法的代码。</span><span class="sxs-lookup"><span data-stu-id="2a735-106">Add an implementing declaration for the partial method, or remove the code that is causing the conditional method to be excluded from compilation.</span></span>  
  
## <a name="example"></a><span data-ttu-id="2a735-107">示例</span><span class="sxs-lookup"><span data-stu-id="2a735-107">Example</span></span>  
 <span data-ttu-id="2a735-108">以下代码在两个位置生成 CS0765：</span><span class="sxs-lookup"><span data-stu-id="2a735-108">The following code generates CS0765 in two locations:</span></span>  
  
```  
// cs0765.cs  
using System;  
using System.Collections;  
using System.Collections.Generic;  
using System.Diagnostics;  
using System.Linq;  
using System.Linq.Expressions;  
  
public delegate void dele();  
  
public class ConClass  
{  
    [Conditional("CONDITION")]  
    public static void TestMethod() { }  
}  
  
public partial class PartClass : IEnumerable  
{  
    List<object> list = new List<object>();  
  
    partial void Add(int x);  
  
    public IEnumerator GetEnumerator()  
    {  
        for (int i = 0; i < list.Count; i++)  
            yield return list[i];  
    }  
  
    static void Main()  
    {  
        Expression<Func<PartClass>> testExpr1 = () => new PartClass { 1, 2 }; // CS0765  
        Expression<dele> testExpr2 = () => ConClass.TestMethod(); // CS0765  
    }  
}  
```
