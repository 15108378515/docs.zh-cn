 <span data-ttu-id="83b96-104">通常情况下，标记为过时的成员不应重写未标记为过时的成员。</span><span class="sxs-lookup"><span data-stu-id="83b96-104">Typically, a member that is marked as obsolete should not override a member that is not marked as obsolete.</span></span> <span data-ttu-id="83b96-105">在生成此警告[!INCLUDE[vs_orcas_long](../../csharp/misc/includes/vs_orcas_long_md.md)]但未显示在[!INCLUDE[vsprvslong](../../csharp/misc/includes/vsprvslong_md.md)]。</span><span class="sxs-lookup"><span data-stu-id="83b96-105">This warning is generated in [!INCLUDE[vs_orcas_long](../../csharp/misc/includes/vs_orcas_long_md.md)] but not in [!INCLUDE[vsprvslong](../../csharp/misc/includes/vsprvslong_md.md)].</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="83b96-106">更正此错误</span><span class="sxs-lookup"><span data-stu-id="83b96-106">To correct this error</span></span>

## <a name="example"></a><span data-ttu-id="83b96-108">示例</span><span class="sxs-lookup"><span data-stu-id="83b96-108">Example</span></span>  
  
```  
// CS0809.cs  
public class Base  
{  
    public virtual void Test1()  
    {  
    }  
}

    [System.Obsolete()]

    }
```
