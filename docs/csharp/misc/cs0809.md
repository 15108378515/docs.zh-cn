 <span data-ttu-id="d3436-104">通常情况下，标记为过时的成员不应重写未标记为过时的成员。</span><span class="sxs-lookup"><span data-stu-id="d3436-104">Typically, a member that is marked as obsolete should not override a member that is not marked as obsolete.</span></span> <span data-ttu-id="d3436-105">在 [!INCLUDE[vs_orcas_long](~/includes/vs-orcas-long-md.md)] 中将生成此警告，但在 [!INCLUDE[vsprvslong](~/includes/vsprvslong-md.md)]中不会生成此警告。</span><span class="sxs-lookup"><span data-stu-id="d3436-105">This warning is generated in [!INCLUDE[vs_orcas_long](~/includes/vs-orcas-long-md.md)] but not in [!INCLUDE[vsprvslong](~/includes/vsprvslong-md.md)].</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="d3436-106">更正此错误</span><span class="sxs-lookup"><span data-stu-id="d3436-106">To correct this error</span></span>

```csharp

    public virtual void Test1()

    }

    }
```
