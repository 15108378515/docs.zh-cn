# <a name="compiler-warning-level-1-cs0809"></a><span data-ttu-id="f949d-102">编译器警告（等级 1）CS0809</span><span class="sxs-lookup"><span data-stu-id="f949d-102">Compiler Warning (level 1) CS0809</span></span>
<span data-ttu-id="f949d-103">过时成员“memberA”重写未过时成员“memberB”。</span><span class="sxs-lookup"><span data-stu-id="f949d-103">Obsolete member 'memberA' overrides non-obsolete member 'memberB'.</span></span>  
  
 <span data-ttu-id="f949d-104">通常情况下，标记为过时的成员不应重写未标记为过时的成员。</span><span class="sxs-lookup"><span data-stu-id="f949d-104">Typically, a member that is marked as obsolete should not override a member that is not marked as obsolete.</span></span> <span data-ttu-id="f949d-105">在 [!INCLUDE[vs_orcas_long](~/includes/vs-orcas-long-md.md)] 中将生成此警告，但在 [!INCLUDE[vsprvslong](~/includes/vsprvslong-md.md)]中不会生成此警告。</span><span class="sxs-lookup"><span data-stu-id="f949d-105">This warning is generated in [!INCLUDE[vs_orcas_long](~/includes/vs-orcas-long-md.md)] but not in [!INCLUDE[vsprvslong](~/includes/vsprvslong-md.md)].</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="f949d-106">更正此错误</span><span class="sxs-lookup"><span data-stu-id="f949d-106">To correct this error</span></span>

## <a name="example"></a><span data-ttu-id="f949d-108">示例</span><span class="sxs-lookup"><span data-stu-id="f949d-108">Example</span></span>  
  
```csharp  
// CS0809.cs  
public class Base  
{  
    public virtual void Test1()  
    {  
    }  
}

{  
    [System.Obsolete()]  
    public override void Test1() // CS0809  
    {  
    }  
}  
```
