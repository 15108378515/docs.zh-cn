# <a name="compiler-error-cs0818"></a>编译器错误 CS0818
隐式类型的局部变量必须已初始化  
  
 在声明它的同时，必须使用值初始化隐式类型的局部变量。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  为变量分配一个值，或为其提供显式类型。  
  
## <a name="example"></a>示例  
 下面的代码生成 CS0818：  
  
```csharp  
// cs0818.cs  
class A  
{  
    public static int Main()  
    {  
        var a; // CS0818  
        return -1;  
    }  
}  
```
