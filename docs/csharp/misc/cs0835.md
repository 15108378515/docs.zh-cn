<span data-ttu-id="9a493-103">无法将 lambda 转换为其类型实参“type”不是委托类型的表达式树。</span><span class="sxs-lookup"><span data-stu-id="9a493-103">Cannot convert lambda to an expression tree whose type argument 'type' is not a delegate type.</span></span>  
  
 <span data-ttu-id="9a493-104">如果将 lambda 表达式转换为表达式树，表达式树必须针对其参数有一个委托类型。</span><span class="sxs-lookup"><span data-stu-id="9a493-104">If a lambda expression is converted to an expression tree, the expression tree must have a delegate type for its argument.</span></span> <span data-ttu-id="9a493-105">此外，lambda 表达式必须可转换为委托类型。</span><span class="sxs-lookup"><span data-stu-id="9a493-105">Furthermore, the lambda expression must be convertible to the delegate type.</span></span>

## <a name="example"></a><span data-ttu-id="9a493-108">示例</span><span class="sxs-lookup"><span data-stu-id="9a493-108">Example</span></span>  
 <span data-ttu-id="9a493-109">以下示例生成 CS0835：</span><span class="sxs-lookup"><span data-stu-id="9a493-109">The following example generates CS0835:</span></span>  
  
```csharp  
// cs0835.cs  
using System;  
using System.Linq;  
using System.Linq.Expressions;  
  
public class C  
{  
    public static int Main()  
    {  
        Expression<int> e = x => x + 1; // CS0835  
  
        // Try the following line instead.  
       // Expression<Func<int,int>> e2 = x => x + 1;  
  
        return 1;  
    }  
}  
```
