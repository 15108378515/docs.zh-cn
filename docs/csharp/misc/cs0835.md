## <a name="example"></a><span data-ttu-id="a1d81-108">示例</span><span class="sxs-lookup"><span data-stu-id="a1d81-108">Example</span></span>

```  
using System.Linq.Expressions;  
  
public class C  
{  
    public static int Main()  
    {  
        Expression<int> e = x => x + 1; // CS0835  
  
        // Try the following line instead.

        return 1;  
    }  
}  
```
