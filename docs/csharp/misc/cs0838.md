# <a name="compiler-error-cs0838"></a><span data-ttu-id="66f72-102">编译器错误 CS0838</span><span class="sxs-lookup"><span data-stu-id="66f72-102">Compiler Error CS0838</span></span>
<span data-ttu-id="66f72-103">表达式树不能包含多维数组初始值设定项。</span><span class="sxs-lookup"><span data-stu-id="66f72-103">An expression tree may not contain a multidimensional array initializer.</span></span>  
  
 <span data-ttu-id="66f72-104">表达式树中的多维数组不能使用数组初始值设定项进行初始化。</span><span class="sxs-lookup"><span data-stu-id="66f72-104">Multidimensional arrays in expression trees cannot be initialized by using an array initializer.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="66f72-105">更正此错误</span><span class="sxs-lookup"><span data-stu-id="66f72-105">To correct this error</span></span>  
  
1.  <span data-ttu-id="66f72-106">在创建表达式树之前，创建并初始化数组。</span><span class="sxs-lookup"><span data-stu-id="66f72-106">Create and initialize the array before creating the expression tree.</span></span>  
  
## <a name="example"></a><span data-ttu-id="66f72-107">示例</span><span class="sxs-lookup"><span data-stu-id="66f72-107">Example</span></span>  
 <span data-ttu-id="66f72-108">下面的示例生成 CS0838：</span><span class="sxs-lookup"><span data-stu-id="66f72-108">The following example generates CS0838:</span></span>  
  
```csharp  
// cs0838.cs  
using System;  
using System.Linq;  
using System.Linq.Expressions;  
  
namespace TestNamespace  
{  
    class Test  
    {  
        static int Main()  
        {  
  
            Expression<Func<int[,]>> expr =  
                () => new int[2, 2] { { 1, 2 }, { 3, 4 } }; // CS0838  
  
            // try the following 2 lines instead  
            int[,] nums = new int[2, 2] { { 1, 2 }, { 3, 4 } };  
            Expression<Func<int[,]>> expr2 = () => nums;   
  
            return 1;  
        }  
    }  
}  
```
