# <a name="compiler-error-cs1007"></a><span data-ttu-id="72b8e-102">编译器错误 CS1007</span><span class="sxs-lookup"><span data-stu-id="72b8e-102">Compiler Error CS1007</span></span>
<span data-ttu-id="72b8e-103">属性访问器已经定义</span><span class="sxs-lookup"><span data-stu-id="72b8e-103">Property accessor already defined</span></span>  
  
 <span data-ttu-id="72b8e-104">声明 [属性](../../csharp/programming-guide/classes-and-structs/using-properties.md)时，还必须声明其访问器方法。</span><span class="sxs-lookup"><span data-stu-id="72b8e-104">When you declare a [property](../../csharp/programming-guide/classes-and-structs/using-properties.md), you must also declare its accessor methods.</span></span> <span data-ttu-id="72b8e-105">但是，属性不能拥有多个 `get` 或 `set` 访问器方法。</span><span class="sxs-lookup"><span data-stu-id="72b8e-105">However, a property cannot have more than one `get` accessor method or more than one `set` accessor method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="72b8e-106">示例</span><span class="sxs-lookup"><span data-stu-id="72b8e-106">Example</span></span>  
 <span data-ttu-id="72b8e-107">下面的示例生成 CS1007：</span><span class="sxs-lookup"><span data-stu-id="72b8e-107">The following sample generates CS1007:</span></span>  
  
```csharp  
// CS1007.cs  
public class clx  
{  
    public int MyProperty  
    {  
        get  
        {  
            return 0;  
        }  
        get   // CS1007, this is the second get method  
        {  
            return 0;  
        }  
    }  
  
    public static void Main() {}  
}  
```
