# <a name="compiler-error-cs1023"></a><span data-ttu-id="4103d-102">编译器错误 CS1023</span><span class="sxs-lookup"><span data-stu-id="4103d-102">Compiler Error CS1023</span></span>
<span data-ttu-id="4103d-103">嵌入的语句不能是声明或标记语句</span><span class="sxs-lookup"><span data-stu-id="4103d-103">Embedded statement cannot be a declaration or labeled statement</span></span>  
  
 <span data-ttu-id="4103d-104">嵌入语句（如跟在 **if** 语句后面的语句）不能包含声明，也不能包含标记语句。</span><span class="sxs-lookup"><span data-stu-id="4103d-104">An embedded statement, such as the statements following an **if** statement, can contain neither declarations nor labeled statements.</span></span>  
  
 <span data-ttu-id="4103d-105">下面的示例生成 CS1023 两次：</span><span class="sxs-lookup"><span data-stu-id="4103d-105">The following sample generates CS1023 twice:</span></span>  
  
```csharp  
// CS1023.cs  
public class a  
{  
   public static void Main()  
   {  
      if (1)  
         int i;      // CS1023, declaration is not valid here  
  
      if (1)  
         xx : i++;   // CS1023, labeled statement is not valid here  
   }  
}  
```
