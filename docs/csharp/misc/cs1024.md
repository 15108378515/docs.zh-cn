# <a name="compiler-error-cs1024"></a>编译器错误 CS1024
应输入预处理器指令  
  
 行以井号 (#) 开头，但后面的字符串不是有效的 [预处理器指令](../../csharp/language-reference/preprocessor-directives/index.md)。  
  
 下面的示例生成 CS1024：  
  
```csharp  
// CS1024.cs  
#import System   // CS1024  
```
