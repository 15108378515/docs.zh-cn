# <a name="compiler-error-cs1055"></a><span data-ttu-id="7c893-102">编译器错误 CS1055</span><span class="sxs-lookup"><span data-stu-id="7c893-102">Compiler Error CS1055</span></span>
<span data-ttu-id="7c893-103">应为 add 访问器或 remove 访问器</span><span class="sxs-lookup"><span data-stu-id="7c893-103">An add or remove accessor expected</span></span>  
  
 <span data-ttu-id="7c893-104">如果 [事件](../../csharp/language-reference/keywords/event.md) 未声明为字段，则它必须同时定义 **add** 和 **remove** 访问器函数。</span><span class="sxs-lookup"><span data-stu-id="7c893-104">If your [event](../../csharp/language-reference/keywords/event.md) is not declared as a field, it must define both **add** and **remove** accessor functions.</span></span>  
  
 <span data-ttu-id="7c893-105">下面的示例生成 CS1055：</span><span class="sxs-lookup"><span data-stu-id="7c893-105">The following sample generates CS1055:</span></span>  
  
```csharp  
// CS1055.cs  
delegate void del();  
class Test  
{  
   public event del MyEvent  
   {  
      int i;   // CS1055  
      // uncomment accessors and delete previous line to resolve  
      // add  
      // {  
      //    MyEvent += value;  
      // }  
      // remove  
      // {  
      //    MyEvent -= value;  
      // }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
