# <a name="compiler-error-cs1057"></a>编译器错误 CS1057
“member”: 静态类不能包含保护成员  
  
 在静态类内部声明受保护成员时，会生成此错误。  
  
## <a name="example"></a>示例  
 下面的示例生成 CS1057。  
  
```csharp  
// CS1057.cs  
using System;  
  
static class Class1  
{  
    protected static int x;   // CS1057  
    public static void Main()  
    {  
    }  
}  
```
