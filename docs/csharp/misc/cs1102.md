# <a name="compiler-error-cs1102"></a><span data-ttu-id="eff4b-102">编译器错误 CS1102</span><span class="sxs-lookup"><span data-stu-id="eff4b-102">Compiler Error CS1102</span></span>
<span data-ttu-id="eff4b-103">参数修饰符“out”不能与“this”一起使用。</span><span class="sxs-lookup"><span data-stu-id="eff4b-103">The parameter modifier 'out' cannot be used with 'this'.</span></span>  
  
 <span data-ttu-id="eff4b-104">当 `this` 关键字修改静态方法的第一个参数时，它向编译器发出信号，指示该方法是扩展方法。</span><span class="sxs-lookup"><span data-stu-id="eff4b-104">When the `this` keyword modifies the first parameter of a static method, it signals to the compiler that the method is an extension method.</span></span> <span data-ttu-id="eff4b-105">不需要或不允许对扩展方法的第一个参数使用任何其他修饰符。</span><span class="sxs-lookup"><span data-stu-id="eff4b-105">No other modifiers are needed or allowed on the first parameter of an extension method.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="eff4b-106">更正此错误</span><span class="sxs-lookup"><span data-stu-id="eff4b-106">To correct this error</span></span>  
  
1.  <span data-ttu-id="eff4b-107">从第一个参数中删除未经授权的修饰符。</span><span class="sxs-lookup"><span data-stu-id="eff4b-107">Remove the unauthorized modifiers from the first parameter.</span></span>  
  
## <a name="example"></a><span data-ttu-id="eff4b-108">示例</span><span class="sxs-lookup"><span data-stu-id="eff4b-108">Example</span></span>  
 <span data-ttu-id="eff4b-109">下面的示例生成 CS1102：</span><span class="sxs-lookup"><span data-stu-id="eff4b-109">The following example generates CS1102:</span></span>  
  
```csharp  
// cs1102.cs  
// Compile with: /target:library.  
public static class Extensions  
{  
    // No type parameters.  
        public static void Test(this out int i) {} // CS1102  
  
    //Single type parameter  
        public static void Test<T>(this out T t) {}// CS1102  
  
    //Multiple type parameters  
        public static void Test<T,U,V>(this out U u) {}// CS1102  
}  
```
