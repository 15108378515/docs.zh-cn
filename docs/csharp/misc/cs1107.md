# <a name="compiler-error-cs1107"></a><span data-ttu-id="566ab-102">编译器错误 CS1107</span><span class="sxs-lookup"><span data-stu-id="566ab-102">Compiler Error CS1107</span></span>
<span data-ttu-id="566ab-103">参数只能具有一个“modifier name”修饰符。</span><span class="sxs-lookup"><span data-stu-id="566ab-103">A parameter can only have one 'modifier name' modifier.</span></span>  
  
 <span data-ttu-id="566ab-104">它是用于类似 `this`、 `ref`和 `out` 的参数修饰符的错误，在参数定义中出现多次。</span><span class="sxs-lookup"><span data-stu-id="566ab-104">It is an error for parameter modifiers such as `this`, `ref`, and `out` to appear more than one time in a parameter definition.</span></span>  
  
## <a name="example"></a><span data-ttu-id="566ab-105">示例</span><span class="sxs-lookup"><span data-stu-id="566ab-105">Example</span></span>  
 <span data-ttu-id="566ab-106">下面的示例生成 CS1107：</span><span class="sxs-lookup"><span data-stu-id="566ab-106">The following example generates CS1107:</span></span>  
  
```  
// cs1107.cs  
public static class Test  
{  
    // Extension methods.  
        public static void TestMethod(this this t) {} // CS1107  
  
    // Regular Instance Method  
        public void TestMethod(ref ref int i) {} // CS1107  
  
}  
```
