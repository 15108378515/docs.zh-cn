## <a name="example"></a><span data-ttu-id="c5104-109">示例</span><span class="sxs-lookup"><span data-stu-id="c5104-109">Example</span></span>  
 <span data-ttu-id="c5104-110">以下示例生成 CS1113：</span><span class="sxs-lookup"><span data-stu-id="c5104-110">The following example generates CS1113:</span></span>  
  
```csharp  
// cs1113.cs  
using System;  
public static class Extensions  
{  
    public static S ExtMethod(this S s)  
    {  
        return s;  
    }  
}  
  
public struct S  
{  
}  
  
public class Test  
{  
    static int Main()  
    {  
        Func<S> f = new S().ExtMethod; // CS1113  
        return 1;  
    }  
}  
```
