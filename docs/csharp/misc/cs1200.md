# <a name="compiler-warning-level-1-cs1200"></a>编译器警告（等级 1）CS1200
功能“invalid feature”已弃用。 请改用“valid feature”。  
  
 尝试使用的功能现已弃用。 请更新代码，以改用有效的功能。
