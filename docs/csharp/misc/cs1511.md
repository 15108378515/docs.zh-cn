 <span data-ttu-id="27c6c-104">[base](../../csharp/language-reference/keywords/base.md) 关键字用在 [静态](../../csharp/language-reference/keywords/static.md) 方法中。</span><span class="sxs-lookup"><span data-stu-id="27c6c-104">The [base](../../csharp/language-reference/keywords/base.md) keyword was used in a [static](../../csharp/language-reference/keywords/static.md) method.</span></span> <span data-ttu-id="27c6c-105">只能在实例构造函数、实例方法或实例访问器中调用`base` 。</span><span class="sxs-lookup"><span data-stu-id="27c6c-105">`base` can only be called in an instance constructor, instance method, or instance accessor.</span></span>  
  
## <a name="example"></a><span data-ttu-id="27c6c-106">示例</span><span class="sxs-lookup"><span data-stu-id="27c6c-106">Example</span></span>  
 <span data-ttu-id="27c6c-107">以下示例生成 CS1511。</span><span class="sxs-lookup"><span data-stu-id="27c6c-107">The following sample generates CS1511.</span></span>  
  
```  
// CS1511.cs  
// compile with: /target:library  
public class A  
{  
   public int j = 0;  
}  
  
class C : A  
{  
   public void Method()  
   {  
      base.j = 3;   // base allowed here  
   }  
  
   public static int StaticMethod()  
   {  
      base.j = 3;   // CS1511  
      return 1;  
   }  
}  
```
