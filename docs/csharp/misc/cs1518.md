# <a name="compiler-error-cs1518"></a>编译器错误 CS1518
应输入 class、delegate、enum、interface 或 struct  
  
 找到在 [命名空间](../../csharp/language-reference/keywords/namespace.md)中不受支持的声明。 在命名空间中，编译器仅接受类、结构、枚举、接口、命名空间和委托。  
  
## <a name="example"></a>示例  
 下面的示例生成 CS1518：  
  
```csharp  
// CS1518.cs  
namespace x  
{  
   sealed class c1 {};      // OK  
   namespace f2 {};         // OK  
   sealed f3 {};            // CS1518  
   }  
```
