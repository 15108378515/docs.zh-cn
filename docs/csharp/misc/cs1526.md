## <a name="example"></a><span data-ttu-id="1dcfd-105">示例</span><span class="sxs-lookup"><span data-stu-id="1dcfd-105">Example</span></span>

```csharp  
// CS1526.cs  
public class y  
{  
   public static int i = 0;  
   public int myi = 0;  
}  
  
public class z  
{  
   public static void Main()  
   {  
      y py = new y;   // CS1526  
      y[] aoys = new y[10];   // Array of Ys  
  
      for (int i = 0; i < aoys.Length; i++)  
         aoys[i] = new y();   // an object of type y  
   }  
}  
```
