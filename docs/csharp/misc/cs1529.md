## <a name="example"></a><span data-ttu-id="6a130-105">示例</span><span class="sxs-lookup"><span data-stu-id="6a130-105">Example</span></span>  
 <span data-ttu-id="6a130-106">下面的示例生成 CS1529:</span><span class="sxs-lookup"><span data-stu-id="6a130-106">The following sample generates CS1529:</span></span>  
  
```  
// CS1529.cs  
namespace X  
{  
    namespace Subspace  
    {  
        using Microsoft;  
  
        class SomeClass  
        {  
        };  
  
        using Microsoft;      // CS1529, place before class definition  
    }  
  
    using System.Reflection;  // CS1529, place before namespace 'Subspace'  
}  
  
using System;                 // CS1529, place at the beginning of the file  
```
