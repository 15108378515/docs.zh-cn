# <a name="compiler-error-cs1556"></a>编译器错误 CS1556
为 Main 方法指定的“construct”必须是有效的类或结构  
  
 [/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) 编译器选项传递不是类名的标识符。
