# <a name="compiler-error-cs1575"></a><span data-ttu-id="fcd3c-102">编译器错误 CS1575</span><span class="sxs-lookup"><span data-stu-id="fcd3c-102">Compiler Error CS1575</span></span>
<span data-ttu-id="fcd3c-103">Stackalloc 表达式要求在类型后有 []</span><span class="sxs-lookup"><span data-stu-id="fcd3c-103">A stackalloc expression requires [] after type</span></span>  
  
 <span data-ttu-id="fcd3c-104">使用 [stackalloc](../../csharp/language-reference/keywords/stackalloc.md)所请求的分配的大小必须在方括号中指定。</span><span class="sxs-lookup"><span data-stu-id="fcd3c-104">The size of the requested allocation, with [stackalloc](../../csharp/language-reference/keywords/stackalloc.md), must be specified in square brackets.</span></span>  
  
 <span data-ttu-id="fcd3c-105">下面的示例生成 CS1575：</span><span class="sxs-lookup"><span data-stu-id="fcd3c-105">The following sample generates CS1575:</span></span>  
  
```  
// CS1575.cs  
// compile with: /unsafe

{  
   unsafe public static void Main()  
   {  
      int *p = stackalloc int (30);   // CS1575  
      // try the following line instead  
      // int *p = stackalloc int [30];  
   }  
}  
```
