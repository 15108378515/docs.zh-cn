 <span data-ttu-id="e3bec-104">编译器检测到传递给 [#line](../../csharp/language-reference/preprocessor-directives/preprocessor-line.md) 指令的值有错误。</span><span class="sxs-lookup"><span data-stu-id="e3bec-104">The compiler detected an error with the value passed to the [#line](../../csharp/language-reference/preprocessor-directives/preprocessor-line.md) directive.</span></span>

      #line "abc.sc"         // CS1576
