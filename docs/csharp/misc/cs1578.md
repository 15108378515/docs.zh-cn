# <a name="compiler-error-cs1578"></a><span data-ttu-id="ef5e2-102">编译器错误 CS1578</span><span class="sxs-lookup"><span data-stu-id="ef5e2-102">Compiler Error CS1578</span></span>

 <span data-ttu-id="ef5e2-104">[#line](../../csharp/language-reference/preprocessor-directives/preprocessor-line.md) 指令后面只能出现文件名（双引号括起）或单行注释。</span><span class="sxs-lookup"><span data-stu-id="ef5e2-104">After a [#line](../../csharp/language-reference/preprocessor-directives/preprocessor-line.md) directive, only a file name (in double quotation marks) or a single-line comment is allowed.</span></span>  
  
 <span data-ttu-id="ef5e2-105">以下示例生成 CS1578：</span><span class="sxs-lookup"><span data-stu-id="ef5e2-105">The following sample generates CS1578:</span></span>  
  
```  
// CS1578.cs  
class MyClass  
{  
   static void Main()  
   {  
      #line 101 abc.cs   // CS1578  
      // try the following line instead  
      //#line 101 "abc.cs"  
      intt i;          // error will be reported on line 101  
   }  
}  
```
