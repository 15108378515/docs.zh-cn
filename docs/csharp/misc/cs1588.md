# <a name="compiler-error-cs1588"></a>编译器错误 CS1588
无法确定公共语言运行时目录 --“reason”  
  
 编译器无法检索 .NET Framework 公共语言运行时的安装目录。 此错误指示公共语言运行时未正确安装。
