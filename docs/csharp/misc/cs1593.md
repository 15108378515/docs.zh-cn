# <a name="compiler-error-cs1593"></a><span data-ttu-id="b6433-102">编译器错误 CS1593</span><span class="sxs-lookup"><span data-stu-id="b6433-102">Compiler Error CS1593</span></span>
<span data-ttu-id="b6433-103">委托“del”不接受“number”参数</span><span class="sxs-lookup"><span data-stu-id="b6433-103">Delegate 'del' does not take 'number' arguments</span></span>  
  
 <span data-ttu-id="b6433-104">传递给 [委托](../../csharp/language-reference/keywords/delegate.md) 调用的参数数目与委托声明中的参数数目不一致。</span><span class="sxs-lookup"><span data-stu-id="b6433-104">The number of arguments passed to a [delegate](../../csharp/language-reference/keywords/delegate.md) invocation does not agree with the number of parameters in the delegate declaration.</span></span>  
  
 <span data-ttu-id="b6433-105">下面的示例生成 CS1593：</span><span class="sxs-lookup"><span data-stu-id="b6433-105">The following sample generates CS1593:</span></span>  
  
```  
// CS1593.cs  
using System;  
delegate string func(int i);   // declare delegate  
  
class a  
{  
   public static void Main()  
   {  
      func dt = new func(z);  
      x(dt);  
   }  
  
   public static string z(int j)  
   {  
      Console.WriteLine(j);  
      return j.ToString();  
   }  
  
   public static void x(func hello)  
   {  
      hello(8, 9);   // CS1593  
      // try the following line instead  
      // hello(8);  
   }  
}  
```
