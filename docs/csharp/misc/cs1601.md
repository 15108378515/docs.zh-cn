# <a name="compiler-error-cs1601"></a><span data-ttu-id="b6288-102">编译器错误 CS1601</span><span class="sxs-lookup"><span data-stu-id="b6288-102">Compiler Error CS1601</span></span>
<span data-ttu-id="b6288-103">方法或委托参数不能是“type”类型</span><span class="sxs-lookup"><span data-stu-id="b6288-103">Method or delegate parameter cannot be of type 'type'</span></span>  
  
 <span data-ttu-id="b6288-104">.NET Framework 类库中的一些类型，例如 <xref:System.TypedReference>、 <xref:System.RuntimeArgumentHandle> 和 <xref:System.ArgIterator> 不能用作 [ref](../../csharp/language-reference/keywords/ref.md) 或 [out](../../csharp/language-reference/keywords/out.md) 参数，因为它们有可能会被用来执行不安全的操作。</span><span class="sxs-lookup"><span data-stu-id="b6288-104">Some types in the .NET Framework class library, for example, <xref:System.TypedReference>, <xref:System.RuntimeArgumentHandle> and <xref:System.ArgIterator> cannot be used as [ref](../../csharp/language-reference/keywords/ref.md) or [out](../../csharp/language-reference/keywords/out.md) parameters because they could potentially be used to perform unsafe operations.</span></span>  
  
 <span data-ttu-id="b6288-105">下面的示例生成 CS1601：</span><span class="sxs-lookup"><span data-stu-id="b6288-105">The following sample generates CS1601:</span></span>  
  
```  
// CS1601.cs  
using System;  
  
class MyClass  
{  
   public void Test1 (ref TypedReference t)   // CS1601  
   {  
   }  
  
   public void Test2 (out ArgIterator t)   // CS1601  
   {  
   }  
}  
```
