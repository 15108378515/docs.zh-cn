# <a name="compiler-error-cs1611"></a><span data-ttu-id="de16e-102">编译器错误 CS1611</span><span class="sxs-lookup"><span data-stu-id="de16e-102">Compiler Error CS1611</span></span>
<span data-ttu-id="de16e-103">Params 参数不能声明为如下所示 ref 或 out</span><span class="sxs-lookup"><span data-stu-id="de16e-103">The params parameter cannot be declared as in ref or out</span></span>  
  
 <span data-ttu-id="de16e-104">关键字[中](../../csharp/language-reference/keywords/in-parameter-modifier.md)， [ref](../../csharp/language-reference/keywords/ref.md)或[出](../../csharp/language-reference/keywords/out-parameter-modifier.md)不能与使用[params](../../csharp/language-reference/keywords/params.md)关键字。</span><span class="sxs-lookup"><span data-stu-id="de16e-104">The keywords [in](../../csharp/language-reference/keywords/in-parameter-modifier.md), [ref](../../csharp/language-reference/keywords/ref.md) or [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) cannot be used with the [params](../../csharp/language-reference/keywords/params.md) keyword.</span></span>  
  
 <span data-ttu-id="de16e-105">下面的示例生成 CS1611：</span><span class="sxs-lookup"><span data-stu-id="de16e-105">The following sample generates CS1611:</span></span>  
  
```csharp  
// CS1611.cs  
public class MyClass  
{  
   public static void Test(params ref int[] a)   // CS1611, remove ref  
   {  
   }  
  
   public static void Main()  
   {  
      Test(1);  
   }  
}  
```
