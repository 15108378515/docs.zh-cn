# <a name="compiler-error-cs1613"></a><span data-ttu-id="f509b-102">编译器错误 CS1613</span><span class="sxs-lookup"><span data-stu-id="f509b-102">Compiler Error CS1613</span></span>
<span data-ttu-id="f509b-103">找不到用于接口 “interface” coclass 包装类 “class” （是否缺少程序集引用？）</span><span class="sxs-lookup"><span data-stu-id="f509b-103">The managed coclass wrapper class 'class' for interface 'interface' cannot be found (are you missing an assembly reference?)</span></span>  
  
 <span data-ttu-id="f509b-104">已尝试实例化来自接口的 COM 对象。</span><span class="sxs-lookup"><span data-stu-id="f509b-104">An attempt was made to instantiate a COM object from an interface.</span></span> <span data-ttu-id="f509b-105">此接口具有 **ComImport** 和 `CoClass` 特性，但编译器找不到为 `CoClass` 特性给定的类型。</span><span class="sxs-lookup"><span data-stu-id="f509b-105">The interface has the **ComImport** and `CoClass` attributes, but the compiler cannot find the type given for the `CoClass` attribute.</span></span>  
  
 <span data-ttu-id="f509b-106">要解决此错误，你可以尝试以下项之一：</span><span class="sxs-lookup"><span data-stu-id="f509b-106">To resolve this error, you can try one of the following:</span></span>  
  
-   <span data-ttu-id="f509b-107">向具有 coclass 的程序集添加一个引用（大多数情况下，接口和 coclass 应位于同一程序集中）</span><span class="sxs-lookup"><span data-stu-id="f509b-107">Add a reference to the assembly that has the coclass (most of the time the interface and coclass should be in the same assembly).</span></span> <span data-ttu-id="f509b-108">请参阅[/参考](../../csharp/language-reference/compiler-options/reference-compiler-option.md)或[添加引用对话框](http://msdn.microsoft.com/library/2feb0fe2-0805-4cc9-8cba-b0315849dfb7)有关信息。</span><span class="sxs-lookup"><span data-stu-id="f509b-108">See [/reference](../../csharp/language-reference/compiler-options/reference-compiler-option.md) or [Add Reference Dialog Box](http://msdn.microsoft.com/library/2feb0fe2-0805-4cc9-8cba-b0315849dfb7) for information.</span></span>  
  
-   <span data-ttu-id="f509b-109">修复接口上的 `CoClass` 特性。</span><span class="sxs-lookup"><span data-stu-id="f509b-109">Fix the `CoClass` attribute on the interface.</span></span>  
  
 <span data-ttu-id="f509b-110">以下示例演示如何正确使用 **CoClassAttribute**：</span><span class="sxs-lookup"><span data-stu-id="f509b-110">The following sample demonstrates correct usage of **CoClassAttribute**:</span></span>  
  
```csharp  
// CS1613.cs  
using System;  
using System.Runtime.InteropServices;  
  
[Guid("1FFD7840-E82D-4268-875C-80A160C23296")]  
[ComImport()]  
[CoClass(typeof(A))]  
public interface IA{}  
public class A : IA {}  
  
public class AA  
{  
   public static void Main()  
   {  
      IA i;  
      i = new IA(); // This is equivalent to new A().  
                    // because of the CoClass attribute on IA  
   }  
}  
```
