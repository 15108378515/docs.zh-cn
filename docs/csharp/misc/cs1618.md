# <a name="compiler-error-cs1618"></a><span data-ttu-id="62d90-102">编译器错误 CS1618</span><span class="sxs-lookup"><span data-stu-id="62d90-102">Compiler Error CS1618</span></span>
<span data-ttu-id="62d90-103">无法用“method”创建委托，因为它具有 Conditional 特性</span><span class="sxs-lookup"><span data-stu-id="62d90-103">Cannot create delegate with 'method' because it has a Conditional attribute</span></span>  
  
 <span data-ttu-id="62d90-104">无法用 conditional 方法创建委托，因为该方法可能不存在于某些生成中。</span><span class="sxs-lookup"><span data-stu-id="62d90-104">You cannot create a delegate with a conditional method because the method might not exist in some builds.</span></span>  
  
 <span data-ttu-id="62d90-105">下面的示例生成 CS1618：</span><span class="sxs-lookup"><span data-stu-id="62d90-105">The following sample generates CS1618:</span></span>  
  
```csharp  
// CS1618.cs  
using System;  
using System.Diagnostics;  
  
delegate void del();  
  
class MakeAnError {  
   public static void Main() {  
      del d = new del(ConditionalMethod);   // CS1618  
      // Invalid because on builds where DEBUG is not set,   
      // there will be no "ConditionalMethod".  
   }  
   // To fix the error, remove the next line:  
   [Conditional("DEBUG")]  
   public static void ConditionalMethod()   
   {  
      Console.WriteLine("Do something only in debug");  
   }  
}  
```
