# <a name="compiler-error-cs1620"></a><span data-ttu-id="e34a6-102">编译器错误 CS1620</span><span class="sxs-lookup"><span data-stu-id="e34a6-102">Compiler Error CS1620</span></span>
<span data-ttu-id="e34a6-103">参数“number”必须使用关键字“keyword”传递</span><span class="sxs-lookup"><span data-stu-id="e34a6-103">Argument 'number' must be passed with the 'keyword' keyword</span></span>  
  
 <span data-ttu-id="e34a6-104">如果将自变量传递给采用的函数，则会发生此错误[ref](../../csharp/language-reference/keywords/ref.md)或[出](../../csharp/language-reference/keywords/out-parameter-modifier.md)参数并且不包括`ref`或`out`关键字在调用时，也包括错误的关键字。</span><span class="sxs-lookup"><span data-stu-id="e34a6-104">This error occurs if you are passing an argument to a function that takes a [ref](../../csharp/language-reference/keywords/ref.md) or [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) parameter and you don't include the `ref` or `out` keyword at the point of call, or you include the wrong keyword.</span></span> <span data-ttu-id="e34a6-105">错误文本指示要使用的相应关键字，以及导致失败的参数。</span><span class="sxs-lookup"><span data-stu-id="e34a6-105">The error text indicates the appropriate keyword to use and which argument caused the failure.</span></span>  
  
 <span data-ttu-id="e34a6-106">下面的示例生成 CS1620：</span><span class="sxs-lookup"><span data-stu-id="e34a6-106">The following sample generates CS1620:</span></span>  
  
```csharp  
// CS1620.cs  
class C  
{  
    void f(ref int i) {}  
    public static void Main()  
    {  
        int x = 1;  
        f(out x);  // CS1620 – f takes a ref parameter, not an out parameter  
        // Try this line instead:  
        // f(ref x);  
    }  
}  
```
