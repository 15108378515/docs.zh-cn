<span data-ttu-id="c1dde-103">无法从迭代器返回值。</span><span class="sxs-lookup"><span data-stu-id="c1dde-103">Cannot return a value from an iterator.</span></span> <span data-ttu-id="c1dde-104">请使用 yield return 语句返回值，或使用 yield break 语句结束迭代。</span><span class="sxs-lookup"><span data-stu-id="c1dde-104">Use the yield return statement to return a value, or yield break to end the iteration.</span></span>  
  
 <span data-ttu-id="c1dde-105">迭代器是通过 yield 语句而不是 return 语句返回值的特殊函数。</span><span class="sxs-lookup"><span data-stu-id="c1dde-105">An iterator is a special function that returns a value via the yield statement rather than the return statement.</span></span> <span data-ttu-id="c1dde-106">有关详细信息，请参见 **迭代器**。</span><span class="sxs-lookup"><span data-stu-id="c1dde-106">For more information, see **iterators**.</span></span>  
  
 <span data-ttu-id="c1dde-107">下面的示例生成 CS1622：</span><span class="sxs-lookup"><span data-stu-id="c1dde-107">The following sample generates CS1622:</span></span>  
  
```csharp  
// CS1622.cs  
// compile with: /target:library  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      return (IEnumerator) this;  // CS1622  
      yield return this;   // OK  
   }  
}  
```
