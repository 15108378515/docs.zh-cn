# <a name="compiler-error-cs1623"></a><span data-ttu-id="6ccd6-102">编译器错误 CS1623</span><span class="sxs-lookup"><span data-stu-id="6ccd6-102">Compiler Error CS1623</span></span>
<span data-ttu-id="6ccd6-103">迭代器不能有 ref 或 out 参数</span><span class="sxs-lookup"><span data-stu-id="6ccd6-103">Iterators cannot have in ref or out parameters</span></span>  
  
 <span data-ttu-id="6ccd6-104">如果迭代器方法采用，则会发生此错误`in`， `ref`，或`out`参数。</span><span class="sxs-lookup"><span data-stu-id="6ccd6-104">This error occurs if an iterator method takes an `in`, `ref`, or `out` parameter.</span></span> <span data-ttu-id="6ccd6-105">若要避免此错误，删除`in`， `ref`，或`out`关键字从方法签名。</span><span class="sxs-lookup"><span data-stu-id="6ccd6-105">To avoid this error, remove the `in`, `ref`, or `out` keyword from the method signature.</span></span>  
  
## <a name="example"></a><span data-ttu-id="6ccd6-106">示例</span><span class="sxs-lookup"><span data-stu-id="6ccd6-106">Example</span></span>  
 <span data-ttu-id="6ccd6-107">以下示例生成 CS1623：</span><span class="sxs-lookup"><span data-stu-id="6ccd6-107">The following sample generates CS1623:</span></span>  
  
```csharp  
// CS1623.cs  
using System.Collections;

class C : IEnumerable
{
    public IEnumerator GetEnumerator()
    {
        yield return 0;
    }

    // To resolve the error, remove in  
    public IEnumerator GetEnumerator(in short i)  // CS1623  
    {
        yield return i;
    }
    // To resolve the error, remove ref  
    public IEnumerator GetEnumerator(ref int i)  // CS1623  
    {
        yield return i;
    }

    // To resolve the error, remove out  
    public IEnumerator GetEnumerator(out float f)  // CS1623  
    {
        f = 0.0F;
        yield return f;
    }
}
```
