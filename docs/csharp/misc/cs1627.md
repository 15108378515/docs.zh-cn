# <a name="compiler-error-cs1627"></a><span data-ttu-id="93235-102">编译器错误 CS1627</span><span class="sxs-lookup"><span data-stu-id="93235-102">Compiler Error CS1627</span></span>
<span data-ttu-id="93235-103">yield return 之后应为表达式</span><span class="sxs-lookup"><span data-stu-id="93235-103">Expression expected after yield return</span></span>  
  
 <span data-ttu-id="93235-104">如果在没有表达式的情况下使用 `yield` ，则会发生此错误。</span><span class="sxs-lookup"><span data-stu-id="93235-104">This error occurs if `yield` is used without an expression.</span></span> <span data-ttu-id="93235-105">若要避免此错误，请在语句中插入适当的表达式。</span><span class="sxs-lookup"><span data-stu-id="93235-105">To avoid this error, insert the appropriate expression in the statement.</span></span>  
  
 <span data-ttu-id="93235-106">以下示例生成 CS1627：</span><span class="sxs-lookup"><span data-stu-id="93235-106">The following sample generates CS1627:</span></span>  
  
```  
// CS1627.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      yield return;   // CS1627  
      // To resolve, add the following line:  
      // yield return 0;  
   }  
}  
  
public class CMain  
{  
   public static void Main() { }  
}  
```
