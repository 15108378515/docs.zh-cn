# <a name="compiler-error-cs1629"></a>编译器错误 CS1629
迭代器中不能出现不安全的代码  
  
 C# 语言规范不允许迭代器中有不安全代码。  
  
 下面的示例生成 CS1629：  
  
```csharp  
// CS1629.cs  
// compile with: /unsafe    
using System.Collections.Generic;  
class C   
{  
   IEnumerator<int> IteratorMeth() {  
      int i;  
      unsafe  // CS1629  
      {  
         int *p = &i;  
         yield return *p;  
      }  
   }  
}  
```
