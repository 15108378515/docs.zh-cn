# <a name="compiler-error-cs1631"></a><span data-ttu-id="2d2df-102">编译器错误 CS1631</span><span class="sxs-lookup"><span data-stu-id="2d2df-102">Compiler Error CS1631</span></span>
<span data-ttu-id="2d2df-103">无法在 catch 子句体中生成值</span><span class="sxs-lookup"><span data-stu-id="2d2df-103">Cannot yield a value in the body of a catch clause</span></span>  
  
 <span data-ttu-id="2d2df-104">在 catch 子句主体内不允许出现 yield 语句。</span><span class="sxs-lookup"><span data-stu-id="2d2df-104">The yield statement is not allowed from within the body of a catch clause.</span></span> <span data-ttu-id="2d2df-105">若要避免此错误，请将 yield 语句移出 catch 子句主体。</span><span class="sxs-lookup"><span data-stu-id="2d2df-105">To avoid this error, move the yield statement outside the body of the catch clause.</span></span>  
  
 <span data-ttu-id="2d2df-106">以下示例生成 CS1631：</span><span class="sxs-lookup"><span data-stu-id="2d2df-106">The following sample generates CS1631:</span></span>  
  
```csharp  
// CS1631.cs  
using System;  
using System.Collections;  
  
public class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()   
   {  
      try  
      {  
      }  
      catch(Exception e)  
      {  
        yield return this;  // CS1631  
      }  
   }    
  
   public static void Main()   
   {  
   }  
}  
```
