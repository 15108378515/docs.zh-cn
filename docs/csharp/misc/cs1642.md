# <a name="compiler-error-cs1642"></a><span data-ttu-id="b43f1-102">编译器错误 CS1642</span><span class="sxs-lookup"><span data-stu-id="b43f1-102">Compiler Error CS1642</span></span>
<span data-ttu-id="b43f1-103">固定大小的缓冲区字段只能是结构的成员。</span><span class="sxs-lookup"><span data-stu-id="b43f1-103">Fixed size buffer fields may only be members of structs.</span></span>  
  
 <span data-ttu-id="b43f1-104">如果你使用 `class`而不是 `struct`中固定大小的缓冲区字段，则会发生此错误。</span><span class="sxs-lookup"><span data-stu-id="b43f1-104">This error occurs if you use a fixed size buffer field in a `class`, instead of a `struct`.</span></span> <span data-ttu-id="b43f1-105">若要解决此错误，请将 `class` 更改为 `struct` ，或将字段声明为一个普通数组。</span><span class="sxs-lookup"><span data-stu-id="b43f1-105">To resolve this error, change the `class` to a `struct` or declare the field as an ordinary array.</span></span>  
  
## <a name="example"></a><span data-ttu-id="b43f1-106">示例</span><span class="sxs-lookup"><span data-stu-id="b43f1-106">Example</span></span>  
 <span data-ttu-id="b43f1-107">下面的示例生成 CS1642。</span><span class="sxs-lookup"><span data-stu-id="b43f1-107">The following sample generates CS1642.</span></span>  
  
```csharp  
// CS1642.cs  
// compile with: /unsafe /target:library  
unsafe class C  
{  
   fixed int a[10];   // CS1642  
}  
  
unsafe struct D  
{  
    fixed int a[10];  
}  
  
unsafe class E  
{  
   public int[] a = null;  
}  
```
