## <a name="example"></a><span data-ttu-id="3590d-106">示例</span><span class="sxs-lookup"><span data-stu-id="3590d-106">Example</span></span>  
 <span data-ttu-id="3590d-107">下面的示例生成 CS1642。</span><span class="sxs-lookup"><span data-stu-id="3590d-107">The following sample generates CS1642.</span></span>  
  
```  
// CS1642.cs  
// compile with: /unsafe /target:library  
unsafe class C  
{  
   fixed int a[10];   // CS1642  
}  
  
unsafe struct D  
{  
    fixed int a[10];  
}  
  
unsafe class E  
{  
   public int[] a = null;  
}  
```
