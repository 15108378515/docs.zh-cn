# <a name="compiler-error-cs1646"></a>编译器错误 CS1646
关键字、 标识符或字符串应原意说明符后面： \@  
  
 请参阅字符串参数的使用情况的原意说明符\@。 仅允许原意说明符位于字符串、关键字或标识符之前。 若要解决此错误，从任何不适当的位置删除 @ 符号，或添加需要的字符串、关键字或标识符。  
  
 下面的示例生成 CS1646：  
  
```csharp  
// CS1646  
class C  
{  
   int i = @5;  // CS1646  
   // Try this line instead:  
   // int i = 5;  
}  
```
