# <a name="compiler-error-cs1647"></a>编译器错误 CS1647
表达式太长或者过于复杂，无法在“code”附近编译  
  
 处理你的代码的编译器中有堆栈溢出。 若要解决此错误，请简化你的代码。 如果你的代码是有效的请与联系[产品支持](http://msdn.microsoft.com/library/77e75b8b-817d-45bf-9c38-458930d873b4)。
