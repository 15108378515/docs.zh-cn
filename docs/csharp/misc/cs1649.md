# <a name="compiler-error-cs1649"></a><span data-ttu-id="a253d-102">编译器错误 CS1649</span><span class="sxs-lookup"><span data-stu-id="a253d-102">Compiler Error CS1649</span></span>
<span data-ttu-id="a253d-103">无法向只读字段“identifier”的成员传递 ref 或 out 参数（构造函数中除外）</span><span class="sxs-lookup"><span data-stu-id="a253d-103">Members of readonly field 'identifier' cannot be passed ref or out (except in a constructor)</span></span>  
  
 <span data-ttu-id="a253d-104">如果将变量作为 `readonly` 或 `ref` 参数传递给一个属于 `out` 字段成员的函数，则会发生此错误。</span><span class="sxs-lookup"><span data-stu-id="a253d-104">This error occurs if you pass a variable to a function that is a member of a `readonly` field as a `ref` or `out` argument.</span></span> <span data-ttu-id="a253d-105">由于该函数可能修改 `ref` 和 `out` 参数，所以这是不允许的。</span><span class="sxs-lookup"><span data-stu-id="a253d-105">Since `ref` and `out` parameters may be modified by the function, this is not allowed.</span></span> <span data-ttu-id="a253d-106">若要解决此错误，请删除字段上的 `readonly` 关键字，或者不要将 `readonly` 字段的成员传递给该函数。</span><span class="sxs-lookup"><span data-stu-id="a253d-106">To resolve this error, remove the `readonly` keyword on the field, or do not pass the members of the `readonly` field to the function.</span></span> <span data-ttu-id="a253d-107">例如，你可以尝试创建一个可修改的临时变量，并将该临时变量作为 `ref` 参数传递，如下面的示例所示。</span><span class="sxs-lookup"><span data-stu-id="a253d-107">For example, you might try creating a temporary variable which can be modifed and passing the temporary as a `ref` argument, as shown in the following example.</span></span>  
  
## <a name="example"></a><span data-ttu-id="a253d-108">示例</span><span class="sxs-lookup"><span data-stu-id="a253d-108">Example</span></span>  
 <span data-ttu-id="a253d-109">下面的示例生成 CS1649：</span><span class="sxs-lookup"><span data-stu-id="a253d-109">The following sample generates CS1649:</span></span>  
  
```csharp  
// CS1649.cs  
public struct Inner  
    {  
        public int i;  
    }  
  
class Outer  
{  
    public readonly Inner inner = new Inner();  
}  
  
class D  
{  
    static void f(ref int iref)  
    {  
    }  
  
    static void Main()  
    {  
        Outer outer = new Outer();   
        f(ref outer.inner.i);  // CS1649  
        // Try this code instead:  
        // int tmp = outer.inner.i;  
        // f(ref tmp);  
    }  
}  
```
