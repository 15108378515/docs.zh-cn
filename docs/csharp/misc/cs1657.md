 <span data-ttu-id="eef1f-104">该错误发生时的变量作为传递[ref](../../csharp/language-reference/keywords/ref.md)或[出](../../csharp/language-reference/keywords/out-parameter-modifier.md)中该变量是只读的上下文参数。</span><span class="sxs-lookup"><span data-stu-id="eef1f-104">This error occurs when a variable is passed as a [ref](../../csharp/language-reference/keywords/ref.md) or [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) argument in a context in which that variable is readonly.</span></span> <span data-ttu-id="eef1f-105">只读上下文包括 [foreach](../../csharp/language-reference/keywords/foreach-in.md) 迭代变量、 [using](../../csharp/language-reference/keywords/using-statement.md) 变量和 `fixed` 变量。</span><span class="sxs-lookup"><span data-stu-id="eef1f-105">Readonly contexts include [foreach](../../csharp/language-reference/keywords/foreach-in.md) iteration variables, [using](../../csharp/language-reference/keywords/using-statement.md) variables, and `fixed` variables.</span></span> <span data-ttu-id="eef1f-106">要解决此错误，请勿调用在 `foreach`块、 `using` 语句和 `fixed` 语句中采用 `ref` 、 `out` 或 `using` 变量作为 `foreach` 或 `fixed` 参数的函数。</span><span class="sxs-lookup"><span data-stu-id="eef1f-106">To resolve this error, do not call functions that take the `foreach`, `using` or `fixed` variable as a `ref` or `out` parameter in `using` blocks, `foreach` statements, and `fixed` statements.</span></span>  
  
## <a name="example"></a><span data-ttu-id="eef1f-107">示例</span><span class="sxs-lookup"><span data-stu-id="eef1f-107">Example</span></span>  
 <span data-ttu-id="eef1f-108">下面的示例生成 CS1657：</span><span class="sxs-lookup"><span data-stu-id="eef1f-108">The following sample generates CS1657:</span></span>  
  
```csharp  
// CS1657.cs  
using System;  
class C : IDisposable  
{  
    public int i;  
    public void Dispose() {}  
}  
  
class CMain  
{  
    static void f(ref C c)  
    {  
    }  
    static void Main()  
    {  
        using (C c = new C())  
        {  
            f(ref c);  // CS1657  
        }  
    }  
}  
```
