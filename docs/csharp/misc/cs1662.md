 <span data-ttu-id="a6e0c-104">如果匿名方法块的 return 语句具有不可隐式转换为委托返回类型的类型，则会发生此错误。</span><span class="sxs-lookup"><span data-stu-id="a6e0c-104">This error occurs if the anonymous method block's return statement had a type that was not implicitly convertible to the return type of the delegate.</span></span>
