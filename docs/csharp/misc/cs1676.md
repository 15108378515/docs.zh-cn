# <a name="compiler-error-cs1676"></a><span data-ttu-id="269b9-102">编译器错误 CS1676</span><span class="sxs-lookup"><span data-stu-id="269b9-102">Compiler Error CS1676</span></span>
<span data-ttu-id="269b9-103">参数“number”必须使用“keyword”关键字进行声明</span><span class="sxs-lookup"><span data-stu-id="269b9-103">Parameter 'number' must be declared with the 'keyword' keyword</span></span>  
  
 <span data-ttu-id="269b9-104">当匿名方法中的参数类型修饰符不同于你要将方法转换为的委托的声明中所用的参数类型修饰符时，将出现此错误。</span><span class="sxs-lookup"><span data-stu-id="269b9-104">This error occurs when the parameter type modifier in an anonymous method is different from that used in the declaration of the delegate you are casting the method to.</span></span>  
  
 <span data-ttu-id="269b9-105">以下示例生成 CS1676：</span><span class="sxs-lookup"><span data-stu-id="269b9-105">The following sample generates CS1676:</span></span>  
  
```  
// CS1676.cs  
delegate void E(ref int i);  
class Errors   
{  
   static void Main()  
   {  
      E e = delegate(out int i) { };   // CS1676  
      // To resolve, use the following line instead:  
      // E e = delegate(ref int i) { };  
   }  
}  
```
