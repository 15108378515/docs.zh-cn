## <a name="example"></a><span data-ttu-id="71db3-105">示例</span><span class="sxs-lookup"><span data-stu-id="71db3-105">Example</span></span>  
 <span data-ttu-id="71db3-106">下面的示例生成 CS1686。</span><span class="sxs-lookup"><span data-stu-id="71db3-106">The following sample generates CS1686.</span></span>  
  
```  
// CS1686.cs  
// compile with: /unsafe /target:library  
class MyClass  
{  
   public unsafe delegate int * MyDelegate();  
  
   public unsafe int * Test()  
   {  
      int j = 0;  
      MyDelegate d = delegate { return &j; };   // CS1686  
      return &j;   // OK  
   }  
}  
```
