# <a name="compiler-warning-level-3-cs1702"></a>编译器警告（等级 3）CS1702
如果程序集引用“Assembly Name #1”与“Assembly Name #2”一致，则可能需要提供运行时策略  
  
 两个程序集引用具有不同的生成和/或修订号，因此不会自动统一。 你可能需要使用应用程序 .config 文件中的指令提供运行时策略，以强制执行统一。
