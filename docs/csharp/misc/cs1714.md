# <a name="compiler-error-cs1714"></a>编译器错误 CS1714
TypeName1 的基类或接口未能解析或无效  
  
 你将从无法解析（例如，编译器无法找到它）或无效的类或接口实现 TypeName1。 解决方法是确定它是这两种情况中的哪一种，或者更准确地指定类型的位置，或者在基类中修复任何编译器错误。
