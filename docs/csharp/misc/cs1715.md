# <a name="compiler-error-cs1715"></a><span data-ttu-id="8b985-102">编译器错误 CS1715</span><span class="sxs-lookup"><span data-stu-id="8b985-102">Compiler Error CS1715</span></span>

 <span data-ttu-id="8b985-104">此错误与 [Compiler Error CS0508](../../csharp/misc/cs0508.md)相同，区别在于 CS0508 目前仅适用于具有返回类型的方法，而 CS1715 适用于仅具有“类型”而非“返回类型”的属性和索引器。</span><span class="sxs-lookup"><span data-stu-id="8b985-104">This error is the same as [Compiler Error CS0508](../../csharp/misc/cs0508.md), except that CS0508 now only applies to methods that have return types, while CS1715 applies to properties and indexers that only have 'types' instead of 'return types'.</span></span>  
  
## <a name="example"></a><span data-ttu-id="8b985-105">示例</span><span class="sxs-lookup"><span data-stu-id="8b985-105">Example</span></span>  
 <span data-ttu-id="8b985-106">下面的代码生成 CS1715。</span><span class="sxs-lookup"><span data-stu-id="8b985-106">The following code generates CS1715.</span></span>  
  
```  
// CS1715.cs  
abstract public class Base  
{  
    abstract public int myProperty  
    {  
        get;  
        set;  
    }  
}  
  
public class Derived : Base  
{  
    int myField;  
    public override double myProperty  // CS1715  
    // try the following line instead  
    // public override int myProperty  
    {  
        get { return myField; }  
        set { myField;= value; }  
    }  
  
    public static void Main()  
    {  
        Derived d = new Derived();  
        d.myProperty = 5;  
    }  
}  
```
