 <span data-ttu-id="33c09-104">如果编写的表达式涉及作为引用类型（例如类）的泛型类型变量的默认值，则会发生此错误。</span><span class="sxs-lookup"><span data-stu-id="33c09-104">If you write an expression involving the default of a generic type variable that is a reference type (for example, a class), this error will occur.</span></span> <span data-ttu-id="33c09-105">考虑下面的表达式：</span><span class="sxs-lookup"><span data-stu-id="33c09-105">Consider the following expression:</span></span>

 <span data-ttu-id="33c09-106">由于 `T` 是引用类型，因此其默认值是 null，并且因此尝试将 <xref:System.Object.ToString%2A> 方法应用于它会引发 <xref:System.NullReferenceException>。</span><span class="sxs-lookup"><span data-stu-id="33c09-106">Since `T` is a reference type, its default value is null, and so attempting to apply the <xref:System.Object.ToString%2A> method to it will throw a <xref:System.NullReferenceException>.</span></span>  
  
## <a name="example"></a><span data-ttu-id="33c09-107">示例</span><span class="sxs-lookup"><span data-stu-id="33c09-107">Example</span></span>

```csharp  
// CS1720.cs  
using System;  
public class Tester   
{  
    public static void GenericClass<T>(T t1) where T : class   
    {  
        Console.WriteLine(default(T).ToString());  // CS1720  
    }  
    public static void Main() {}  
}  
```
