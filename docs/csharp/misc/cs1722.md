## <a name="example"></a><span data-ttu-id="2f5a3-105">示例</span><span class="sxs-lookup"><span data-stu-id="2f5a3-105">Example</span></span>  
 <span data-ttu-id="2f5a3-106">下面的示例生成 CS1722。</span><span class="sxs-lookup"><span data-stu-id="2f5a3-106">The following sample generates CS1722.</span></span>  
  
```csharp  
// CS1722.cs

public class A {}  
interface I {}  
  
public class MyClass : I, A {}   // CS1722  
public class MyClass2 : A, I {}   // OK  
```
