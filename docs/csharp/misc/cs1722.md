# <a name="compiler-error-cs1722"></a>编译器错误 CS1722
基类“class”必须在任何接口之前  
  
 指定从中继承的类和要实现的接口时，必须首先指定类名。  
  
## <a name="example"></a>示例  
 下面的示例生成 CS1722。  
  
```csharp  
// CS1722.cs  
// compile with: /target:library  
public class A {}  
interface I {}  
  
public class MyClass : I, A {}   // CS1722  
public class MyClass2 : A, I {}   // OK  
```
