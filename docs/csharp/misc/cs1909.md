## <a name="example"></a><span data-ttu-id="8ac37-105">示例</span><span class="sxs-lookup"><span data-stu-id="8ac37-105">Example</span></span>  
 <span data-ttu-id="8ac37-106">以下示例生成 CS1909。</span><span class="sxs-lookup"><span data-stu-id="8ac37-106">The following sample generates CS1909.</span></span>  
  
```csharp  
// CS1909.cs

public interface ISomeInterface  
{  
   void Test1([DefaultParameterValue(new int[] {1, 2})] int[] arr1);   // CS1909

}  
```
