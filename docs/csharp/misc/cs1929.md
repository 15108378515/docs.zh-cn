# <a name="compiler-error-cs1929"></a><span data-ttu-id="ee37d-102">编译器错误 CS1929</span><span class="sxs-lookup"><span data-stu-id="ee37d-102">Compiler Error CS1929</span></span>
<span data-ttu-id="ee37d-103">实例参数：无法从“typeA”转换为“typeB”。</span><span class="sxs-lookup"><span data-stu-id="ee37d-103">Instance argument: cannot convert from 'typeA' to 'typeB'.</span></span>  
  
 <span data-ttu-id="ee37d-104">当从不会扩展的类中调用扩展方法时，则会生成此错误。</span><span class="sxs-lookup"><span data-stu-id="ee37d-104">This error is generated when you try to invoke an extension method from a class that it does not extend.</span></span> <span data-ttu-id="ee37d-105">在此处所示示例中，是为派生的类 `A`而不是为基类 `B`定义扩展方法。</span><span class="sxs-lookup"><span data-stu-id="ee37d-105">In the example shown here, the extension method is defined for the derived class `A`, but not for the base class `B`.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="ee37d-106">更正此错误</span><span class="sxs-lookup"><span data-stu-id="ee37d-106">To correct this error</span></span>  
  
1.  <span data-ttu-id="ee37d-107">在需要调用扩展方法的位置为该类型创建一个新的扩展方法，或者则将调用移到现有方法扩展的类型的对象中。</span><span class="sxs-lookup"><span data-stu-id="ee37d-107">Create a new extension method for the type where you have to invoke it, or else move the call into an object of the type that the existing method extends.</span></span>  
  
## <a name="example"></a><span data-ttu-id="ee37d-108">示例</span><span class="sxs-lookup"><span data-stu-id="ee37d-108">Example</span></span>  
 <span data-ttu-id="ee37d-109">以下代码生成 CS1928 和 CS1929：</span><span class="sxs-lookup"><span data-stu-id="ee37d-109">The following code generates CS1928 and CS1929:</span></span>  
  
```csharp  
// cs1929.cs  
using System.Linq;  
    using System.Collections;  
  
    static class Ext  
    {  
        public static void ExtMethod(this A a)  
        {  
        }  
    }  
  
    class A : B  
    {  
    }  
  
    class B  
    {  
        static void Main()  
        {  
            B b = new B();  
            b.ExtMethod(); // CS1929  
        }  
    }  
```
