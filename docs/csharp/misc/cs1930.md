<span data-ttu-id="a2157-103">已声明范围变量“name”</span><span class="sxs-lookup"><span data-stu-id="a2157-103">The range variable 'name' has already been declared</span></span>  
  
 <span data-ttu-id="a2157-104">查询表达式中的范围变量会一直位于范围内，直到该查询表达式终止。</span><span class="sxs-lookup"><span data-stu-id="a2157-104">The range variable in a query expression is in scope until the query expression terminates.</span></span> <span data-ttu-id="a2157-105">因此，该变量必须具有唯一标识符。</span><span class="sxs-lookup"><span data-stu-id="a2157-105">It must therefore have a unique identifier.</span></span>

## <a name="example"></a><span data-ttu-id="a2157-108">示例</span><span class="sxs-lookup"><span data-stu-id="a2157-108">Example</span></span>  
 <span data-ttu-id="a2157-109">下面的示例生成 CS1930，因为同时对 `num` 子句中的范围变量和 `from` 子句所引入的范围变量使用了标识符 `let` 。</span><span class="sxs-lookup"><span data-stu-id="a2157-109">The following example generates CS1930 because the identifier `num` is used for the range variable in the `from` clause and for the range variable introduced by the `let` clause.</span></span>  
  
```  
// cs1930.cs  
using System.Linq;  
class Program  
{  
    static void Main()  
    {  
        int[] nums = { 0, 1, 2, 3, 4, 5 };  
        var query = from num in nums  
                    let num = 3 // CS1930  
                    select num;   
    }  
}  
```
