 <span data-ttu-id="2dd9b-108">下面的代码生成 CS1931，因为该标识符 `x` 既用作 `Main` 中的本地变量，也用作查询表达式中的范围变量：</span><span class="sxs-lookup"><span data-stu-id="2dd9b-108">The following code generates CS1931 because the identifier `x` is used both as a local variable in `Main` and as the range variable in the query expression:</span></span>
