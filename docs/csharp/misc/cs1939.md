# <a name="compiler-error-cs1939"></a><span data-ttu-id="8ff83-102">编译器错误 CS1939</span><span class="sxs-lookup"><span data-stu-id="8ff83-102">Compiler Error CS1939</span></span>

 <span data-ttu-id="8ff83-104">范围变量是只读变量，在查询表达式中引入，并作为源序列中每个后续元素的标识符。</span><span class="sxs-lookup"><span data-stu-id="8ff83-104">A range variable is a read-only variable that is introduced in a query expression and serves as an identifier for each successive element in a source sequence.</span></span> <span data-ttu-id="8ff83-105">由于它不能以任何方式进行修改，因此通过 `ref` 或 `out`传递它没有任何意义。</span><span class="sxs-lookup"><span data-stu-id="8ff83-105">Because it cannot be modified in any way, there is no point in passing it by `ref` or `out`.</span></span> <span data-ttu-id="8ff83-106">因此，这两种操作都无效。</span><span class="sxs-lookup"><span data-stu-id="8ff83-106">Therefore, both operations are not valid.</span></span>

## <a name="example"></a><span data-ttu-id="8ff83-109">示例</span><span class="sxs-lookup"><span data-stu-id="8ff83-109">Example</span></span>  
 <span data-ttu-id="8ff83-110">下面的示例生成 CS1939：</span><span class="sxs-lookup"><span data-stu-id="8ff83-110">The following example generates CS1939:</span></span>  
  
```csharp  
// cs1939.cs  
using System.Linq;  
class Test  
{  
    public static void F(ref int i)  
    {  
    }  
    public static void Main()  
    {  
        var list = new int[] { 0, 1, 2, 3, 4, 5 };  
        var q = from x in list  
                let k = x  
                select Test.F(ref x); // CS1939  
    }  
}  
```
