# <a name="compiler-error-cs1940"></a><span data-ttu-id="18f85-102">编译器错误 CS1940</span><span class="sxs-lookup"><span data-stu-id="18f85-102">Compiler Error CS1940</span></span>
<span data-ttu-id="18f85-103">找到源类型“type”的多个查询模式实现。</span><span class="sxs-lookup"><span data-stu-id="18f85-103">Multiple implementations of the query pattern were found for source type 'type'.</span></span> <span data-ttu-id="18f85-104">对“method”的调用不明确。</span><span class="sxs-lookup"><span data-stu-id="18f85-104">Ambiguous call to 'method'.</span></span>  
  
 <span data-ttu-id="18f85-105">当定义了查询方法的多个实现，而编译器无法确定最适合的查询时，则会生成此错误。</span><span class="sxs-lookup"><span data-stu-id="18f85-105">This error is generated when multiple implementations of a query method are defined and the compiler cannot disambiguate which one is best to use for the query.</span></span> <span data-ttu-id="18f85-106">在下面的示例中， `Select` 的两个版本具有相同的签名，因为它们都接受同一 `int` 作为输入参数，并且返回值均为 `int` 。</span><span class="sxs-lookup"><span data-stu-id="18f85-106">In the following example, both versions of `Select` have the same signature, because they both accept one `int` as an input parameter and have `int` as a return value.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="18f85-107">更正此错误</span><span class="sxs-lookup"><span data-stu-id="18f85-107">To correct this error</span></span>  
  
1.  <span data-ttu-id="18f85-108">仅为每个方法提供一个实现。</span><span class="sxs-lookup"><span data-stu-id="18f85-108">Provide only one implementation for each method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="18f85-109">示例</span><span class="sxs-lookup"><span data-stu-id="18f85-109">Example</span></span>  
 <span data-ttu-id="18f85-110">以下代码生成 CS1940：</span><span class="sxs-lookup"><span data-stu-id="18f85-110">The following code generates CS1940:</span></span>  
  
```csharp  
// cs1940.cs  
using System; //must include explicitly for types defined in 3.5  
class Test  
{  
    public delegate int Dele(int x);  
    int num = 0;  
    public int Select(Func<int, int> d)  
    {  
        return d(this.num);  
    }  
    public int Select(Dele d) // CS1940  
    {  
        return d(this.num) + 1;  
    }  
    public static void Main()  
    {  
        var q = from x in new Test()  
        select x;  
    }  
}  
```
