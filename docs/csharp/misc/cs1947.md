## <a name="example"></a><span data-ttu-id="abd89-109">示例</span><span class="sxs-lookup"><span data-stu-id="abd89-109">Example</span></span>  
 <span data-ttu-id="abd89-110">以下代码生成 CS1947：</span><span class="sxs-lookup"><span data-stu-id="abd89-110">The following code generates CS1947:</span></span>  
  
```  
// cs1947.cs  
using System.Linq;  
class Test  
{  
    static void Main()  
    {  
        int[] array = new int[] { 1, 2, 3, 4, 5 };  
        var x = from i in array  
                let k = i

        x.ToList();  
    }  
}  
```
