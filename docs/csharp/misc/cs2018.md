# <a name="compiler-error-cs2018"></a>编译器错误 CS2018
无法找到消息文件“cscmsgs.dll”  
  
 未找到包含编译器的错误和警告消息的 .dll 文件。 此文件必须位于与另一个编译器支持文件相同的目录中。
