# <a name="compiler-error-cs2035"></a>编译器错误 CS2035
命令行语法错误： 缺少:\<编号 > 的 compiler_option 选项  
  
 某些编译器选项需要一个值。  
  
## <a name="example"></a>示例  
 下面的示例生成 CS2035。  
  
```csharp  
// CS2035.cs  
// compile with: /baseaddress  
// CS2035 expected  
```
