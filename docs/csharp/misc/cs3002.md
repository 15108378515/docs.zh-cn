# <a name="compiler-warning-level-1-cs3002"></a>编译器警告（等级 1）CS3002
“method”的返回类型不符合 CLS  
  
 A[公共](../../csharp/language-reference/keywords/public.md)，[保护](../../csharp/language-reference/keywords/protected.md)，或`protected internal`方法必须返回的值的类型符合公共语言规范 (CLS)。 有关 CLS 符合性的详细信息，请参阅[编写符合 CLS 的代码](http://msdn.microsoft.com/library/4c705105-69a2-4e5e-b24e-0633bc32c7f3)和[语言独立性和与语言无关的组件](../../../docs/standard/language-independence-and-language-independent-components.md)。  
  
## <a name="example"></a>示例  
 下面的示例生成 CS3002：  
  
```csharp  
// CS3002.cs  
  
[assembly:System.CLSCompliant(true)]  
public class a  
{  
    public ushort bad()   // CS3002, public method  
    {  
        ushort a;  
        a = ushort.MaxValue;  
        return a;  
    }  
  
    private ushort OK()   // OK, private method  
    {  
        ushort a;  
        a = ushort.MaxValue;  
        return a;  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
