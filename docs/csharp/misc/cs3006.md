# <a name="compiler-warning-level-1-cs3006"></a>编译器警告（等级 1）CS3006
仅 ref 或 out，或者仅数组秩不同的重载方法“method”不符合 CLS  
  
 一种方法不会不无法重载基于[ref](../../csharp/language-reference/keywords/ref.md)或[出](../../csharp/language-reference/keywords/out-parameter-modifier.md)参数，仍然遵循公共语言规范 (CLS)。 有关 CLS 符合性的详细信息，请参阅[编写符合 CLS 的代码](http://msdn.microsoft.com/library/4c705105-69a2-4e5e-b24e-0633bc32c7f3)和[语言独立性和与语言无关的组件](../../../docs/standard/language-independence-and-language-independent-components.md)。  
  
## <a name="example"></a>示例  
 以下示例生成 CS3006。 若要解决此警告，注释掉该程序集级别的特性，或删除其中一个方法定义。  
  
```csharp  
// CS3006.cs  
  
using System;  
  
[assembly: CLSCompliant(true)]  
public class MyClass  
{  
    public void f(int i)  
    {  
    }  
  
    public void f(ref int i)   // CS3006  
    {  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
