## <a name="example"></a><span data-ttu-id="28f03-108">示例</span><span class="sxs-lookup"><span data-stu-id="28f03-108">Example</span></span>  
 <span data-ttu-id="28f03-109">下面的示例生成 CS3018。</span><span class="sxs-lookup"><span data-stu-id="28f03-109">The following sample generates CS3018.</span></span>  
  
```  
// CS3018.cs  
// compile with: /target:library  
using System;  
  
[assembly: CLSCompliant(true)]  
[CLSCompliant(false)]  
public class Outer  
{  
   [CLSCompliant(true)]   // CS3018  
   public class Nested {}  
  
   // OK  
   public class Nested2 {}  
  
   [CLSCompliant(false)]  
   public class Nested3 {}  
}  
```
