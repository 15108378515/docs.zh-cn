## <a name="example"></a><span data-ttu-id="575b5-105">示例</span><span class="sxs-lookup"><span data-stu-id="575b5-105">Example</span></span>  
 <span data-ttu-id="575b5-106">下面的示例包含的接口中有一个方法，该方法在其签名中使用不符合 CLS 的类型，致使类型不符合 CLS。</span><span class="sxs-lookup"><span data-stu-id="575b5-106">The following sample contains an interface with a method that uses a non-CLS compliant type in its signature, making the type non-CLS compliant.</span></span>

```csharp  
// compile with: /target:library

   void IMethod(uint i);  
}  
```

 <span data-ttu-id="575b5-108">下面的示例生成 CS3027。</span><span class="sxs-lookup"><span data-stu-id="575b5-108">The following sample generates CS3027.</span></span>
