## <a name="associativity"></a><span data-ttu-id="bd203-254">结合性</span><span class="sxs-lookup"><span data-stu-id="bd203-254">Associativity</span></span>  
 <span data-ttu-id="bd203-255">当表达式中出现两个或两个以上具有相同优先级的运算符时，将根据结合性计算它们。</span><span class="sxs-lookup"><span data-stu-id="bd203-255">When two or more operators that have the same precedence are present in an expression, they are evaluated based on associativity.</span></span> <span data-ttu-id="bd203-256">左结合运算符按从左到右的顺序计算。</span><span class="sxs-lookup"><span data-stu-id="bd203-256">Left-associative operators are evaluated in order from left to right.</span></span> <span data-ttu-id="bd203-257">例如，`x * y / z` 将计算为 `(x * y) / z`。</span><span class="sxs-lookup"><span data-stu-id="bd203-257">For example, `x * y / z` is evaluated as `(x * y) / z`.</span></span> <span data-ttu-id="bd203-258">右结合运算符按从右到左的顺序计算。</span><span class="sxs-lookup"><span data-stu-id="bd203-258">Right-associative operators are evaluated in order from right to left.</span></span> <span data-ttu-id="bd203-259">例如，赋值运算符是右关联的。</span><span class="sxs-lookup"><span data-stu-id="bd203-259">For example, the assignment operator is right associative.</span></span> <span data-ttu-id="bd203-260">如果不是，下面的代码将导致错误。</span><span class="sxs-lookup"><span data-stu-id="bd203-260">If it were not, the following code would result in an error.</span></span>  
  
```csharp

//(a = b) = c;  
```  
  
 <span data-ttu-id="bd203-261">再举一个例子，三元运算符 ([?:](../../../csharp/language-reference/operators/conditional-operator.md)) 是右结合运算符。</span><span class="sxs-lookup"><span data-stu-id="bd203-261">As another example the ternary operator ([?:](../../../csharp/language-reference/operators/conditional-operator.md)) is right associative.</span></span> <span data-ttu-id="bd203-262">大多数的二元运算符是左结合运算符。</span><span class="sxs-lookup"><span data-stu-id="bd203-262">Most binary operators are left associative.</span></span>
