# <a name="coreresponsedata-class"></a>CoreResponseData 类

`CoreResponseData`类表示的 HTTP 标头和响应正文的分析。

## <a name="syntax"></a>语法
  
```csharp
internal class CoreResponseData
```

> [!WARNING]
> 此 API 是内部，方法，但并不是在代码中直接使用。 相反，应使用<xref:System.Diagnostics.DiagnosticSource>连接网络的代码。 请参阅[DiagnosticSource 用户指南](https://github.com/dotnet/corefx/blob/master/src/System.Diagnostics.DiagnosticSource/src/DiagnosticSourceUsersGuide.md)。
> 
> Microsoft 不支持在生产应用程序在任何情况下使用此类。
