# <a name="-equals-entity-sql"></a>=（等于）(Entity SQL)
比较两个表达式是否相等。  
  
## <a name="syntax"></a>语法  
  
```  
expression = expression  
or   
expression == expression  
```  
  
## <a name="arguments"></a>参数  
 `expression`  
 任何有效表达式。 两个表达式都必须具有可隐式转换的数据类型。  
  
## <a name="result-types"></a>结果类型  
 如果左侧表达式等于右侧表达式，则为`true` ；否则为 `false`。  
  
## <a name="remarks"></a>备注  
 == 运算符等效于 =。
