# <a name="sqlclient-for-entity-framework-functions"></a>用于实体框架函数的 SqlClient
用于实体框架的 SQL Server .NET Framework 数据提供程序 (SqlClient) 提供一组执行数学和聚合运算的函数，以及执行 `System.DateTime` 和 `string` 运算的函数。 这些函数位于 `SQLServer` 命名空间中。  
  
 与任何提供程序搭配使用的函数的列表，请参阅[规范函数](../../../../../docs/framework/data/adonet/ef/language-reference/canonical-functions.md)。  
  
 有关如何规范函数映射到 SQL Server 函数的信息，请参阅[概念模型规范 SQL Server 函数映射](../../../../../docs/framework/data/adonet/ef/conceptual-model-canonical-to-sql-server-functions-mapping.md)。
