# Boolean 数据类型
布尔运算符在公共语言运行库 \(CLR\) 中按预期方式工作，但其短路行为不会进行转换。  例如，Visual Basic `AndAlso` 运算符在行为上类似于 `And` 运算符。  C\# `&&` 运算符在行为上类似于 `&` 运算符。  
  
 [!INCLUDE[vbtecdlinq](../../../../../../includes/vbtecdlinq-md.md)] 支持以下运算符。  
  
|Visual Basic|C\#|  
|------------------|---------|  
|[And 运算符](../Topic/And%20Operator%20\(Visual%20Basic\).md)|[& 运算符](../Topic/&%20Operator%20\(C%23%20Reference\).md)|  
|[AndAlso 运算符](../Topic/AndAlso%20Operator%20\(Visual%20Basic\).md)|[&& 运算符](../Topic/&&%20Operator%20\(C%23%20Reference\).md)|  
|[Or 运算符](../Topic/Or%20Operator%20\(Visual%20Basic\).md)|[&#124; 运算符](../Topic/%7C%20Operator%20\(C%23%20Reference\).md)|  
|[OrElse 运算符](../Topic/OrElse%20Operator%20\(Visual%20Basic\).md)|[&#124;&#124; 运算符](../Topic/%7C%7C%20Operator%20\(C%23%20Reference\).md)|  
|[异或运算符](../Topic/Xor%20Operator%20\(Visual%20Basic\).md)|[^ 运算符](../Topic/%5E%20Operator%20\(C%23%20Reference\).md)|
