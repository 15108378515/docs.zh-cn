# <a name="creating-the-object-model"></a>创建对象模型

 如果你使用[!INCLUDE[vs_current_short](../../../../../../includes/vs-current-short-md.md)]，你可以使用[!INCLUDE[vs_ordesigner_long](../../../../../../includes/vs-ordesigner-long-md.md)]来创建对象模型。
