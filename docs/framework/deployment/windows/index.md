# <a name="installing-the-net-framework-on-windows"></a>在 Windows 上安装 .NET Framework

以下文档介绍如何为每个 Windows 版本安装 .NET Framework。

## <a name="version-differences"></a>版本差异

Windows 附带 .NET Framework。 每个 Windows 版本都包含一个 .NET Framework 版本，可能该版本不是所需版本。 如果想要对给定版本的 Windows 使用不同版本的 .NET Framework，则必须安装它。

一般情况下，较高版本的 .NET Framework 将运行为较低 .NET Framework 版本构建的应用程序。 例如，.NET Framework 4.6.2 将运行为 .NET Framework 4.0 构建的应用程序。 此规则有一个例外。 为 .NET Framework 1.0 到 3.5 构建的应用程序可在 .NET Framework 3.5 上运行，但无法在 .NET Framework 4.0 或更高版本上运行。
