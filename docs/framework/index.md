# .NET Framework 4.6 和 4.5
> [!NOTE]
>  此 .NET Framework 内容集包括适用于 .NET Framework 4.5、4.5.1、4.5.2、4.6 和 4.6.1 的信息。 若要下载 .NET Framework，请参阅[安装 .NET Framework](../../docs/framework/install/guide-for-developers.md)。 有关 NET Framework 4.5、[!INCLUDE[net_v46](../../includes/net-v46-md.md)] 及其单点版本的新增功能和更新列表，请参阅[新增功能](../../docs/framework/whats-new/index.md)。 有关受支持的平台列表，请参阅 [.NET Framework 系统要求](../../docs/framework/get-started/system-requirements.md)。  
  
 .NET Framework 是用于为 Windows、Windows Phone、Windows Server 和 Microsoft Azure 构建应用的开发平台。 它包括公共语言运行时 \(CLR\) 和 .NET Framework 类库（包含支持多种技术的类、接口和值类型）。 .NET Framework 提供托管执行环境、简化的开发和部署以及与各种编程语言（包括 Visual Basic 和 Visual C\#）的集成。  
  
> [!NOTE]
>  若要下载 .NET Framework，请参阅[安装 .NET Framework](../../docs/framework/install/guide-for-developers.md)。  
  
 有关面向用户和开发人员的 .NET Framework 的常规介绍，请参阅[入门](../../docs/framework/get-started/index.md)。 有关 .NET Framework 的体系结构和主要功能的介绍，请参阅[概述](../../docs/framework/get-started/overview.md)。  
  
 .NET Framework 的文档包含多种类库参考、概念性概述、分步过程以及有关示例、编译器和命令行工具的信息。 若要查找感兴趣的信息，请参见以下主要主题部分列表。
