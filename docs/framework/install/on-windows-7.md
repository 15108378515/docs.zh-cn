# <a name="install-the-net-framework-on-windows-7-sp1"></a>在 Windows 7 SP1 上安装 .NET Framework

必须安装 .NET Framework，才能在 Windows 7 SP1 上运行应用程序。 可以按照以下说明操作，安装最新的受支持版本。 这些说明因需要安装的 .NET Framework 版本而异。

## <a name="net-framework-462"></a>.NET Framework 4.6.2

可以在 Windows 7 上安装 [.NET Framework 4.6.2](https://go.microsoft.com/fwlink/?linkid=845529&source=dotnetdocs) 或更高版本。

.NET Framework 4.6.2 包含在 [Windows 10 周年更新](https://www.microsoft.com/software-download/windows10)中。

.NET Framework 4.6.2 可用于运行针对 .NET Framework 4.0 或更高版本生成的应用程序。

## <a name="net-framework-35"></a>.NET Framework 3.5

.NET Framework 3.5 包含在 Windows 7 中。

.NET Framework 3.5 可用于运行针对 .NET Framework 1.0-3.5 生成的应用程序。

## <a name="help"></a>帮助

如果无法确定已安装 .NET Framework 的正确版本，可以[联系 Microsoft 获取帮助](mailto:dotnet-install-help@microsoft.com?subject=Help installing .NET Framework)。
