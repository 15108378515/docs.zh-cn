# 网络编程示例
本节包含了和链接到 <xref:System.Net>、 <xref:System.Net.Cache>、 <xref:System.Net.Configuration>、 <xref:System.Net.Mail>、 <xref:System.Net.Mime>、 <xref:System.Net.NetworkInformation>、 <xref:System.Net.PeerToPeer>、 <xref:System.Net.Security>、 <xref:System.Net.Sockets>和相关命名空间使用选件类的可下载的网络编程示例。  
  
## 本节内容  
 [下载进度指示器技术示例](http://go.microsoft.com/fwlink/?LinkID=179556)  
 演示如何显示文件的进度下载。  
  
 [FTP客户端技术示例](http://go.microsoft.com/fwlink/?LinkID=179557)  
 如何显示上载以及向\/从FTP服务器下载文件。  
  
 [HttpListener技术示例](http://go.microsoft.com/fwlink/?LinkID=179558)  
 演示如何处理HTTP请求从应用程序内部使用。  
  
 [HttpListener ASPX宿主应用程序示例](http://go.microsoft.com/fwlink/?LinkID=179560)  
 演示如何使用 <xref:System.Net.HttpListener> 选件类的功能创建路由调用承载的ASP.NET应用程序的HTTP服务器。  
  
 [消息程序技术示例](http://go.microsoft.com/fwlink/?LinkID=179561)  
 演示如何从客户端应用程序发送电子邮件。  
  
 [NetStat工具技术示例](http://go.microsoft.com/fwlink/?LinkID=179562)  
 演示NCLNetStat网络信息工具。  
  
 [网络信息技术示例](http://go.microsoft.com/fwlink/?LinkID=179564)  
 演示如何监视和显示网络信息。  
  
 [Ping客户端技术示例](http://go.microsoft.com/fwlink/?LinkID=179565)  
 演示可以ping远程主机的客户端应用程序。  
  
 [web客户端技术示例](http://go.microsoft.com/fwlink/?LinkID=179566)  
 演示如何执行常见操作，如上载或下载文件或数据。  
  
 [安全流示例](http://go.microsoft.com/fwlink/?LinkID=179567)  
 演示如何使用安全流通信在客户端和服务器之间。  
  
 [IPv6套接字示例](http://go.microsoft.com/fwlink/?LinkID=179568)  
 ，当IPv6启用时，演示如何使用套接字。  
  
 [FTP资源管理器技术示例](http://go.microsoft.com/fwlink/?LinkID=179569)  
 演示如何列出FTP服务器的内容。  
  
 [存储性能技术示例](http://go.microsoft.com/fwlink/?LinkID=179570)  
 在 <xref:System.Net.Sockets.Socket> 选件类演示如何使用提高生成使用异步网络I\/O完成高性能的服务器应用程序。  
  
 [对技术示例](http://go.microsoft.com/fwlink/?LinkID=179571)  
 在 <xref:System.Net.PeerToPeer> 命名空间演示如何使用新选件类注册和公布对等类名称然后解决对等类名称。
