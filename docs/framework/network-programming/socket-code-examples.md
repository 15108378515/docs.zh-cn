# Socket 代码示例
下面的代码示例演示如何使用 <xref:System.Net.Sockets.Socket> 选件类作为客户端连接到远程网络服务和作为侦听来自远程客户端的连接的服务器。  
  
## 本节内容  
 [同步客户端套接字示例](../../../docs/framework/network-programming/synchronous-client-socket-example.md)  
 演示如何实现连接到服务器并显示从服务器返回的数据的同步 <xref:System.Net.Sockets.Socket> 客户端。  
  
 [同步服务器套接字示例](../../../docs/framework/network-programming/synchronous-server-socket-example.md)  
 演示如何实现接受来自客户端的连接，然后echo返回从客户端收到的数据的同步 <xref:System.Net.Sockets.Socket> 服务器。  
  
 [异步客户端套接字示例](../../../docs/framework/network-programming/asynchronous-client-socket-example.md)  
 演示如何实现连接到服务器并显示从服务器返回的数据的异步 <xref:System.Net.Sockets.Socket> 客户端。  
  
 [异步服务器套接字示例](../../../docs/framework/network-programming/asynchronous-server-socket-example.md)  
 演示如何实现接受来自客户端的连接，然后echo返回从客户端接收数据的异步 <xref:System.Net.Sockets.Socket> 服务器。
