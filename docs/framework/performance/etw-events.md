# .NET Framework 中的 ETW 事件
Windows 事件跟踪 \(ETW\) 是 Windows 操作系统提供的高性能、低开销的可伸缩跟踪系统。  它对 .NET Framework 所提供的分析和调试支持提供了补充，可用于排除各种方案的问题。  有关 ETW 的常规信息，请参见 MSDN Library上的[提高调整使用 ETW 的调试和性能](http://go.microsoft.com/fwlink/?LinkID=161142) 。  
  
 在 .NET Framework 中，ETW 事件跟踪适用于公共语言运行时 \(CLR\)、[任务并行库](../../../docs/standard/parallel-programming/task-parallel-library-tpl.md)和[Parallel LINQ \(PLINQ\)](../../../docs/standard/parallel-programming/parallel-linq-plinq.md)。
