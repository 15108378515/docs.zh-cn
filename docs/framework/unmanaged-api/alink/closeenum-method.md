# <a name="closeenum-method"></a>CloseEnum 方法
关闭指定的枚举并释放关联的资源。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT CloseEnum(  
    HALINKENUM hEnum  
) PURE;  
```  
  
#### <a name="parameters"></a>参数  
 `hEnum`  
 枚举即将关闭句柄。  
  
## <a name="return-value"></a>返回值  
 如果该方法成功，则返回，则为 S_OK。  
  
## <a name="requirements"></a>要求  
 需要 alink.h
