# <a name="getassemblyrefhash-method"></a>GetAssemblyRefHash 方法
检索给定的程序集哈希 blob。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT GetAssemblyRefHash(  
    mdToken FileToken,  
    const void** ppvHash,  
    DWORD* pcbHash  
) PURE;  
```  
  
#### <a name="parameters"></a>参数  
 `FileToken`  
 哈希将引用的程序集的 ID。  
  
 `ppvHash`  
 接收生成的哈希 blob。  
  
 `pcbHash`  
 接收大小，以字节为单位的哈希 blob。  
  
## <a name="return-value"></a>返回值  
 如果该方法成功，则返回，则为 S_OK。  
  
## <a name="requirements"></a>要求  
 需要 alink.h
