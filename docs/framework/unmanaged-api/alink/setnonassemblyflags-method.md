# <a name="setnonassemblyflags-method"></a>SetNonAssemblyFlags 方法
设置不是程序集特定的标志。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT SetNonAssemblyFlags(  
    AssemblyFlags afFlags  
) PURE;  
```  
  
#### <a name="parameters"></a>参数  
 `afFlags`  
 ALink 标志。  
  
## <a name="return-value"></a>返回值  
 如果该方法成功，则返回，则为 S_OK。  
  
## <a name="requirements"></a>惠?  
 需要 alink.h
