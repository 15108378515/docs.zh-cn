# <a name="cordebugsteprange-structure"></a>COR_DEBUG_STEP_RANGE 结构
包含代码区域的偏离量信息。  
  
 此结构可由[icordebugstepper:: Steprange](../../../../docs/framework/unmanaged-api/debugging/icordebugstepper-steprange-method.md)方法。  
  
## <a name="syntax"></a>语法  
  
```  
typedef struct {  
    ULONG32 startOffset;  
    ULONG32 endOffset;  
} COR_DEBUG_STEP_RANGE;  
```  
  
## <a name="members"></a>成员  
  
|成员|描述|  
|------------|-----------------|  
|`startOffset`|范围的开始处的偏移量。|  
|`endOffset`|偏移量范围的末尾。|
