# <a name="icordebuginitialize-method"></a>ICorDebug::Initialize 方法
初始化 `ICorDebug` 对象。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT Initialize ();  
```  
  
## <a name="remarks"></a>备注  
 调试器必须调用`Initialize`在创建时间来初始化调试服务。 必须对任何其他方法之前调用此方法`ICorDebug`调用。
