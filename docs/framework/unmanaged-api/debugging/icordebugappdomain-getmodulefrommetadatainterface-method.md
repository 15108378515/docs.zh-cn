# <a name="icordebugappdomaingetmodulefrommetadatainterface-method"></a>ICorDebugAppDomain::GetModuleFromMetaDataInterface 方法
获取对应于给定的元数据接口的模块。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT GetModuleFromMetaDataInterface (  
    [in] IUnknown           *pIMetaData,  
    [out] ICorDebugModule  **ppModule  
);  
```  
  
#### <a name="parameters"></a>参数  
 `pIMetaData`  
 [in]指向一个对象，它之一[元数据接口](../../../../docs/framework/unmanaged-api/metadata/metadata-interfaces.md)。  
  
 `ppModule`  
 [out]指向一个到给定的元数据接口表示相对应的模块的 icor 调试模块对象的地址的指针。
