# <a name="icordebugappdomaingetprocess-method"></a>ICorDebugAppDomain::GetProcess 方法
获取包含应用程序域的进程。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT GetProcess (  
    [out] ICorDebugProcess   **ppProcess  
);  
```  
  
#### <a name="parameters"></a>参数  
 `ppProcess`  
 [out]指向 ICorDebugProcess 对象表示进程的地址的指针。
