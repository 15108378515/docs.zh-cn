# <a name="icordebugappdomain4-interface"></a>ICorDebugAppDomain4 接口
合理扩展 ICorDebugAppDomain 接口以从 COM 可调用包装器获取托管的对象。  
  
## <a name="method"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[GetObjectForCCW 方法](../../../../docs/framework/unmanaged-api/debugging/icordebugappdomain4-getobjectforccw-method.md)|从 COM 可调用包装器 (CCW) 指针获取托管对象。|
