# <a name="icordebugblockingobjectenum-interface"></a>ICorDebugBlockingObjectEnum 接口
有关的列表中提供的枚举器[CorDebugBlockingObject](../../../../docs/framework/unmanaged-api/debugging/cordebugblockingobject-structure.md)结构。 此接口是 ICorDebugEnum 接口的子类。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[Next 方法](../../../../docs/framework/unmanaged-api/debugging/icordebugblockingobjectenum-next-method.md)|枚举的列表[CorDebugBlockingObject](../../../../docs/framework/unmanaged-api/debugging/cordebugblockingobject-structure.md)结构。|  
  
## <a name="remarks"></a>备注  
 每个 `CorDebugBlockingObject` 结构均表示一个阻塞线程的对象。  
  
> [!NOTE]
>  此接口不支持跨计算机或跨进程远程调用。
