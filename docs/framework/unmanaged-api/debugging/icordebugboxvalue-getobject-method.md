# <a name="icordebugboxvaluegetobject-method"></a>ICorDebugBoxValue::GetObject 方法
获取已装箱的值。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT GetObject (  
    [out] ICorDebugObjectValue **ppObject  
);  
```  
  
#### <a name="parameters"></a>参数  
 `ppObject`  
 [out]指向一个 ICorDebugObjectValue 对象，表示装箱的值的地址的指针。
