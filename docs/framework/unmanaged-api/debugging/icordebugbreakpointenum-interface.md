# <a name="icordebugbreakpointenum-interface1"></a>ICorDebugBreakpointEnum 接口 1
实现 ICorDebugEnum 方法，并枚举 ICorDebugBreakpoint 数组。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[Next 方法](../../../../docs/framework/unmanaged-api/debugging/icordebugbreakpointenum-next-method.md)|获取指定的数目的`ICorDebugBreakpoint`枚举，从当前位置开始中的实例。|  
  
## <a name="remarks"></a>备注  
  
> [!NOTE]
>  此接口不支持跨计算机或跨进程远程调用。
