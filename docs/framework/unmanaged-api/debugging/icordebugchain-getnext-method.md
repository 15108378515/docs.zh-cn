# <a name="icordebugchaingetnext-method"></a>ICorDebugChain::GetNext 方法
获取线程的下一步帧链。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT GetNext (  
    [out] ICorDebugChain     **ppChain  
);  
```  
  
#### <a name="parameters"></a>参数  
 `ppChain`  
 [out]指向一个表示下一个线程的帧链 ICorDebugChain 对象的地址的指针。 如果此链是最后一个链，`ppChain`为 null。
