# <a name="icordebugchaingetprevious-method"></a>ICorDebugChain::GetPrevious 方法
获取线程的上一帧链。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT GetPrevious (  
    [out] ICorDebugChain     **ppChain  
);  
```  
  
#### <a name="parameters"></a>参数  
 `ppChain`  
 [out]指向一个表示前一个为此线程的帧链 ICorDebugChain 对象的地址的指针。 如果此证书链的第一个链，`ppChain`为 null。
