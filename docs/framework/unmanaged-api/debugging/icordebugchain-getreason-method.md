# <a name="icordebugchaingetreason-method"></a>ICorDebugChain::GetReason 方法
获取此调用链的生成的原因。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT GetReason (  
    [out] CorDebugChainReason *pReason  
);  
```  
  
#### <a name="parameters"></a>参数  
 `pReason`  
 [out]指向 CorDebugChainReason 枚举，该值指示此调用链的生成的原因的值 （按位组合） 的指针。
