# <a name="icordebugclassgetmodule-method"></a>ICorDebugClass::GetModule 方法
获取定义此类的模块。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT GetModule (  
    [out] ICorDebugModule    **pModule  
);  
```  
  
#### <a name="parameters"></a>参数  
 `pModule`  
 [out]指向一个表示在其中定义此类模块的 icor 调试模块对象的地址的指针。
