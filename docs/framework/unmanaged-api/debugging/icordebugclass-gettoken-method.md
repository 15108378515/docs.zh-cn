# <a name="icordebugclassgettoken-method"></a>ICorDebugClass::GetToken 方法
获取`TypeDef`引用此类定义的元数据标记。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT GetToken (  
    [out] mdTypeDef          *pTypeDef  
);  
```  
  
#### <a name="parameters"></a>参数  
 `pTypeDef`  
 [out]指向的指针`mdTypeDef`引用此类定义的令牌。
