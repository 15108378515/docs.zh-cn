# <a name="icordebugcodegetaddress-method"></a>ICorDebugCode::GetAddress 方法
此"icor 调试代码"接口表示的代码段中获取的相对虚拟地址 (RVA)。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT GetAddress (  
    [out] CORDB_ADDRESS *pStart  
);  
```  
  
#### <a name="parameters"></a>参数  
 `pStart`  
 [out]代码段的 RVA 指向的指针。
