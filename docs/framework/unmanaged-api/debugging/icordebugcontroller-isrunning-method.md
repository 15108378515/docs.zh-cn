# <a name="icordebugcontrollerisrunning-method"></a>ICorDebugController::IsRunning 方法
获取一个值，该值指示是否在进程中的线程当前自由地运行。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT IsRunning (  
    [out] BOOL *pbRunning  
);  
```  
  
#### <a name="parameters"></a>参数  
 `pbRunning`  
 [out]指向一个值，是`true`如果自由; 否则为运行进程中的线程`false`。
