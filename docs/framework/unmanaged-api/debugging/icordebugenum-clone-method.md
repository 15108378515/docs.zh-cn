# <a name="icordebugenumclone-method"></a>ICorDebugEnum::Clone 方法
创建此 ICorDebugEnum 对象的副本。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT Clone (  
    [out] ICorDebugEnum **ppEnum  
);  
```  
  
#### <a name="parameters"></a>参数  
 `ppEnum`  
 [out]指向的地址的指针`ICorDebugEnum`对象，它是一份`ICorDebugEnum`对象。
