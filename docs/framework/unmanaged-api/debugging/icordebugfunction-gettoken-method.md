# <a name="icordebugfunctiongettoken-method"></a>ICorDebugFunction::GetToken 方法
获取此函数的元数据令牌。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT GetToken (  
    [out] mdMethodDef *pMethodDef  
);  
```  
  
#### <a name="parameters"></a>参数  
 `pMethodDef`  
 [out]指向的指针`mdMethodDef`引用此函数的元数据的令牌。
