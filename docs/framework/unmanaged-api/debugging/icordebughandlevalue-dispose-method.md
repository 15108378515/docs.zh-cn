# <a name="icordebughandlevaluedispose-method"></a>ICorDebugHandleValue::Dispose 方法
释放此 ICorDebugHandleValue 对象而不显式释放的接口指针引用的句柄。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT Dispose ();  
```
