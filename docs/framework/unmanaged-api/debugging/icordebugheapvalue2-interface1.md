# <a name="icordebugheapvalue2-interface1"></a>ICorDebugHeapValue2 接口 1
ICorDebugHeapValue 提供支持公共语言运行时 (CLR) 处理的扩展。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[CreateHandle 方法](../../../../docs/framework/unmanaged-api/debugging/icordebugheapvalue2-createhandle-method.md)|此创建指定类型的句柄`ICorDebugHeapValue2`对象。|  
  
## <a name="remarks"></a>备注  
  
> [!NOTE]
>  此接口不支持跨计算机或跨进程远程调用。
