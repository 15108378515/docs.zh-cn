# <a name="icordebugmodulegetprocess-method"></a>ICorDebugModule::GetProcess 方法
获取此模块包含进程。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT GetProcess (  
    [out] ICorDebugProcess **ppProcess  
);  
```  
  
#### <a name="parameters"></a>参数  
 `ppProcess`  
 [out]指向一个 ICorDebugProcess 对象，表示包含此模块的过程的地址的指针。
