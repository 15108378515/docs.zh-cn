# <a name="icordebugprocessenablelogmessages-method"></a>ICorDebugProcess::EnableLogMessages 方法
启用和禁用的到调试器的日志消息传输。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT EnableLogMessages([in]BOOL fOnOff);  
```  
  
#### <a name="parameters"></a>参数  
 `fOnOff`  
 [in]`true`启用的日志消息; 传输`false`禁用传输。  
  
## <a name="remarks"></a>备注  
 此方法无效后，才[icordebugmanagedcallback:: Createprocess](../../../../docs/framework/unmanaged-api/debugging/icordebugmanagedcallback-createprocess-method.md)回调时发生。
