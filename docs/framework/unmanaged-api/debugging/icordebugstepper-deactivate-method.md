# <a name="icordebugstepperdeactivate-method"></a>ICorDebugStepper::Deactivate 方法
导致此 ICorDebugStepper 取消它收到的最后一个步骤命令。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT Deactivate ();  
```  
  
## <a name="remarks"></a>备注  
 已取消在最近收到的步骤命令后，可以发出一个新的单步执行命令。
