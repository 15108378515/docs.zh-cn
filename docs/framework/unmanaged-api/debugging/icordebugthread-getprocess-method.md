# <a name="icordebugthreadgetprocess-method"></a>ICorDebugThread::GetProcess 方法
获取此 ICorDebugThread 构成一部分的进程的接口指针。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT GetProcess (  
    [out] ICorDebugProcess   **ppProcess  
);  
```  
  
#### <a name="parameters"></a>参数  
 `ppProcess`  
 [out]ICorDebugProcess 接口对象表示进程的地址指针。
