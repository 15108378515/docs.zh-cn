# <a name="icordebugtypegetrank-method"></a>ICorDebugType::GetRank 方法
获取一个数组类型中的维度数。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT GetRank (  
    [out] ULONG32   *pnRank  
);  
```  
  
#### <a name="parameters"></a>参数  
 `pnRank`  
 [out]指向的维度数的指针。
