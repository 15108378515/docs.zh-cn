# <a name="icorpublishappdomaingetid-method"></a>ICorPublishAppDomain::GetID 方法
获取此唯一标识符[ICorPublishAppDomain](../../../../docs/framework/unmanaged-api/debugging/icorpublishappdomain-interface.md)。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT GetID (  
    [out] ULONG32   *puId  
);  
```  
  
#### <a name="parameters"></a>参数  
 `puId`  
 [out]指向应用程序域的标识符的指针。  
  
## <a name="remarks"></a>备注  
 仅在包含进程的作用域中的标识符是唯一的。
