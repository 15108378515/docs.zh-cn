# <a name="isymunmanageddispose-interface"></a>ISymUnmanagedDispose 接口
释放非托管资源。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[Destroy 方法](../../../../docs/framework/unmanaged-api/diagnostics/isymunmanageddispose-destroy-method.md)|导致要释放所有内部引用，并在任何后续方法调用返回失败的基础对象。|  
  
## <a name="requirements"></a>要求  
 **标头：** CorSym.idl、 CorSym.h
