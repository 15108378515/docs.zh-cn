# <a name="iassemblyenum-interface"></a>IAssemblyEnum 接口
表示为数组的枚举`IAssemblyName`对象。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[Clone 方法](../../../../docs/framework/unmanaged-api/fusion/iassemblyenum-clone-method.md)|创建此的浅表副本`IAssemblyEnum`对象。|  
|[GetNextAssembly 方法](../../../../docs/framework/unmanaged-api/fusion/iassemblyenum-getnextassembly-method.md)|获取一个指针指向下一步`IAssemblyName`包含在此`IAssemblyEnum`对象。|  
|[Reset 方法](../../../../docs/framework/unmanaged-api/fusion/iassemblyenum-reset-method.md)|重置此`IAssemblyEnum`到它的起始位置的对象。|
