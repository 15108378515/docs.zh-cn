# <a name="corgcthreadstatstypes-enumeration"></a>COR_GC_THREAD_STATS_TYPES 枚举
指示线程垃圾回收统计信息。  
  
## <a name="syntax"></a>语法  
  
```  
typedef enum {  
    COR_GC_THREAD_HAS_PROMOTED_BYTES  = 0x00000001  
} COR_GC_THREAD_STATS_TYPES;  
```  
  
## <a name="members"></a>成员  
  
|成员|描述|  
|------------|-----------------|  
|`COR_GC_THREAD_HAS_PROMOTED_BYTES`|线程有在最新的垃圾回收提升的字节。|
