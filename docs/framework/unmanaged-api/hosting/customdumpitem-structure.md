# <a name="customdumpitem-structure"></a>CustomDumpItem 结构
描述要添加到自定义转储错误报告中的项。  
  
## <a name="syntax"></a>语法  
  
```  
struct {  
    ECustomDumpItemKind itemKind;   
    union {  
        UINT_PTR pReserved;  
    }  
} CustomDumpItem;  
```  
  
## <a name="members"></a>成员  
  
|成员|描述|  
|------------|-----------------|  
|`itemKind`|[ECustomDumpItemKind](../../../../docs/framework/unmanaged-api/hosting/ecustomdumpitemkind-enumeration.md)值，该值指示要添加项的种类。|  
|`pReserved`|当前未使用。 任何项添加到联合必须不大于指针大小。 如果`struct`是必需的必须单独分配，并指向它。|  
  
## <a name="remarks"></a>备注  
 [Iclrerrorreportingmanager:: Begincustomdump](../../../../docs/framework/unmanaged-api/hosting/iclrerrorreportingmanager-begincustomdump-method.md)采用类型的参数`CustomDumpItem`。
