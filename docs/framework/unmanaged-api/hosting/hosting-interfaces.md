# <a name="hosting-interfaces"></a>承载接口
本部分描述的接口以及非托管主机可用于将公共语言运行时 (CLR) 集成到其应用程序。  
  
 .NET Framework 2.0 版承载接口取代了.NET Framework 版本 1.0 和 1.1 接口。 有两个集的接口之间的重要差异。 混合使用这些接口可能会导致意外的行为，不建议使用。  
  
 .NET Framework 版本 3.0 和 3.5 使用.NET Framework 2.0 版承载接口，并且并未引入任何宿主功能。  
  
 [!INCLUDE[net_v40_long](../../../../includes/net-v40-long-md.md)]承载接口取代.NET Framework 2.0 接口。
