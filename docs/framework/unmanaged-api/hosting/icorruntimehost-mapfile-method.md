# <a name="icorruntimehostmapfile-method"></a>ICorRuntimeHost::MapFile 方法
将指定的文件映射到内存中。 此方法已过时。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT MapFile(  
    [in]  HANDLE    hFile,  
    [out] HMODULE*  hMapAddress  
);  
```  
  
#### <a name="parameters"></a>参数  
 `hFile`  
 [in]要映射的文件句柄。  
  
 `hMapAddress`  
 [out]从此处开始映射文件起始内存地址。
