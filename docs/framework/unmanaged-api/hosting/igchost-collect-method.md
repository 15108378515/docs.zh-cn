# <a name="igchostcollect-method"></a>IGCHost::Collect 方法
强制回收进行回收对给定的代，而无论当前的垃圾回收的状态。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT Collect (  
    [in] LONG Generation  
);  
```  
  
#### <a name="parameters"></a>参数  
 `Generation`  
 [in]要对其执行垃圾回收生成。 值-1 指示所有代上将都进行垃圾回收。
