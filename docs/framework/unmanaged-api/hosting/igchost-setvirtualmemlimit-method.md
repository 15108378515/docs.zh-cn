# <a name="igchostsetvirtualmemlimit-method"></a>IGCHost::SetVirtualMemLimit 方法
设置运行时的虚拟内存的最大大小。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT SetVirtualMemLimit (  
    [in] SIZE_T sztMaxVirtualMemMB  
);  
```  
  
#### <a name="parameters"></a>参数  
 `sztMaxVirtualMemMB`  
 [in]最大的大小，以兆字节为单位，运行时的虚拟内存。  
  
## <a name="remarks"></a>备注  
 可以动态更改运行时的虚拟内存的最大大小。
