|方法|描述|  
|------------|-----------------|  
|[RequestVirtualMemLimit 方法](../../../../docs/framework/unmanaged-api/hosting/igchostcontrol-requestvirtualmemlimit-method.md)|请求主机后，若要更改的虚拟内存限制。|
