# <a name="iobjecthandleunwrap-method"></a>IObjectHandle::Unwrap 方法
解包中间接寻址的封送按值对象。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT Unwrap (  
    [out, retval] VARIANT *ppv  
);  
```  
  
#### <a name="parameters"></a>参数  
 `ppv`  
 [out]指向要为解包的对象的指针。
