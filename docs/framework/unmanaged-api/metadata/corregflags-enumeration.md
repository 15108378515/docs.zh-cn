# <a name="corregflags-enumeration"></a>CorRegFlags 枚举
提供在安装模块或复合图像时用于注册的标志值。  
  
## <a name="syntax"></a>语法  
  
```  
typedef enum   
{  
    regNoCopy  = 0x00000001,  
    regConfig  = 0x00000002,  
    regHasRefs = 0x00000004  
} CorRegFlags;  
```  
  
## <a name="members"></a>成员  
  
|成员|描述|  
|------------|-----------------|  
|`regNoCopy`|指定不应将文件复制到目标。|  
|`regConfig`|指定模块或复合是一种配置。|  
|`regHasRefs`|指定该模块或复合具有类引用。|
