# <a name="iceegenallocatemethodbuffer-method"></a>ICeeGen::AllocateMethodBuffer 方法
创建方法，指定大小的缓冲区并获取该方法的相对虚拟地址。  
  
 此方法已过时，不应使用。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT AllocateMethodBuffer (   
    [in]  ULONG    cchBuffer,   
    [out] UCHAR    **lpBuffer,  
    [out] ULONG    *RVA  
);  
```  
  
#### <a name="parameters"></a>参数  
 `cchBuffer`  
 [in]要创建的缓冲区长度。  
  
 `lpBuffer`  
 [out]返回的缓冲区。  
  
 `RVA`  
 [out]相对虚拟地址的方法。
