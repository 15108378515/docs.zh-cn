# <a name="imetadataemitsavetomemory-method"></a>IMetaDataEmit::SaveToMemory 方法
将所有元数据保存到指定的内存区域的当前作用域中。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT SaveToMemory (   
    [out]  void        *pbData,   
    [in]   ULONG       cbData   
);  
```  
  
#### <a name="parameters"></a>参数  
 `pbData`  
 [out]从此处开始写入元数据地址。  
  
 `cbData`  
 [in]以字节为单位分配的内存大小。
