# <a name="imetadataemit2savedelta-method"></a>IMetaDataEmit2::SaveDelta 方法
将更改从当前的编辑并继续会话保存到指定的文件。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT SaveDelta (  
    [in] LPCWSTR     szFile,   
    [in] DWORD       dwSaveFlags  
);  
```  
  
#### <a name="parameters"></a>参数  
 `szFile`  
 [in]要保存在其下的文件名称更改。  
  
 `dwSaveFlags`  
 [in]保留。 必须为零。
