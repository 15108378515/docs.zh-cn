# <a name="imetadataemit2savedeltatostream-method"></a>IMetaDataEmit2::SaveDeltaToStream 方法
将更改从当前的编辑并继续会话保存到指定的流。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT SaveDeltaToStream (  
    [in] IStream     *pIStream,   
    [in] DWORD       dwSaveFlags  
);  
```  
  
#### <a name="parameters"></a>参数  
 `pIStream`  
 [in]指向要将更改保存到的可写流的接口指针。  
  
 `dwSaveFlags`  
 [in]保留。 此值必须为零。
