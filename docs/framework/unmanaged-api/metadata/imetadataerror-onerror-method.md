# <a name="imetadataerroronerror-method"></a>IMetaDataError::OnError 方法
提供在元数据合并过程中发生的错误的通知。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT OnError (  
    [in] HRESULT   hrError,   
    [in] mdToken   token  
);  
```  
  
#### <a name="parameters"></a>参数  
 `hrError`  
 [in]返回到调用方法的 HRESULT 错误值。  
  
 `token`  
 [in]出现错误时合并的代码对象的元数据标记。
