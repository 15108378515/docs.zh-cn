# <a name="imetadatafilteristokenmarked-method"></a>IMetaDataFilter::IsTokenMarked 方法
获取一个值，该值指定元数据标记是否已标记为已处理。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT IsTokenMarked (  
    [in]  mdToken  tk,   
    [out] BOOL     *pIsMarked  
);  
```  
  
#### <a name="parameters"></a>参数  
 `tk`  
 [in]要检查处理标记的标记。  
  
 `pIsMarked`  
 [out]一个值，是`true`如果`tk`处理; 否则为`false`。
