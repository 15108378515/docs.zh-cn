# <a name="imetadatafiltermarktoken-method"></a>IMetaDataFilter::MarkToken 方法
设置一个值，该值指定元数据标记已被处理。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT MarkToken (  
    [in] mdToken   tk  
);  
```  
  
#### <a name="parameters"></a>参数  
 `tk`  
 [in]要标记为已处理的标记。
