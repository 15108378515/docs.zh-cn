# <a name="imetadataimportfindtyperef-method"></a>IMetaDataImport::FindTypeRef 方法
获取一个指向 TypeRef 标记，表示<xref:System.Type>中指定的作用域并具有指定的名称的引用。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT FindTypeRef (  
   [in] mdToken        tkResolutionScope,  
   [in]  LPCWSTR       szName,  
   [out] mdTypeRef     *ptr  
);  
```  
  
#### <a name="parameters"></a>参数  
 `tkResolutionScope`  
 [in]指定模块、 程序集或类型的 ModuleRef、 AssemblyRef 或 TypeRef 令牌分别在类型引用该定义。  
  
 `szName`  
 [in]要搜索的类型引用的名称。  
  
 `ptr`  
 [out]指向匹配的 TypeRef 标记的指针。
