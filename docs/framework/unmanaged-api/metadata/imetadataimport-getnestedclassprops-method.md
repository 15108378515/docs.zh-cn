# <a name="imetadataimportgetnestedclassprops-method"></a>IMetaDataImport::GetNestedClassProps 方法
获取的 TypeDef 标记的父<xref:System.Type>指定嵌套类型。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT GetNestedClassProps (  
   [in]   mdTypeDef      tdNestedClass,  
   [out]  mdTypeDef      *ptdEnclosingClass  
);  
```  
  
#### <a name="parameters"></a>参数  
 `tdNestedClass`  
 [in]一个 TypeDef 令牌表示<xref:System.Type>要返回的父类的令牌。  
  
 `ptdEnclosingClass`  
 [out]TypeDef 标记的指针<xref:System.Type>，`tdNestedClass`嵌套在。
