# <a name="imetadataimportgetuserstring-method"></a>IMetaDataImport::GetUserString 方法
获取指定元数据标记所表示的文字字符串。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT GetUserString (  
   [in]   mdString    stk,  
   [out]  LPWSTR      szString,  
   [in]   ULONG       cchString,  
   [out]  ULONG       *pchString  
);  
```  
  
#### <a name="parameters"></a>参数  
 `stk`  
 [in]要返回的关联的字符串的字符串标记。  
  
 `szString`  
 [out]请求的字符串的副本。  
  
 `cchString`  
 [in]以宽字符为单位的所请求的最大大小`szString`。  
  
 `pchString`  
 [out]在返回的宽字符的大小`szString`。
