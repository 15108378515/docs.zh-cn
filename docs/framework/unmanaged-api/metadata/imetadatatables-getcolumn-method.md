# <a name="imetadatatablesgetcolumn-method"></a>IMetaDataTables::GetColumn 方法
获取一个指向包含的指定的列和行中给定的表的单元格的值。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT GetColumn (   
    [in]  ULONG   ixTbl,  
    [in]  ULONG   ixCol,  
    [in]  ULONG   rid,  
    [out] ULONG   *pVal  
);  
```  
  
#### <a name="parameters"></a>参数  
 `ixTbl`  
 [in]表的索引。  
  
 `ixCol`  
 [in]表中列的索引。  
  
 `rid`  
 [in]表中的行的索引。  
  
 `pVal`  
 [out]指向单元格中的值的指针。
