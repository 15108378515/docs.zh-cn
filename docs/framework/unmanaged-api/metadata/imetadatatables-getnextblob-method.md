# <a name="imetadatatablesgetnextblob-method"></a>IMetaDataTables::GetNextBlob 方法
获取表中的下一步二进制大型对象 (BLOB) 的索引。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT GetNextBlob (  
    [in]  ULONG   ixBlob,  
    [out] ULONG   *pNext  
);  
```  
  
#### <a name="parameters"></a>参数  
 `ixBlob`  
 [in]返回的 Blob 列的索引。  
  
 `pNext`  
 [out]指向下一个 BLOB 的索引的指针。
