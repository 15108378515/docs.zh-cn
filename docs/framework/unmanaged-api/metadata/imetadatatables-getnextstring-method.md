# <a name="imetadatatablesgetnextstring-method"></a>IMetaDataTables::GetNextString 方法
获取当前的表列中的下一步的字符串的索引。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT GetNextString (   
    [in]  ULONG   ixString,  
    [out] ULONG   *pNext  
);  
```  
  
#### <a name="parameters"></a>参数  
 `ixString`  
 [in]字符串表列中的索引值。  
  
 `pNext`  
 [out]指向列中的下一步字符串的索引的指针。
