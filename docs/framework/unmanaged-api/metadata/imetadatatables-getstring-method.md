# <a name="imetadatatablesgetstring-method"></a>IMetaDataTables::GetString 方法
获取当前引用作用域中从表的列的指定索引处的字符串。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT GetString (   
    [in]  ULONG       ixString,  
    [out] const char  **ppString  
);  
```  
  
#### <a name="parameters"></a>参数  
 `ixString`  
 [in]若要开始搜索下一个值处的索引。  
  
 `ppString`  
 [out]指向返回的字符串值的指针指向的指针。
