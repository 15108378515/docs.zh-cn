# <a name="imetadatatablesgetstringheapsize-method"></a>IMetaDataTables::GetStringHeapSize 方法
获取用字节表示，字符串堆的大小。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT GetStringHeapSize (  
    [out] ULONG   *pcbStrings  
);  
```  
  
#### <a name="parameters"></a>参数  
 `pcbStrings`  
 [out]指向以字节为单位，在字符串堆的大小的指针。
