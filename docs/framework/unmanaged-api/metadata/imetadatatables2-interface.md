# <a name="imetadatatables2-interface"></a>IMetaDataTables2 接口
扩展[IMetaDataTables](../../../../docs/framework/unmanaged-api/metadata/imetadatatables-interface.md)包括用于处理元数据流的方法。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[GetMetaDataStorage 方法](../../../../docs/framework/unmanaged-api/metadata/imetadatatables2-getmetadatastorage-method.md)|获取的大小和存储在指定的节中的元数据的内容。|  
|[GetMetaDataStreamInfo 方法](../../../../docs/framework/unmanaged-api/metadata/imetadatatables2-getmetadatastreaminfo-method.md)|获取名称、 大小和元数据流的指定索引处的内容。|
