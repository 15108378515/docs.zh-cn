# <a name="corprfcodeinfo-structure"></a>COR_PRF_CODE_INFO 结构
表示存储在内存中的一个本机代码连续块。  
  
## <a name="syntax"></a>语法  
  
```  
typedef struct _COR_PRF_CODE_INFO {  
    UINT_PTR startAddress;  
    SIZE_T size;  
} COR_PRF_CODE_INFO;  
```  
  
## <a name="members"></a>成员  
  
|成员|描述|  
|------------|-----------------|  
|`startAddress`|代码连续块起始地址。|  
|`size`|块的大小。|
