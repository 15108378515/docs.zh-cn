# <a name="icorprofilercallbackthreaddestroyed-method"></a>ICorProfilerCallback::ThreadDestroyed 方法
通知探查器已销毁线程。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT ThreadDestroyed(  
    [in] ThreadID threadId);  
```  
  
#### <a name="parameters"></a>参数  
 `threadId`  
 [in]已销毁线程的 ID。  
  
## <a name="remarks"></a>备注  
 `threadId`值将不再有效在此调用时。
