# <a name="icorprofilerobjectenumskip-method"></a>ICorProfilerObjectEnum::Skip 方法
因此，跳过指定的数量的元素，请从其当前位置前移此枚举器的光标。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT Skip (  
    [in] ULONG   celt  
);  
```  
  
#### <a name="parameters"></a>参数  
 `celt`  
 [in]要跳过的元素数。  
  
## <a name="remarks"></a>备注  
 此枚举器的光标的新位置是: （当前位置） + `celt` 。
