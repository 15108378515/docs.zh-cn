# <a name="3821---routingserviceclosefailed"></a>3821 - RoutingServiceCloseFailed
## <a name="properties"></a>属性  
  
|||  
|-|-|  
|ID|3821|  
|关键字|RoutingServices|  
|级别|警告|  
|通道|Microsoft-Windows-应用程序服务器-应用程序/调试|  
  
## <a name="description"></a>描述  
 当路由服务无法关闭出站客户端时，将发出此事件。  
  
## <a name="message"></a>消息  
 关闭出站客户端“%1”时，路由服务失败。
