## <a name="properties"></a>属性  
  
|||  
|-|-|  
|ID|4016|  
|关键字|ActivationServices|  
|级别|错误|  
|通道|Microsoft-Windows-应用程序服务器-应用程序/分析|  
  
## <a name="description"></a>描述  
 当消息队列复制套接字失败时，将发出此事件。  
  
## <a name="message"></a>消息  
 消息队列无法复制套接字。
