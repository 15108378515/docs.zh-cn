|||  
|-|-|  
|ID|4801|  
|关键字|发现|  
|级别|警告|  
|通道|Microsoft-Windows-应用程序服务器-应用程序/调试|  
  
## <a name="description"></a>描述  
 在 DiscoveryClientChannel 内部创建的 DiscoveryClient 无法关闭，因此已中止时，发出此事件。  
  
## <a name="message"></a>消息  
 在 DiscoveryClientChannel 内部创建的 DiscoveryClient 无法关闭，因此已中止。
