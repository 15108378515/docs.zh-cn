|||  
|-|-|  
|ID|4804|  
|关键字|发现|  
|级别|信息|  
|通道|Microsoft-Windows-应用程序服务器-应用程序/调试|  
  
## <a name="description"></a>描述

## <a name="message"></a>消息  
 messageId 为“%2”的 %1 消息已被 DiscoveryClient 丢弃，因为相应的 %3 操作已完成。
