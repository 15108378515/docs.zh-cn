# <a name="complustlbimporterror"></a>ComPlusTLBImportError
ID：132  
  
 严重性：错误  
  
 类别：ServiceModel  
  
## <a name="description"></a>描述  
 此事件指示导入类型库时发生错误。 该事件列出了 IID、类型库 ID、异常、进程名及进程 ID。
