# <a name="failedtologmessage"></a>FailedToLogMessage
Id: 113  
  
 严重性：错误  
  
 类别：ServiceModel  
  
## <a name="description"></a>描述  
 此事件指示消息未记入日志。 此事件列出了异常、进程名称和进程 ID。
