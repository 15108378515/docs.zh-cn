# <a name="failedtotraceeventwithexception"></a>FailedToTraceEventWithException
ID：105  
  
 严重性：错误  
  
 类别：跟踪  
  
## <a name="description"></a>描述  
 未跟踪一个或多个事件。  
  
 此事件将列出原始事件字符串、异常、进程名及进程 ID。
