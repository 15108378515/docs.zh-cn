# <a name="messageauthenticationsuccess"></a>MessageAuthenticationSuccess
Id: 169  
  
 严重性：信息  
  
 类别：SecurityAudit  
  
## <a name="description"></a>描述  
 此事件指示消息身份验证成功。 此事件将列出服务、操作、客户端标识和活动 ID。
