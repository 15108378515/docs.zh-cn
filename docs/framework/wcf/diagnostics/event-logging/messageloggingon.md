# <a name="messageloggingon"></a>MessageLoggingOn
ID：116  
  
 严重性：错误  
  
 类别：ServiceModel  
  
## <a name="description"></a>描述  
 已打开消息日志记录。 将记录敏感信息，即使此敏感信息已在网络上加密也是如此。 例如，消息正文。 此事件将列出 AppID、异常、进程名称和进程 ID。
