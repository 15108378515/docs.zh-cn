# <a name="piiloggingon"></a>PiiLoggingOn
Id: 107  
  
 严重性：错误  
  
 类别：跟踪  
  
## <a name="description"></a>描述  
 此事件指示 PII 日志记录已打开。 将记录敏感信息。 该事件将列出进程名称和进程 ID。
