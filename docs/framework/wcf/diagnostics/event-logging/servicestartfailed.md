# <a name="servicestartfailed"></a>ServiceStartFailed
Id: 164  
  
 严重性：错误  
  
 类别：SMSvcHost  
  
## <a name="description"></a>描述  
 此事件指示请求启动服务失败。 此事件列出错误代码、进程名称和进程 ID。
