# <a name="sslnoaccessibleprivatekey"></a>SslNoAccessiblePrivateKey
Id: 155  
  
 严重性：错误  
  
 类别：TransactionBridge  
  
## <a name="description"></a>描述  
 此事件指示带有特定主题名称和指纹的标识证书不具有可访问的私钥。 该事件将列出进程名称和进程 ID。
