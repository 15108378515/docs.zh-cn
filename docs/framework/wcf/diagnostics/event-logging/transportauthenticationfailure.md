# <a name="transportauthenticationfailure"></a>TransportAuthenticationFailure
Id: 174  
  
 严重性：错误  
  
 类别：SecurityAudit  
  
## <a name="description"></a>描述  
 此事件指示传输身份验证已失败，并列出了客户端标识和活动 ID。
