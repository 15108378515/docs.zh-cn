# <a name="unknownlisteneradaptererror"></a>UnknownListenerAdapterError
ID：161  
  
 严重性：错误  
  
 类别：SMSvcHost  
  
## <a name="description"></a>描述  
 此事件指示在处理 Windows 进程激活服务 (WAS) 通知时侦听器适配器中出现了意外错误。 已终止该进程。 该事件将列出协议、异常、进程名称和进程 ID。
