# <a name="wasdisconnected"></a>WasDisconnected
Id: 162  
  
 严重性：错误  
  
 类别：SMSvcHost  
  
## <a name="description"></a>描述  
 此事件指示 Windows 进程激活 (WAS) 尚未断开连接。 此事件将列出 HRESULT、进程名称和进程 ID。
