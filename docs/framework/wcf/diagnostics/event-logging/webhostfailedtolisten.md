# <a name="webhostfailedtolisten"></a>WebHostFailedToListen
Id: 112  
  
 严重性：错误  
  
 类别：ServiceModel  
  
## <a name="description"></a>描述  
 此事件指示在尝试侦听特定 URL 时出现错误。 已终止该辅助进程。 此事件列出了异常、进程名称和进程 ID。
