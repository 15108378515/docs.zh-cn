# <a name="webhosthttperror"></a>WebHostHttpError
ID：110  
  
 严重性：错误  
  
 类别：ServiceModel  
  
## <a name="description"></a>描述  
 此事件指示发生了 WebHost HTTP 异常。 该事件将列出发送方信息、HtmlErrorMessage、异常、进程名称和进程 ID。
