# <a name="wmiputinstancefailed"></a>WmiPutInstanceFailed
Id: 121  
  
 严重性：错误  
  
 类别：ServiceModel  
  
## <a name="description"></a>描述  
 此事件指示未处理 WMI PutInstance 请求。 此事件将列出 WMI 对象、异常、进程名称和进程 ID。
