# <a name="calls-failed"></a>Calls Failed（失败的调用次数）
计数器名称：Calls Failed（失败的调用次数）  
  
## <a name="description"></a>描述  
 此操作中出现未处理异常的调用数量。 该操作中每出现一个未处理的异常，此计数器就会增加。
