# <a name="endpoint-transactions-flowed"></a>终结点：流动的事务数
计数器名称：Transactions Flowed（流动的事务数）  
  
## <a name="description"></a>描述  
 流向此终结点上的操作的事务数。 只要发送到终结点的消息中出现事务 ID，此计数器即会递增。
