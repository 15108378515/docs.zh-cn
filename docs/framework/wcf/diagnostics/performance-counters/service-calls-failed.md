# <a name="service-calls-failed"></a>服务：Calls Failed（失败的调用次数）
计数器名称：Calls Failed（失败的调用次数）。  
  
## <a name="description"></a>描述  
 此服务中出现未处理异常的调用次数。 此服务中每出现一个未处理异常，此计数器就会递增。
