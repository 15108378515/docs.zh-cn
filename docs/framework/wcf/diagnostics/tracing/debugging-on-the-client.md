# <a name="debugging-on-the-client"></a>在客户端上调试
若要使用户更轻松地编写您的 WCF 服务的客户端应用程序，你可以添加[ \<serviceDebug >](../../../../../docs/framework/configure-apps/file-schema/wcf/servicedebug.md)服务为你的服务配置文件的行为。 此行为可用于发布帮助页，以及在返回到客户端的 SOAP 错误的详细信息中返回托管异常信息。
