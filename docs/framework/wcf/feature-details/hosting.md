# <a name="hosting"></a>宿主
本节中的主题介绍服务承载。 可以承载服务，通过 Internet 信息服务 (IIS)、 Windows 进程激活服务 (WAS)、 Windows Server AppFabric、 Windows 服务，或托管的应用程序-此选项通常称为*自承载*。  
  
 值得注意的是，从不受信任的主机运行服务或任何扩展会危害安全。
