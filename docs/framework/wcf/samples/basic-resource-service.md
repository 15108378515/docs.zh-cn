# <a name="basic-resource-service"></a><span data-ttu-id="f50c0-102">基本资源服务</span><span class="sxs-lookup"><span data-stu-id="f50c0-102">Basic Resource Service</span></span>
<span data-ttu-id="f50c0-103">此示例演示如何使用 [!INCLUDE[indigo1](../../../../includes/indigo1-md.md)] REST 编程模型实现基于 HTTP 的服务，该模型公开支持检索、添加、删除和替换操作的客户集合。</span><span class="sxs-lookup"><span data-stu-id="f50c0-103">This sample demonstrates how to implement a HTTP-based service using the [!INCLUDE[indigo1](../../../../includes/indigo1-md.md)] REST Programming model that exposes a collection of customers that supports the retrieve, add, delete and replace operations.</span></span> <span data-ttu-id="f50c0-104">此示例由两个组件组成：一个自承载 [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] HTTP 服务 (Service.cs) 和一个创建服务并对该服务进行调用的控制台应用程序 (program.cs)。</span><span class="sxs-lookup"><span data-stu-id="f50c0-104">This sample consists of 2 components - a self-hosted [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] HTTP service (Service.cs) and a console application (program.cs) that creates the service and makes calls to it.</span></span>  
  
## <a name="sample-details"></a><span data-ttu-id="f50c0-105">示例详细信息</span><span class="sxs-lookup"><span data-stu-id="f50c0-105">Sample Details</span></span>  
 <span data-ttu-id="f50c0-106">[!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] 服务以面向资源的/REST 方式公开客户集合。</span><span class="sxs-lookup"><span data-stu-id="f50c0-106">The [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] service exposes a collection of customers in a resource-oriented/REST manner.</span></span> <span data-ttu-id="f50c0-107">简而言之，这涉及拥有客户集合以及该集合中每个客户的唯一 URI。</span><span class="sxs-lookup"><span data-stu-id="f50c0-107">In short, this involves having unique URIs for the collection of customers and every customer in the collection.</span></span> <span data-ttu-id="f50c0-108">该服务支持在集合 URI 上发送 HTTP `GET` 以检索整个集合，并在结合 URI 上发送 HTTP `POST` 以向该集合添加新客户。</span><span class="sxs-lookup"><span data-stu-id="f50c0-108">The service supports sending an HTTP `GET` at the collection URI to retrieve the entire collection and HTTP `POST` at the collection URI to add a new customer to the collection.</span></span> <span data-ttu-id="f50c0-109">另外，单个客户的 URI 上，它还支持 HTTP `GET` 以获取客户详细信息，支持 HTTP `PUT` 以替换该客户的详细信息，并支持 HTTP `DELETE` 以将该客户从集合中移除。</span><span class="sxs-lookup"><span data-stu-id="f50c0-109">Also at the URI for an individual customer, it supports HTTP `GET` to get the customer details, HTTP `PUT` to replace the details of the customer and HTTP `DELETE` to remove the customer from the collection.</span></span> <span data-ttu-id="f50c0-110">将新客户添加到集合中时，该服务会向其分配一个唯一的 URI，并将该 URI 作为客户详细信息的一部分存储。</span><span class="sxs-lookup"><span data-stu-id="f50c0-110">When a new customer is added to the collection, the service assigns it a unique URI and stores the URI as part of the customer’s details.</span></span> <span data-ttu-id="f50c0-111">此外，该服务还使用响应的位置 HTTP 标头将该 URI 通知客户端。</span><span class="sxs-lookup"><span data-stu-id="f50c0-111">Also, it communicates the URI to the client using the Location HTTP header of the response.</span></span>  
  
 <span data-ttu-id="f50c0-112">App.config 文件使用 [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] 属性设置为 <xref:System.ServiceModel.Description.WebHttpEndpoint> 的默认 <xref:System.ServiceModel.Description.WebHttpEndpoint.HelpEnabled%2A> 来配置 `true` 服务。</span><span class="sxs-lookup"><span data-stu-id="f50c0-112">The App.config file configures the [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] service with a default <xref:System.ServiceModel.Description.WebHttpEndpoint> that has the <xref:System.ServiceModel.Description.WebHttpEndpoint.HelpEnabled%2A> property set to `true`.</span></span> <span data-ttu-id="f50c0-113">因此，[!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] 将在 `http://localhost:8000/Customers/help` 上创建一个基于 HTML 的自动帮助页，该帮助页可提供有关如何构造服务的 HTTP 请求以及如何访问服务的 HTTP 响应的信息 – 例如，如何以 XML 和 JSON 表示客户详细信息的示例。</span><span class="sxs-lookup"><span data-stu-id="f50c0-113">As a result, the [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] creates an automatic HTML-based help page at `http://localhost:8000/Customers/help` that provides information about how to construct HTTP requests to the service and how to access the service’s HTTP response – for instance, an example of how the customer details is represented in XML and JSON.</span></span>  
  
 <span data-ttu-id="f50c0-114">通过以这种方式公开客户集合（一般来说是任何资源），客户端可以使用 URI 和 HTTP `GET`、`PUT`、`DELETE` 和 `POST` 以一种统一的方式与服务进行交互。</span><span class="sxs-lookup"><span data-stu-id="f50c0-114">Exposing the authored using <xref:System.Net.HttpWebRequest>.</span></span> <span data-ttu-id="f50c0-116">请注意，这只是访问 [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] REST 服务的一种方式。</span><span class="sxs-lookup"><span data-stu-id="f50c0-116">Note that this is just one way to access a [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] REST service.</span></span> <span data-ttu-id="f50c0-117">还可以使用其他 .NET Framework 类（如 <xref:System.ServiceModel.ChannelFactory> 和 <xref:System.Net.WebClient>）来访问该服务。</span><span class="sxs-lookup"><span data-stu-id="f50c0-117">It is data-ttu-id="f50c0-118">SDK 中的其他示例 (如[基本 HTTP 服务](../../../../docs/framework/wcf/samples/basic-http-service.md)示例和[自动格式选择](../../../../docs/framework/wcf/samples/automatic-format-selection.md)示例) 演示如何使用这些类与通信[!INCLUDE[indigo2](../../../../includes/indigo2-md.md)]服务。</span><span class="sxs-lookup"><span data-stu-id="f50c0-118">Other samples in the SDK (such as the [Basic HTTP Service](../../../../docs/framework/wcf/samples/basic-http-service.md) sample and the [Automatic Format Selection](../../../../docs/framework/wcf/samples/automatic-format-selection.md) sample) show
