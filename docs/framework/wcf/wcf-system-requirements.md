# <a name="wcf-system-requirements"></a>WCF 系统需求
Windows Communication Foundation (WCF) 是用于创建分布式应用程序的通信基础结构。 下面列出了启用 WCF 运行的要求。  
  
## <a name="system-requirements"></a>系统要求  
 默认情况下在安装 WCF [!INCLUDE[wv](../../../includes/wv-md.md)]。  
  
 此外可上安装 WCF [!INCLUDE[wxpsp2](../../../includes/wxpsp2-md.md)]， [!INCLUDE[ws2003r2](../../../includes/ws2003r2-md.md)]，或[!INCLUDE[ws2003sp1](../../../includes/ws2003sp1-md.md)]。  
  
 **请注意**仅在支持的 WCF 的消息队列 (MSMQ) 功能[!INCLUDE[wv](../../../includes/wv-md.md)]， [!INCLUDE[ws2003r2](../../../includes/ws2003r2-md.md)]， [!INCLUDE[ws2003sp1](../../../includes/ws2003sp1-md.md)]，和[!INCLUDE[wxppro](../../../includes/wxppro-md.md)]。
