# <a name="1003---workflowinstancecanceled"></a>1003 - WorkflowInstanceCanceled
## <a name="properties"></a>属性  
  
|||  
|-|-|  
|ID|1003|  
|关键字|WFRuntime|  
|级别|信息|  
|通道|Microsoft-Windows-应用程序服务器-应用程序/调试|  
  
## <a name="description"></a>描述  
 指示工作流实例在“已取消”状态下完成。  
  
## <a name="message"></a>消息  
 WorkflowInstance Id“%1”在“已取消”状态下完成。
