# <a name="1004---workflowinstanceaborted"></a>1004 - WorkflowInstanceAborted
## <a name="properties"></a>属性  
  
|||  
|-|-|  
|ID|1004|  
|关键字|WFRuntime|  
|级别|信息|  
|通道|Microsoft-Windows-应用程序服务器-应用程序/调试|  
  
## <a name="description"></a>描述  
 指示工作流实例已中止并且具有异常。  
  
## <a name="message"></a>消息  
 WorkflowInstance Id“%1”因出现异常而中止。
