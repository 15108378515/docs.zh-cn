# <a name="1005---workflowapplicationidled"></a>1005 - WorkflowApplicationIdled
## <a name="properties"></a>属性  
  
|||  
|-|-|  
|ID|1005|  
|关键字|WFRuntime|  
|级别|信息|  
|通道|Microsoft-Windows-应用程序服务器-应用程序/调试|  
  
## <a name="description"></a>描述  
 指示工作流应用程序已处于空闲状态。  
  
## <a name="message"></a>消息  
 WorkflowApplication ID“%1”变为空闲状态。
