# <a name="1147---flowchartswitchdefault"></a>1147 - FlowchartSwitchDefault
## <a name="properties"></a>属性  
  
|||  
|-|-|  
|ID|1147|  
|关键字|WFActivities|  
|级别|信息|  
|通道|Microsoft-Windows-应用程序服务器-应用程序/调试|  
  
## <a name="description"></a>描述
