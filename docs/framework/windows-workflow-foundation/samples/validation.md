# <a name="validation"></a>验证
本节包含演示活动验证的示例。  
  
## <a name="in-this-section"></a>本节内容  
 [外部活动验证](../../../../docs/framework/windows-workflow-foundation/samples/external-activity-validation.md)  
 此示例演示如何向不是由您创作的内置活动添加验证逻辑。  
  
 [基本验证](../../../../docs/framework/windows-workflow-foundation/samples/basic-validation.md)  
 演示 `CreateProduct` 活动，该活动验证它的 `Cost` 参数是否小于或等于它的 `Price` 参数。  
  
 [OverloadGroups](../../../../docs/framework/windows-workflow-foundation/samples/overloadgroups.md)  
 演示 `CreateLocation` 活动，该活动具有必需的参数和可选的参数，并允许用户选择提供两个不同的参数集中的一个。  
  
 [活动关系验证](../../../../docs/framework/windows-workflow-foundation/samples/activity-relationships-validation.md)  
 演示如何验证活动之间的关系。
