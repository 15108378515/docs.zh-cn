# <a name="how-to-change-the-value-of-an-existing-setting-at-design-time"></a>如何：在设计时更改现有设置的值
你可以使用 Visual Studio 项目中编辑现有设置的值。  
  
### <a name="to-change-the-value-of-an-existing-setting-at-design-time-in-c"></a>若要更改在设计时在 C# 中的现有设置的值  
  
1.  在**解决方案资源管理器**，展开**属性**你的项目节点。  
  
2.  双击想要添加新设置的.settings 文件。 此文件的默认名称是 Settings.settings。  
  
3.  在设置设计器中，找到你想要更改值，并在值列中键入新值的设置。  
  
### <a name="to-change-the-value-of-an-existing-setting-at-design-time-in-visual-basic"></a>若要更改在设计时在 Visual Basic 中的现有设置的值  
  
1.  在**解决方案资源管理器**，右键单击你的项目节点，然后选择**属性**。  
  
2.  在**属性**页上，选择**设置**选项卡。  
  
3.  在设置设计器中，找到你想要更改值，并在值列中键入新值的设置。
