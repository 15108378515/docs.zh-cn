# <a name="datetimepicker-control-windows-forms"></a>DateTimePicker 控件（Windows 窗体）
Windows 窗体`DateTimePicker`控件允许用户从日期或时间的列表中选择单个项。 当用于表示的日期，它将显示在两个部分： 文本，并单击列表旁的向下箭头时出现的网格中所表示日期的下拉列表。
