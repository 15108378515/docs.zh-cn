# <a name="how-to-add-or-remove-imagelist-images-with-the-designer"></a>如何：使用设计器添加或移除 ImageList 图像
你可以将映像添加到<xref:System.Windows.Forms.ImageList>组件几种不同方式。 您可以通过使用与关联的智能标记非常快速地添加图像<xref:System.Windows.Forms.ImageList>，或如果你要设置多个其他属性上<xref:System.Windows.Forms.ImageList>，你可能会发现更方便地通过属性窗口添加图像。 此外可以通过使用代码来添加映像。 有关如何添加与代码的映像的详细信息，请参阅[如何： 添加或移除图像使用 Windows 窗体 ImageList 组件](../../../../docs/framework/winforms/controls/how-to-add-or-remove-images-with-the-windows-forms-imagelist-component.md)。 通常填充<xref:System.Windows.Forms.ImageList>映像之前关联与控件，但这不是必需组件。  
  
> [!NOTE]
>  显示的对话框和菜单命令可能会与“帮助”中的描述不同，具体取决于你现用的设置或版本。 若要更改设置，请在 **“工具”** 菜单上选择 **“导入和导出设置”** 。 有关详细信息，请参阅[在 Visual Studio 中自定义开发设置](http://msdn.microsoft.com/library/22c4debb-4e31-47a8-8f19-16f328d7dcd3)。  
  
### <a name="to-add-or-remove-images-by-using-the-properties-window"></a>若要添加或删除映像，通过使用属性窗口  
  
1.  选择<xref:System.Windows.Forms.ImageList>组件，或者添加到窗体的某一。  
  
2.  在属性窗口中，单击省略号按钮 (![VisualStudioEllipsesButton 屏幕快照](../../../../docs/framework/winforms/media/vbellipsesbutton.png "vbEllipsesButton")) 旁边<xref:System.Windows.Forms.ImageList.Images%2A>属性。  
  
3.  在**图像集合编辑器**，单击**添加**或**删除**添加或从列表中删除映像。  
  
### <a name="to-add-or-remove-images-using-the-smart-tag"></a>若要添加或移除使用智能标记的图像  
  
1.  选择<xref:System.Windows.Forms.ImageList>组件，或者添加到窗体的某一。  
  
2.  单击智能标记标志符号 (![智能标记标志符号](../../../../docs/framework/winforms/controls/media/vs-winformsmttagglyph.gif "VS_WinFormSmtTagGlyph"))  
  
3.  在**ImageList 任务**对话框中，选择**选择映像**。  
  
4.  在**映像集合编辑器**单击**添加**或**删除**添加或从列表中删除映像。
