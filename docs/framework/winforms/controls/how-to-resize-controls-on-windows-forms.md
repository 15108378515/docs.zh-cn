# <a name="how-to-resize-controls-on-windows-forms"></a>如何：调整 Windows 窗体上控件的大小
你可以调整单个控件的大小和如调整大小相同或不同类型的多个控件<xref:System.Windows.Forms.Button>和<xref:System.Windows.Forms.GroupBox>控件。  
  
### <a name="to-resize-a-control"></a>调整控件的大小  
  
1.  单击控件可调整大小并拖动其中一个八个大小调整手柄。  
  
    > [!NOTE]
    >  选择的控件，然后在按住 SHIFT 键来一次调整控件的一个像素的同时按箭头键。 在按住 SHIFT 和 CTRL 键，若要调整大小大幅度中的控件的同时按向下或向右箭头键。  
  
### <a name="to-resize-multiple-controls-on-a-form"></a>若要调整窗体上的多个控件的大小  
  
1.  按住 CTRL 或 SHIFT 键并选择你想要调整大小的控件。 你选择的第一个控件的大小适用于其他控件。  
  
2.  上**格式**菜单上，选择**使大小相同**，并选择四个选项之一。 前三个命令更改以匹配第一个选择控件的控件的尺寸。
