# 入门 (WPF)
Windows Presentation Foundation \(WPF\) 是一个可创建丰富的交互式客户端应用程序的 UI 框架。  WPF 开发平台支持广泛的应用程序开发功能，包括应用程序模型、资源、控件、图形、布局、数据绑定、文档和安全性。  它是 .NET Framework 的子集，因此，如果你曾经通过 ASP.NET 或 Windows 窗体使用 .NET Framework 构建应用程序，应该会熟悉此编程体验。  WPF 使用可扩展应用程序标记语言 \(XAML\) 为应用程序编程提供声明性模型。  本节包含引导和帮助你开始使用 WPF 的主题。
