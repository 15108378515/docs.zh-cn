# <a name="f-guide"></a>F# 指南

F # 是一功能的编程语言，它在.NET 上运行。  除了支持函数编程构造，它还具有对象编程功能。  此混合函数编程和面向对象的功能使 F # 完成任何任务的编程语言。

## <a name="if-youre-new-to-f"></a>如果您是初次接触 F # #

如果你是新手 F #，以开始[教程的 F #](tour.md)若要获取的语言的概述和一些其编程概念。  如果你使用 Visual Studio，该教程的项目模板将包含相同的内容。

## <a name="if-youre-experienced-with-f"></a>如果你有使用 F # 经验 #

如果你知道方面 F # 中，或者想要了解有关特定语言构造的详细信息，请参阅[语言参考](language-reference/index.md)。  它是所有的 F # 语言功能的全面指南。

此外， [F # 核心库参考](https://msdn.microsoft.com/visualfsharpdocs/conceptual/fsharp-core-library-reference)是可用于了解 FSharp.Core，这是属于 F # 核心库的重大资源。

## <a name="the-f-software-foundation"></a>F# 软件基金会

尽管 Microsoft 是主 F # 语言和其工具的开发人员，F # 还支持独立的基础，F # 软件 Foundation (FSSF)。

F# 软件基金会的任务是提升、保护和改进 F# 编程语言，以及支持和促进国际化的多元 F# 程序员社区的成长。

若要了解详细信息并参与，请查看 [fsharp.org](http://fsharp.org)。
