# <a name="getting-started-with-f"></a>F# 入门# #

F# 入门的方式主要有三种：

如果使用 Windows，请查看[在 Visual Studio 中入门 F#](getting-started-visual-studio.md)。

如果对在 Windows、Linux 或 macOS 上的轻量型 IDE 中编写代码感兴趣，请查看[在 Visual Studio Code 中通过 Ionide 入门 F#](getting-started-vscode.md)。

如果想要在任何操作系统上使用命令行，请查看[通过命令行工具入门 F#](getting-started-command-line.md)。
