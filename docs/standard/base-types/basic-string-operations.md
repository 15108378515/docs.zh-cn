# <a name="basic-string-operations"></a>基本字符串操作

应用程序经常通过构造基于用户输入的消息来响应用户。 例如，网站用包含用户名的专用问候语来响应新登录的用户的情况并不少见。 使用 [System.String](xref:System.String) 和 [System.Text.StringBuilder](xref:System.Text.StringBuilder) 类中的多个方法可以动态构造要在用户界面中显示的自定义字符串。 借助这些方法还可执行许多基本字符串操作，例如，从字节数组创建新字符串，比较字符串的值和修改现有字符串。

[填充字符串](padding.md) - 介绍如何将字符或空格插入到字符串中。

[比较字符串](comparing.md) - 介绍如何比较两个或更多个字符串的内容。

[如何：执行基本字符串操作](basic-manipulations.md) - 演示基本字符串操作的用法。
