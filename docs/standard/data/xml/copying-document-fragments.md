# <a name="copying-document-fragments"></a>复制文档片段
你可以创建**XmlDocumentFragment**节点并将其下的节点。 当**XmlDocumentFragment**插入与**InsertNode**方法， **XmlDocumentFragment**不复制节点，但在 XML 中插入及其子节点文档对象模型 (DOM)。
