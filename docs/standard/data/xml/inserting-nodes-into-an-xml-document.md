# <a name="inserting-nodes-into-an-xml-document"></a>将节点插入 XML 文档中
可以通过插入新节点来修改文档。 节点可以是新创建的，存在于同一文档中，也可以从另一个文档导入。 有关创建新节点的详细信息，请参阅[DOM 中创建新节点](../../../../docs/standard/data/xml/create-new-nodes-in-the-dom.md)。 复制现有节点的详细信息，请参阅[复制现有节点](../../../../docs/standard/data/xml/copy-existing-nodes.md)，[复制现有节点从一个文档复制到另一个](../../../../docs/standard/data/xml/copying-existing-nodes-from-one-document-to-another.md)，或[复制文档片段](../../../../docs/standard/data/xml/copying-document-fragments.md)。
