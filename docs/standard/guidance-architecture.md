# <a name="net-architecture-guidance"></a>.NET 的体系结构指南

## <a name="containerized-docker-application-lifecycle-with-the-microsoft-platform-and-toolscontainerized-lifecycle-architectureindexmd"></a>[使用 Microsoft 平台和工具的 Docker 容器化应用程序生命周期](./containerized-lifecycle-architecture/index.md)

本指南将介绍如何将用于开发、 验证和部署使用 Visual Studio 和 Microsoft Azure 的容器化的 Docker 应用程序的建议的端到端生命周期进程。

## <a name="modernize-existing-net-applications-with-azure-cloud-and-windows-containersmodernize-with-azure-and-containersindexmd"></a>[更新现有的.NET 应用程序与 Azure 云和 Windows 容器](./modernize-with-azure-and-containers/index.md)

本指南将介绍你将需要迁移到 Azure 云和 Windows 容器的现有 web 应用程序的策略。 你将了解代码策略、 数据迁移、 orchestrators，并 CI/CD 处理。

## <a name="architect-modern-web-applications-with-aspnet-core-and-azuremodern-web-apps-azure-architectureindexmd"></a>[使用 ASP.NET Core 和 Azure 的架构师现代 web 应用程序](modern-web-apps-azure-architecture/index.md)

本指南将介绍建议体系结构、 设计和部署过程将用于生成 ASP.NET 和 ASP.NET Core 应用程序并托管在 Azure 中的这些应用程序。

## <a name="architecting-container-and-microservice-based-applicationsmicroservices-architectureindexmd"></a>[构建容器化微服务应用](microservices-architecture/index.md)

本指南介绍如何使用容器开发基于微服务的应用程序并对其进行管理。 本指南探讨使用 .NET Core 和 Docker 容器的体系结构设计和实现方法。
