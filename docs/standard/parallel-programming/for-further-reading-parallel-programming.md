# <a name="for-further-reading-parallel-programming"></a>其他阅读材料（并行编程）
下面的资源提供了有关 .NET Framework 中并行编程的其他信息：  
  
-   MSDN 网站上的[并行计算开发人员中心](https://msdn.microsoft.com/vstudio/bb964701)包含一些链接，指向有关如何在 .NET Framework 和本机 C++ 中进行并行编程的内容和论坛帖子。  
  
-   MSDN 博客网站上的[使用 .NET 并行编程](https://blogs.msdn.microsoft.com/pfxteam/)博客包含许多有关 .NET Framework 中并行编程的深入文章。  
  
-   可以在 MSDN 论坛网站上的[并行扩展](https://social.msdn.microsoft.com/Forums/en-US/home?forum=parallelextensions)论坛中提出和回答有关并行编程的问题。  
  
-   MSDN 代码库网站上的[并行扩展示例](https://code.msdn.microsoft.com/ParExtSamples)页面包含许多示例，展示了中级和高级并行编程技术。
