# <a name="for-further-reading-parallel-programming"></a>其他阅读材料（并行编程）
下面的资源提供了有关 .NET Framework 中并行编程的其他信息：  
  
-   MSDN 网站上的[并行计算开发人员中心](http://go.microsoft.com/fwlink/?LinkID=160570)包含一些链接，链接到有关在 .NET Framework 和本机 C++ 中进行并行编程的最新内容和论坛帖子。  
  
-   MSDN 博客网站上的[使用 .NET 并行编程](http://go.microsoft.com/fwlink/?LinkID=169627)博客包含许多有关 .NET Framework 中并行编程的深入文章。  
  
-   MSDN 博客网站上的[并发可视化工具](http://go.microsoft.com/fwlink/?LinkID=169630)博客介绍了 [!INCLUDE[vs_dev10_long](../../../includes/vs-dev10-long-md.md)] 一些版本随附的新性能分析工具。  
  
-   可以在 MSDN 论坛网站上的[并行扩展](http://go.microsoft.com/fwlink/?LinkID=169628)论坛中提出和回答有关并行编程的问题。  
  
-   MSDN 代码库网站上的[并行扩展示例](http://go.microsoft.com/fwlink/?LinkID=165717)页面包含许多示例，展示了中级和高级并行编程技术。
