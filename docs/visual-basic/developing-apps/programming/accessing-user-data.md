# <a name="accessing-user-data-visual-basic"></a>访问用户数据 (Visual Basic)
本部分包含有关处理 `My.User` 对象的主题以及可通过该操作完成的任务。  
  
 `My.User` 对象通过返回实现 <xref:System.Security.Principal.IPrincipal> 接口的对象，提供有关登录用户的信息的访问权限。
