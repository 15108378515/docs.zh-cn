# <a name="visual-basic-intellisense-code-snippets"></a>Visual Basic IntelliSense 代码段
[!INCLUDE[vbprvb](../../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]包括包含几个几百个代码段，名为 IntelliSense 代码段，就可以在您的应用程序中插入一个代码库。 每个代码段都执行完整的编程任务，如创建自定义例外、 发送电子邮件，或绘制一个圆圈。 您可以将代码段插入源代码中用点击几下鼠标。 此外可以创建您自己的代码段适应业务需求，将它们添加到库中，然后使用它们在需要时。 有关详细信息，请参阅[代码片段](https://docs.microsoft.com/visualstudio/ide/code-snippets)。  
  
## <a name="a-productivity-tool-for-visual-basic-users"></a>Visual Basic 用户工作效率工具  
 这些代码段提高工作效率，减少了您查找代码示例所花的时间、 减少来了解如何使用熟悉的功能所需的时间和支持代码重用。  
  
 代码库支持以下功能︰  
  
-   插入到代码段之一**代码编辑器**。  
  
-   创建新任务，可以在您的项目中重用它们。  
  
-   创建新任务，以与工作组和同事共享。  
  
-   编辑任务。  
  
-   从第三方下载更多的任务。  
  
 这些代码块是在整个可用[!INCLUDE[vsprvs](../../../csharp/includes/vsprvs_md.md)]:  
  
-   您可以通过的快捷菜单添加它们**代码编辑器**。  
  
-   您可以将 XML 代码文件从**文件资源管理器**拖动到源代码文件。
