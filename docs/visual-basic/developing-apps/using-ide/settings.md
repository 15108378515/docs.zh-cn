# <a name="visual-basic-settings"></a>Visual Basic 设置
Visual Basic 设置旨在优化用户体验，并以最大化的 Visual Basic 开发人员工作效率。 它属于[Visual Studio 中自定义开发设置](http://msdn.microsoft.com/en-us/22c4debb-4e31-47a8-8f19-16f328d7dcd3)功能，它使您可以自定义和保存在 Visual Studio 集成开发环境 (IDE) 的设置。  
  
 此外可以端口或重新加载您在其他计算机上的 IDE 设置。 有关详细信息，请参阅[How to︰ 计算机之间共享设置](http://msdn.microsoft.com/en-us/1131fb10-35c1-42da-9cd8-91aa3235b882)。
