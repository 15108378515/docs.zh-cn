# <a name="constant-expression-not-representable-in-type-39lttypenamegt39"></a>无法在类型中表示的常量表达式&#39; &lt;typename&gt;&#39;
你尝试评估一个常数，而不能适应目标类型，通常是因为它溢出范围。  
  
 **错误 ID:** BC30439  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  目标类型更改为一个可以处理该常量。
