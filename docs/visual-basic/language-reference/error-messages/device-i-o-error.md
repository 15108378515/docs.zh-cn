# <a name="device-io-error"></a>设备 I/O 错误
输入或输出出错程序已使用例如打印机或磁盘驱动器的设备。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   请确保该设备正常，然后重试该操作。
