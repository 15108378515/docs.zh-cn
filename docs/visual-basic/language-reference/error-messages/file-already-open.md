# <a name="file-already-open"></a>文件已打开
有时必须在另一部分之前关闭的文件`FileOpen`或位置执行其他操作。 此错误的可能原因包括：  
  
-   顺序输出模式`FileOpen`已打开的文件执行操作  
  
-   某个语句引用打开的文件。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  执行语句之前关闭该文件。
