# <a name="file-not-found-visual-basic-run-time-error"></a>找不到文件（Visual Basic 运行时错误）
指定在找不到该文件。 错误具有以下可能的原因：  
  
-   语句引用不存在的文件。  
  
-   尝试调用动态链接库 (DLL) 中的过程，但在指定库`Lib`子句`Declare`找不到语句。  
  
-   您尝试打开的项目或加载不存在的文本文件。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  检查文件名和路径规范的拼写。
