# <a name="39ltmethodnamegt39-has-multiple-definitions-with-identical-signatures"></a>&#39;&lt;methodname&gt;&#39; 具有多个完全相同的签名定义
A`Function`或`Sub`过程声明与以前的声明将使用相同的过程名称和自变量列表。 一个可能的原因是尝试重载原始的过程。 重载的过程必须具有不同的自变量列表。  
  
 **错误 ID:** BC30269  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   更改过程名称或自变量列表中，或删除重复的声明。
