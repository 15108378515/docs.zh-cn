# <a name="39ltname1gt39-is-ambiguous-imported-from-the-namespaces-or-types-39ltname2gt39"></a>&#39;&lt;name1&gt;&#39; 不明确，从命名空间或类型 &#39; 导入&lt;name2&gt;&#39;
你提供的名称不明确，因此与另一个名称冲突。 [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] 编译器没有任何冲突解决规则；你必须自己区分名称。  
  
 **错误 ID:** BC30561  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  删除命名空间导入，消除名称歧义。  
  
2.  完全限定名称。
