# <a name="need-property-array-index"></a>需要属性数组索引
该属性值由数组而不是由单个值组成。 没有为尝试访问的属性数组指定索引。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   检查组件的文档，以查找适合于数组的索引范围。 在属性访问语句中指定合适的索引。
