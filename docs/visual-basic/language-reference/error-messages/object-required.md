# <a name="object-required-visual-basic"></a>需要对象 (Visual Basic)
对属性和方法的引用通常需要使用显式对象识别符。 这是这种情况。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  检查到的对象属性或方法的引用具有有效的对象限定符。 如果未提供一个，指定的对象限定符。  
  
2.  检查对象限定符的拼写正确并确保该对象是在计划在其中你正在引用它的一部分中可见。  
  
3.  如果将路径提供给主机应用程序的**文件打开**命令，请检查参数是否正确。  
  
4.  检查对象的文档，并确保操作无效。
