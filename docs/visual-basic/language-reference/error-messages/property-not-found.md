# <a name="property-not-found"></a>找不到属性
此对象不支持指定的属性。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  检查属性的名称的拼写。  
  
2.  检查以确保不尝试访问类似于"text"属性时对象实际支持的"标题"或类似的命名属性的对象的文档。
