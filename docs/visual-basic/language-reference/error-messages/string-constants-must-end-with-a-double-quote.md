# <a name="string-constants-must-end-with-a-double-quote"></a>字符串常量必须以双引号结束
字符串常量必须开始和结束用引号引起来。  
  
 **错误 Id:** BC30648  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   请确保以引号 （'） 结尾的字符串文本。 如果你执行粘贴操作从其他文本编辑器的值，请确保粘贴的字符是有效的引号，而不是一种类似，如"智能"或"大"引号字符 （") 或两个单引号 （'）。
