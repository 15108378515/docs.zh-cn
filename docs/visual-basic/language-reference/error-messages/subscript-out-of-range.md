# <a name="subscript-out-of-range-visual-basic"></a>下标超出范围 (Visual Basic)
数组下标无效，因为它超出了允许的范围。 维度的最低下标值始终是 0，而最高的下标值由`GetUpperBound`该维度的方法。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   更改下标，使它在有效范围内。
