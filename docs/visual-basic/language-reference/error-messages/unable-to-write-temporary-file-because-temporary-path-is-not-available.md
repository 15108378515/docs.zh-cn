[!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] 未能确定临时文件的存储路径。  
  
 **错误 ID:** BC30698  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  重新启动 [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)]。  
  
2.  如果问题依然存在，请重新安装 [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)]。
