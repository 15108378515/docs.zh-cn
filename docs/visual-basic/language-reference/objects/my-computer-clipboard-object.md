提供了用于操控剪贴板的方法。

 有关的方法和属性的信息`My.Computer.Clipboard`对象，请参阅<xref:Microsoft.VisualBasic.MyServices.ClipboardProxy>。  
  
 有关详细信息，请参阅[存储到的数据以及从剪贴板读取](../../../visual-basic/developing-apps/programming/computer-resources/storing-data-to-and-reading-from-the-clipboard.md)。
