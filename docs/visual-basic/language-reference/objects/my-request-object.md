# <a name="myrequest-object"></a>My.Request 对象

## <a name="remarks"></a>备注  
 `My.Request` 对象包含当前 HTTP 请求的相关信息。  
  
 `My.Request` 对象仅适用于 ASP.NET 应用程序。
