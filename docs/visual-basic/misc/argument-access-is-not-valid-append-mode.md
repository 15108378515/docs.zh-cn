# <a name="argument-39access39-is-not-valid-valid-values-for-append-mode-are-39openaccesswrite39-and-39openaccessdefault39"></a>自变量&#39;访问&#39;无效。 追加模式的有效值是&#39;OpenAccess.Write&#39;和&#39;OpenAccess.Default&#39;
访问值在追加模式中无效。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  将值更改为 `OpenAccess.Write` 或 `OpenAccess.Default`。
