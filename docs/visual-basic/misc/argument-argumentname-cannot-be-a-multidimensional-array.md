# <a name="argument-39ltargumentnamegt39-cannot-be-a-multidimensional-array"></a>自变量 &#39;&lt;argumentname&gt;&#39; 不能是多维数组
数组参数的秩大于 1，但它只能是一维数组。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   将参数秩更改为 1。
