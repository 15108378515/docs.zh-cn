# <a name="argument-ltargumentnamegt-cannot-be-an-empty-string-or-nothing"></a>自变量&lt;argumentname&gt;不能为空字符串或 Nothing
该参数需要除空字符串 ("") 或 `Nothing`以外的值。 如果计算值时有误，则可能会出现此结果。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   为参数提供有效值。
