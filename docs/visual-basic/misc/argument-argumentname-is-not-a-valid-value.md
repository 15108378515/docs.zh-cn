# <a name="argument-39ltargumentnamegt39-is-not-a-valid-value"></a>自变量&#39; &lt;argumentname&gt; &#39;不是有效的值
参数的值无效。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  检查参数的值。  
  
2.  如果参数由某个计算提供，则检查该计算所处理的值。
