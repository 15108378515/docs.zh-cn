# <a name="argument-39ltargumentnamegt39-is-not-valid-for-the-array"></a>自变量 &#39;&lt;argumentname&gt;&#39; 不是对于的数组无效
数组包含无效参数。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   检查表达式中参数的拼写是否正确。 拼写错误的变量名可能会隐式创建一个初始化为零的数值变量。
