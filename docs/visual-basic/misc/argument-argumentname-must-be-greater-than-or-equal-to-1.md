# <a name="argument-39ltargumentnamegt39-must-be-greater-than-or-equal-to-1"></a>自变量&#39; &lt;argumentname&gt; &#39;必须大于或等于 1
参数值小于 1。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  确保参数的值在有效范围内。  
  
2.  如果参数由某个计算提供，则检查该计算所处理的值。
