# <a name="argument-ltargumentnamegt-must-be-greater-than-or-equal-to-zero"></a>自变量&lt;argumentname&gt;必须大于或等于零
参数值小于零。 参数值必须大于或等于零。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  确保参数的值大于或等于零。  
  
2.  如果参数由某个计算提供，则检查该计算所处理的值。
