# <a name="argument-cannot-be-less-than-zero"></a>参数不能小于零
提供了一个小于零的值，但参数不能小于零。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   检查有问题的值的计算方式，并确保其正确无误。
