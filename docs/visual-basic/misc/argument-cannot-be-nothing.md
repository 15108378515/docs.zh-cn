# <a name="argument-cannot-be-nothing"></a>参数不能为 Nothing
向必须具有值的参数提供了 null 值。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   你可能试图在未先提供对象的实例的情况下使用对象。 在这种情况下，请使用 `New` 关键字创建实例。  
  
-   检查值是否计算正确。
