# <a name="argument-39path39-is-nothing-or-empty"></a>自变量 &#39;路径 &#39;是 Nothing 或是空
路径名称无效，因为它被初始化为 `Nothing` 或为空。 路径必须具有特定的数据类型和有效的名称。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   添加有效的路径名称。
