# <a name="arguments-are-not-valid"></a>参数无效
传递给函数或子例程的参数无效。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  检查参数以确定它们为何无效。  
  
2.  如果这些参数由计算提供，则检查计算处理的值。
