# <a name="file-ltfilenamegt-could-not-be-found"></a>文件&lt;filename&gt;找不到
无法定位指定的文件。 这可能是提供不正确的文件路径的结果。  
  
 **错误 ID：** BC2001  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   检查文件名和路径是否正确。
