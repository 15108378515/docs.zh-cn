# <a name="option-ltoptionnamegt-can-be-followed-only-by-3939-or-39-39"></a>选项&lt;选项名称&gt;可以后面只能接 &#39; + （& a) #39; 或 &#39;-&#39;
该选项后的文本不是加号 (+) 或减号 (-)。  
  
 **错误 ID：** BC2009  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   将此选项后的文本替换为加号 (+) 以打开该选项。  
  
     — 或 —  
  
-   将此选项后的文本替换为减号 (-) 以关闭该选项。
