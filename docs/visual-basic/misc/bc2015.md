# <a name="the-file-39ltfilenamegt39-is-not-a-text-file"></a>文件 &#39;&lt;filename&gt;&#39; 不是文本文件
无法写入指定的文件，因为它是二进制文件。  
  
 **错误 ID：** BC2015  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   检查你所提供的文件名和路径是否正确。
