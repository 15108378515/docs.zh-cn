# <a name="39end-function39-expected"></a>&#39;结束函数 &#39;预期
`End` 语句必须与它终止的块匹配。 未正确终止 `Function` 过程定义。  
  
 **错误 ID：** BC30027  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   将 `End Function` 语句添加到过程末尾。
