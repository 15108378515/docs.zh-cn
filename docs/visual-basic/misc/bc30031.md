# <a name="fully-qualified-names-for-types-in-namespaces-cannot-be-longer-than-ltnumbergt-characters"></a>命名空间中的类型的完全限定的名不能超过&lt;数&gt;字符
该类型的完全限定名不能超过指定数目的字符。  
  
 **错误 ID：** BC30031  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   缩短类型的名称。
