# <a name="bracketed-identifier-is-missing-closing-3939"></a>用括号括起来的标识符缺少右边&#39;]&#39;
转义名称中的方括号必须配对出现。  
  
 **错误 ID：** BC30034  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   在转义名称的结尾处插入右方括号。
