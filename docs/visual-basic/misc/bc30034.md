# <a name="bracketed-identifier-is-missing-closing-3939"></a>结束 &#39; 缺少括在括号中的标识符]&#39;
转义名称中的方括号必须配对出现。  
  
 **错误 ID：** BC30034  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   在转义名称的结尾处插入右方括号。
