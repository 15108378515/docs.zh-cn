# <a name="maximum-number-of-errors-has-been-exceeded"></a>已超出最大错误数
编译错误的最大数目为 100。 编译器进行此限制，以减少错误报告的大小。  
  
 **错误 ID：** BC30041  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   查看此编译生成的其他错误并尽可能多地解决。 如果源代码包含 100 个以上的错误，则在下一次编译中，可能会看到更多的错误。
