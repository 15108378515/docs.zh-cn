# <a name="too-many-arguments-to-39ltprocedurenamegt39"></a>参数过多 &#39;&lt;过程名称&gt;&#39;
过程调用中提供的参数数量超过为过程定义的参数数量。  
  
 **错误 ID：** BC30057  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   请查阅过程的定义，并从调用中删除多余的参数。
