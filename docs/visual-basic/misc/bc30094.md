# <a name="label-39ltlabelnamegt39-is-already-defined-in-the-current-methodmultiline-lambda"></a>标签&#39;&lt;标签名&gt;&#39;已在当前方法/多行 lambda 中定义
过程中的标签名称重复。 过程中的所有名称均必须唯一。  
  
 **错误 ID：** BC30094  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  重命名重复的标签。
