# “Stop”语句在即时窗口中无效
`Stop` 和 `End` 语句暂停执行，且不允许在调试上下文中使用它们。  
  
 **错误 ID：**BC30122  
  
### 更正此错误  
  
-   不要在“即时”窗口中发出 `Stop` 或 `End` 语句。
