# <a name="39goto39-statements-are-not-valid-in-the-immediate-window"></a>&#39;GoTo&#39;语句中不能使用即时窗口
`GoTo` 语句执行分支，但不允许用于调试上下文。  
  
 **错误 ID：** BC30133  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   不要在“即时” `GoTo`**窗口中发出** 语句。
