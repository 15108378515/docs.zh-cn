# <a name="cannot-create-an-instance-of-module-39ltmodulenamegt39"></a>无法创建模块 &#39; 的实例&lt;modulename&gt;&#39;
模块仅作为单个共享实例存在，不能创建其他实例。  
  
 **错误 ID：** BC30166  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   将模块更改为类，或在 `New` 子句中将其替换为类名。
