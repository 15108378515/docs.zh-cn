# <a name="lttype1gt-39lttypenamegt39-and-lttype2gt-39lttypenamegt39-declared-in-39ltnamegt39-conflict-in-ltnamespacegt-39ltnamespacenamegt39"></a>&lt;type1&gt; &#39;&lt;typename&gt;&#39; 和&lt;type2&gt; &#39;&lt;typename&gt;&#39; 中声明 &#39;&lt;名称&gt;&#39; 中的冲突&lt;命名空间&gt;&#39;&lt;命名空间名称&gt;&#39;
已声明的两个元素（如类和模块）在同一命名空间中具有相同的名称。  
  
 **错误 ID：** BC30175  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   将其中一个元素的声明移到其他命名空间。  
  
-   更改其中一个元素的名称。
