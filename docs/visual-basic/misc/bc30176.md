# <a name="only-one-of-39public39-39private39-39protected39-39friend39-or-39protected-friend39-can-be-specified"></a>只有一个&#39;公共&#39;，&#39;私有&#39;，&#39;受保护&#39;，&#39;友元&#39;，或&#39;Protected Friend&#39;可以指定
在元素声明中发生冲突的可访问性。 只能指定一个可访问性。  
  
 **错误 ID：** BC30176  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   选择一个可访问性，并删除声明中的其他关键字。
