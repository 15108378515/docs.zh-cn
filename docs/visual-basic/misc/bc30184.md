# <a name="39end-enum39-must-be-preceded-by-a-matching-39enum39"></a>&#39;End Enum &#39;前面必须是匹配 &#39;枚举 &#39;
出现 `End Enum` 语句而没有相应的 `Enum` 语句。 `End Enum` 前面必须有相应的 `Enum` 语句。  
  
 **错误 ID：** BC30184  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  确保前面的 `Enum` 语句无拼写错误或有效。  
  
2.  确保 `Enum` 成员的格式正确。
