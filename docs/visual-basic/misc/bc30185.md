# <a name="39enum39-must-end-with-a-matching-39end-enum39"></a>&#39;枚举 &#39;必须以匹配 &#39; 结尾End Enum &#39;
出现 `Enum` 语句而没有相应的 `End Enum` 语句。 必须用 `End Enum` 语句结束枚举。  
  
 **错误 ID：** BC30185  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  检查 `Enum` 成员的格式是否正确。  
  
2.  将 `End Enum` 语句添加到枚举末尾。
