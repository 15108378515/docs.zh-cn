# <a name="comma-expected"></a>需要逗号
参数列表中两个参数之间缺少逗号 (,)。  
  
 **错误 ID：** BC30196  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   确保所有参数之间用逗号分隔。
