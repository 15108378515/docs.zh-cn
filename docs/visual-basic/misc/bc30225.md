# <a name="39option-ltspecifiergt39-statement-can-only-appear-once-per-file"></a>&#39; 选项&lt;说明符&gt;&#39; 每个文件只能一次出现的语句可以
`Option Compare`、 `Option Explicit`和 `Option Strict` 语句只能在同一源文件中各使用一次。  
  
 **错误 ID：** BC30225  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除重复的 `Option` 语句。
