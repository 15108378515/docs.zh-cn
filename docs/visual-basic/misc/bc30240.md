# <a name="39exit39-must-be-followed-by-39sub39-39function39-39property39-39do39-39for39-39while39-39select39-or-39try39"></a>&#39;退出&#39;后面必须跟&#39;子&#39;，&#39;函数&#39;，&#39;属性&#39;，&#39;执行&#39;，&#39;为&#39;，&#39;时&#39;，&#39;选择&#39;，或&#39;重&#39;
`Exit` 语句包含无效的关键字。 `Exit` 必须指定要将控制权从其转移到其后的语句的块，例如 `End While`。  
  
 **错误 ID：** BC30240  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   在 `Exit`后添加有效关键字，或删除 `Exit` 语句。
