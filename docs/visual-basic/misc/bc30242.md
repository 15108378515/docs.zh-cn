# <a name="39ltmodifiergt39-is-not-valid-on-a-method-declaration"></a>&#39;&lt;修饰符&gt;&#39; 在方法声明上无效
`Function` 或 `Sub` 语句包含无效的关键字。  
  
 **错误 ID：** BC30242  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   将无效的关键字从 `Function` 或 `Sub` 语句中删除。
