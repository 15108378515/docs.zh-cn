# <a name="39ltmodifiergt39-is-not-valid-on-an-event-declaration"></a>&#39;&lt;修饰符&gt;&#39; 在事件声明上无效
`Event` 语句包含无效的关键字，如 `ReadOnly`。  
  
 **错误 ID：** BC30243  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  从 `Event` 语句中删除无效的关键字。
