# <a name="39ltmodifiergt39-is-not-valid-on-a-local-constant-declaration"></a>&#39;&lt;修饰符&gt;&#39; 在局部常量声明上无效
过程内的 `Const` 语句包含无效的关键字，如 `Overloads`。  
  
 **错误 ID：** BC30246  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  从 `Const` 语句中删除无效的关键字。
