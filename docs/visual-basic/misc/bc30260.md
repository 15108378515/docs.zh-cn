# <a name="39ltnamegt39-is-already-declared-as-39ltdeclarationgt39-in-this-ltdeclarationspacegt"></a>&#39;&lt;名称&gt;&#39; 已被声明为 &#39;&lt;声明&gt;&#39; 在此&lt;declarationspace&gt;
在同一声明空间（类、模块、接口或结构）中，声明了两个同名编程元素。  
  
 **错误 ID：** BC30260  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   将其中一个声明移动到其他声明空间，或更改其中某个声明元素的名称。
