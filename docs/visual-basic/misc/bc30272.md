# <a name="39ltargumentnamegt39-is-not-a-parameter-of-39ltmethoddeclarationgt39"></a>&#39;&lt;argumentname&gt;&#39; 的参数 &#39; 不是&lt;methoddeclaration&gt;&#39;
过程调用提供一个不是由过程定义的附加参数。  
  
 **错误 ID：** BC30272  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   从参数列表中删除多余的参数。
