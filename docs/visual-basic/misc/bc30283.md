# <a name="39sub-new39-cannot-be-declared-39overrides39"></a>&#39;Sub New&#39;不能声明为&#39;重写&#39;
构造函数显示它重写继承构造函数。 不能重写构造函数。  
  
 **错误 ID：** BC30283  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   从 `Overrides` 声明中删除 `Sub` 关键字。
