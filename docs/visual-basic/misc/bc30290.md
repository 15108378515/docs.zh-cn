# <a name="local-variable-cannot-have-the-same-name-as-the-function-containing-it"></a>局部变量不能与包含它的函数同名
`Dim` 语句指定的某个变量与包含该变量的 `Function` 过程同名。  
  
 **错误 ID：** BC30290  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除变量声明，或更改变量的名称。
