# <a name="lterrorgt-39ltstructurename1gt39-contains-39ltstructurename2gt39"></a>&lt;错误&gt;: &#39; &lt;structurename1&gt; &#39;包含&#39; &lt;structurename2&gt;&#39;
当一个结构将自身指定为自己的成员之一时，会出现循环结构定义。  
  
 **错误 ID：** BC30293  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   更改结构成员的名称。
