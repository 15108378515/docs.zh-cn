# <a name="type-character-39ltcharactergt39-cannot-be-used-in-a-declaration-with-an-explicit-type"></a>类型字符 &#39;&lt;字符&gt;&#39; 不能在具有显式类型的声明
你在显式指定类型时使用了类型字符，如 `$`。  
  
 **错误 ID：** BC30302  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除此类型字符或从声明中删除 `As <Type>` 子句。
