# <a name="39inherits39-statements-must-precede-all-declarations-in-an-interface"></a>&#39;继承&#39;语句必须位于所有声明都之前在接口中
`Inherits` 语句必须是类定义中的第一个非空非注释行。  
  
 **错误 ID：** BC30357  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   移动 `Inherits` 语句，使其位于所有声明之前。
