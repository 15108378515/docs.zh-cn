# <a name="39default39-can-be-applied-to-only-one-property-name-in-a-lttypegt"></a>&#39;默认&#39;可应用于中只有一个属性名&lt;类型&gt;
尝试在类、接口或结构中对多个属性名应用 `Default` 。  
  
 **错误 ID：** BC30359  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除多余的 `Default` 声明，只保留一个。
