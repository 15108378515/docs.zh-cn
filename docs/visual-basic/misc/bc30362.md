# <a name="39ltpropertyname1gt39-cannot-override-39ltpropertyname2gt39-because-they-differ-by-39readonly39-or-39writeonly39"></a>&#39;&lt;propertyname1&gt;&#39; 不能重写 &#39;&lt;propertyname2&gt;&#39; 因为它们的差异在于 &#39;ReadOnly &#39;或 &#39;WriteOnly &#39;
已尝试用仅与第一个属性在 `ReadOnly` 或 `WriteOnly`上不同的第二个属性重写属性。  
  
 **错误 ID：** BC30362  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   请确保属性不仅由 `ReadOnly` 和 `WriteOnly`区分。
