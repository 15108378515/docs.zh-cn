# <a name="39sub-new39-cannot-be-declared-39ltmodifiergt39"></a>&#39;新的子 &#39;不能声明为 &#39;&lt;修饰符&gt;&#39;
已尝试使用修饰符声明 `Sub New` ，例如 `Overrides`，但是无效。  
  
 **错误 ID：** BC30364  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除此修饰符。
