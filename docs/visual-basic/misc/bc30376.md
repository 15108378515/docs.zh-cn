# <a name="39new39-cannot-be-used-on-class-39ltclassnamegt39-because-it-contains-a-39mustoverride39-member-that-has-not-been-overridden"></a>&#39;新 &#39;不在类 &#39; 上使用&lt;classname&gt;&#39; 因为它包含 &#39;MustOverride &#39;不重写的成员
已在包含未被重写的 `New` 成员的类上尝试使用 `MustOverride` 。  
  
 **错误 ID：** BC30376  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除 `New` 语句。
