# <a name="39finally39-can-only-appear-once-in-a-39try39-statement"></a>&#39;最后 &#39;只能出现一次在 &#39;请尝试 &#39;语句
`Finally` 用于完成 `Try...Catch...Finally` 块；因此它只能在块的末尾出现一次。  
  
 **错误 ID：** BC30381  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  找到并删除冗余 `Finally`。
