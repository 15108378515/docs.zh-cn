# <a name="39ltqualifiedmethodgt39-is-not-accessible-in-this-context-because-it-is-39ltmodifiergt39"></a>&#39;&lt;qualifiedmethod&gt; &#39;因为它是不在此上下文中可访问&#39;&lt;修饰符&gt;&#39;
尝试访问一个由于应用了修饰符而在此上下文中不可访问的方法。  
  
 **错误 ID：** BC30390  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除此修饰符。
