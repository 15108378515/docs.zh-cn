# <a name="39ltspecifiergt39-is-not-valid-on-a-structure-declaration"></a>&#39;&lt;说明符&gt;&#39; 在结构声明上无效
你使用了在 `Dim` 声明上无效的说明符，例如 `Override`或 `Structure` 。  
  
 **错误 ID：** BC30395  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除说明符。
