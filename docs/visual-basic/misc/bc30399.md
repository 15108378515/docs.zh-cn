# <a name="39mybase39-cannot-be-used-with-method-39ltmethodnamegt39-because-it-is-declared-39mustoverride39"></a>&#39;MyBase&#39;不能与方法使用&#39; &lt;methodname&gt; &#39;因为它声明为&#39;MustOverride&#39;
试图在声明为 `MyBase` 的方法中使用 `MustOverride`。  
  
 **错误 ID：** BC30399  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除 `MustOverride` 声明。
