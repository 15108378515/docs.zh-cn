# <a name="39ltinterfacename1gt39-cannot-implement-39ltmethodnamegt39-because-there-is-no-matching-ltmethodgt-on-interface-39ltinterfacename2gt39"></a>&#39;&lt;interfacename1&gt;&#39; 不能实现 &#39;&lt;methodname&gt;&#39; 因为不存在匹配&lt;方法&gt;上接口 &#39;&lt;interfacename2&gt;&#39;
当第一个接口继承的接口上没有匹配的方法时，你已尝试在该接口上实现方法。  
  
 **错误 ID：** BC30401  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   请确保你没有键错方法的名称或提供缺少的方法。
