# <a name="39withevents39-variables-must-have-an-39as39-clause"></a>&#39;WithEvents&#39;变量必须具有&#39;作为&#39;子句
未向 `As` 子句提供关键字 `WithEvents`。 将变量声明为可引发事件的特定类。  
  
 **错误 ID：** BC30412  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  添加指定可引发事件的类的 `As` 子句。
