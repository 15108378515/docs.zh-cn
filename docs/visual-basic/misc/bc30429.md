# <a name="39end-sub39-must-be-preceded-by-a-matching-39sub39"></a>&#39;End Sub&#39;前面必须是匹配的&#39;Sub&#39;
代码中的 `End Sub` 语句前面没有与其匹配的 `Sub` 过程定义。  
  
 **错误 ID：** BC30429  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除多余的 `End Sub` 语句。  
  
-   提供缺少的 `Sub` 过程（如果有）。  
  
-   将 `End Sub` 移到代码中的适当位置。
