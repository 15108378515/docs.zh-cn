# <a name="39end-property39-must-be-preceded-by-a-matching-39property39"></a>&#39;End Property&#39;前面必须是匹配的&#39;属性&#39;
你的代码中出现 `End Property` 语句，其前面没有任何匹配的 `Property` 声明。  
  
 **错误 ID：** BC30431  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除多余的 `End Property` 语句。  
  
-   提供缺少的 `Property` 过程（如果有）。  
  
-   将 `End Property` 移到代码中的适当位置。
