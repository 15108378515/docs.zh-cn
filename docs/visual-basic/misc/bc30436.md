# <a name="members-in-a-module-cannot-be-declared-39ltspecifiergt39"></a>不能声明为模块中的成员 （& a) #39;&lt;说明符&gt;&#39;
你使用了在 `Module` 语句内的成员上无效的说明符。 模块永远不能实例化，不支持继承且不能实现接口。  
  
 **错误 ID：** BC30436  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除说明符。
