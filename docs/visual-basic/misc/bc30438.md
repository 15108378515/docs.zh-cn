# <a name="constants-must-have-a-value"></a>常量必须具有值
你声明了一个常量但未对其赋值。  
  
 **错误 ID：** BC30438  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   为常量赋值。
