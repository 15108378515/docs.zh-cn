# <a name="39catch39-must-end-with-a-matching-39end-try39"></a>&#39;Catch &#39;必须以匹配 &#39; 结尾结束 Try &#39;
代码中出现的 `Catch` 语句出现没有匹配的 `End Try` 语句。 `Catch` 语句必须以 `End Try` 语句结束。  
  
 **错误 ID：** BC30441  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  删除 `Catch` 语句。  
  
2.  添加 `End Try` 语句来结束块。
