# <a name="operator-39ltoperatornamegt39-is-not-defined-for-types-39lttype1gt39-and-39lttype2gt39"></a>运算符&#39;&lt;运算符名称&gt;&#39;没有为类型定义&#39; &lt;type1&gt; &#39;和&#39; &lt;type2&gt;&#39;
你已尝试使用二元运算符执行操作，但该操作对于涉及两个返回值的代码元素无效。  
  
 **错误 ID：** BC30452  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   检查这两个元素，并确保它们互相兼容。
