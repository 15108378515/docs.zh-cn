# <a name="classes-cannot-be-declared-39ltspecifiergt39"></a>类不能声明 &#39;&lt;说明符&gt;&#39;
你试图使用无效的说明符声明 `Class` 或 `Module` 。  
  
 **错误 ID：** BC30461  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除说明符。
