# <a name="39myclass39-cannot-be-used-outside-of-a-class"></a>&#39;MyClass&#39;不能在类的外部
你已在类结构外部使用 `MyClass` 关键字。  
  
 **错误 ID：** BC30470  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   将 `MyClass` 声明移动到类结构中。
