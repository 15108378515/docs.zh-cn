# <a name="shared-39sub-new39-cannot-be-declared-39ltspecifiergt39"></a>共享&#39;Sub New&#39;不能声明为&#39;&lt;说明符&gt;&#39;
试图在 `Shared Sub New` 声明中使用说明符，而这样是无效的。  
  
 **错误 ID：** BC30480  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除说明符。
