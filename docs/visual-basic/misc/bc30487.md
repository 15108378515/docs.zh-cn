# <a name="operator-39ltoperatornamegt39-is-not-defined-for-type-39lttypenamegt39"></a>运算符 &#39;&lt;运算符名称&gt;&#39; 没有为类型 &#39; 定义&lt;typename&gt;&#39;
尝试对某个类型（如`+`）使用没有为其定义的一元运算符（ `-`、 `Not`或 `String`）。  
  
 **错误 ID：** BC30487  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   将一元运算符替换为适用于该表达式的相应运算符。
