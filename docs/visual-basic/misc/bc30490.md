# <a name="39default39-cannot-be-combined-with-39ltspecifiergt39"></a>&#39;默认&#39;不能与组合&#39;&lt;说明符&gt;&#39;
试图将 `Default` 关键字与此上下文中无效的说明符（例如 `Shared` 或 `Private`组合起来。  
  
 **错误 ID：** BC30490  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除 `Default`。  
  
     -或-  
  
-   删除冲突的说明符。
