# <a name="39shared39-cannot-be-combined-with-39ltspecifiergt39-on-a-method-declaration"></a>&#39;共享 &#39;不能与组合 &#39;&lt;说明符&gt;&#39; 方法声明上
你试图将 `Shared` 修饰符与方法声明上的此类组合中无效的说明符（例如 `Overridable`、 `NotOverridable`或 `MustOverride`）组合。  
  
 **错误 ID：** BC30501  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除说明符。
