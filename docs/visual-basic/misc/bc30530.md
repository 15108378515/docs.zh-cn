# <a name="parameter-cannot-have-the-same-name-as-its-defining-function"></a>参数不能与它的定义函数同名
你在参数及其定义函数中使用了相同名称。  
  
 **错误 ID：** BC30530  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   检查函数内的参数名并更改冲突参数的名称。
