# <a name="39ltnamespacenamegt39-is-ambiguous"></a>&#39;&lt;namespacename&gt;&#39; 不明确
你提供的名称不明确，因此与另一个名称冲突。 [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] 编译器没有任何冲突解决规则；你必须自己区分名称。  
  
 **错误 ID：** BC30554  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   完全限定名称。
