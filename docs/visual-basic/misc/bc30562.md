# <a name="39ltnamegt39-is-ambiguous-between-declarations-in-modules-39ltmodulename1gt39-and-39ltmodulename2gt39"></a>&#39;&lt;名称&gt;&#39;模块中的声明之间不明确&#39; &lt;modulename1&gt; &#39;和&#39; &lt;modulename2&gt;&#39;
你提供的名称不明确，因此与另一个名称冲突。 Visual Basic 编译器没有任何冲突解决规则;你必须自己区分名称。  
  
 **错误 ID：** BC30562  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   完全限定名称。
