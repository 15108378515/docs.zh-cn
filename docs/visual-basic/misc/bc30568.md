# <a name="array-initializer-has-ltnumbergt-too-many-elements"></a>数组初始值设定项&lt;数&gt;元素过多
你的数组初始值设定项包含过多元素。  
  
 **错误 ID：** BC30568  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   使用 `ReDim` 更改数组大小。
