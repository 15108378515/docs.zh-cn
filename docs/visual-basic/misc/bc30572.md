# <a name="alias-39ltnamespacenamegt39-is-already-declared"></a>别名&#39; &lt;namespacename&gt; &#39;已声明
你已尝试使用 `Imports` 导入已声明的命名空间。  
  
 **错误 ID：** BC30572  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   检查你尝试导入的命名空间。
