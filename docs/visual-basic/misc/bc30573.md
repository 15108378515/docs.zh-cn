# <a name="xml-namespace-prefix-39prefix39-is-already-declared"></a>XML 命名空间前缀 &#39; 前缀 &#39;已声明
`Imports` 语句定义的 XML 命名空间前缀与之前定义的 XML 命名空间前缀匹配。  
  
 **错误 ID：** BC30573  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   对 XML 命名空间前缀使用唯一名称。
