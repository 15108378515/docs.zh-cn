# <a name="omitted-argument-cannot-match-a-paramarray-parameter"></a>省略的参数不能匹配 ParamArray 参数
省略了匹配 `ParamArray` 参数所需的参数。 `ParamArray`（它使你可以将任意数量的参数传递给过程）只能在参数列表的最后一个参数上使用。  
  
 **错误 ID：** BC30588  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   提供省略的参数。
