# <a name="39withevents39-variable-does-not-raise-any-events"></a>&#39;WithEvents &#39;变量不会引发任何事件
使用 `WithEvents` 关键字声明的对象变量不引用可以引发事件的类实例。  
  
 **错误 ID：** BC30591  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  确保对象变量引用可以引发事件的类实例。
