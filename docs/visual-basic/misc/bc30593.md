# <a name="variables-in-modules-cannot-be-declared-39ltspecifiergt39"></a>模块中的变量不能声明为&#39;&lt;说明符&gt;&#39;
你在 `MustInherit`语句中的某个变量上使用了一个说明符（如 `Module` ）。 模块永远不能实例化，不支持继承且不能实现接口。  
  
 **错误 ID：** BC30593  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除说明符。
