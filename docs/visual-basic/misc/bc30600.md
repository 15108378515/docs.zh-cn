# <a name="39withevents39-variable-does-not-raise-any-instance-events-that-are-accessible-to-39ltcontainernamegt39"></a>&#39;WithEvents&#39;变量不会引发可访问的任何实例事件&#39; &lt;containername&gt;&#39;
正在引用的对象不会引发可访问的事件。  
  
 **错误 ID：** BC30600  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  修改事件的访问修饰符，或选择在当前上下文中可访问的事件。
