# <a name="39option-strict39-can-be-followed-only-by-39on39-or-39off39"></a>&#39;选项 Strict&#39;可以后面只能接&#39;上&#39;或&#39;关闭&#39;
`Option Strict` 语句必须指定 `On` 或 `Off`。 其他所有值均无效。  
  
 **错误 ID：** BC30620  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   为 `On` 语句指定 `Off` 或 `Option Strict` 。
