# <a name="39end-namespace39-must-be-preceded-by-a-matching-39namespace39"></a>&#39;结束 Namespace &#39;前面必须是匹配 &#39;Namespace &#39;
命名空间块以 `Namespace` 语句开头且以 `End Namespace` 构造结尾。  
  
 **错误 ID：** BC30623  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   确保命名空间块以 `Namespace` 语句开头。
