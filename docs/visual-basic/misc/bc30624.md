# <a name="39structure39-statement-must-end-with-a-matching-39end-structure39"></a>&#39;结构&#39;语句必须以匹配结束&#39;结束结构&#39;
结构以 `Structure` 语句开头且以 `End Structure` 构造结尾。  
  
 **错误 ID：** BC30624  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   请确保结构块以 `End Structure` 语句结尾。
