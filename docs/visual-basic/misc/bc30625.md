# <a name="39module39-statement-must-end-with-a-matching-39end-module39"></a>&#39;模块&#39;语句必须以匹配结束&#39;结束模块&#39;
Module 块必须结束`End Module`语句。  
  
 **错误 ID：** BC30625  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   将 `End Module` 语句添加到 `Module` 块末尾。
