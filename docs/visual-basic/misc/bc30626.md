# <a name="39namespace39-statement-must-end-with-a-matching-39end-namespace39"></a>&#39;Namespace&#39;语句必须以匹配结束&#39;结束 Namespace&#39;
`Namespace` 块必须以 `End Namespace` 语句结束。  
  
 **错误 ID：** BC30626  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   将 `End Namespace` 语句添加到 `Namespace` 块末尾。
