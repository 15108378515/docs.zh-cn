# <a name="39option39-statements-must-precede-any-declarations-or-39imports39-statements"></a>&#39;选项&#39;语句必须位于任何声明之前或&#39;导入&#39;语句
`Option` 语句应为你的源代码中第一个非注释语句。  
  
 **错误 ID：** BC30627  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   将 `Option` 语句移动到源代码顶部 `Imports` 和 `Namespace` 语句之前。
