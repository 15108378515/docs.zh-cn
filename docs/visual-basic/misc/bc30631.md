# <a name="39get39-statement-must-end-with-a-matching-39end-get39"></a>&#39;Get &#39;语句必须以匹配 &#39; 结束结束 Get &#39;
`Get` 属性过程必须以 `End Get` 语句结束。  
  
 **错误 ID：** BC30631  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   确保 `Get` 属性过程以 `End Get` 构造结束。
