# <a name="39set39-statement-must-end-with-a-matching-39end-set39"></a>&#39;集 &#39;语句必须以匹配 &#39; 结束结束集 &#39;
`Set` 属性过程必须以 `End Set` 语句结束。  
  
 **错误 ID：** BC30633  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   确保 `Set` 属性过程以 `End Set` 构造结束。
