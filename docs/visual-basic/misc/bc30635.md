# <a name="39readonly39-and-39writeonly39-cannot-be-combined"></a>&#39;ReadOnly &#39;和 &#39;WriteOnly &#39;不能组合
`ReadOnly` 和 `WriteOnly` 修饰符互相排斥。  
  
 **错误 ID：** BC30635  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   指定 `ReadOnly` 或 `WriteOnly`，但不能同时指定这两者。
