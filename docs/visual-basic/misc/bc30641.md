# <a name="39byval39-and-39byref39-cannot-be-combined"></a>&#39;ByVal&#39;和&#39;ByRef&#39;不能组合
`ByVal` 和 `ByRef` 关键字互斥。  
  
 **错误 ID：** BC30641  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   指定 `ByVal` 或 `ByRef`，但不能同时指定这两者。
