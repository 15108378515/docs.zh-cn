# <a name="39optional39-and-39paramarray39-cannot-be-combined"></a>&#39;可选&#39;和&#39;ParamArray&#39;不能组合
`Optional` 和 `ParamArray` 修饰符互相排斥。  
  
 **错误 ID：** BC30642  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   指定 `ParamArray` 或 `Optional`，但不能同时指定这两者。
