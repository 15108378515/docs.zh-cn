# <a name="property-39ltpropertynamegt39-is-of-an-unsupported-type"></a>属性 &#39;&lt;propertyname&gt;&#39; 的类型不受支持
引用的程序集中的属性包含不能表示的类型。  
  
 **错误 ID：** BC30643  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   指定可与应用程序一同使用的类型。
