# <a name="39lttypenamegt39-is-an-unsupported-type"></a>&#39;&lt;typename&gt; &#39;是类型不受支持
不能通过 Visual Studio 表示指定的类型。  
  
 **错误 ID：** BC30649  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   使用受支持的类型。  
  
-   开发新类型。
