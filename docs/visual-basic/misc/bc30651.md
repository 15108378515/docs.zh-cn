# <a name="ltpropertygt-parameters-cannot-be-declared-39byref39"></a>&lt;属性&gt;参数不能声明 &#39;ByRef &#39;
属性参数必须使用 `ByVal` 修饰符进行声明。  
  
 **错误 ID：** BC30651  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   将属性参数定义为 `ByVal`。
