# <a name="field-39ltfieldnamegt39-is-of-an-unsupported-type"></a>字段&#39; &lt;fieldname&gt; &#39;的不受支持的类型
字段声明为类型中不能表示由 Visual Studio。  
  
 **错误 ID：** BC30656  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  将字段定义为支持的类型。  
  
2.  开发一种可以描述字段中数据的新类型。
