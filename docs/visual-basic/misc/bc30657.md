# <a name="39ltprocedurenamegt39-has-a-return-type-that-is-not-supported-or-parameter-types-that-are-not-supported"></a>&#39;&lt;过程名称&gt;&#39;有返回类型不受支持或不支持的参数类型
使用参数或不能通过 Visual Studio 中表示的返回类型声明的过程。  
  
 **错误 ID：** BC30657  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   将返回类型或参数定义为受支持的类型。  
  
-   开发用于描述此过程所使用数据的新类型。
