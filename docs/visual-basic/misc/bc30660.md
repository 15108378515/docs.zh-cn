# <a name="attributes-cannot-be-applied-to-local-variables"></a>特性不能应用于局部变量
特性不能应用于局部变量。  
  
 **错误 ID：** BC30660  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  在下一更高级别的作用域声明此变量，例如类或模块级别。
