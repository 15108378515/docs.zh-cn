# <a name="conversions-from-39lttypename1gt39-to-39lttypename2gt39-must-be-explicit"></a>从转换 &#39;&lt;typename1&gt;&#39; 为 &#39;&lt;typename2&gt;&#39; 必须显式
无法隐式实现此转换。  
  
 **错误 ID：** BC30664  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   使用 `CType`或某个特定转换函数，如 `CInt` 或 `CDec`。
