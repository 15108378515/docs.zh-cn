# <a name="39synclock39-statement-must-end-with-a-matching-39end-synclock39"></a>&#39;SyncLock &#39;语句必须以匹配 &#39; 结束End SyncLock &#39;
`SyncLock` 块以 `SyncLock` 关键字开头，以 `End SyncLock` 构造结尾。  
  
 **错误 ID：** BC30675  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   请确保 `SyncLock` 块以 `End SyncLock` 构造结尾。
