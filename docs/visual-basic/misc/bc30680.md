# <a name="39end-region39-must-be-preceded-by-a-matching-39region39"></a>&#39;#End 区域&#39;前面必须是匹配的&#39;#Region&#39;
当使用 `#Region` 代码编辑器的大纲显示功能时，可使用 [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)] 指定要展开或折叠的代码块。 `#Region` 语句的开始和结束必须在同一个代码块中。  
  
 **错误 ID：** BC30680  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   在相应的 `#Region`  `#End``Region` 。
