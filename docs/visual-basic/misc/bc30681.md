# <a name="39region39-statement-must-end-with-a-matching-39end-region39"></a>&#39;#Region&#39;语句必须以匹配结束&#39;#End 区域&#39;
使用`#Region`指令指定的可以展开或折叠时使用的 Visual Studio 代码编辑器的大纲显示功能的代码块。 `#Region` 语句的开始和结束必须在同一个代码块中。  
  
 **错误 ID:** BC30681  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  在代码中 `#End Region` 语句后的适当位置插入 `#Region`
