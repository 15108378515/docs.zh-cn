# <a name="39region39-statement-must-end-with-a-matching-39end-region39"></a>&#39; #Region &#39;语句必须以匹配 #39; #End 区域 &#39; 结束
使用 `#Region` 指令指定使用 [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)] 代码编辑器的大纲显示功能时要展开或折叠的代码块。 `#Region` 语句的开始和结束必须在同一个代码块中。  
  
 **错误 ID:** BC30681  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  在代码中 `#End Region` 语句后的适当位置插入 `#Region`
