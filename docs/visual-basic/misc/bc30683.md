# <a name="39inherits39-statement-must-precede-all-declarations-in-a-class"></a>&#39;继承&#39;语句必须位于所有声明都之前类中
`Inherits` 语句必须位于类定义的开头。  
  
 **错误 ID：** BC30683  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   将 `Inherits` 语句移动到类的顶部。
