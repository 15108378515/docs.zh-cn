# <a name="events-in-interfaces-cannot-be-declared-39ltimplementsgt39"></a>不能声明为接口中的事件 （& a) #39;&lt;实现&gt;&#39;
在接口中声明的事件不能实现其他接口的事件。  
  
 **错误 ID：** BC30688  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  删除 `Implements` 语句。  
  
2.  实现类或结构中的事件。
