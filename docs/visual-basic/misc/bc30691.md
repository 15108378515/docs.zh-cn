# <a name="39lttypenamegt39-is-a-type-in-39ltcontainernamegt39-and-cannot-be-used-as-an-expression"></a>&#39;&lt;typename&gt; &#39;是中的类型&#39; &lt;containername&gt; &#39;和不能用作表达式
使用中的其中一个名称已声明为类型的名称，且不能在表达式中使用。  
  
 **错误 ID：** BC30691  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   对成员进行唯一命名。  
  
-   用方括号括起成员名称。
