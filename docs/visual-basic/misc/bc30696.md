# <a name="39ltmembername1gt39-and-39ltmembername2gt39-cannot-overload-each-other-because-they-differ-only-by-the-types-of-optional-parameters"></a>&#39;&lt;m e 1&gt; &#39;和&#39; &lt;m e 2&gt; &#39;无法相互重载，因为它们的差异仅在于可选参数的类型
重载的成员必须为一个或多个所需参数提供不同数据类型。  
  
 **错误 ID：** BC30696  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   将至少一个所需参数定义为不同的数据类型。
