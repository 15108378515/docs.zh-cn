# <a name="39ltmembername1gt39-cannot-override-39ltmembername2gt39-because-they-differ-by-the-types-of-optional-parameters"></a>&#39;&lt;m e 1&gt;&#39; 不能重写 &#39;&lt;m e 2&gt;&#39; 因为它们在可选参数的类型上存在差异
重载的成员必须为一个或多个所需参数提供不同数据类型。  
  
 **错误 ID：** BC30697  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   将至少一个所需参数定义为不同的数据类型。
