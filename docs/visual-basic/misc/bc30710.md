# <a name="end-of-expression-expected"></a>应为表达式结尾
在否则完成表达式的末尾找到额外字符。  
  
 **错误 ID：** BC30710  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   验证对象或语句的语法。
