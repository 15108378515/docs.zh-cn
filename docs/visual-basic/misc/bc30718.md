# <a name="39select39-statements-are-not-valid-in-the-immediate-window"></a>&#39;选择&#39;语句中不能使用即时窗口
`Select` 语句只能在源代码中使用。  
  
 **错误 ID：** BC30718  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   从调试代码中删除 `Select` 语句。
