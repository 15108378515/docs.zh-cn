# <a name="39on-error39-statements-are-not-valid-in-the-immediate-window"></a>&#39; 上错误 &#39;语句中不能使用即时窗口
`On Error` 语句只能在源代码中使用。  
  
 **错误 ID：** BC30720  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   从调试代码中删除 `On Error` 语句。
