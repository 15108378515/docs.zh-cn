# <a name="39throw39-statements-are-not-valid-in-the-immediate-window"></a>&#39;引发&#39;语句中不能使用即时窗口
`Throw` 语句只能在源代码中使用。  
  
 **错误 ID：** BC30725  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   从调试代码中删除 `Throw` 语句。
