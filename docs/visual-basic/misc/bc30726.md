# <a name="39with39-contexts-and-statements-are-not-valid-in-debug-windows"></a>&#39;使用 &#39;上下文语句不能和在调试窗口中
`With` 语句只能在源代码中使用。  
  
 **错误 ID：** BC30726  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   从调试代码中删除 `With` 语句。
