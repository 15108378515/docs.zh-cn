# <a name="39addressof39-expressions-are-not-valid-in-debug-windows"></a>&#39;AddressOf &#39;表达式在调试窗口中不可有效
`AddressOf` 语句只能在源代码中使用。  
  
 **错误 ID：** BC30731  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  从调试代码中删除 `AddressOf` 语句。
