# <a name="39ltnamegt39-is-already-declared-as-a-parameter-of-this-method"></a>&#39;&lt;名称&gt;&#39; 已被声明为此方法的一个参数
已为此方法声明了指定的参数名。  
  
 **错误 ID：** BC30734  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   重命名参数，以使其唯一。
