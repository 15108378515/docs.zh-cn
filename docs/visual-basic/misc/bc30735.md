# <a name="type-in-a-module-cannot-be-declared-39ltspecifiergt39"></a>模块中的类型不能声明为 &#39;&lt;说明符&gt;&#39;
在模块中声明了具有无效修饰符的类型。  
  
 **错误 ID：** BC30735  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   不要对模块中声明的类型使用 `Protected` 或 `Protected Friend` 修饰符。
