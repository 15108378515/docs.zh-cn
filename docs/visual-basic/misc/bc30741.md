# <a name="39lttypename1gt39-cannot-be-converted-to-39lttypename2gt39"></a>&#39;&lt;typename1&gt; &#39;不能转换为&#39; &lt;typename2&gt;&#39;
在两个不兼容的类型之间尝试执行了类型转换。  
  
 **错误 ID：** BC30741  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   验证正在被强制转换的类型。
