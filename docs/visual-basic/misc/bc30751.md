# <a name="unable-to-get-type-information-for-39lttypenamegt39"></a>无法获取类型信息 &#39;&lt;typename&gt;&#39;
引用了公共语言运行时未加载的类型。  
  
 **错误 ID：** BC30751  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   重构代码，使该类型定义在当前范围内。
