# <a name="referenced-39ltmembernamegt39-has-a-value-of-39nothing39"></a>引用&#39; &lt;membername&gt; &#39;的值为&#39;执行任何操作&#39;
引用的项未被分配值，或分配的值为 `Nothing`。  
  
 **错误 ID：** BC30759  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   验证项名称的拼写及设置是否都正确。
