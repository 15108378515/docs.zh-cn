# <a name="the-targeted-version-of-the-net-compact-framework-does-not-support-the-39end39-statement"></a>.NET Compact Framework 的目标的版本不支持 &#39;结束 &#39;语句
你正在使用的 .NET Compact Framework 的版本不支持 `End` 语句。  
  
 **错误 ID：** BC30769  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除该语句。
