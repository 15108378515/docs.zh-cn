# <a name="39continue39-statements-are-not-valid-in-the-immediate-window"></a>&#39;继续&#39;语句中不能使用即时窗口
`Continue` 语句只能在源代码中使用。  
  
 **错误 ID：** BC30780  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   从调试代码中删除 `Continue` 语句。
