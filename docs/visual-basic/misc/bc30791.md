# <a name="39declare39-statements-in-a-structure-cannot-be-declared-39ltspecifiergt39"></a>&#39;声明&#39;结构中的语句不能声明&#39;&lt;说明符&gt;&#39;
`Declare` 声明具有在 `Structure` 声明中无效的说明符。  
  
 **错误 ID：** BC30791  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除说明符。
