# <a name="39type39-statements-are-no-longer-supported-use-39structure39-statements-instead"></a>&#39;类型&#39;语句不再受支持;使用&#39;结构&#39;语句改为
`Type` 关键字不能再用于定义用户定义的类型。  
  
 **错误 ID：** BC30802  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   使用 `Structure` 语句来定义新类型。
