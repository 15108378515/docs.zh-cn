# <a name="39currency39-is-no-longer-a-supported-type-use-the-39decimal39-type-instead"></a>&#39;货币 &#39;不再支持的类型;使用 &#39; 小数 &#39;类型
`Currency` 数据类型已由 `Decimal` 数据类型代替。  
  
 **错误 ID：** BC30803  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   使用 `Decimal` 数据类型而非 `Currency`。
