# <a name="39let39-and-39set39-assignment-statements-are-no-longer-supported"></a>&#39;让 &#39;和 &#39;集 &#39;不再支持赋值语句
不能在赋值中使用`Let` 和 `Set` 。  
  
 **错误 ID：** BC30807  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   省略赋值语句中的 `Let` 和 `Set` ；例如：  
  
     `X = 2`
