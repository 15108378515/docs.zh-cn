# <a name="39wend39-statements-are-no-longer-supported-use-39end-while39-statements-instead"></a>&#39;Wend&#39;语句不再受支持;使用&#39;End While&#39;语句改为
`End While` 构建可替换 `Wend` 关键字。  
  
 **错误 ID:** BC30809  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   使用 `End While` 结束 `While` 块。
