# <a name="39on-goto39-and-39on-gosub39-statements-are-no-longer-supported"></a>&#39; 上转到 &#39;和 &#39; 上 GoSub &#39;不再支持语句
将变量或表达式的值用于 `GoTo` 和 `GoSub` 语句以控制程序流不再有效。  
  
 **错误 ID:** BC30817  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   重构应用程序以使用 `If...Then...Else` 或 `Case` 语句。
