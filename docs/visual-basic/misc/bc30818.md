# <a name="39ltfunctiongt39-is-not-declared"></a>&#39;&lt;函数&gt;&#39;未声明
\<函数 > 未声明。 函数已移到 Microsoft.VisualBasic 命名空间。  
  
 **错误 ID：** BC30818  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   使用 `Microsoft.VisualBasic` 命名空间中的适当函数。
