# <a name="39ltmathfunction1gt39-is-not-declared"></a>&#39;&lt;mathfunction1&gt; &#39;未声明
\<mathfunction1 > 未声明。 函数已移动到 System.Math 类，现称\<mathfunction2 >。  
  
 在早期版本中固有的 Visual Basic 的几个函数均已移至<xref:System.Math?displayProperty=nameWithType>命名空间。 这使得其功能更普遍适用于所有编程语言。  
  
 **错误 ID：** BC30819  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   使用中声明的方法<xref:System.Math?displayProperty=nameWithType>。
