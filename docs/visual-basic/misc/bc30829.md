# <a name="39get39-statements-are-no-longer-supported"></a>&#39;Get &#39;不再支持语句
不再支持`Get` 语句。 文件 I/O 功能在 `Microsoft.VisualBasic` 命名空间中可用。  
  
 文件操作不支持`Get` ，只能在属性过程语法中使用它。  
  
 **错误 ID：** BC30829  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  使用 `System.IO`、 `FileSystemObject`和 [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] 运行时函数的成员执行文件操作。
