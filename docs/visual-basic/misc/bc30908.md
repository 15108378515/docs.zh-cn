# <a name="type-39lttypenamegt39-cannot-inherit-from-a-type-nested-within-it"></a><span data-ttu-id="03e9e-102">类型 '&lt;typename&gt;' 不能从其内部嵌套的类型继承</span><span class="sxs-lookup"><span data-stu-id="03e9e-102">Type &#39;&lt;typename&gt;&#39; cannot inherit from a type nested within it</span></span>
<span data-ttu-id="03e9e-103">类或接口定义包括[继承语句](../../visual-basic/language-reference/statements/inherits-statement.md)，它指定一种类型嵌套在其中。</span><span class="sxs-lookup"><span data-stu-id="03e9e-103">A class or interface definition includes an [Inherits Statement](../../visual-basic/language-reference/statements/inherits-statement.md) that specifies a type nested within it.</span></span>  
  
 <span data-ttu-id="03e9e-104">继承必须采用线性方式，而不能采用循环方式。</span><span class="sxs-lookup"><span data-stu-id="03e9e-104">Inheritance must be linear, not circular.</span></span> <span data-ttu-id="03e9e-105">类型不能从它本身继承的类型继承。</span><span class="sxs-lookup"><span data-stu-id="03e9e-105">A type cannot inherit from a type that inherits from it.</span></span>  
  
 <span data-ttu-id="03e9e-106">一个相关的限制为：类型不能从尚未定义的类型中继承。</span><span class="sxs-lookup"><span data-stu-id="03e9e-106">A related restriction is that a type cannot inherit from a type that is not yet defined.</span></span> <span data-ttu-id="03e9e-107">继承涉及到重用基类成员的能力，这反过来又要求这些成员已定义。</span><span class="sxs-lookup"><span data-stu-id="03e9e-107">Inheritance involves the ability to reuse members of the base class, which in turn requires that these members be defined.</span></span> <span data-ttu-id="03e9e-108">因此， [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] ，如下面的示例代码将无法编译。</span><span class="sxs-lookup"><span data-stu-id="03e9e-108">Therefore, [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] cannot compile code such as the following example.</span></span>  
  
```  
Public Class outerClass  
    ' The following statement is INVALID because innerClass is not defined.  
    Inherits innerClass  
    Public Class innerClass  
        Public Sub doSomething()  
        End Sub  
    End Class  
End Class  
```
