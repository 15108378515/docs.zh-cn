# <a name="39ltderivedtypenamegt39-cannot-inherit-from-lttypegt-39ltconstructedbasetypenamegt39-because-it-expands-the-access-of-type-39ltinternaltypenamegt39-to-ltregiongt-39ltregionnamegt39"></a><span data-ttu-id="896a4-102">&#39;&lt;derivedtypename&gt;&#39; 不能继承自&lt;类型&gt;&#39;&lt;constructedbasetypename&gt;&#39; 因为它将扩展的访问权限的类型 &#39;&lt;internaltypename&gt;&#39; 为&lt;区域&gt;&#39;&lt;regionname&gt;&#39;</span><span class="sxs-lookup"><span data-stu-id="896a4-102">&#39;&lt;derivedtypename&gt;&#39; cannot inherit from &lt;type&gt; &#39;&lt;constructedbasetypename&gt;&#39; because it expands the access of type &#39;&lt;internaltypename&gt;&#39; to &lt;region&gt; &#39;&lt;regionname&gt;&#39;</span></span>
<span data-ttu-id="896a4-103">派生的类或接口尝试通过将内部类型作为基类或接口的类型参数来扩展其访问级别。</span><span class="sxs-lookup"><span data-stu-id="896a4-103">A derived class or interface attempts to expand the access level of an internal type by using it as a type argument to a base class or interface.</span></span>  
  
 <span data-ttu-id="896a4-104">以下代码会生成此错误。</span><span class="sxs-lookup"><span data-stu-id="896a4-104">The following code can generate this error.</span></span>  
  
```  
Public Class containingClass  
    Public Class baseClass(Of t)  
    End Class  
    Friend Class derivedClass  
        Inherits baseClass(Of internalStructure)  
    End Class  
    Private Structure internalStructure  
        Dim firstMember As Integer  
    End Structure  
End Class  
```  
  
 <span data-ttu-id="896a4-105">不允许 `containingClass` 外部的代码访问 `internalStructure`。</span><span class="sxs-lookup"><span data-stu-id="896a4-105">Code outside `containingClass` is not allowed to access `internalStructure`.</span></span> <span data-ttu-id="896a4-106">但是，可从同一程序集中的任何代码访问 `derivedClass` 。</span><span class="sxs-lookup"><span data-stu-id="896a4-106">However, `derivedClass` can be accessed from any code in the same assembly.</span></span> <span data-ttu-id="896a4-107">因此，如果 `derivedClass` 使用 `baseClass` 作为类型参数，则它将不能继承 `internalStructure` ，因为这样会将 `internalStructure` 公开到整个代码定义区域。</span><span class="sxs-lookup"><span data-stu-id="896a4-107">Therefore, `derivedClass` cannot inherit `baseClass` if it uses `internalStructure` as a type argument, because that could expose `internalStructure` throughout the defining code region.</span></span>  
  
 <span data-ttu-id="896a4-108">**错误 ID：** BC30921</span><span class="sxs-lookup"><span data-stu-id="896a4-108">**Error ID:** BC30921</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="896a4-109">更正此错误</span><span class="sxs-lookup"><span data-stu-id="896a4-109">To correct this error</span></span>  
  
-   <span data-ttu-id="896a4-110">调整类或接口的访问级别，这样派生的类型将不会扩展内部类型的访问级别。</span><span class="sxs-lookup"><span data-stu-id="896a4-110">Adjust the access levels of the classes or interfaces so that the derived type does not expand the access level of the internal type.</span></span>  
  
     <span data-ttu-id="896a4-111">- 或 -</span><span class="sxs-lookup"><span data-stu-id="896a4-111">-or-</span></span>  
  
-   <span data-ttu-id="896a4-112">如果你不能调整访问级别，请勿在构建基类或接口时将内部类型用作类型参数。</span><span class="sxs-lookup"><span data-stu-id="896a4-112">If you cannot adjust the access levels, then do not use the internal type as a type argument when constructing the base class or interface.</span></span>
