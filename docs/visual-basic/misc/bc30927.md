# <a name="39mustoverride39-cannot-be-specified-on-39ltprocedurenamegt39-because-it-is-in-a-partial-type-that-is-declared-39notinheritable39-in-another-partial-definition"></a><span data-ttu-id="17b34-102">&#39;MustOverride &#39;不能对指定 &#39;&lt;过程名称&gt;&#39; 因为它是在分部类型声明 &#39;NotInheritable &#39;在另一个分部定义</span><span class="sxs-lookup"><span data-stu-id="17b34-102">&#39;MustOverride&#39; cannot be specified on &#39;&lt;procedurename&gt;&#39; because it is in a partial type that is declared &#39;NotInheritable&#39; in another partial definition</span></span>
<span data-ttu-id="17b34-103">在类中将过程或属性声明为 `MustOverride` ，该类在多个分部声明中定义，但其中一个分部定义为该类指定了 `NotInheritable` 。</span><span class="sxs-lookup"><span data-stu-id="17b34-103">A procedure or property is declared as `MustOverride` within a class that is defined in multiple partial declarations, but one of the partial definitions specifies `NotInheritable` for the class.</span></span>  
  
 <span data-ttu-id="17b34-104">在多个分部声明间划分类的定义时，编译器会将该类视为它的所有分部声明的联合。</span><span class="sxs-lookup"><span data-stu-id="17b34-104">When you divide the definition of a class among several partial declarations, the compiler treats the class as the union of all its partial declarations.</span></span> <span data-ttu-id="17b34-105">这不仅适用于成员，还适用于实现、继承和访问级别。</span><span class="sxs-lookup"><span data-stu-id="17b34-105">This applies not only to the members but also to the implementation, inheritance, and access level.</span></span>  
  
 <span data-ttu-id="17b34-106">若要重写过程或属性，类必须从基类继承它。</span><span class="sxs-lookup"><span data-stu-id="17b34-106">To override a procedure or property, a class must inherit it from a base class.</span></span> <span data-ttu-id="17b34-107">因此，若要在基类中为过程或属性指定 `MustOverride` ，必须为类指定 `MustInherit` 。</span><span class="sxs-lookup"><span data-stu-id="17b34-107">Therefore, to specify `MustOverride` for a procedure or property in a base class, you must specify `MustInherit` for the class.</span></span> <span data-ttu-id="17b34-108">不能为同一个类同时指定 `MustInherit` 和 `NotInheritable` ，因为它们互相矛盾。</span><span class="sxs-lookup"><span data-stu-id="17b34-108">Because they are mutually contradictory, you cannot specify both `MustInherit` and `NotInheritable` for the same class.</span></span>  
  
 <span data-ttu-id="17b34-109">**错误 ID：** BC30927</span><span class="sxs-lookup"><span data-stu-id="17b34-109">**Error ID:** BC30927</span></span>
