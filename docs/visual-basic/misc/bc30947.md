# <a name="suspend-request-is-pending"></a>挂起请求被挂起
在 Visual Studio 调试器中，表达式指定了一个过程调用，但存在挂起线程的请求。  
  
 调试器不会尝试在未处于活动状态的线程上调用过程。  
  
 **错误 ID：** BC30947  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   如果可能，请确定挂起线程的请求的源，以便可以防止它再次出现。  
  
-   终止并重新启动调试。
