# <a name="wrong-number-of-type-arguments-passed-to-method-39ltgenericprocedurenamegt39"></a>传递给方法的类型参数的数目不正确&#39; &lt;genericprocedurename&gt;&#39;
已调用泛型过程，其中类型实参的数量与定义它的类型形参的数量不匹配。  
  
 **错误 ID：** BC30951  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   向为泛型过程定义的各个类型形参均提供一个类型实参。  
  
     -或-  
  
-   调用不带任何类型实参的泛型过程，并让编译器尝试推断类型实参。
