# <a name="39for39-loop-control-variable-already-in-use-by-an-enclosing-39for39-loop"></a>&#39;有关&#39;已在使用被封闭的循环控制变量&#39;为&#39;循环
所有嵌套 `For` 循环必须使用唯一的控制变量。  
  
 **错误 ID：** BC30975  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   更改其中一个控制变量名。
