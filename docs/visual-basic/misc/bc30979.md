# <a name="39option-infer39-can-be-followed-only-by-39on39-or-39off39"></a>&#39; 选项推断 &#39;可以后面只能接 &#39; 在 &#39;或 &#39; 关闭 &#39;
`Option Infer` 语句必须指定 `On` 或 `Off`。 其他值均无效。  
  
 **错误 ID：** BC30979  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   指定 `Option Infer On` 或 `Option Infer Off`。
