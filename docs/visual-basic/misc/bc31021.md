# <a name="visual-basic-compiler-is-unable-to-recover-from-the-following-error-lterrorgt"></a>Visual Basic 编译器不能从以下的错误恢复：&lt;错误&gt;
Visual Basic 编译器不能从以下的错误恢复： <`error`>。  
  
 已禁用某些 IDE 功能。  
  
 在 Visual Basic 编译器中发生内部错误。  
  
 **错误 ID：** BC31021  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  保存所做的工作并重新启动 [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)]。  
  
2.  如果错误重复出现，请重新安装 [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)]。  
  
3.  如果在重新安装之后依然出现错误，请通知 Microsoft 产品支持服务。
