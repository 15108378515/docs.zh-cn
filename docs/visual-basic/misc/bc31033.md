# <a name="interface-39ltinterfacenamegt39-can-be-implemented-only-once-by-this-type"></a>接口 &#39;&lt;interfacename&gt;&#39; 可以实现此类型的一次
指定的接口已由此类型实现，不能再次实现。  
  
 **错误 ID：** BC31033  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   只能使用此类型实现此接口一次。
