# <a name="statement-is-not-valid-in-an-interface"></a>接口中语句无效
在接口中遇到无效语句。  
  
 **错误 ID：** BC31041  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   验证接口中的语句定义元素但不提供实现。
