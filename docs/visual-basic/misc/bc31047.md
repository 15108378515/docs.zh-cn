# <a name="protected-types-can-only-be-declared-inside-of-a-class"></a>受保护的类型只能在类内部声明
模块中的类型被声明为 `Protected`。  
  
 **错误 ID：** BC31047  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  声明类中的类型。  
  
2.  删除该 `Protected` 修饰符。
