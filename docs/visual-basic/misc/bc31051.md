# <a name="namespace-or-type-39ltnamegt39-has-already-been-imported"></a>Namespace 或类型和 #39;&lt;名称&gt;&#39; 已导入
多次尝试导入命名空间。  
  
 **错误 ID：** BC31051  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除多余的 `Imports` 语句。
