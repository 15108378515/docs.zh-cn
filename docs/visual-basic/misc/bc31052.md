# <a name="modules-cannot-be-declared-39ltspecifiergt39"></a>模块不能声明&#39;&lt;说明符&gt;&#39;
已使用无效的访问修饰符声明模块。  
  
 **错误 ID:** BC31052  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   用有效的访问修饰符（如 `Public` 或 `Friend`）替换该访问修饰符。
