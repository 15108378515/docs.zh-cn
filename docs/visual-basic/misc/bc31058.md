# <a name="conflicts-with-39ltname1gt39-which-is-implicitly-created-for-39ltname2gt39-in-lttypegt-39ltname3gt39"></a>与冲突&#39; &lt;name1&gt;&#39;，这隐式创建&#39; &lt;name2&gt; &#39;中&lt;类型&gt; &#39; &lt;name3&gt;&#39;
类型成员的名称与另一个隐式创建的成员相冲突。  
  
 **错误 ID：** BC31058  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   重命名显式声明的成员，以消除命名冲突。
