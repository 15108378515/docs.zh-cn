# <a name="39ltpropertynamegt39-implicitly-defines-39ltmembernamegt39-which-conflicts-with-a-member-of-the-same-name-in-lttypegt-39lttypenamegt39"></a>&#39;&lt;propertyname&gt;&#39; 隐式定义 &#39;&lt;membername&gt;&#39;，这与中的同名成员冲突&lt;类型&gt;&#39;&lt;typename&gt;&#39;
类型成员名称与为属性隐式创建的成员名称相冲突。 属性隐式创建多个隐式变量，这些变量可能与定义的成员冲突。  
  
 **错误 ID：** BC31060  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   重命名显式声明的成员，以消除命名冲突。
