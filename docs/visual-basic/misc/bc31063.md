# <a name="39set39-method-cannot-have-more-than-one-parameter"></a>&#39;集 &#39;方法不能有多个参数
为属性 `Set` 过程定义了多个参数。  
  
 **错误 ID：** BC31063  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   从属性 `Set` 过程中删除所有多余参数。
