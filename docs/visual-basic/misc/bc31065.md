# <a name="39set39-parameter-cannot-be-declared-39ltspecifiergt39"></a>&#39;设置&#39;参数不能声明&#39;&lt;说明符&gt;&#39;
在 `Set` 属性过程使用参数指定了无效的修饰符。  
  
 **错误 ID：** BC31065  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除无效的修饰符，如 `ByRef`、 `ParamArray`和 `Optional`。
