# <a name="method-in-a-structure-cannot-be-declared-39protected39-or-39protected-friend39"></a>不能声明为结构中的方法&#39;受保护&#39;或&#39;Protected Friend&#39;
结构中的方法被声明为 `Protected` 或 `Protected Friend`。  
  
 **错误 ID：** BC31067  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   将结构中的方法声明为 `Public` 或 `Private`。
