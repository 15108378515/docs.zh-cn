# <a name="class-in-an-interface-cannot-be-declared-39ltspecifiergt39"></a>接口中的类不能声明为&#39;&lt;说明符&gt;&#39;
使用在接口中声明的类指定了无效的修饰符。  
  
 **错误 ID：** BC31070  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除修饰符，如 `Public`、 `Private`、 `Shared`、 `Friend`、 `Protected`、 `Protected Friend`或 `Overridable`。
