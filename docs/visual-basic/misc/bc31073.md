# <a name="39ltname1gt39-conflicts-with-a-lttypegt-by-the-same-name-defined-in-39ltname2gt39"></a>&#39;&lt;name1&gt; &#39;与冲突&lt;类型&gt;中定义的相同名称的&#39; &lt;name2&gt;&#39;
遇到与具有相同名称的类型冲突的名称。  
  
 **错误 ID：** BC31073  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  声明唯一的名称。  
  
2.  在冲突成员名称两端使用方括号。
