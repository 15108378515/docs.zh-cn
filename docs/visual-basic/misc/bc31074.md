# 无法找到实现委托所需的类型“&lt;typename&gt;”的委托构造函数
包含该类型的程序集可能被损坏或无效。  
  
 **错误 ID：**BC31074  
  
### 更正此错误  
  
1.  验证包含该类型的程序集是有效的。  
  
2.  再次编译此程序以查看错误是否重复出现。  
  
3.  如果仍出现错误，请保存你的工作并重新启动 [!INCLUDE[vsprvs](../../csharp/includes/vsprvs-md.md)]。
