# <a name="delegate-constructor-for-type-39lttypenamegt39-necessary-for-the-implementation-of-delegates-cannot-be-found"></a>委托类型 &#39; 的构造函数&lt;typename&gt;&#39; 所必需的实现委托，找不到
包含该类型的程序集可能被损坏或无效。  
  
 **错误 ID：** BC31074  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  验证包含该类型的程序集是有效的。  
  
2.  再次编译此程序以查看错误是否重复出现。  
  
3.  如果仍出现错误，请保存你的工作并重新启动 [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)]。
