# <a name="delegate-constructor-for-type-39lttypenamegt39-necessary-for-the-implementation-of-delegates-cannot-be-found"></a>类型的委托构造函数&#39; &lt;typename&gt;&#39;找不到实现委托，根据需要，
包含该类型的程序集可能被损坏或无效。  
  
 **错误 ID：** BC31074  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  验证包含该类型的程序集是有效的。  
  
2.  再次编译此程序以查看错误是否重复出现。  
  
3.  如果错误仍然存在，保存你的工作并重新启动 Visual Studio。
