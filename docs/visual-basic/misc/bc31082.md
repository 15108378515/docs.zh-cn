# <a name="39ltvariablenamegt39-is-not-a-local-variable-or-parameter-and-so-cannot-be-used-as-a-39catch39-variable"></a>&#39;&lt;variablename&gt; &#39;不是本地变量或参数，并因此不能用作&#39;捕获&#39;变量
`Try...Catch...Finally` 语句中的变量必须在本地声明，或者是当前过程的参数。  
  
 **错误 ID：** BC31082  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  声明 `Try...Catch...Finally` 语句的本地变量或参数。
