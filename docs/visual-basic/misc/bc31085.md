# <a name="date-constant-is-not-valid"></a>日期常量无效
声明日期常量时使用的日期无效。  
  
 **错误 ID：** BC31085  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   更正常量中所用的日期。
