# <a name="39notoverridable39-cannot-be-specified-on-methods-that-do-not-override-another-method"></a>&#39;NotOverridable&#39;不能对指定不重写另一种方法的方法
默认情况下，属性和方法 `NotOverridable` 。 `NotOverridable` 修饰符仅可用于重写另一属性或方法的方法。  
  
 **错误 ID:** BC31088  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   从不重写另一成员的属性和方法中删除 `NotOverridable` 修饰符。
