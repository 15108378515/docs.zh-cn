# <a name="member-39ltmembernamegt39-cannot-be-found-in-class-39ltclassnamegt39"></a>成员 &#39;&lt;membername&gt;&#39; 不能位于类 &#39;&lt;类名&gt;&#39;
成员\<成员名称 > 不能在类的中找到\<类名 >。 这种情况通常是“Microsoft.VisualBasic.dll”不匹配的结果。  
  
 未能找到定义的成员。  
  
 **错误 ID：** BC31097  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  再次编译此程序以查看错误是否重复出现。  
  
2.  如果错误重复出现，请保存你的工作并重新启动 [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)]。  
  
3.  如果仍发生错误，请重新安装 [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)]。  
  
4.  如果在重新安装之后依然出现错误，请通知 Microsoft 产品支持服务。
