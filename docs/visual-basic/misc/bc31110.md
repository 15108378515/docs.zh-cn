# <a name="arrays-used-as-attribute-arguments-are-required-to-explicitly-specify-values-for-all-elements"></a>用作特性参数的数组需要显式指定所有元素的值
用作特性参数的数组具有带未指定值的元素。  
  
 **错误 ID：** BC31110  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   指定所有元素值。
