# <a name="statement-cannot-appear-within-an-event-body"></a>语句不能出现在事件体内
语句不能出现在事件体内。 假定已到达事件末尾。  
  
 在事件体内无效的语句将出现在其中一个事件体内。  
  
 **错误 ID：** BC31112  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   将 `End Event` 添加到语句之前。
