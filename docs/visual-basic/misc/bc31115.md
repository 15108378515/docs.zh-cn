# <a name="39addhandler39-declaration-must-end-with-a-matching-39end-addhandler39"></a>&#39;AddHandler&#39;声明必须以匹配结束&#39;End AddHandler&#39;
`AddHandler` 声明必须以 `End AddHandler` 语句结束。  
  
 **错误 ID：** BC31115  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   检查 `AddHandler` 声明是否以 `End AddHandler` 语句结束。
