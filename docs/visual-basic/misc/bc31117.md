# <a name="39raiseevent39-declaration-must-end-with-a-matching-39end-raiseevent39"></a>&#39;RaiseEvent &#39;声明必须以匹配 &#39; 结束End RaiseEvent &#39;
`RaiseEvent` 声明必须以 `End RaiseEvent` 语句结束。  
  
 **错误 ID：** BC31117  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   检查 `RaiseEvent` 声明是否以 `End RaiseEvent` 语句结束。
