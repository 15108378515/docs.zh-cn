# <a name="39end-addhandler39-must-be-the-first-statement-on-a-line"></a>&#39;End AddHandler &#39;必须是行的第一个语句
`End AddHandler` 语句跟在冒号 (:) 语句分隔符之后。 `End AddHandler` 必须是其源行的唯一语句。  
  
 **错误 ID：** BC31118  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   将多个语句分隔到不同的行。
