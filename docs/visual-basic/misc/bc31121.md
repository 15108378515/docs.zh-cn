# “Custom”修饰符在接口中声明的事件上无效
无法在接口中声明自定义事件，因为自定义事件必须提供其 `AddHandler`、`RemoverHandler` 和 `RaiseEvent` 方法的实现。  
  
 `Custom` 关键字可用于实现事件的派生类中。  
  
 **错误 ID：**BC31121  
  
### 更正此错误  
  
-   从接口的事件声明中删除 `Custom` 关键字。
