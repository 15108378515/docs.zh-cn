# <a name="39end-removehandler39-must-be-preceded-by-a-matching-39removehandler39-declaration"></a>&#39;End RemoveHandler &#39;前面必须是匹配 &#39;RemoveHandler &#39;声明
出现 `End RemoveHandler` 语句而没有相应的 `RemoveHandler` 语句。 `End RemoveHandler` 前面必须有相应的 `RemoveHandler` 语句。  
  
 **错误 ID：** BC31125  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   检查前面的 `RemoveHandler` 语句是否有效且拼写正确。
