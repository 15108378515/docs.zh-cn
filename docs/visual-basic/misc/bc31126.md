# <a name="39end-raiseevent39-must-be-preceded-by-a-matching-39raiseevent39-declaration"></a>&#39;End RaiseEvent&#39;前面必须是匹配的&#39;RaiseEvent&#39;声明
出现 `End RaiseEvent` 语句而没有相应的 `RaiseEvent` 语句。 `End RaiseEvent` 前面必须有相应的 `RaiseEvent` 语句。  
  
 **错误 ID：** BC31126  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   检查前面的 `RaiseEvent` 语句是否有效且拼写正确。
