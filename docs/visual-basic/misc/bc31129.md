# <a name="39raiseevent39-is-already-declared"></a>&#39;RaiseEvent &#39;已声明
自定义事件声明中出现多个 `RaiseEvent` 声明。 `RaiseEvent` 声明会声明用于引发事件的过程。  
  
 **错误 ID：** BC31129  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除多余的 `RaiseEvent` 语句。
