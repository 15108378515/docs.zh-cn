# <a name="required-attribute-39version39-missing-from-xml-declaration"></a>必需的特性&#39;版本&#39;XML 声明中缺少
XML 文档文本的 `version` 特性丢失。  
  
 **错误 ID：** BC31153  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   向 XML 文档声明添加 `version` 特性，如以下示例所示： `<?xml version="1.0"?>`
