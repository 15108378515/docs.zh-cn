# <a name="xml-attribute-39version39-must-be-the-first-attribute-in-xml-declaration"></a>XML 属性 &#39; 版本 &#39;必须在 XML 声明中的第一个属性
XML 文档文本的 `version` 特性不是指定的第一个特性。 根据 XML 1.0 规范， `version` 特性必须是为 XML 文档文本指定的第一个特性。  
  
 **错误 ID：** BC31156  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   重新定位 XML 文档文本的 `version` 特性，使其成为第一个特性。
