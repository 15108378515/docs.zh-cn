# <a name="expected-closing-39gt39-for-xml-tag"></a>预期的右 &#39;&gt;&#39; 对于 XML 标记
XML 文本的 XML 标记缺少必需的结束字符“>”。  
  
 **错误 ID：** BC31166  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   将结束字符“>”添加到 XML 文本的 XML 标记的结束位置。
