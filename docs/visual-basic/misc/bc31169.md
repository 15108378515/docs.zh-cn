# <a name="character-39character39-0xlthexadecimalgt-is-not-allowed-at-the-beginning-of-an-xml-name"></a>字符 &#39; 字符 &#39;(0x&lt;十六进制&gt;) 不允许在 XML 名称的开头
XML 文本或 XML 轴属性的名称开头包含了不受支持的字符。  
  
 **错误 ID：** BC31169  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   从 XML 文本或 XML 轴属性的名称开头删除不受支持的字符。
