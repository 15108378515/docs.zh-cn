# <a name="character-39character39-0xlthexadecimalgt-is-not-allowed-in-an-xml-name"></a>字符 &#39; 字符 &#39;(0x&lt;十六进制&gt;) XML 名称中不允许
XML 文本或 XML 轴属性的名称中包含了不受支持的字符。  
  
 **错误 ID：** BC31170  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   从 XML 文本或 XML 轴属性的名称中删除不受支持的字符。
