# <a name="character-sequence-39--39-is-not-allowed-in-an-xml-comment"></a>字符序列&#39;-&#39; XML 注释中不允许
XML 注释文本包含字符序列“--”，这不受支持。  
  
 **错误 ID：** BC31171  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   从 XML 注释文本中删除字符序列“--”。
