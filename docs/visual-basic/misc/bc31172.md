# <a name="an-embedded-expression-may-not-be-used-here"></a>不能在此处使用嵌入式表达式
嵌入式的表达式包括了在 XML 文本中不受支持的位置。 请注意，不能使用嵌入式的表达式来指定 XML 命名空间值。  
  
 **错误 ID：** BC31172  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   从不受支持的位置中删除嵌入的表达式。
