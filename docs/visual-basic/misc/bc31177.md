# <a name="white-space-may-not-appear-here"></a>此处不能出现空白
XML 元素文本的名称中含有一个或多个空格。  
  
 **错误 ID：** BC31177  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   从 XML 元素文本名称中删除空白。
