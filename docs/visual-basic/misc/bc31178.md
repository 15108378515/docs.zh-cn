# <a name="expected-closing-3939-for-xml-entity"></a>预期结束 &#39;&#39;XML 实体
XML 实体引用（例如， `>`）缺少右分号 (;)。  
  
 **错误 ID：** BC31178  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   在 XML 实体引用末尾添加分号。
