# <a name="attribute-value-is-not-valid-expecting-39value39"></a>属性值不是有效，则为应为&#39;值&#39;
为 XML 文档文本中的特性指定了无效的值。 在 XML 1.0 规范中应指定有效的 XML 文档文本特性值。  
  
 **错误 ID：** BC31181  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   将特性值更新为在 XML 1.0 规范中指定的有效值。
