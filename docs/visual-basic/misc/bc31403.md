# <a name="imports-alias-39ltnamegt39-conflicts-with-39ltname2gt39-declared-in-the-root-namespace"></a>导入别名 &#39;&lt;名称&gt;&#39; 与冲突 &#39;&lt;name2&gt;&#39; 中的根命名空间声明
`Imports` 语句中使用的别名名称与根命名空间中声明的另一个名称冲突。  
  
 **错误 ID：** BC31403  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   选择 `Imports` 语句使用的唯一别名名称。
