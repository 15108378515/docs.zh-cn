# <a name="39ltmethodnamegt39-cannot-override-a-method-that-has-been-shadowed"></a>&#39;&lt;methodname&gt; &#39;不能重写已被隐藏的方法
某个方法试图重写隐藏其他某个成员的方法。  
  
 **错误 ID：** BC31406  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   使用 `Shadows` 修饰符重写隐藏成员。
