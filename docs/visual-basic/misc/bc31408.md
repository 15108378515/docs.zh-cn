# <a name="39ltspecifier1gt39-and-39ltspecifier2gt39-cannot-be-combined"></a>&#39;&lt;specifier1&gt; &#39;和&#39; &lt;specifier2&gt; &#39;不能组合
同时使用了两个不兼容的说明符（如 `Shadows` 和 `Overrides` ）。  
  
 **错误 ID：** BC31408  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除其中一个说明符。
