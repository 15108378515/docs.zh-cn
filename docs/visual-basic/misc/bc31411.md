# <a name="39lttypenamegt39-must-be-declared-39mustinherit39-because-it-contains-methods-declared-39mustoverride39"></a>&#39;&lt;typename&gt; &#39;必须声明&#39;MustInherit&#39;因为它包含声明的方法&#39;MustOverride&#39;
包含 `MustOverride` 成员的类型必须声明为 `MustInherit`。  
  
 **错误 ID：** BC31411  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   将 `MustInherit` 修饰符添加到类型。
