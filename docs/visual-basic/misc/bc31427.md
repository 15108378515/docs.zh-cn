# <a name="syntax-error-in-conditional-compilation-expression"></a>条件编译表达式中有语法错误
条件编译表达式生成非特定的语法错误。  
  
 **错误 ID：** BC31427  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  检查条件编译以及用于表达式的关键字的文档。 将文档中的示例代码与生成此错误的源行进行比较。  
  
2.  如果无法识别错误的原因，则收集有关该情况的信息并通知 Microsoft 产品支持服务。
