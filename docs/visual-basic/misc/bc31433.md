# <a name="method-39ltmethodnamegt39-cannot-be-declared-39partial39-because-only-one-method-39ltmethodnamegt39-can-be-marked-39partial39"></a>方法&#39; &lt;methodname&gt; &#39;不能声明为&#39;部分&#39;因为只有一个方法&#39; &lt;methodname&gt; &#39;可以标记&#39;部分&#39;
两个分部方法声明不能具有相同名称和签名。  
  
 **错误 ID:** BC31433  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  更改其中一个分部方法声明的名称或签名。
