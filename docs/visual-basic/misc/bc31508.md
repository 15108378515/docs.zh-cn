# <a name="cannot-find-net-framework-directory-lterrorgt"></a>找不到.NET Framework 目录：&lt;错误&gt;
无法找到存储 [!INCLUDE[dnprdnshort](~/includes/dnprdnshort-md.md)] 程序集的目录。  
  
 **错误 ID：** BC31508  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  再次编译此程序以查看错误是否重复出现。  
  
2.  如果仍出现错误，请保存你的工作并重新启动 [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)]。  
  
3.  如果错误重复出现，请重新安装 [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)]。  
  
4.  如果在重新安装之后依然出现错误，请通知 Microsoft 产品支持服务。
