# <a name="unable-to-open-resource-file-39ltfilenamegt39-lterrorgt"></a>无法打开资源文件&#39; &lt;filename&gt;&#39;:&lt;错误&gt;
打开指定的资源文件时出错。  
  
 **错误 ID：** BC31509  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  验证指定的名称和位置是否正确。  
  
2.  验证该文件是否已损坏。
