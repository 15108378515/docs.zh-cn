# <a name="type-39lttypenamegt39-cannot-be-used-as-an-attribute-because-it-is-not-declared-39public39"></a>类型&#39; &lt;typename&gt; &#39;因为它未声明为不能用作特性&#39;公共&#39;
特性必须是公开的。  
  
 **错误 ID：** BC31516  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  向特性类中添加 `Public` 修饰符。
