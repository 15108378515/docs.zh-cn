# “&lt;filename&gt;”不是程序集，因此无法引用它。
引用了不是 .NET 程序集的文件。  
  
 **错误 ID：**BC31519  
  
### 更正此错误  
  
1.  验证程序集的名称和位置。  
  
2.  验证该程序集是否已损坏。
