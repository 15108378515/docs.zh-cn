# <a name="39ltkeywordgt39-is-valid-only-within-a-class"></a>&#39;&lt;关键字&gt;&#39; 仅在类有效
与类相关的关键字，如 `Me` 或 `MyClass`，在类定义之外使用。  
  
 **错误 ID：** BC32002  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   如果使用关键字的代码涉及类实例，则将其移到类实现。  
  
-   如果使用关键字的代码不适用于类，则删除无效的关键字。
