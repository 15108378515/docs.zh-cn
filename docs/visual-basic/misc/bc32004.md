# <a name="39ltnamegt39-is-declared-in-project-39ltprojectname1gt39-which-is-not-referenced-by-project-39ltprojectname2gt39"></a>&#39;&lt;名称&gt;&#39; 在项目 &#39; 中声明&lt;projectname1&gt;&#39;，不引用由项目 &#39;&lt;项目名称 2>&gt;&#39;
在当前项目未引用的项目中定义了一个编程元素。  
  
 **错误 ID：** BC32004  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   在当前项目中，添加对包含所需元素的项目的引用。
