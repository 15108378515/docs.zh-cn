# <a name="parameter-39ltparameternamegt39-in-39ltmethodnamegt39-already-has-a-matching-omitted-argument"></a>参数&#39; &lt;parametername&gt; &#39;中&#39; &lt;methodname&gt; &#39;已经有一个相匹配的省略自变量
按位置省略了一个参数后，过程调用又按名称提供了同一个参数，例如：  
  
```  
Public Sub ABC(ByVal X As Byte, Optional ByVal Y As Byte = 0, _  
                                Optional ByVal Z As Byte = 0)  
' ...  
Call ABC(6, , Y:=3)   ' Argument Y omitted by position, supplied by name.  
```  
  
 **错误 ID：** BC32021  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   按位置提供参数，或删除省略它的逗号。
