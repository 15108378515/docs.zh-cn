# <a name="the-39ltkeywordgt39-keyword-is-used-to-overload-inherited-members-do-not-use-the-39ltkeywordgt39-keyword-when-overloading-39sub-new39"></a>&#39;&lt;关键字&gt;&#39; 关键字用于重载继承的成员; 不要使用 &#39;&lt;关键字&gt;&#39; 关键字重载时 &#39;新的子 &#39;
使用 `Overloads` 关键字声明了一个构造函数。  
  
 Visual Basic 不支持继承或重载构造函数。  
  
 **错误 ID：** BC32040  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   从所有构造函数声明中删除 `Overloads` 关键字。
