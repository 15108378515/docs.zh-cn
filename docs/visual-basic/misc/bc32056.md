<span data-ttu-id="8a12a-103">泛型类型中的 `Implements` 子句将类型形参指定为要实现的成员。</span><span class="sxs-lookup"><span data-stu-id="8a12a-103">An `Implements` clause in a generic type specifies a type parameter as the member to be implemented.</span></span>
