# <a name="type-arguments-unexpected"></a><span data-ttu-id="d31d8-102">不应为类型参数</span><span class="sxs-lookup"><span data-stu-id="d31d8-102">Type arguments unexpected</span></span>
<span data-ttu-id="d31d8-103">`Implements` 子句为它正在实现的接口成员提供类型参数。</span><span class="sxs-lookup"><span data-stu-id="d31d8-103">An `Implements` clause supplies type arguments for the interface member it is implementing.</span></span>  
  
 <span data-ttu-id="d31d8-104">`Implements` 子句应只标识它正在实现的接口和成员。</span><span class="sxs-lookup"><span data-stu-id="d31d8-104">The `Implements` clause should only identify the interface and member it is implementing.</span></span> <span data-ttu-id="d31d8-105">这意味着，如果你声明一个泛型过程，则 `Of` 子句和类型参数应在该声明的主要部分出现，正如你未实现接口过程它们也会出现一样。</span><span class="sxs-lookup"><span data-stu-id="d31d8-105">This means that if you are declaring a generic procedure, the `Of` clause and the type arguments should appear in the main part of the declaration, just as they would if you were not implementing an interface procedure.</span></span>  
  
 <span data-ttu-id="d31d8-106">以下代码会生成此错误。</span><span class="sxs-lookup"><span data-stu-id="d31d8-106">The following code can generate this error.</span></span>  
  
```  
Public Interface testInterface  
    Sub testSub(Of t)()  
End Interface  
Public Class testClass  
    Implements testInterface  
    Public Sub testSub() Implements testInterface.testSub(Of t)()  
    End Sub  
End Class  
```  
  
 <span data-ttu-id="d31d8-107">`Implements` 子句前面的声明除了可能添加访问或过程修饰符，其他应看起来像接口定义。</span><span class="sxs-lookup"><span data-stu-id="d31d8-107">The declaration preceding the `Implements` clause should look like the interface definition, except for the possible addition of access or procedure modifiers.</span></span> <span data-ttu-id="d31d8-108">以下代码可避免此错误。</span><span class="sxs-lookup"><span data-stu-id="d31d8-108">The following code avoids the error.</span></span>  
  
```  
Public Sub testSub(Of t)() Implements testInterface.testSub  
```  
  
 <span data-ttu-id="d31d8-109">**错误 ID：** BC32088</span><span class="sxs-lookup"><span data-stu-id="d31d8-109">**Error ID:** BC32088</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="d31d8-110">更正此错误</span><span class="sxs-lookup"><span data-stu-id="d31d8-110">To correct this error</span></span>  
  
-   <span data-ttu-id="d31d8-111">从 `Implements` 子句中删除类型参数列表。</span><span class="sxs-lookup"><span data-stu-id="d31d8-111">Remove the type argument list from the `Implements` clause.</span></span>  
  
-   <span data-ttu-id="d31d8-112">如果要实现接口的泛型成员，则将类型参数列表放在声明的主要部分， `Implements` 关键字之前。</span><span class="sxs-lookup"><span data-stu-id="d31d8-112">If you are implementing a generic member of the interface, then put the type argument list in the main part of the declaration, preceding the `Implements` keyword.</span></span>
