# <a name="39as39-comma-or-3939-expected"></a>&#39;作为&#39;，逗号或&#39;)&#39;预期
已提供的类型参数缺少 `As`、逗号 (`,`) 或右括号 ( `)` )。  
  
 **错误 ID：** BC32100  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   提供缺少的 `As`、逗号或右括号。
