<span data-ttu-id="a2bf4-103">泛型接口继承自两个或多个泛型接口，对于某些类型实参的值，两个继承可能冲突。</span><span class="sxs-lookup"><span data-stu-id="a2bf4-103">A generic interface inherits from two or more generic interfaces, and two of the inheritances could conflict for certain values of type arguments.</span></span>  
  
 <span data-ttu-id="a2bf4-104">以下语句可能会生成此错误。</span><span class="sxs-lookup"><span data-stu-id="a2bf4-104">The following statements can generate this error.</span></span>  
  
```  
Public Interface interfaceA(Of u)  
    Inherits interfaceX(Of u)  
End Interface  
Public Interface interfaceX(Of v)  
End Interface  
Public Interface derivedInterface(Of t1, t2)  
    Inherits interfaceA(Of t1), interfaceX(Of t2)  
End Interface  
```  
  
 <span data-ttu-id="a2bf4-105">如果构造或实现 `derivedInterface` 时同时向 `t1` 和 `t2`提供相同类型，则它必须继承类型参数相同的两个版本的 `interfaceX` 。</span><span class="sxs-lookup"><span data-stu-id="a2bf4-105">If `derivedInterface` is constructed or implemented supplying the same type to both `t1` and `t2`, it must inherit two versions of `interfaceX` with identical type arguments.</span></span> <span data-ttu-id="a2bf4-106">如果这样做，则在访问哪个版本方面存在歧义。</span><span class="sxs-lookup"><span data-stu-id="a2bf4-106">Doing so would produce an ambiguity about which version to access.</span></span>
