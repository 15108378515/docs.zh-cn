# <a name="cannot-implement-39ltinterfacename1gtltmembernamegt39-because-its-implementation-could-conflict-with-the-implementation-for-39ltinterfacename2gtltmembernamegt39-for-some-type-arguments"></a><span data-ttu-id="42b1d-102">不能实现 &#39;&lt;interfacename1&gt;。&lt;membername&gt;&#39;，因为其实现可能使用的实现 &#39; 冲突&lt;interfacename2&gt;。&lt;membername&gt;&#39; 对于某些类型参数</span><span class="sxs-lookup"><span data-stu-id="42b1d-102">Cannot implement &#39;&lt;interfacename1&gt;.&lt;membername&gt;&#39; because its implementation could conflict with the implementation for &#39;&lt;interfacename2&gt;.&lt;membername&gt;&#39; for some type arguments</span></span>
<span data-ttu-id="42b1d-103">一个类可实现多个泛型接口（相互继承），对于类型参数的某些值，一个接口成员的两个实现可能相互冲突。</span><span class="sxs-lookup"><span data-stu-id="42b1d-103">A class implements more than one generic interface, one of which inherits from another, and two implementations of an interface member could conflict for certain values of type arguments.</span></span>  
  
 <span data-ttu-id="42b1d-104">以下语句可能会生成此错误。</span><span class="sxs-lookup"><span data-stu-id="42b1d-104">The following statements can generate this error.</span></span>  
  
```  
Public Interface iFace1(Of t)  
    Sub testSub()  
End Interface  
Public Interface iFace2(Of u)  
    Inherits iFace1(Of u)  
End Interface  
Public Class testClass(Of y, z)  
    Implements iFace1(Of y), iFace2(Of z)  
    Public Sub testSuby() Implements iFace1(Of y).testSub  
    End Sub  
    Public Sub testSubz() Implements iFace1(Of z).testSub  
    End Sub  
End Class  
```  
  
 <span data-ttu-id="42b1d-105">由于 `iFace2` 继承自 `iFace1` ，后者使用其自己的类型参数 (`u`)，因此如果将相同的类型参数传递给 `testClass` 和 `iFace1.testSub` ，则 `y` 将实现带有相同签名的两个版本的 `z`。</span><span class="sxs-lookup"><span data-stu-id="42b1d-105">Because `iFace2` inherits from `iFace1` using its own type parameter (`u`), `testClass` would implement two versions of `iFace1.testSub` with identical signatures if the same type argument were passed to `y` and `z`.</span></span> <span data-ttu-id="42b1d-106">这将产生关于要访问哪个版本的多义性。</span><span class="sxs-lookup"><span data-stu-id="42b1d-106">This would produce an ambiguity about which version to access.</span></span>  
  
 <span data-ttu-id="42b1d-107">**错误 ID：** BC32125</span><span class="sxs-lookup"><span data-stu-id="42b1d-107">**Error ID:** BC32125</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="42b1d-108">更正此错误</span><span class="sxs-lookup"><span data-stu-id="42b1d-108">To correct this error</span></span>

     <span data-ttu-id="42b1d-110">- 或 -</span><span class="sxs-lookup"><span data-stu-id="42b1d-110">-or-</span></span>
