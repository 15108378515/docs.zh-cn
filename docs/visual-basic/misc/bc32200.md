# <a name="39ltmembernamegt39-cannot-be-declared-39shadows39-outside-of-a-class-structure-or-interface"></a>&#39;&lt;membername&gt; &#39;不能声明为&#39;阴影&#39;外部类、 结构或接口
`Shadows` 关键字出现在命名空间、模块或文件级别的声明中。 隐藏仅在可从基元素继承的编程元素内有意义。  
  
 **错误 ID：** BC32200  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除 `Shadows` 关键字，或在类、结构或接口内部移动声明。
