# <a name="property-parameters-cannot-have-the-name-39value39"></a>属性参数不能具有名称 &#39;值 &#39;
`Property` 语句中的参数名为 `Value`。 此名称保留在 `Set` 过程中被分配给该属性的新值的属性定义内。  
  
 **错误 ID：** BC32201  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   更改 `Property` 语句中的参数名称。
