# <a name="project-has-been-closed"></a>项目已关闭
[!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] 编译器内部发生代码模型访问错误。  
  
 **错误 ID：** BC32301  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  再次编译此程序以查看错误是否重复出现。  
  
2.  如果错误重复出现，请重新安装 [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] 编译器。  
  
3.  如果重新安装后仍然出现错误，请收集有关该情况的信息并通知 Microsoft 产品支持服务。
