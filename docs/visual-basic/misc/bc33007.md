# <a name="39end-operator39-must-be-preceded-by-a-matching-39operator39"></a>&#39;End Operator&#39;前面必须是匹配的&#39;运算符&#39;
`End Operator` 语句前面没有任何匹配的 `Operator` 声明。  
  
 **错误 ID：** BC33007  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除多余的 `End Operator` 语句。  
  
-   提供缺少的 `Operator` 过程（如果有）。  
  
-   将 `End Operator` 语句移到代码中的适当位置。
