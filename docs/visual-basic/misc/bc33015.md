# <a name="operator-39ltoperatorgt39-must-have-two-parameters"></a>运算符 &#39;&lt;运算符&gt;&#39; 必须有两个参数
用两个以上或以下的参数定义了二元运算符。  
  
 二元运算符必须有且只有两个参数。  
  
 **错误 ID：** BC33015  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   将定义调整为指定恰好两个参数。  
  
-   如果你只需要一个参数，则必须定义一元运算符。  
  
-   如果您需要任何参数或两个以上，则必须使用[Function 语句](../../visual-basic/language-reference/statements/function-statement.md)定义`Function`过程而不是运算符。
