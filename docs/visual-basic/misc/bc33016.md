# <a name="operator-39ltoperatorgt39-must-have-either-one-or-two-parameters"></a>运算符 &#39;&lt;运算符&gt;&#39; 必须有一个或两个参数
未使用参数或使用两个以上的参数定义了可能是一元或二元的运算符。  
  
 一元/二元运算符必须有一个或两个参数。  
  
 **错误 ID：** BC33016  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   调整定义以指定一个或两个参数。  
  
-   如果您需要任何参数或两个以上，则必须使用[Function 语句](../../visual-basic/language-reference/statements/function-statement.md)定义`Function`过程而不是运算符。
