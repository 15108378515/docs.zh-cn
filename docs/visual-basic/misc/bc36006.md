# <a name="attribute-constructor-has-a-39byref39-parameter-of-type-39lttypenamegt39-cannot-use-constructors-with-byref-parameters-to-apply-the-attribute"></a>特性构造函数具有&#39;ByRef&#39;类型的参数&#39; &lt;typename&gt;&#39;; 不能使用构造函数使用 byref 参数将该属性应用
使用接受 `ByRef` 参数的特性构造函数向编程元素应用了特性。  
  
 特性在编译时应用，编译器需要具体值来传递给特性构造函数。 `ByRef` 参数将指针传递到值，此操作不能在编译时进行计算。  
  
 你可以定义接受 `ByRef` 参数的特性构造函数并将其用于继承等目的，但应用该特性时必须使用不接受任何 `ByRef` 参数的构造函数。  
  
 **错误 ID：** BC36006  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   使用不接受任何 `ByRef` 参数的构造函数来应用该特性，或不应用该特性。
