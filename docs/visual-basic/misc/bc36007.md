# <a name="39end-using39-must-be-preceded-by-a-matching-39using39"></a>&#39;结束使用&#39;前面必须是匹配的&#39;使用&#39;
`End Using` 语句前面没有任何匹配的 `Using` 声明。  
  
 **错误 ID：** BC36007  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除多余的 `End Using` 语句。  
  
-   提供缺少[Using 语句](../../visual-basic/language-reference/statements/using-statement.md)缺少。  
  
-   将 `End Using` 语句移到代码中的适当位置。
