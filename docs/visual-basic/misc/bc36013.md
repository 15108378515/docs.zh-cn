# <a name="39on-error39-statements-are-not-valid-within-39using39-statements"></a>&#39;错误&#39;语句不在有效&#39;使用&#39;语句
`On Error` 语句出现在 `Using` 语句中，但在该上下文中无效。  
  
 **错误 ID：** BC36013  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   使用结构化的错误处理（如 `Try…Catch` 块）代替 `On Error` 语句。
