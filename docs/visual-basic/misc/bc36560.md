# <a name="type-characters-cannot-be-used-in-anonymous-type-declarations"></a><span data-ttu-id="d5030-102">不能在匿名类型声明中使用类型字符</span><span class="sxs-lookup"><span data-stu-id="d5030-102">Type characters cannot be used in anonymous type declarations</span></span>
<span data-ttu-id="d5030-103">声明匿名类型的实例时，不能在属性名称中使用类型字符。</span><span class="sxs-lookup"><span data-stu-id="d5030-103">You cannot use a type character on a property name when you declare an instance of an anonymous type.</span></span> <span data-ttu-id="d5030-104">属性的数据类型是从赋给它的值推断出来的。</span><span class="sxs-lookup"><span data-stu-id="d5030-104">The data type of the property is inferred from the value assigned to it.</span></span> <span data-ttu-id="d5030-105">例如，下面的声明无效。</span><span class="sxs-lookup"><span data-stu-id="d5030-105">For example, the following declarations are not valid.</span></span>  
  
```vb

'Dim anon2 = New With {.ID$ = 42}  
```  
  
 <span data-ttu-id="d5030-106">**错误 ID：** BC36560</span><span class="sxs-lookup"><span data-stu-id="d5030-106">**Error ID:** BC36560</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="d5030-107">更正此错误</span><span class="sxs-lookup"><span data-stu-id="d5030-107">To correct this error</span></span>
