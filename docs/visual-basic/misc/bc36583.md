# <a name="parameter-39ltparameternamegt39-in-extension-method-39ltmethodnamegt39-defined-in-39lttypenamegt39-already-has-a-matching-omitted-argument"></a>参数&#39; &lt;parametername&gt; &#39;的扩展方法中&#39; &lt;methodname&gt; &#39;中定义&#39; &lt;typename&gt; &#39;已具有匹配的忽略的实参
扩展方法的过程调用按位置忽略实参，然后按名称提供实参。 例如，对扩展方法 `ABC` 的以下调用首先忽略参数的实参 `Y`，然后按名称提供。  
  
```  
<Extension()> _  
Public Sub ABC(ByVal X As Integer, Optional ByVal Y As Byte = 0, _  
               Optional ByVal Z As Byte = 0)  
End Sub  
' . . .  
' Calling extension method ABC.  
Dim number As Integer  
' Not valid.  
' number.ABC(, 4, Y:=5)  
```  
  
 **错误 ID：** BC36583  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   按位置提供参数，或删除省略它的逗号。
