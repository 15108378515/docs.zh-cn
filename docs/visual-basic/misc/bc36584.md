# <a name="parameter-39ltparameternamegt39-in-extension-method-39ltmethodnamegt39-defined-in-39lttypenamegt39-already-has-a-matching-argument"></a>参数 &#39;&lt;parametername&gt;&#39; 在扩展方法 &#39;&lt;methodname&gt;&#39; 在中定义 &#39;&lt;typename&gt;&#39; 已具有匹配的实参
参数在同一过程调用中按名称传递了多次。  
  
 **错误 ID：** BC36584  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   检查以确定是否其中一个参数名拼写错误。  
  
-   从参数列表中删除重复的参数。
