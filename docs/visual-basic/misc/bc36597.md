# <a name="goto-to-label-39ltlabelnamegt39-cannot-be-used-within-lambda-expressions"></a>转至标签&#39;&lt;标签名&gt;&#39;不能在 lambda 表达式中使用
Lambda 表达式的主体必须是计算并返回单个值的表达式。 不允许任何语句，包括 `GoTo` 语句。  
  
 **错误 ID：** BC36597
