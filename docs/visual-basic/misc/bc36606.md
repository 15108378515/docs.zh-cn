# <a name="range-variable-name-cannot-match-the-name-of-a-member-of-the-39object39-class"></a>范围变量名称不能与匹配的成员的名称&#39;对象&#39;类
LINQ 查询中的范围变量与 <xref:System.Object> 类的成员名匹配。 这会导致与 Visual Basic 为 LINQ 查询创建的对象冲突。  
  
 **错误 ID：** BC36606  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  为不与 <xref:System.Object> 类的任何成员名匹配的范围变量选择新名称。
