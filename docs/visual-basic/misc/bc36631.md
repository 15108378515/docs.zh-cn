 <span data-ttu-id="00597-104">**错误 ID：** BC36631</span><span class="sxs-lookup"><span data-stu-id="00597-104">**Error ID:** BC36631</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="00597-105">更正此错误</span><span class="sxs-lookup"><span data-stu-id="00597-105">To correct this error</span></span>  
  
1.  <span data-ttu-id="00597-106">将 `Join` 关键字添加到 `Group Join` 子句中，如下面的示例中所示：</span><span class="sxs-lookup"><span data-stu-id="00597-106">Add the `Join` keyword to the `Group Join` clause, as shown in the following example:</span></span>  
  
```vb

            Join var2 In collection2 _  
            Group Join var3 In collection3 _  
            On var2.ID Equals var3 Into Matches = Group _  
            On var1 Equals var2.ID _  
            Select JoinID = var1, var2.Name, Matches  
```
