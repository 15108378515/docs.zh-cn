# <a name="instance-of-restricted-type-39lttypenamegt39-cannot-be-used-in-a-lambda-expression"></a>受限类型的实例&#39; &lt;typename&gt; &#39;不能在 lambda 表达式
不能在 lambda 表达式或查询表达式中使用受限类型的实例，例如 <xref:System.ArgIterator>、 <xref:System.RuntimeArgumentHandle>和 <xref:System.TypedReference>。  
  
 **错误 ID：** BC36640
