# <a name="option-strict-on-does-not-allow-narrowing-in-implicit-type-conversions-between-the-lambda-expression-and-delegate-39ltdelegatenamegt39"></a><span data-ttu-id="f7600-102">Option Strict On 不允许在 lambda 表达式和委托之间的隐式类型转换收缩 '&lt;委托名&gt;</span><span class="sxs-lookup"><span data-stu-id="f7600-102">Option Strict On does not allow narrowing in implicit type conversions between the lambda expression and delegate &#39;&lt;delegatename&gt;&#39;</span></span>
<span data-ttu-id="f7600-103">启用 `Option Strict` 时，你不能在委托中参数的数据类型与分配给该委托类型的变量的 lambda 表达式的相应参数之间进行收缩转换。</span><span class="sxs-lookup"><span data-stu-id="f7600-103">With `Option Strict` on, you cannot have a narrowing conversion between the data type of a parameter in a delegate and the corresponding parameter of a lambda expression assigned to a variable of that delegate type.</span></span> <span data-ttu-id="f7600-104">例如，在下面的代码中，委托 `Del` 具有一个 `Integer`类型的参数。</span><span class="sxs-lookup"><span data-stu-id="f7600-104">For example, in the following code, delegate `Del` has one parameter of type `Integer`.</span></span>  
  
```vb  
Delegate Function Del(ByVal p As Integer) As String  
```  
  
 <span data-ttu-id="f7600-105">因此，分配给 `Del` 类型的变量的任何 lambda 表达式的相应参数可为 `Integer` 或任何数据类型，以确保存在 `Integer`的扩大转换。</span><span class="sxs-lookup"><span data-stu-id="f7600-105">Therefore, the corresponding parameter of any lambda expression assigned to a variable of type `Del` can be an `Integer` or any data type for which there is a widening conversion from `Integer`.</span></span>  
  
```vb  
' Valid.  
Dim example1 As Del = Function(n As Integer) "Valid"  
Dim example2 As Del = Function(n As Long) "Valid"  
  
' Not valid.  
Dim example3 As Del = Function(n As Short) "Not Valid"  
```  
  
 <span data-ttu-id="f7600-106">**错误 ID：** BC36662</span><span class="sxs-lookup"><span data-stu-id="f7600-106">**Error ID:** BC36662</span></span>

