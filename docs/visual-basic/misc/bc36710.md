# <a name="extension-method-39ltmethodnamegt39-defined-in-39ltmodulenamegt39-does-not-have-a-signature-compatible-with-delegate-39ltdelegatenamegt39"></a>扩展方法 &#39;&lt;methodname&gt;&#39; 在中定义 &#39;&lt;modulename&gt;&#39; 没有与委托 &#39; 兼容的签名&lt;delegatename&gt;&#39;
扩展方法的签名与你正尝试使用的委托签名不匹配。 `Delegate` 语句定义参数类型和委托类的返回类型。 任何具有兼容参数、参数类型和返回类型的过程都可能用于创建此委托类型的实例。  
  
 **错误 ID：** BC36710
