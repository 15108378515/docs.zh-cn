# <a name="argument-matching-parameter-39ltparameternamegt39-narrows-to-39lttypenamegt39"></a>自变量匹配参数&#39; &lt;parametername&gt; &#39;收缩转换为&#39; &lt;typename&gt;&#39;
已调用重载方法，但编译器找不到无需收缩转换便可调用的方法。 *“收缩转换”* 将值更改为可能无法保存某些可能值的数据类型。  
  
 **错误 ID：** BC36711
