# <a name="type-39lttypenamegt39-and-partial-type-39lttypenamegt39-declared-in-39ltfilenamegt39-conflict-in-container-39ltcontainernamegt39-but-are-being-merged-because-one-of-them-is-declared-partial"></a>类型&#39; &lt;typename&gt; &#39;和分部类型&#39; &lt;typename&gt; &#39;中声明&#39; &lt;filename&gt; &#39;容器中的冲突&#39; &lt;containername&gt;&#39;，但由于其中一个被声明为 partial，因此正在合并
类或结构出现在相同容器类型的多个定义中，并且多个定义未标记 `Partial`。  
  
 你必须至少在类或结构的多个定义中的一个上使用 [Partial](../../visual-basic/language-reference/modifiers/partial.md) 关键字，但是建议你将其用于所有分部定义。  
  
 默认情况下，此消息是一个警告。 有关隐藏警告或将警告视为错误的信息，请参见 [Configuring Warnings in Visual Basic](/visualstudio/ide/configuring-warnings-in-visual-basic)。  
  
 **错误 ID:** BC40047  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   在类或结构的每个分部定义中使用 [Partial](../../visual-basic/language-reference/modifiers/partial.md) 关键字。
