# <a name="only-one-xml-comment-block-is-allowed-per-language-element"></a>每个语言元素只能有一个 XML 注释块
多个 XML 注释块已应用到某个语言元素。  
  
 **错误 ID:** BC42301  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除多余的 XML 注释块。
