# <a name="xml-comment-tag-39lttaggt39-is-not-permitted-on-a-39ltelementgt39-language-element"></a>XML 注释标记 &#39;&lt;标记&gt;&#39; 上不允许 &#39;&lt;元素&gt;&#39; 语言元素
XML 注释标记\<标记 > 中不允许\<元素 > 元素。 将忽略 XML 注释。  
  
 应用 XML 注释标记的语言元素不支持该标记。  
  
 **错误 ID：** BC42306  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除不受支持的 XML 注释标记。
