# <a name="xml-comment-tag-39include39-must-have-a-39ltattributegt39-attribute"></a>XML 注释标记&#39;包括&#39;必须具有&#39;&lt;属性&gt;&#39;属性
XML 注释标记 include 必须具有\<属性 > 特性。 将忽略 XML 注释。  
  
 传递给 `file` include `path` 标记的 `<`或`>` 特性缺少或不完整。  
  
 **错误 ID：** BC42310  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   添加或补充所需的特性\<包括 > 标记。
