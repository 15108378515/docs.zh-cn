# <a name="xml-comment-tag-39returns39-is-not-permitted-on-a-39writeonly39-property"></a>XML 注释标记 #39; 返回 &#39;中不允许 &#39;WriteOnly &#39;属性
WriteOnly 属性中不允许有 XML 注释标记 "returns"。 将忽略 XML 注释。  
  
 只写属性声明的 XML 注释不能包含\<返回 > 标记。  
  
 **错误 ID：** BC42313  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除不受支持\<返回 > 标记。
