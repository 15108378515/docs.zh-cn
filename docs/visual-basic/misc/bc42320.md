# <a name="unable-to-include-xml-fragment-39ltelementpathgt39-of-file-39ltcommentfilegt39"></a>无法包括 XML 片段&#39; &lt;elementpath&gt; &#39;文件的&#39; &lt;l e&gt;&#39;
XML 注释中的 `<include>` 标记包含不存在元素的 XPath 表达式，或此文件包含格式不正确的 XML。  
  
 **错误 ID：** BC42320  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  验证该文件中的 XML 是否有效。  
  
2.  验证文件中是否存在 XML 片段。
