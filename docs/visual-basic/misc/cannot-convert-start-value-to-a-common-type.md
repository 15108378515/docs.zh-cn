# <a name="cannot-convert-start-value-of-type-39lttype1gt39-limit-value-of-39lttype2gt39-and-step-value-of-39lttype3gt39-to-a-common-type"></a>无法转换类型的起始值&#39; &lt;type1&gt;&#39;，限制值&#39; &lt;type2&gt; &#39;和的步长值&#39; &lt;type3&gt; &#39;到通用类型
指定的 `For...Next` 控制值具有不同的类型，因此无效，其原因是它们不能转换为通用类型。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  更改一个或多个值的类型，以便所有值都可以转换为常用类型。
