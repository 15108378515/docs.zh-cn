# <a name="cannot-rename-with-different-drive"></a>无法用不同的驱动器重命名
`Rename` 语句不能指定文件路径中的驱动器号。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   从文件路径中删除驱动器号。
