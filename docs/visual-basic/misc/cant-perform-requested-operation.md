# <a name="can39t-perform-requested-operation-visual-basic"></a>可以&#39;t 执行请求的操作 (Visual Basic)
无法执行某项操作，因为它会使该项目的当前状态无效。 例如，你可能尝试以编程方式修改正在运行的代码。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   停止执行代码，必要时进行修改，然后重试该操作。
