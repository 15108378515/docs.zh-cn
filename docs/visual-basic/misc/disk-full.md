# <a name="disk-full-visual-basic"></a>磁盘已满 (Visual Basic)
磁盘上没有足够的空间，无法完成 `Print`、 `Write`或 `FileClose` 操作，也无法创建所需的文件。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   通过将一些文件移到另一个磁盘或删除一些文件，可在磁盘上创建更多空间。
