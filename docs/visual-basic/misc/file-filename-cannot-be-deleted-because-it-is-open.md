# <a name="file-39ltfilenamegt39-cannot-be-deleted-because-it-is-open"></a>文件 &#39;&lt;filename&gt;&#39; 无法删除，因为它已打开
正在尝试删除打开的文件。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   在删除之前关闭该文件。
