# <a name="file-39ltfilenamegt39-is-write-protected"></a>文件&#39; &lt;filename&gt; &#39;已被写保护
尝试写入的文件受写保护，因此不能更改。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  将针对文件的权限更改为读/写，或写入另一个文件。
