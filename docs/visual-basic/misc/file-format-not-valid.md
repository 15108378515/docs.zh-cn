# <a name="file-format-not-valid"></a>文件格式无效
要么你尝试将一个文件加载到组件中，但是文件中的数据格式与该组件不兼容，要么你尝试将组件数据保存到文件中，但是数据格式与文件格式不兼容。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  查阅有关该组件的文档以确定磁盘文件数据的正确格式。  
  
2.  查阅有关该组件的文档以确定它是否支持转换格式。
