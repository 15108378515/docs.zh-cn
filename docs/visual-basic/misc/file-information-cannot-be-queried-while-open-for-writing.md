# <a name="file-information-cannot-be-queried-while-open-for-writing"></a>当文件打开以进行写入时无法查询文件信息
已尝试获取有关当前打开以进行写入的文件的信息。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   关闭文件，然后重试该操作。
