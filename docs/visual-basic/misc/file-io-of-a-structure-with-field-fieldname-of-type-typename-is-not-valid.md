# <a name="file-io-of-a-structure-with-field-39ltfieldnamegt39-of-type-39lttypenamegt39-is-not-valid"></a>文件 I/O 的具有字段 &#39; 的结构&lt;fieldname&gt;&#39; 类型 &#39;&lt;typename&gt;&#39; 不是有效
试图向文件写入的结构包含不支持的类型。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   检查结构来确定不支持的类型出现的位置，然后将其更改为有效类型。
