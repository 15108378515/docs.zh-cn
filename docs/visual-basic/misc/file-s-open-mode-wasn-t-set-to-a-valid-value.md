# <a name="file39s-open-mode-wasn39t-set-to-a-valid-value"></a>文件&#39;s 打开模式不是&#39;t 为有效的值设置
为文件的打开模式提供的值无效。 下表显示 <xref:Microsoft.VisualBasic.OpenMode> 枚举的有效值。  
  
|值|模式|  
|-----------|----------|  
|1|`OpenMode.Input`|  
|2|`OpenMode.Output`|  
|4|`OpenMode.Random`|  
|8|`OpenMode.Append`|  
|32|`OpenMode.Binary`|  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   验证为文件的打开模式提供的值。
