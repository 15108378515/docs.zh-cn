# <a name="file-specified-by-filename-does-not-use-the-encoding-specified-by-fileencoding"></a>FileName 指定的文件不使用 FileEncoding 指定的编码
已尝试从文件中读取，但该文件不使用指定的编码。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   检查以确保提供正确的文件编码。
