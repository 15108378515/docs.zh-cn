# <a name="insufficient-security-permissions-to-set-the-system-date"></a>没有足够的安全权限设置系统日期。
安全权限不足以设置系统日期。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   更改安全权限以允许设置系统日期。
