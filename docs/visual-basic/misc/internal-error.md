# <a name="internal-error-visual-basic"></a>内部错误 (Visual Basic)
在 Visual Basic 中出现内部故障。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   请确保此错误不是由 `Error` 语句或 `Raise` 方法生成的。 除非是，否则请联系 Microsoft 产品支持服务报告出现此消息的情况。
