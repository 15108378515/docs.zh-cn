# <a name="line-ltnumbergt-cannot-be-parsed-using-the-current-delimiters"></a>行&lt;数&gt;无法使用当前的 Delimiter 分析
无法分析指定的行，因为它使用的不是那些指定的分隔符。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   调整 `Delimiters` 以便正确分析此行，或插入异常处理代码以处理此行。
