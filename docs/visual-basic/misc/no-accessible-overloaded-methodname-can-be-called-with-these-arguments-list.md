# <a name="no-accessible-overloaded-39ltmethodnamegt39-can-be-called-with-these-arguments-ltlistgt"></a>没有可访问重载 &#39;&lt;methodname&gt;&#39; 可以使用这些自变量调用：&lt;列表&gt;
已调用重载的方法，但没有方法可以与提供的参数列表匹配。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   对照你想要调用的重载与参数列表进行匹配。  
  
-   调整参数列表，以便签名与该重载匹配。
