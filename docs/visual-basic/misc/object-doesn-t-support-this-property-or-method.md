# <a name="object-doesn39t-support-this-property-or-method-visual-basic"></a>对象没有 &#39; 不支持此属性或方法 (Visual Basic)
你指定的属性或方法不在此自动对象中。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   阅读有关对象详细信息的文档并检查属性和方法的拼写。
