# <a name="root-folder-cannot-be-renamed"></a>无法重命名根文件夹。
尝试重命名根目录。 这可能是由于不正确地分析文件路径造成的。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   检查文件路径，以确保它是正确无误的。
