# <a name="set-not-supported-read-only-property"></a>不支持 Set 语句(只读属性)
试图设置或更改只读属性。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  从代码中删除对该属性的引用。  
  
2.  更改引用以便在运行时仅返回该属性的值。
