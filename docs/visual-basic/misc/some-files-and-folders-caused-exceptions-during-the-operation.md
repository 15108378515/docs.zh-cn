# <a name="some-files-and-folders-caused-exceptions-during-the-operation"></a>某些文件和文件夹在操作期间导致异常
当尝试执行操作，但用户缺少必要的权限时，目录和文件可能会引发异常。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   添加访问所有文件和文件夹所需的权限。
