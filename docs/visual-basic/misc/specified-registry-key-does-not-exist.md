# <a name="specified-registry-key-does-not-exist"></a>指定的注册表项不存在
已指定的注册表项不存在。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   检查当前提供的注册表项，确保已正确指定了路径。  
  
-   在尝试写入之前创建注册表项。
