# <a name="target-folder-is-a-file"></a>目标文件夹是一个文件
在尝试执行与文件夹相关的操作时指定了文件。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   检查所提供的路径，以确保指定的是文件夹而不是文件。
