# <a name="the-file-is-currently-closed"></a>此文件当前已关闭
试图执行只能对打开的文件执行的操作，但此文件当前已关闭。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   打开此文件，然后再次尝试该操作。
