# <a name="the-folder-cannot-be-created-since-a-file-already-exists-with-the-same-path"></a>无法创建该文件夹，因为存在具有相同路径的文件
尝试创建一个目录，但已存在具有该路径的文件。 这可能是由于不正确地分析路径所导致，特别是使用相对路径时。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   检查提供的路径是否正确。  
  
-   如果不再需要此冗余文件，请将其删除。
