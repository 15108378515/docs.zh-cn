# <a name="the-path-has-not-been-set"></a>尚未设置路径
试图进行文件操作（例如，移动或复制文件），但未提供必要的路径。 这可能是由于未正确分析路径造成的。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   检查是否提供了正确的文件路径。
