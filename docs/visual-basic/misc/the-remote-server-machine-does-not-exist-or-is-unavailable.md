# <a name="the-remote-server-machine-does-not-exist-or-is-unavailable-visual-basic"></a>远程服务器计算机不存在或不可用 (Visual Basic)
尝试连接到不可用或不存在的远程服务器。 此错误可能由网络连接断开引起。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   重新连接网络，然后重试。
