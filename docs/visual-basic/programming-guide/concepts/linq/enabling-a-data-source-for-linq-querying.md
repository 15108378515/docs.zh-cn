## <a name="iqueryable-linq-providers"></a><span data-ttu-id="f4e03-124">IQueryable LINQ 提供程序</span><span class="sxs-lookup"><span data-stu-id="f4e03-124">IQueryable LINQ Providers</span></span>  
 [!INCLUDE[vbteclinq](../../../../csharp/includes/vbteclinq_md.md)]<span data-ttu-id="f4e03-125">提供程序实现<xref:System.Linq.IQueryable%601>范围很广的其复杂程度。</xref:System.Linq.IQueryable%601></span><span class="sxs-lookup"><span data-stu-id="f4e03-125"> providers that implement <xref:System.Linq.IQueryable%601> can vary widely in their complexity.</span></span> <span data-ttu-id="f4e03-126">本节讨论这些不同程度的复杂性。</span><span class="sxs-lookup"><span data-stu-id="f4e03-126">This section discusses the different levels of complexity.</span></span>
