# <a name="how-to-query-an-assembly39s-metadata-with-reflection-linq-visual-basic"></a><span data-ttu-id="260cf-102">如何︰ 查询使用反射 (LINQ) (Visual Basic 中) 的程序集的元数据</span><span class="sxs-lookup"><span data-stu-id="260cf-102">How to: Query An Assembly&#39;s Metadata with Reflection (LINQ) (Visual Basic)</span></span>
<span data-ttu-id="260cf-103">下面的示例演示 LINQ 可以与反射一起用于检索有关与指定的搜索条件匹配的方法的特定元数据。</span><span class="sxs-lookup"><span data-stu-id="260cf-103">The following example shows how LINQ can be used with reflection to retrieve specific metadata about methods that match a specified search criterion.</span></span> <span data-ttu-id="260cf-104">在这种情况下，该查询将返回可枚举的类型，如数组的程序集中查找的所有方法的名称。</span><span class="sxs-lookup"><span data-stu-id="260cf-104">In this case, the query will find the names of all the methods in the assembly that return enumerable types such as arrays.</span></span>  
  
## <a name="example"></a><span data-ttu-id="260cf-105">示例</span><span class="sxs-lookup"><span data-stu-id="260cf-105">Example</span></span>  
  
```vb  
Imports System.Reflection  
Imports System.IO  
Imports System.Linq  
Module Module1  
  
    Sub Main()  
        Dim asmbly As Assembly =   
            Assembly.Load("System.Core, Version=3.5.0.0, Culture=neutral, PublicKeyToken= b77a5c561934e089")  
  
        Dim pubTypesQuery = From type In asmbly.GetTypes()   
                            Where type.IsPublic   
                            From method In type.GetMethods()   
                            Where method.ReturnType.IsArray = True   
                            Let name = method.ToString()   
                            Let typeName = type.ToString()   
                            Group name By typeName Into methodNames = Group  
  
        Console.WriteLine("Getting ready to iterate")  
        For Each item In pubTypesQuery  
            Console.WriteLine(item.methodNames)  
  
            For Each type In item.methodNames  
                Console.WriteLine(" " & type)  
            Next  
        Next  
        Console.ReadKey()  
    End Sub  
  
End Module  
```  
  
 <span data-ttu-id="260cf-106">该示例使用<xref:System.Reflection.Assembly.GetTypes%2A>方法以返回类型的数组中指定的程序集。</xref:System.Reflection.Assembly.GetTypes%2A></span><span class="sxs-lookup"><span data-stu-id="260cf-106">The example uses the <xref:System.Reflection.Assembly.GetTypes%2A> method to return an array of types in the specified assembly.</span></span> <span data-ttu-id="260cf-107">[Where 子句](../../../../visual-basic/language-reference/queries/where-clause.md)应用筛选器以便仅将公共类型返回。</span><span class="sxs-lookup"><span data-stu-id="260cf-107">The [Where Clause](../../../../visual-basic/language-reference/queries/where-clause.md) filter is applied so that only public types are returned.</span></span> <span data-ttu-id="260cf-108">对于每个公共类型，子查询通过使用生成<xref:System.Reflection.MethodInfo>从返回的数组<xref:System.Type.GetMethods%2A>调用。</xref:System.Type.GetMethods%2A> </xref:System.Reflection.MethodInfo></span><span class="sxs-lookup"><span data-stu-id="260cf-108">For each public type, a subquery is generated by using the <xref:System.Reflection.MethodInfo> array that is returned from the <xref:System.Type.GetMethods%2A> call.</span></span> <span data-ttu-id="260cf-109">这些结果进行筛选，返回其返回类型是数组或其他类型的实现<xref:System.Collections.Generic.IEnumerable%601>。</xref:System.Collections.Generic.IEnumerable%601>这些方法</span><span class="sxs-lookup"><span data-stu-id="260cf-109">These results are filtered to return only those methods whose return type is an array or else a type that implements <xref:System.Collections.Generic.IEnumerable%601>.</span></span> <span data-ttu-id="260cf-110">最后，这些结果进行分组所使用的类型名称作为键。</span><span class="sxs-lookup"><span data-stu-id="260cf-110">Finally, these results are grouped by using the type name as a key.</span></span>
