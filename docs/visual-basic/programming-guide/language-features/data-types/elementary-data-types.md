# <a name="elementary-data-types-visual-basic"></a>基本数据类型 (Visual Basic)
Visual Basic 提供一组预定义的数据类型，你可以使用很多编程元素。 本部分介绍这些类型以及如何使用它们。  
  
> [!NOTE]
>  在 Visual Basic 中的每个基本数据类型支持的结构或类中<xref:System>命名空间。 编译器会使用每个数据类型关键字作为别名的基础结构或类。 例如，通过使用保留的字声明一个变量`Byte`等同于使用的完全限定的结构名称声明<xref:System.Byte?displayProperty=nameWithType>。
