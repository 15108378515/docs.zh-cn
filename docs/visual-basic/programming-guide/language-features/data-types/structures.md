# <a name="structures-visual-basic"></a>结构 (Visual Basic)
A*结构*是用户定义支持的类型 (UDT) 的早期版本的泛化[!INCLUDE[vbprvb](~/includes/vbprvb-md.md)]。 字段中，除了结构可以公开属性、 方法和事件。 结构可以实现一个或多个接口，而您可以声明的每个字段的单独的访问级别。  
  
 你可以组合来创建结构的不同类型的数据项。 一个结构将相关联，这是一个或多个*元素*和与结构本身。 当你声明一个结构时，它将成为*复合数据类型*，你可以声明该类型的变量。  
  
 当你想单个变量来保存多个相关的信息片段时，结构很有用。 例如，你可能想要将员工的名称、 电话分机和薪金放在一起。 你可以使用多个变量进行此信息，或无法定义结构并将其用于单个雇员变量。 当有许多员工，因此的变量的多个实例时，会了然-结构的优点。
